Create an organisation contact

Steps to create an organisation contact record in Cuba

This table describes how to create record for a contact within an
organisation in Cuba .

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Start point - Create a new        |
|                                   | contact + Read more ...           |
|                                   |                                   |
|                                   | Start point: Organisation         |
|                                   | Contacts window                   |
+-----------------------------------+-----------------------------------+
| 2                                 | Create the new contact record +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Click the [New icon] icon, or     |
|                                   | click the File menu and select    |
|                                   | New, to begin recording the new   |
|                                   | contact's details.                |
+-----------------------------------+-----------------------------------+
| 3                                 | Enter the contact details + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | In the Contact Details group box: |
|                                   |                                   |
|                                   | -   type the position the contact |
|                                   |     has within the organisation   |
|                                   |     in the Position field. This   |
|                                   |     is a mandatory field          |
|                                   | -   select the relevant title     |
|                                   |     from the Title drop down list |
|                                   | -   type the contacts First Name  |
|                                   |     , Other Given , and Surnames  |
|                                   |     in the relevant fields. The   |
|                                   |     First Name is a mandatory     |
|                                   |     field                         |
|                                   | -   type the Phone , Mobile and   |
|                                   |     Fax numbers in the relevant   |
|                                   |     fields                        |
|                                   | -   type the E-mail address for   |
|                                   |     the employer in the relevant  |
|                                   |     field                         |
|                                   | -   in the Mail to be directed    |
|                                   |     group box, select the         |
|                                   |     appropriate radio button for  |
|                                   |     the contact to receive the    |
|                                   |     specified mail                |
|                                   | -   in the Legal Contacts group   |
|                                   |     box, select the appropriate   |
|                                   |     radio button for the contact  |
|                                   |     to receive the specified mail |
+-----------------------------------+-----------------------------------+
| 4                                 | Save the contact's record + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | -   Click the                     |
|                                   |     icon to save the record       |
|                                   |                                   |
|                                   | Click the                         |
|                                   |                                   |
|                                   | icon to return to the             |
|                                   | Organisation window               |
+-----------------------------------+-----------------------------------+
| 5                                 | Document the details + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | From the Organisation window:     |
|                                   |                                   |
|                                   | -   click the Go menu             |
|                                   | -   select Organisation Detail to |
|                                   |     open the Organisation Detail  |
|                                   |     window                        |
|                                   | -   click the                     |
|                                   |     icon and create a note        |
|                                   |                                   |
|                                   | See Documenting Child Support     |
|                                   | information in Cuba for more      |
|                                   | information.                      |
+-----------------------------------+-----------------------------------+

View an organisation contact

Steps to view an organisation contact's details

This table describes how to find and view an organisation's contact
person as recorded in Cuba.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Start point - View a contact +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Start point: Organisation         |
|                                   | Contacts window                   |
+-----------------------------------+-----------------------------------+
| 2                                 | View a contact + Read more ...    |
|                                   |                                   |
|                                   | Select a record in the Details    |
|                                   | list box. The contact’s details   |
|                                   | will be displayed in the Contact  |
|                                   | Details group box.                |
|                                   |                                   |
|                                   | Note : if the                     |
|                                   |                                   |
|                                   | button is enabled on initial      |
|                                   | access to the Organisation        |
|                                   | Contacts window, this means that  |
|                                   | there are more records that can   |
|                                   | initially be displayed. Click to  |
|                                   | view more records. Alternatively  |
|                                   | go to the next step to find a     |
|                                   | specific contact.                 |
+-----------------------------------+-----------------------------------+
| 3                                 | Find a specific contact + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | -   To locate a specific contact: |
|                                   | -   click the                     |
|                                   |     icon. This opens the          |
|                                   |     Organisation Contacts Find    |
|                                   |     dialogue box                  |
|                                   | -   type the surname of the       |
|                                   |     contact in the Surname field  |
|                                   | -   Note : wild card(s) (*) can   |
|                                   |     be used if the whole surname  |
|                                   |     is not known                  |
|                                   | -   type a range to search in the |
|                                   |     Effective Date and To fields  |
|                                   | -   click the Find button. This   |
|                                   |     returns you to the            |
|                                   |     Organisation Contacts window, |
|                                   |     with the results of your      |
|                                   |     search displayed in the       |
|                                   |     Details list box              |
|                                   | -   click the record in the       |
|                                   |     Details list box to view the  |
|                                   |     contacts details in the       |
|                                   |     Contact Details group box     |
+-----------------------------------+-----------------------------------+

Update an organisation contact

Steps to update a record for a contact within an organisation in Cuba

This table describes how to update a record for a contact within an
organisation in Cuba.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Start point - Update a contact's  |
|                                   | details + Read more ...           |
|                                   |                                   |
|                                   | Start point : Organisation        |
|                                   | Contacts window                   |
+-----------------------------------+-----------------------------------+
| 2                                 | Update a contact's record + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | -   Select the relevant record in |
|                                   |     the Details List box. This    |
|                                   |     will display the details of   |
|                                   |     the contact in the Contact    |
|                                   |     Details group box             |
|                                   |                                   |
|                                   | Note: use the                     |
|                                   |                                   |
|                                   | button to view more records, or   |
|                                   | go to Find a specific contact if  |
|                                   | unable to locate the contact      |
|                                   | record in the List box.           |
|                                   |                                   |
|                                   | To change:                        |
|                                   |                                   |
|                                   | -   the position, overtype the    |
|                                   |     position the contact has      |
|                                   |     within the organisation in    |
|                                   |     the Position field            |
|                                   | -   the title, select the         |
|                                   |     relevant title from the Title |
|                                   |     drop down list                |
|                                   | -   the contact name, overtype    |
|                                   |     the contacts First Name ,     |
|                                   |     Other Given , and Surnames in |
|                                   |     the relevant fields. The      |
|                                   |     First Name is a mandatory     |
|                                   |     field                         |
|                                   | -   the contact numbers, overtype |
|                                   |     the Phone , Mobile and Fax    |
|                                   |     numbers in the relevant       |
|                                   |     fields                        |
|                                   | -   the E-mail address, overtype  |
|                                   |     the address for the employer  |
|                                   |     in the E-mail field           |
|                                   | -   where the mail is directed,   |
|                                   |     select the relevant radio     |
|                                   |     button for the contact to     |
|                                   |     receive the specified mail in |
|                                   |     the Mail to be directed group |
|                                   |     box                           |
|                                   | -   the legal contact mail,       |
|                                   |     select the relevant radio     |
|                                   |     button for the contact to     |
|                                   |     receive the specified mail in |
|                                   |     the Legal Contacts group box  |
+-----------------------------------+-----------------------------------+
| 3                                 | Save the updated record + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | -   Click the                     |
|                                   |     icon to save the record       |
|                                   | -   Click the                     |
|                                   |     icon to return to the         |
|                                   |     Organisation window           |
+-----------------------------------+-----------------------------------+
| 4                                 | Document the details + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | From the Organisation window:     |
|                                   |                                   |
|                                   | -   click the Go menu             |
|                                   | -   select Organisation Detail to |
|                                   |     open the Organisation Detail  |
|                                   |     window                        |
|                                   | -   click the                     |
|                                   |     icon and create a note        |
|                                   |                                   |
|                                   | See Documenting Child Support     |
|                                   | information in Cuba for more      |
|                                   | information.                      |

Delete an organisation contact

Steps to delete a record for a contact within an organisation in Cuba

This table describes how to delete a record for a contact within an
organisation in Cuba.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Start point - Delete a contact +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Start point : Organisation        |
|                                   | Contacts window                   |
+-----------------------------------+-----------------------------------+
| 2                                 | Delete an organisation contact’s  |
|                                   | record + Read more ...            |
|                                   |                                   |
|                                   | Note : only a current record can  |
|                                   | be deleted.                       |
|                                   |                                   |
|                                   | -   Select the relevant record in |
|                                   |     the List box                  |
|                                   | -   Note : use the Get More       |

|                                   |     contact if unable to locate   |
|                                   |     the contact record in the     |
|                                   |     List box.                     |
|                                   | -   Click the File menu, and      |
|                                   |     select Delete                 |
|                                   | -   Click the Yes button in the   |
|                                   |     confirmation dialogue box.    |
|                                   |     The record will be end dated  |
|                                   |     with the current date         |
|                                   | -   Click the                     |
|                                   |     icon to return to the         |
|                                   |     Organisation window           |

| 3                                 | Document the details + Read more  |
|                                   | ...                               |

|                                   | -   select Organisation Detail to |
|                                   |     open the Organisation Detail  |
|                                   |     window                        |
|                                   | -   click the                     |
|                                   |     icon and create a note        |
|                                   |                                   |
|                                   | See Documenting Child Support     |
|                                   | information in Cuba for more      |
