Review evidence

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Review evidence + Read more ...   |
|                                   |                                   |
|                                   | Discuss information, supporting   |
|                                   | documentation and evidence that   |
|                                   | was considered in the care        |
|                                   | decision with each customer       |
|                                   | before finalising the decision.   |
|                                   | Whether submitted by either       |
|                                   | customer, a third party or        |
|                                   | obtained from agency initiated    |
|                                   | searches.                         |
|                                   |                                   |
|                                   | Note: if a customer or third      |
|                                   | party does not want the evidence  |
|                                   | or information to be discussed or |
|                                   | made available to all customers   |
|                                   | for comment. Do not consider this |
|                                   | evidence or information when      |
|                                   | making the care determination.    |
|                                   |                                   |
|                                   | Discuss all conflicting           |
|                                   | information so that each customer |
|                                   | has an opportunity to respond to  |
|                                   | relevant information provided by  |
|                                   | the other customer.               |
|                                   |                                   |
|                                   | The other customer may change     |
|                                   | details originally given and/or   |
|                                   | gives extra information. This may |
|                                   | mean contacting customers on more |
|                                   | than one occasion to discuss the  |
|                                   | information and evidence received |
|                                   | to reach the most appropriate     |
|                                   | decision.                         |
+-----------------------------------+-----------------------------------+
| 2                                 | Sufficient information/evidence   |
|                                   | to make a decision + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | Decide whether there is           |
|                                   | sufficient information/evidence   |
|                                   | to make a care determination      |
|                                   | about the notified change in      |
|                                   | care.                             |
|                                   |                                   |
|                                   | If the customers:                 |
|                                   |                                   |
|                                   | -   provide sufficient evidence   |
|                                   |     for a care determination to   |
|                                   |     be made, go to Integrated     |
|                                   |     care – Making and recording   |
|                                   |     care determination            |
|                                   | -   do not provide supporting     |
|                                   |     evidence, go to Step 3        |
|                                   | -   are unable to agree on the    |
|                                   |     date the care changed, go to  |
|                                   |     Step 4                        |
+-----------------------------------+-----------------------------------+
| 3                                 | No supporting evidence received + |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | There should be sufficient and    |
|                                   | reliable information/evidence to  |
|                                   | support the decision to accept or |
|                                   | reject the change in care. If no  |
|                                   | supporting evidence is received,  |
|                                   | make the decision on the best     |
|                                   | information available.            |
|                                   |                                   |
|                                   | If there is any reason to doubt   |
|                                   | the information provided, seek    |
|                                   | further evidence from the         |
|                                   | notifying customer to support     |
|                                   | their claims.                     |
|                                   |                                   |
|                                   | Is there sufficient information   |
|                                   | and/or evidence to support the    |
|                                   | decision?                         |
|                                   |                                   |
|                                   | -   Yes, go to Integrated care -  |
|                                   |     Making and recording care     |
|                                   |     determination                 |
|                                   | -   No, go to Step 4              |
