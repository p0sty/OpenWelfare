Coding requirements for cancellation of JSP/YA

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Job seeker is on standard         |
|                                   | reporting cycle + Read more ...   |
|                                   |                                   |
|                                   | If the job seeker normally        |
|                                   | reports weekly or fortnightly but |
|                                   | does not intend to report         |
|                                   | anymore:                          |
|                                   |                                   |
|                                   | -   Cancel their payment with the |
|                                   |     date of event (DOV) as DPT +  |
|                                   |     1                             |
|                                   | -   Go to Step 3                  |
+-----------------------------------+-----------------------------------+
| 2                                 | Job seeker is on variable         |
|                                   | reporting cycle + Read more ...   |
|                                   |                                   |
|                                   | If the job seeker has an extended |
|                                   | reporting period and does not     |
|                                   | intend to report any more:        |
|                                   |                                   |
|                                   | -   Cancel their payment with the |
|                                   |     date of event as the day      |
|                                   |     following the 'authorised to  |
|                                   |     date', that is, the EPED + 1  |
|                                   |     of the last Reporting         |
|                                   |     Statement lodged. For coding  |
|                                   |     details, go to Step 3         |
|                                   | -   If the job seeker has         |
|                                   |     received any payments after   |
|                                   |     that date the system will     |

|                                   |     and create a debt             |
|                                   | -   Record details on a DOC       |

|                                   |                                   |

|                                   | Claiming and CAN CLR              |
|                                   |                                   |
|                                   | CAN CLR is not restorable. Where  |

|                                   | customer must submit a new online |
|                                   | claim if they wish to receive JSP |
|                                   | or Youth Allowance (job seeker).  |

|                                   | Customers claiming JSP or Youth   |
|                                   | Allowance (job seeker) online     |

|                                   |                                   |

|                                   |                                   |
|                                   | If a JSP or Youth Allowance (job  |
|                                   | seeker) payment is REJ or         |
|                                   | CAN/CLR, the customer has no      |
|                                   | right to a review as there is no  |
|                                   | legislative decision. The         |
|                                   | customer will have to reclaim.    |
|                                   |                                   |
|                                   | If CAN is for a reason other than |
|                                   | CLR, customer may seek a review   |
|                                   | of the decision. For more details |
|                                   | about the different types of      |
|                                   | cancellation reasons, see         |
|                                   | Cancellation, suspension and      |
|                                   | rejection reason codes for        |
|                                   | Newstart System (NSS).            |

|                                   |                                   |

|                                   | the estimate of income for Family |

|                                   | Subsidy (CCS). Estimates may be   |
|                                   | given over the phone              |
|                                   |                                   |

|                                   | issue a general cancellation      |
