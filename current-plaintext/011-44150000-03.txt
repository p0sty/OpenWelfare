|                                   | been residing outside of          |

|                                   |                                   |

|                                   |                                   |

|                                   | personal details held in CDMS     |

|                                   | If an error message displays      |
|                                   | Replacement not Allowed see       |
|                                   | remove a return mail indicator    |

Stopping a Medicare card

Table 6

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+

|                                   | insufficient documents + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | If an applicant is eligible for   |
|                                   | Medicare but has been enrolled in |
|                                   | Medicare without providing        |

Medicare card issued in error

Table 7

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Medicare card has been issued in  |
|                                   | error + Read more ...             |
|                                   |                                   |
|                                   | If the:                           |
|                                   |                                   |
|                                   | -   Service Officer has created a |
|                                   |     duplicate Consumer ID (no     |
|                                   |     programs attached) go to Step |
|                                   |     2                             |
|                                   | -   applicant is already enrolled |
|                                   |     and a duplicate PIN has been  |
|                                   |     created, go to Step 3         |
|                                   | -   card has been issued in error |
|                                   |     and the applicant is eligible |
|                                   |     for Medicare, go to Step 4    |
|                                   | -   applicant has been enrolled   |
|                                   |     in error and is not eligible  |
|                                   |     for Medicare, go to Step 5    |
|                                   |                                   |
|                                   | See Duplicate Medicare enrolment  |
|                                   | and Consumer IDs in Medicare for  |
|                                   | more details.                     |
+-----------------------------------+-----------------------------------+
| 2                                 | Duplicate Consumer IDs + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Contact the Medicare Public team  |
|                                   | to request the Consumer ID be     |
|                                   | deleted.                          |
+-----------------------------------+-----------------------------------+
| 3                                 | Duplicate PIN + Read more ...     |
|                                   |                                   |
|                                   | See Duplicate Medicare enrolment  |
|                                   | and Consumer IDs in Medicare      |
|                                   | process for more details.         |
+-----------------------------------+-----------------------------------+
| 4                                 | Issued in error and eligible for  |
|                                   | Medicare + Read more ...          |
|                                   |                                   |
|                                   | Extract the card by completing a  |
|                                   | card extraction form no later     |
|                                   | than 12 noon (AEST) the following |
|                                   | day. The form is available on the |
|                                   | Resources page.                   |
|                                   |                                   |
|                                   | The extracted card will not need  |
|                                   | to be returned to the Service     |
|                                   | Office as the error must be       |
|                                   | rectified immediately.            |
|                                   |                                   |
|                                   | If the cut-off time of 12 o’clock |
|                                   | is missed, a Medicare card will   |
|                                   | be issued and sent to the person. |
|                                   | In this case the Service Officer  |
|                                   | must:                             |
|                                   |                                   |
|                                   | -   contact the group contact of  |
|                                   |     the Medicare card concerned   |
|                                   | -   apologise for the error       |
|                                   |     related to the enrolment and  |
|                                   |     advise the group contact they |
|                                   |     will receive an incorrect     |
|                                   |     Medicare card                 |
|                                   | -   advise the group contact this |
|                                   |     card must not be used, and    |
|                                   |     must be destroyed             |
+-----------------------------------+-----------------------------------+
| 5                                 | Enrolled but not eligible + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | The consumer must have a consumer |
|                                   | end date keyed and a comment put  |
|                                   | in Personal CDMS. Extract the     |
|                                   | card by completing a card         |
|                                   | extraction form no later than 12  |
|                                   | noon (AEST) the following day.    |
|                                   | The form is available on the      |
|                                   | Resources page.                   |
|                                   |                                   |
|                                   | If the error was identified after |
|                                   | the card extraction timeframe,    |
|                                   | contact the group card holder to  |
|                                   | inform them they will be          |
|                                   | receiving a card and advise them  |
|                                   | they are not eligible.            |
