|                                   |     welfare payment for at least  |
|                                   |     52 weeks during the past 104  |
|                                   |     week period                   |

|                                   | Management will be applied to     |

|                                   |                                   |
|                                   | Explain how money can be accessed |
|                                   | for Income Management customers   |
|                                   | accessing funds while travelling  |
|                                   |                                   |
|                                   | Explain the process for           |
|                                   | contacting the agency, including  |
|                                   | self-service, in regards to       |
|                                   | Income Management and BasicsCard. |

|                                   | there are insufficient funds to   |
|                                   | cover an expense                  |
|                                   |                                   |
|                                   | Explain how lump sum payments     |

|                                   | reconciliation payments are 100%  |
|                                   | income managed                    |
|                                   |                                   |
