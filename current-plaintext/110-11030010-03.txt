|                                   | more ...                          |
|                                   |                                   |
|                                   | If the Commonwealth Director of   |
|                                   | Public Prosecution (CDPP) advises |

| 5                                 | Requesting CCTV footage + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | If CCTV is needed, staff can      |
|                                   | request the footage at the        |
|                                   | commencement of the intelligence  |
|                                   | assessment or investigation to    |
|                                   | make sure any available footage   |

|                                   | deletion off the system.          |

|                                   | To request CCTV footage staff     |

|                                   |                                   |
|                                   | -   Complete the request template |
|                                   | -   Get supervisor’s approval,    |
|                                   |     with a valid business reason  |
|                                   |     for the request               |
|                                   | -   Send an email to the local    |
|                                   |     authorised person through the |
|                                   |     team’s positional mailbox.    |
|                                   |     CCTV footage request          |
|                                   |     guidelines are available in   |
|                                   |     the Investigation Toolkit. If |
|                                   |     the authorised person is      |
|                                   |     unavailable and the request   |
|                                   |     is urgent, staff can email    |
|                                   |     the request to an authorised  |
|                                   |     person in another site        |
|                                   |                                   |
|                                   | Note: If the supervisor is listed |
|                                   | as a person who can access the    |
|                                   | Samsung Smart View CCTV system,   |
|                                   | they cannot sign the request      |
|                                   | template. There must be a         |
|                                   | separation of duties between the  |
|                                   | approver and the actioner, this   |
|                                   | is to make sure accountability    |
|                                   | and transparency. In these        |
|                                   | circumstances, staff must seek    |
|                                   | alternative approval from another |
|                                   | supervisor onsite.                |
|                                   |                                   |
|                                   | On receipt of CCTV footage        |
|                                   |                                   |
|                                   | -   analyse CCTV information      |
