This page contains information about the records disposal, bulk bill
adjustment delegates and a link to the Application for late lodgement of
a claim for assigned Medicare benefits form (DB019). It also provides
contact details for Bulk bill claims and My Health Record.

Disposal action of records

This table outlines the disposal action of records as stated in Entry
20173 of the National Archives of Australia - Records Authority -
Medicare.

+-----------------------------------+-----------------------------------+
| Description of records            | Disposal action                   |
+-----------------------------------+-----------------------------------+
| Claim forms for health and        | Destroy 2 years after date of     |
| medical payments including:       | claim if not required for         |
|                                   | investigation.                    |
| -   Medicare claim forms          |                                   |
| -   Bulk billing forms            |                                   |
| -   Pharmaceutical benefits claim |                                   |
|     forms                         |                                   |
| -   Authority prescriptions       |                                   |
+-----------------------------------+-----------------------------------+
