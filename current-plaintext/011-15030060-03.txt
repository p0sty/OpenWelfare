Process care costs

Table 2: this process follows on from Step 6 in Table 1 - Service
Support Officer (SSO) (APS5) or above trained in Care costs
calculations.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Calculate care costs + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | The claim was redirected to the   |
|                                   | nursing home queue.               |
|                                   |                                   |
|                                   | The SSO5 picks up the statement   |
|                                   | of claim case to:                 |
|                                   |                                   |
|                                   | -   process the statement         |
|                                   | -   identify the relevant         |
|                                   |     Medicare benefits             |
|                                   | -   calculate the applicable      |
|                                   |     amount of care costs on the   |
|                                   |     NOPB/NOPB-AP/NOC              |
|                                   | -   add the calculated care costs |
|                                   |     to the SBC details in MCRS    |
|                                   | -   file the calculation          |
|                                   |     spreadsheet and associated    |
|                                   |     breakdown in the shared drive |
|                                   |     folder - CompRec Care Costs   |
|                                   |     (nursing home residential     |
|                                   |     care home care)               |
+-----------------------------------+-----------------------------------+
| 3                                 | Send the documents to the         |
|                                   | recipient + Read more ...         |
|                                   |                                   |
|                                   | Send the NOPB/NOPB-AP/NOC to the  |
|                                   | recipient via regular mail        |
|                                   | (post).                           |
+-----------------------------------+-----------------------------------+
| 4                                 | Add case notes + Read more ...    |
|                                   |                                   |
|                                   | Add case notes to the claim and   |
|                                   | include details of all actions    |
|                                   | taken.                            |
