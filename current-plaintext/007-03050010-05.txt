Effect of FTB payment choice on payments and services

Table 3

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Advance payments of FTB Part A    |
|                                   |                                   |
|                                   | Eligible customers can request to |
|                                   | be paid lump sum FTB Part A       |
|                                   | advance payments of part of their |
|                                   | future FTB Part A entitlement.    |
|                                   | Ongoing FTB payments will be      |
|                                   | reduced to repay the advance      |
|                                   | payment.                          |
|                                   |                                   |
|                                   | To be eligible for an FTB advance |
|                                   | payment a person must receive FTB |
|                                   | Part A instalments. This includes |
|                                   | FTB instalment customers who      |
|                                   | elect to defer part or all of     |
|                                   | their FTB Part A entitlement. All |
|                                   | advance calculations are based on |
|                                   | the person's legislative rate of  |
|                                   | FTB, prior to any reductions due  |
|                                   | to their FTB payment choice.      |
|                                   |                                   |
|                                   | If they have an FTB regular       |
|                                   | advance arrangement in place,     |
|                                   | they may wish to also cease the   |
|                                   | regular advance payment           |
|                                   | arrangement.                      |
|                                   |                                   |
|                                   | If their legislative rate of FTB  |
|                                   | Part A rate falls below the       |
|                                   | advance payment recovery rate,    |
|                                   | all of their on-going FTB Part A  |
|                                   | entitlement will go towards       |
|                                   | repaying the advance/s and will   |
|                                   | extend the advance repayment      |
|                                   | period.                           |
|                                   |                                   |
|                                   | A debt will be raised for any FTB |
|                                   | advance amount that has not been  |
|                                   | recovered where the customer's:   |
|                                   |                                   |
|                                   | -   FTB Part A is cancelled, or   |
|                                   | -   Rate of FTB Part A reduces    |
|                                   |     and it will take an           |
|                                   |     unreasonable amount of time   |
|                                   |     to repay the FTB advance      |
|                                   |                                   |
|                                   | Customers may receive an FTB Part |
|                                   | A advance payment if entitled     |
|                                   | based on their income estimate.   |
+-----------------------------------+-----------------------------------+
| 2                                 | Child support                     |
|                                   |                                   |
|                                   | Persons receiving FTB Part A for  |
|                                   | a child from a previous           |
|                                   | relationship are still required   |
|                                   | to take reasonable action to      |
|                                   | obtain child support during the   |
|                                   | financial year, even if they      |
|                                   | choose to receive the base rate   |
|                                   | or nil rate of FTB Part A         |
|                                   | fortnightly.                      |
|                                   |                                   |
|                                   | If they do not take reasonable    |
|                                   | action to obtain child support    |
|                                   | during the financial year, they   |
|                                   | will not be entitled to more than |
|                                   | the base rate of FTB Part A after |
|                                   | the end of the financial year.    |
+-----------------------------------+-----------------------------------+
| 3                                 | Debt withholdings                 |
|                                   |                                   |
|                                   | If the customer's adjusted rate   |
|                                   | is less than their current        |
|                                   | withholding rate from FTB, they   |
|                                   | will receive only $1 as the rest  |
|                                   | will be withheld.                 |
|                                   |                                   |
|                                   | If they choose to receive all FTB |
|                                   | after the end of the financial    |
|                                   | year (zero ongoing rate), they    |
|                                   | will need to make a new repayment |
|                                   | arrangement.                      |
+-----------------------------------+-----------------------------------+
| 4                                 | Energy Supplement                 |
|                                   |                                   |
|                                   | The Energy Supplement is          |
|                                   | calculated separately for FTB     |
|                                   | Part A and FTB Part B.            |
|                                   |                                   |
|                                   | Those receiving FTB fortnightly   |
|                                   | can choose to receive the Energy  |
|                                   | Supplement fortnightly or         |
|                                   | quarterly in arrears. Quarterly   |
|                                   | payments are made in October,     |
|                                   | January, April and July based on  |
|                                   | their first Entitlement Period    |
|                                   | Start Date (EPSD) after the end   |
|                                   | of the quarter. They can change   |
|                                   | their choice at any time. The     |
|                                   | change takes effect in line with  |
|                                   | other FTB payment choices.        |
|                                   |                                   |
|                                   | How the Energy Supplement is paid |
|                                   | depends on FTB Part A and Part B  |
|                                   | payment choices:                  |
|                                   |                                   |
|                                   | -   FTB Part A fortnightly: the   |
|                                   |     Part A Energy Supplement is   |
|                                   |     paid fortnightly by default.  |
|                                   |     They can choose to receive    |
|                                   |     the Energy Supplement         |
|                                   |     quarterly in arrears instead  |
|                                   | -   Base FTB Part A: the Part A   |
|                                   |     Energy Supplement is paid at  |
|                                   |     base rate, fortnightly by     |
|                                   |     default. They can choose to   |
|                                   |     receive the Energy Supplement |
|                                   |     quarterly in arrears instead. |
|                                   |     Any extra Part A Energy       |
|                                   |     Supplement will be included   |
|                                   |     in the FTB reconciliation     |
|                                   |     calculation                   |
|                                   | -   FTB Part A after financial    |
|                                   |     year: The Part A Energy       |
|                                   |     Supplement will be included   |
|                                   |     in the FTB reconciliation     |
|                                   |     calculation                   |
|                                   | -   FTB Part B fortnightly: The   |
|                                   |     Part B Energy Supplement is   |
|                                   |     paid fortnightly by default.  |
|                                   |     They can choose to receive    |
|                                   |     the Energy Supplement         |
|                                   |     quarterly in arrears instead  |
|                                   | -   FTB Part B after financial    |
|                                   |     year: The Part B Energy       |
|                                   |     Supplement will be included   |
|                                   |     in the FTB reconciliation     |
|                                   |     calculation                   |
|                                   |                                   |
|                                   | Note: persons receiving both FTB  |
|                                   | Part A and Part B fortnightly     |
|                                   | must receive the Part A Energy    |
|                                   | Supplement and Part B Energy      |
|                                   | Supplement the same way (either   |
|                                   | both fortnightly or both          |
|                                   | quarterly).                       |
|                                   |                                   |
|                                   | The FTB Energy Supplement is      |
|                                   | included when calculating their   |
|                                   | correct FTB entitlement at FTB    |
|                                   | reconciliation.                   |
|                                   |                                   |
|                                   | To update the Energy Supplement   |
|                                   | payment choice, run the Family    |
|                                   | Income and Choices workflow.      |
|                                   |                                   |
|                                   | To view history, go to the Energy |
|                                   | Supplement Quarterly Payment      |
|                                   | Choice History (FYCQ) screen. If  |
|                                   | nothing is recorded, the Energy   |
|                                   | Supplement is being paid          |
|                                   | fortnightly or annually depending |
|                                   | on their FTB payment choice. If   |
|                                   | only a start date is recorded,    |
|                                   | the Energy Supplement is being    |
|                                   | paid quarterly (if they have not  |
|                                   | chosen to receive FTB annually).  |
|                                   | If an end date is recorded, they  |
|                                   | have chosen to no longer receive  |
|                                   | the Energy Supplement quarterly.  |
|                                   |                                   |
|                                   | FTB lump sum claims               |
|                                   |                                   |
|                                   | Those claiming FTB as a lump sum  |
|                                   | after the end of the financial    |
|                                   | year (for 2013-14 onwards) will   |
|                                   | have their Energy Supplement      |
|                                   | entitlement calculated when their |
|                                   | claim is finalised.               |

Examples - revoking payment choice, financial hardship and arrears when changing payment choice

Table 4

+-----------------------------------+-----------------------------------+
| Item                              | Example                           |
+-----------------------------------+-----------------------------------+
| 1                                 | Financial hardship                |
|                                   |                                   |
|                                   | Financial hardship needs to be    |
|                                   | considered on a case-by-case      |
|                                   | basis. For arrears to be paid a   |
|                                   | reasonable explanation must be    |
|                                   | provided by the person,           |
|                                   | supporting the fact that          |
|                                   | financial hardship would result   |
|                                   | if the payment of arrears was not |
|                                   | granted. This information can be  |
|                                   | provided verbally, evidence is    |
|                                   | not required.                     |
|                                   |                                   |
|                                   | Revoking payment choice due to    |
|                                   | financial hardship                |
|                                   |                                   |
|                                   | Kim and Todd have just had their  |
|                                   | first child and are uncertain of  |
|                                   | what their income will be for     |
|                                   | this coming year. Rather than     |
|                                   | take the risk of receiving an     |
|                                   | overpayment, they elect to        |
|                                   | receive the base rate of FTB Part |
|                                   | A and wait until the end of the   |
|                                   | financial year to receive any     |
|                                   | additional FTB Part A they may be |
|                                   | entitled to.                      |
|                                   |                                   |
|                                   | Part way through the year the     |
|                                   | family's income stabilises and    |
|                                   | they decide to revoke their       |
|                                   | choice, as the risk of an         |
|                                   | overpayment is now minimal.       |
|                                   | However, due to unforeseen        |
|                                   | medical expenses relating to      |
|                                   | their baby Toby, Kim and Todd ask |
|                                   | for arrears of FTB A to be paid.  |
|                                   | Services Australia assesses the   |
|                                   | family's income and expenses and  |
|                                   | determines that the family could  |
|                                   | not reasonably meet their medical |
|                                   | expenses and would experience     |
|                                   | considerable financial hardship   |
|                                   | if arrears were not paid.         |
|                                   |                                   |
|                                   | The family's entitlement to FTB   |
|                                   | Part A will be determined based   |
|                                   | on their estimated family income  |
|                                   | and they will be paid arrears     |
|                                   | amounting to the difference       |
|                                   | between the entitlement and what  |
|                                   | they have already received.       |
+-----------------------------------+-----------------------------------+
| 2                                 | Example of revoking payment       |
|                                   | choice for Newborn Upfront        |
|                                   | Payment (NBU)/Newborn Supplement  |
|                                   | (NBS) to be paid                  |
|                                   |                                   |
|                                   | Joseph and Louise are receiving   |
|                                   | FTB for their first child and     |
|                                   | currently have a nil rate FTB     |
|                                   | choice in place. They claim FTB   |
|                                   | on 1 April 2014 for their 2nd     |
|                                   | child born on 4 March 2014. In    |
|                                   | this claim they have requested to |
|                                   | change their choice to receive    |
|                                   | all FTB payments fortnightly.     |
|                                   |                                   |
|                                   | In order for NBU to be paid, the  |
|                                   | payment choice would need to be   |
|                                   | revoked from the date of birth    |
|                                   | (DOB) of the 2nd child (4 March   |
|                                   | 2014).                            |
|                                   |                                   |
|                                   | If the choice is not revoked, the |
|                                   | change would only take effect     |
|                                   | from the date paid to (DPT) + 1,  |
|                                   | therefore resulting in NBS paying |
|                                   | for the remaining period, but NBU |
|                                   | would not be paid due to the      |
|                                   | change not including the first    |
|                                   | day of NBS eligibility (for       |
|                                   | example, child Date of Birth      |
|                                   | (DOB)).                           |
