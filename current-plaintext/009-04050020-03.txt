Eligibility for CP (child) under combined care provisions – assessment of claim

This table describes the steps to follow when assessing eligibility for
Carer Payment (CP) (child) for 2 to 4 children under the combined care
provisions.

+-----------------------+-----------------------+-----------------------+
| Step                  | Action                |                       |
+-----------------------+-----------------------+-----------------------+
| 1                     | Examine the Carer     | []                    |
|                       | Payment Medical       |                       |
|                       | Report (for a child   |                       |
|                       | under 16 years) +     |                       |
|                       | Read more ...         |                       |
|                       |                       |                       |
|                       | Do at least 2 of the  |                       |
|                       | children have either  |                       |
|                       | a disability or a     |                       |
|                       | medical condition?    |                       |
|                       |                       |                       |
|                       | -   Yes, go to Step 2 |                       |
|                       | -   No, go to Step 5  |                       |
+-----------------------+-----------------------+-----------------------+
| 2                     | Examine the duration  | []                    |
|                       | of care required in   |                       |
|                       | the carer payment     |                       |
|                       | Medical Report + Read |                       |
|                       | more ...              |                       |
|                       |                       |                       |
|                       | Do any of the         |                       |
|                       | children require care |                       |
|                       | for a period of at    |                       |
|                       | least 3 months but    |                       |
|                       | less than 6 months?   |                       |
|                       |                       |                       |
|                       | Yes, they may qualify |                       |
|                       | under the provisions  |                       |
|                       | for:                  |                       |
|                       |                       |                       |
|                       | -   short term care   |                       |
|                       | -   episodic care, go |                       |
|                       |     to Step 3         |                       |
|                       |                       |                       |
|                       | No, (or a child       |                       |
|                       | suffers from a        |                       |
|                       | terminal illness), go |                       |
|                       | to Step 3             |                       |
+-----------------------+-----------------------+-----------------------+
| 3                     | Examine the duration  | []                    |
|                       | of care required in   |                       |
|                       | the Carer Payment     |                       |
|                       | Medical Report + Read |                       |
|                       | more ...              |                       |
|                       |                       |                       |
|                       | Do the conditions     |                       |
|                       | indicate that each of |                       |
|                       | the children need     |                       |
|                       | continuous personal   |                       |
|                       | care for either:      |                       |
|                       |                       |                       |
|                       | -   at least 3 months |                       |
|                       |     but less than 6   |                       |
|                       |     months, or        |                       |
|                       | -   6 months or more, |                       |
|                       |     or                |                       |
|                       | -   for the duration  |                       |
|                       |     of a terminal     |                       |
|                       |     illness           |                       |
|                       |                       |                       |
|                       | Note: The terminal    |                       |
|                       | illness must be       |                       |
|                       | supported by written  |                       |
|                       | evidence from a       |                       |
|                       | medical practitioner. |                       |
|                       |                       |                       |
|                       | -   Yes, go to Step 4 |                       |
|                       | -   No, go to Step 5  |                       |
+-----------------------+-----------------------+-----------------------+
| 4                     | Each child has a      | []                    |
|                       | disability or medical |                       |
|                       | condition + Read more |                       |
|                       | ...                   |                       |
|                       |                       |                       |
|                       | -   The carer may be  |                       |
|                       |     considered for CP |                       |
|                       |     (child) under the |                       |
|                       |     combined care     |                       |
|                       |     provisions        |                       |
|                       | -   If the carer is   |                       |
|                       |     also caring for a |                       |
|                       |     disabled adult    |                       |
|                       |     (together with 1  |                       |
|                       |     or 2 children     |                       |
|                       |     with a disability |                       |
|                       |     or medical        |                       |
|                       |     condition) they   |                       |
|                       |     may qualify under |                       |
|                       |     the multiple care |                       |
|                       |     provisions        |                       |
|                       | -   If the carer is a |                       |
|                       |     separated or      |                       |
|                       |     divorced parent   |                       |
|                       |     of the children,  |                       |
|                       |     but does not      |                       |
|                       |     provide care to   |                       |
|                       |     the same          |                       |
|                       |     child/ren every   |                       |
|                       |     day, they may be  |                       |
|                       |     considered for CP |                       |
|                       |     (child) under the |                       |
|                       |     exchanged care    |                       |
|                       |     provisions        |                       |
|                       | -   If the care is    |                       |
|                       |     for more than 3   |                       |
|                       |     months but less   |                       |
|                       |     than 6 months,    |                       |
|                       |     the carer may     |                       |
|                       |     qualify for CP    |                       |
|                       |     (child) under the |                       |
|                       |     short term care   |                       |
|                       |     or the episodic   |                       |
|                       |     care provisions   |                       |
|                       |                       |                       |
|                       | Code the claim for    |                       |
|                       | the 2 to 4 children   |                       |
|                       | combined claim.       |                       |
|                       |                       |                       |
|                       | Go to Step 6.         |                       |
+-----------------------+-----------------------+-----------------------+
| 5                     | None of the children  | []                    |
|                       | qualify + Read more   |                       |
|                       | ...                   |                       |
|                       |                       |                       |
|                       | The claim will be     |                       |
|                       | rejected during the   |                       |
|                       | new claim coding.     |                       |
+-----------------------+-----------------------+-----------------------+
| 6                     | Complete activity +   | []                    |
|                       | Read more ...         |                       |
|                       |                       |                       |
|                       | Record all details on |                       |
|                       | a DOC on the carer's  |                       |
|                       | record, and finalise  |                       |
|                       | the claim via the     |                       |
|                       | Assessment Results    |                       |
|                       | (AR) screen.          |                       |
