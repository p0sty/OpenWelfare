Changing study details to cancel payment when stopping studies

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Changing study details when       |
|                                   | stopping studies + Read more ...  |
|                                   |                                   |
|                                   | If a YA or Austudy (Student)      |
|                                   | stops full-time study they must   |
|                                   | notify within 14 days             |
|                                   | (Notification Period) to receive  |
|                                   | backdating of JobSeeker Payment   |
|                                   | (JSP) or YA (Job seeker) if they  |
|                                   | are eligible.                     |
|                                   |                                   |
|                                   | If they fail to notify they have  |
|                                   | stopped studying full time within |
|                                   | 14 days, payment of JSP/YA (Job   |
|                                   | seeker) may only start from the   |
|                                   | date the job seeker claim was     |
|                                   | submitted.                        |
|                                   |                                   |
|                                   | Update the customer record as     |
|                                   | follows:                          |
|                                   |                                   |
|                                   | go to Customer Study Details      |
|                                   | (EDCH) screen in Process Direct   |
|                                   | or Education Course History       |
|                                   | (EDCH) screen in Customer First   |
|                                   | and correct the Student End Date: |
|                                   | field, to the date the student:   |
|                                   |                                   |
|                                   | -   last studied, or              |
|                                   | -   stopped to have an intention  |
|                                   |     to return to study, or        |
|                                   | -   is no longer undertaking a    |
|                                   |     sufficient study load to      |
|                                   |     qualify for payment. That is, |
|                                   |     the day before their study    |
|                                   |     load reduced                  |
|                                   |                                   |
|                                   | insert a Newstart Education       |
|                                   | Status (NES) entry:               |
|                                   |                                   |
|                                   | -   Event Date: field, code the   |
|                                   |     student end date plus 1 day   |
|                                   | -   Student Status: field, code   |
|                                   |     'NST' (not a full-time        |
|                                   |     student)                      |
|                                   | -   Education Level Attained:     |
|                                   |     field, update if necessary    |
|                                   |                                   |
|                                   | For a dependent student under 22  |
|                                   | years of age receiving the away   |
|                                   | from home rate, determine if they |
|                                   | continue to be entitled to the    |
|                                   | away from home rate.              |
|                                   |                                   |
|                                   | Go to Step 2                      |
+-----------------------------------+-----------------------------------+
| 2                                 | Determining if a manual           |
|                                   | cancellation is required + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | If the customer is:               |
|                                   |                                   |
|                                   | under 22 years of age, and        |
|                                   |                                   |
|                                   | -   no longer wants to continue   |
|                                   |     to receive YA as a jobseeker, |
|                                   |     go to Step 4                  |
|                                   | -   wants to continue to receive  |
|                                   |     YA as a jobseeker, go to Step |
|                                   |     3                             |
|                                   | -   failed to complete the Course |
|                                   |     End Date review, see Student  |
|                                   |     to job seeker transfers       |
|                                   |     (Table 2)                     |
|                                   |                                   |
|                                   | 22 years or over and ceased       |
|                                   | studying:                         |
|                                   |                                   |
|                                   | -   before their 22nd birthday,   |
|                                   |     go to Step 4                  |
|                                   | -   after their 22nd birthday,    |
|                                   |     manual cancellation is not    |
|                                   |     needed. The payments will     |
|                                   |     cancel automatically from the |
|                                   |     updates made to the NES and   |
|                                   |     EDCH screens. Go to Step 5    |
+-----------------------------------+-----------------------------------+
| 3                                 | YA (Student) Transfer to YA as a  |
|                                   | jobseeker + Read more ...         |
|                                   |                                   |
|                                   | Customers under 22 years who      |
|                                   | advise of stopping study within   |
|                                   | the 14 day notification period,   |
|                                   | and are seeking another income    |
|                                   | support payment do not need to    |
|                                   | lodge a claim.                    |
|                                   |                                   |
|                                   | If the customer has contacted:    |
|                                   |                                   |
|                                   | within their Notification Period, |
|                                   | and they want to receive further  |
|                                   | financial assistance:             |
|                                   |                                   |
|                                   | -   for a job seeker payment, see |
|                                   |     Student to Job seeker         |
|                                   |     transfers                     |
|                                   | -   for an alternative income     |
|                                   |     support, see Contact in       |
|                                   |     relation to an intended claim |
|                                   |     (CLK)                         |
|                                   |                                   |
|                                   | outside their Notification Period |
|                                   | and want to receive further       |
|                                   | financial assistance, they must   |
|                                   | lodge a new claim. Go to Step 4   |
+-----------------------------------+-----------------------------------+
| 4                                 | Coding a manual cancellation +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Manual cancellation of the        |
|                                   | student payment is needed for     |
|                                   | customers:                        |
|                                   |                                   |
|                                   | -   22 years or over who ceased   |
|                                   |     studying before their 22nd    |
|                                   |     birthday, and                 |
|                                   | -   customers under 22 years of   |
|                                   |     age who do not want to        |
|                                   |     continue to receive YA as a   |
|                                   |     jobseeker                     |
|                                   |                                   |
|                                   | To manually cancel the student    |
|                                   | payment:                          |
|                                   |                                   |
|                                   | -   go to the Benefit Action (BA) |
|                                   |     screen                        |
|                                   | -   Service Reason: field, select |
|                                   |     the customers payment type    |
|                                   | -   Action: field, CAN - Cancel   |
|                                   | -   Reason: field 'NST' - not a   |
|                                   |     full-time student             |
|                                   | -   Effect Date: field, new study |
|                                   |     end date plus 1 day           |
|                                   |                                   |
|                                   | Record the details of the         |
|                                   | decision to cancel on a DOC.      |
|                                   |                                   |
|                                   | Go to Step 5.                     |
+-----------------------------------+-----------------------------------+
| 5                                 | Further financial assistance +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | If the customer wants to receive  |
|                                   | further financial assistance and  |
|                                   | contacts:                         |
|                                   |                                   |
|                                   | within their Notification Period  |
|                                   |                                   |
|                                   | -   for a job seeker payment, see |
|                                   |     Student to Job seeker         |
|                                   |     transfers                     |
|                                   | -   for an alternative income     |
|                                   |     support, see Contact in       |
|                                   |     relation to an intended claim |
|                                   |     (CLK)                         |
|                                   |                                   |
|                                   | outside their Notification Period |
|                                   | they must lodge a new claim. Go   |
|                                   | to Step 6                         |
|                                   |                                   |
|                                   | If the customer does not want to  |
|                                   | receive further financial         |
|                                   | assistance, go to step 6          |
+-----------------------------------+-----------------------------------+
| 6                                 | Debt incurred for failing to      |
|                                   | notify + Read more ...            |
|                                   |                                   |
|                                   | Determine if the customer has     |
|                                   | incurred a debt for failing to    |
|                                   | notify of stopping studies, see:  |
|                                   |                                   |
|                                   | -   Raising debts resulting from  |
|                                   |     a failure to report or notify |
|                                   |     and                           |
|                                   | -   Raising debts for Centrelink  |
|                                   |     administered payments for     |
|                                   |     students                      |
|                                   |                                   |
|                                   | Record details of the decision to |
|                                   | cancel and any other action taken |
|                                   | on a DOC.                         |
