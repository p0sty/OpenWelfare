| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Part 1 financial vulnerability    |
|                                   | test + Read more ...              |
|                                   |                                   |
|                                   | There are two parts to the        |
|                                   | process of assessing exemptions   |
|                                   | for customers with dependent      |
|                                   | children.                         |
|                                   |                                   |
|                                   | Part one of the exemption process |
|                                   | requires a customer to            |
|                                   | demonstrate there have been no    |
|                                   | indications of financial          |
|                                   | vulnerability during the 12 month |
|                                   | period ending immediately before  |
|                                   | the date of the exemption         |
|                                   | application.                      |
|                                   |                                   |
|                                   | The principles for determining    |
|                                   | that no indications of financial  |
|                                   | vulnerability were present during |
|                                   | the previous 12 months are that   |
|                                   | the customer:                     |
|                                   |                                   |
|                                   | -   Has been applying appropriate |
|                                   |     resources to meet their and   |
|                                   |     their dependents priority     |
|                                   |     needs                         |
|                                   | -   Had stable payment patterns   |
|                                   |     and budgeting practices and   |
|                                   |     is meeting priority needs     |
|                                   |     from their income support and |
|                                   |     family assistance payments    |
|                                   | -   Had control over their money  |
|                                   |     and was not subject to        |
|                                   |     financial exploitation        |
|                                   | -   Did not regularly require     |
|                                   |     urgent funds to pay for       |
|                                   |     foreseeable costs, or did not |
|                                   |     frequently change their       |
|                                   |     income support pay dates.     |
|                                   |     Consideration is given to the |
|                                   |     reason for seeking the urgent |
|                                   |     payment                       |
|                                   |                                   |
|                                   | For more information about the    |
|                                   | Financial Vulnerability test, see |
|                                   | Conducting the financial          |
|                                   | vulnerability test for an         |
|                                   | exemption from Income Management  |
|                                   | for customers with dependent      |
|                                   | children.                         |
+-----------------------------------+-----------------------------------+
| 2                                 | Part 2 participation              |
|                                   | requirements + Read more ...      |
|                                   |                                   |
|                                   | Part two of the exemption process |
|                                   | requires a customer with          |
|                                   | dependent children of compulsory  |
|                                   | school age for the purposes of    |
|                                   | Income Management or younger to   |
|                                   | demonstrate they are undertaking  |
|                                   | responsible parenting.            |
|                                   |                                   |
|                                   | This is done by meeting           |
|                                   | attendance or participation       |
|                                   | requirements relating to          |
|                                   | education, health care and/or     |
|                                   | other activities.                 |
+-----------------------------------+-----------------------------------+
| 3                                 | 12 month period for exemptions +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Any exemption granted will remain |
|                                   | valid for a period of 12 months.  |
|                                   |                                   |
|                                   | At the end of the 12 months the   |
|                                   | customer will be advised they are |
|                                   | eligible for Income Management    |
|                                   | and must contact Services         |
|                                   | Australia for an initial          |
|                                   | assessment. The customer can      |
|                                   | reapply for an exemption from     |
|                                   | Income Management at this time    |
|                                   | and will have access to the same  |
|                                   | engagement and grace periods as   |
|                                   | any other 'new' customer.         |
|                                   |                                   |
|                                   | Although exemptions are not       |
|                                   | reviewed during the 12 month      |
|                                   | period, it may be reassessed      |
|                                   | prior to the end of the 12 month  |
|                                   | exemption period in cases where   |
|                                   | an individual's circumstances     |
|                                   | have significantly changed. See   |
|                                   | the Resources page for examples   |
|                                   | of when a review of an exemption  |
|                                   | would be appropriate.             |
+-----------------------------------+-----------------------------------+
| 4                                 | Determining responsible           |
|                                   | parenting - child of compulsory   |
|                                   | school age + Read more ...        |
|                                   |                                   |
|                                   | The principles for determining    |
|                                   | responsible parenting in relation |
|                                   | to each child of compulsory       |
|                                   | school age for the purposes of    |
|                                   | Income Management require one of  |
|                                   | the following to be met:          |
|                                   |                                   |
|                                   | School attendance (less than 5    |
|                                   | satisfactory absences):           |
|                                   |                                   |
|                                   | -   The child is enrolled in and  |
|                                   |     attending school              |
|                                   | -   Has no more than 5 absences   |
|                                   |     for each of the previous 2    |
|                                   |     school terms that are not     |
|                                   |     satisfactory to the person    |
|                                   |     responsible for the operation |
|                                   |     of the school, and            |
|                                   | -   The customer provides         |
|                                   |     evidence of school            |
|                                   |     attendance. See Step 1 in the |
|                                   |     Evidence required for         |
|                                   |     activity table                |
|                                   |                                   |
|                                   | School attendance (five or more   |
|                                   | unsatisfactory absences):         |
|                                   |                                   |
|                                   | -   The child is enrolled in and  |
|                                   |     attending school              |
|                                   | -   Has 5 or more absences that   |
|                                   |     are not satisfactory to a     |
|                                   |     person responsible for the    |
|                                   |     operation of the school, and  |
|                                   | -   The customer is taking        |
|                                   |     reasonable steps to ensure    |
|                                   |     the child attends school as   |
|                                   |     required. For example, they   |
|                                   |     have taken action to stop     |
|                                   |     truancy by addressing         |
|                                   |     bullying at school            |
|                                   |                                   |
|                                   | Home schooling:                   |
|                                   |                                   |
|                                   | -   The child is approved to      |
|                                   |     receive home based schooling, |
|                                   |     or other alternative          |
|                                   |     schooling arrangement, in     |
|                                   |     accordance with the relevant  |
|                                   |     state or territory            |
|                                   |     legislation, and              |
|                                   | -   The customer must provide     |
|                                   |     evidence of home schooling.   |
|                                   |     See Step 2 in the Evidence    |
|                                   |     required for activity table   |
|                                   |                                   |
|                                   | Special circumstances:            |
|                                   |                                   |
|                                   | -   If there are special          |
|                                   |     circumstances preventing      |
|                                   |     participation in schooling,   |
|                                   |     the customer must provide     |
|                                   |     evidence of special           |
|                                   |     circumstances for             |
|                                   |     non-enrolment and attendance. |
|                                   |     See Step 3 in the Evidence    |
|                                   |     required for activity table   |
|                                   |                                   |
|                                   | In the case of changed care       |
|                                   | arrangements, a foster parent or  |
|                                   | dependent carer should not be     |
|                                   | held responsible for attendance   |
|                                   | patterns prior to the time the    |
|                                   | child entered their care. School  |
|                                   | attendance should be considered   |
|                                   | only from the time the child      |
|                                   | commences care with the foster    |
|                                   | parent or dependent carer.        |
+-----------------------------------+-----------------------------------+
| 5                                 | Determining responsible           |
|                                   | parenting - child under           |
|                                   | compulsory school age + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | The principles for determining    |
|                                   | responsible parenting in relation |
|                                   | to each child under compulsory    |
|                                   | school age for the purposes of    |
|                                   | Income Management are:            |
|                                   |                                   |
|                                   | -   The child is participating in |
|                                   |     the required number and kind  |
|                                   |     of activities relating to     |
|                                   |     their intellectual, physical  |
|                                   |     or social development         |
|                                   | -   This depends on the child's   |
|                                   |     age as follows.               |
|                                   |                                   |
|                                   | Child aged 0 - 36 months          |
|                                   |                                   |
|                                   | -   At least two Health Related   |
|                                   |     Activities                    |
|                                   |                                   |
|                                   | Child aged 36 - 48 months         |
|                                   |                                   |
|                                   | -   At least two Health Related   |
|                                   |     Activities, and               |
|                                   | -   At least one Engagement       |
|                                   |     Activity                      |
|                                   |                                   |
|                                   | Child aged 48 months to           |
|                                   | compulsory school age for the     |
|                                   | purposes of Income Management     |
|                                   |                                   |
|                                   | -   At least two Health Related   |
|                                   |     Activities, and               |
|                                   | -   Regular participation in      |
|                                   |     preschool, or                 |
|                                   | -   At least two other Engagement |
|                                   |     Activities. One of these      |
|                                   |     activities to be a formal     |
|                                   |     activity requiring evidence   |
|                                   |     such as child care            |
|                                   |                                   |
|                                   | In cases where a child under      |
|                                   | compulsory school age for the     |
|                                   | purposes of Income Management has |
|                                   | significant disabilities, three   |
|                                   | Health Related Activities would   |
|                                   | be appropriate, due to the        |
|                                   | difficulty of finding             |
|                                   | appropriately supported social    |
|                                   | activities.                       |
|                                   |                                   |
|                                   | Evidence of significant           |
|                                   | disabilities would need to be     |
|                                   | provided to the agency in the     |
|                                   | form of a Doctor's Certificate or |
|                                   | Treating Doctor's Report (TDR).   |
|                                   | The Income Management Exemption   |
|                                   | and Assessment Team (IMEAT) are   |
|                                   | the primary decision makers in    |
|                                   | granting an exemption in these    |
|                                   | circumstances.                    |
|                                   |                                   |
|                                   | Note: Additional updates are      |
|                                   | required for customers impacted   |
|                                   | by the Healthy Start for School   |
|                                   | initiative. If they lodge         |
|                                   | evidence to support their Income  |
|                                   | Management exemption application  |
|                                   | which proves the child has        |
|                                   | completed a relevant health check |
|                                   | since turning 3, their health     |
|                                   | check status should also be       |
|                                   | updated. For more information,    |
|                                   | see Healthy Start for School.     |
+-----------------------------------+-----------------------------------+
| 6                                 | Health Related Activities -       |
|                                   | Immunisation and health checks +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | The following are the acceptable  |
|                                   | Health Related Activities for     |
|                                   | children under compulsory school  |
|                                   | age for the purposes of an        |
|                                   | exemption from Income Management: |
|                                   |                                   |
|                                   | Age appropriate level of          |
|                                   | immunisation                      |
|                                   |                                   |
|                                   | In accordance with Family         |
|                                   | Assistance Act section 6, the     |
|                                   | immunisations are scheduled at:   |
|                                   |                                   |
|                                   | -   2 months                      |
|                                   | -   4 months                      |
|                                   | -   6 months                      |
|                                   | -   12 months                     |
|                                   | -   18 months, and                |
|                                   | -   4 years                       |
|                                   |                                   |
|                                   | Or one of the following:          |
|                                   |                                   |
|                                   | -   A child has a medical reason  |
|                                   |     not to have a particular      |
|                                   |     vaccination                   |
|                                   | -   A child has had a disease and |
|                                   |     has a natural immunity        |
|                                   | -   A particular vaccine is       |
|                                   |     unavailable                   |
|                                   | -   A child is on a 'catch up'    |
|                                   |     vaccination schedule for a    |
|                                   |     range of reasons              |
|                                   |                                   |
|                                   | For a link to the forms and the   |
|                                   | Australian Immunisation Register  |
|                                   | (AIR) Immunisation Schedule see   |
|                                   | the Resources page.               |
|                                   |                                   |
|                                   | The customer needs to provide     |
|                                   | evidence of immunisation.         |
|                                   |                                   |
|                                   | Regular child health checks       |
|                                   |                                   |
|                                   | As set out in the personal health |
|                                   | and immunisation record as        |
|                                   | published by the Northern         |
|                                   | Territory Government. This        |
|                                   | requires health checks at:        |
|                                   |                                   |
|                                   | -   Up to 10 days                 |
|                                   | -   4 weeks                       |
|                                   | -   2 months                      |
|                                   | -   6 - 8 months                  |
|                                   | -   12 months                     |
|                                   | -   18 months                     |
|                                   | -   2 years 6 months, and         |
|                                   | -   3 - 4 years                   |
|                                   |                                   |
|                                   | The customer needs to provide     |
|                                   | evidence of regular child health  |
|                                   | checks.                           |
|                                   |                                   |
|                                   | Regular health checks or          |
|                                   | immunisation requirements not met |
|                                   |                                   |
|                                   | If a customer has not met the     |
|                                   | regular health checks or          |
|                                   | immunisation requirements listed  |
|                                   | above they should not be granted  |
|                                   | an exemption in the first         |
|                                   | instance.                         |
|                                   |                                   |
|                                   | If the customer reapplies for an  |
|                                   | exemption, the customer must      |
|                                   | provide evidence showing they     |
|                                   | have undertaken a full child      |
|                                   | health check along with all       |
|                                   | required catch up immunisations   |
|                                   | to qualify for an exemption.      |
|                                   |                                   |
|                                   | Participation in a program        |
|                                   |                                   |
|                                   | The child is participating in a   |
|                                   | program of speech therapy,        |
|                                   | occupational therapy, physical    |
|                                   | therapy or other approved         |
|                                   | therapy. The customer needs to    |
|                                   | provide evidence from a health    |
|                                   | professional.                     |
|                                   |                                   |
|                                   | Participation in this case means  |
|                                   | reasonable participation as       |
|                                   | assessed by the health            |
|                                   | professional.                     |

|                                   | -   Occasional care               |

|                                   | Age Appropriate Activities        |

|                                   |                                   |
|                                   | If the child is under compulsory  |
|                                   | school age and attending school   |

|                                   | as evidence of an engagement      |
|                                   | activity. Attendance at school    |
|                                   | must be verified, refer to        |
|                                   | Evidence of School attendance.    |

| 2                                 | Grace period to gather evidence + |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Customers, with dependent         |
|                                   | children, not currently income    |
|                                   | managed but eligible for Income   |
|                                   | Management, are given a 28 day    |
|                                   | grace period to gather the        |
|                                   | evidence required to be granted   |
|                                   | an exemption.                     |
|                                   |                                   |
|                                   | This means, once the customer     |
|                                   | applies for an exemption, they    |
|                                   | will have 28 days to gather the   |
|                                   | required information before       |
|                                   | Income Management is applied.     |
|                                   |                                   |
|                                   | This grace period is in addition  |
|                                   | the 28/56 day engagement period.  |
|                                   |                                   |
|                                   | At the end of the grace period,   |
|                                   | if the exemption has not been     |
|                                   | granted, the customer is given an |
|                                   | additional engagement period to   |
|                                   | complete an initial assessment    |
|                                   | for Income Management before Auto |
|                                   | Income Management is applied.     |
|                                   |                                   |
|                                   | Customers who are already income  |
|                                   | managed and are applying for an   |
|                                   | exemption, have 28 days to        |
|                                   | complete their application. At    |
|                                   | the end of the 28 days, if the    |
|                                   | required evidence has not been    |
|                                   | provided, the application will be |
|                                   | rejected and Income Management    |
|                                   | will continue to apply.           |
|                                   |                                   |
|                                   | See the Resources page for an     |
|                                   | example.                          |

| 1                                 | Evidence of school attendance +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Evidence of attendance at school  |
|                                   | is taken from the child's school  |
|                                   | report which, in the Northern     |
|                                   | Territory, includes the total     |
|                                   | number of absences.               |
|                                   |                                   |
|                                   | Where the school reports show     |
|                                   | there have been more than 5       |
|                                   | absences in either of the last    |
|                                   | two terms or semester, the        |
|                                   | customer will be asked to obtain  |
|                                   | evidence directly from the        |
|                                   | school. This is to determine      |
|                                   | whether the absences were         |
|                                   | satisfactory to the person        |
|                                   | responsible for the operation of  |
|                                   | the school.                       |
|                                   |                                   |
|                                   | The school report, together with  |
|                                   | other evidence provided by the    |
|                                   | school, would be considered       |
|                                   | definitive.                       |
|                                   |                                   |
|                                   | -   Parent/principal carer would  |
|                                   |     not be able to provide        |
|                                   |     evidence to the agency where  |
|                                   |     the school has wrongly        |
|                                   |     reported their child's        |
|                                   |     attendance                    |
|                                   | -   If the parent/principal carer |
|                                   |     felt the school had           |
|                                   |     misrepresented their child's  |
|                                   |     attendance, they should be    |
|                                   |     advised to follow this up     |
|                                   |     with the school, and seek to  |
|                                   |     have an amended report issued |
|                                   |     or a letter from the school   |
|                                   |     correcting the information    |
|                                   |                                   |
|                                   | Customers with children attending |
|                                   | a Northern Territory Department   |
|                                   | of Education school               |
|                                   |                                   |
|                                   | -   The parent/principal carer    |
|                                   |     has to request the child's    |
|                                   |     attendance report for Income  |
|                                   |     Management exemption purposes |
|                                   | -   The individual student        |
|                                   |     attendance record will be     |
|                                   |     produced by school staff and  |
|                                   |     forwarded to the Principal of |
|                                   |     the school                    |
|                                   | -   When the Principal approves   |
|                                   |     the report it is emailed to   |
|                                   |     the Income Management         |
|                                   |     Exemption and Assessment Team |
|                                   |     (IMEAT) and where possible    |
|                                   |     the parent/principal carer    |
|                                   |     will be cc'd into the email   |
|                                   | -   It is the responsibility of   |
|                                   |     parent/principal carer to     |
|                                   |     follow up with the agency to  |
|                                   |     make sure the report has been |
|                                   |     received                      |
|                                   | -   Where a parent/principal      |
|                                   |     carer disputes the record,    |
|                                   |     they must contact the School  |
|                                   |     Enrolment and Attendance Team |
|                                   |     and lodge their dispute       |
|                                   | -   Burden of proof rests with    |
|                                   |     the parent/principal carer    |
|                                   | -   Where a parent/principal      |
|                                   |     carer can prove their child   |
|                                   |     was present, a school can     |
|                                   |     change the record             |
|                                   | -   Where a parent/principal      |
|                                   |     carer attains a medical       |
|                                   |     certificate or other evidence |
|                                   |     for dates marked with a 'U'   |
|                                   |     or 'X' code after a 10 week   |
|                                   |     period, the school will not   |
|                                   |     change the record as this is  |
|                                   |     not evidence of the timely    |
|                                   |     notification of the absence   |
|                                   | -   See the Resources page for a  |
|                                   |     link to the contact details   |
