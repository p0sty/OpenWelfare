Respond to pharmacist enquiry

This table describes how to respond to a call from a pharmacist on the
PBS Authorities Line.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Record approval number            |
|                                   |                                   |
|                                   | Record the approval number of the |
|                                   | pharmacist calling.               |
+-----------------------------------+-----------------------------------+
| 2                                 | Advise correct number to call     |
|                                   |                                   |
|                                   | Tell the pharmacist the correct   |
|                                   | number to call. For contact       |
|                                   | details, see:                     |
|                                   |                                   |
|                                   | -   PBS - General enquires line   |
|                                   | -   PBS - Authority approvals     |
+-----------------------------------+-----------------------------------+
| 3                                 | Transfer the call                 |
|                                   |                                   |
|                                   | Transfer the call to the correct  |
|                                   | number. For contact details, see: |
|                                   |                                   |
|                                   | -   PBS - General enquires line   |
|                                   | -   PBS - Authority approvals     |
+-----------------------------------+-----------------------------------+
