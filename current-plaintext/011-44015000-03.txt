|                                   | ...                               |
|                                   |                                   |
|                                   | Once DVS request complete, an     |

|                                   | the outcome.                      |

|                                   | Match results include:            |
|                                   |                                   |
|                                   | Yes,                              |
|                                   |                                   |
|                                   | -   Verification successful       |

|                                   |                                   |

|                                   |                                   |

|                                   |     unsuccessful (2 attempts to   |

|                                   |     letter                        |
|                                   |                                   |

|                                   | with issuer, but cannot be        |
|                                   | verified)                         |
|                                   |                                   |

|                                   | -   Return the original document  |
|                                   |     to the customer with relevant |

Face to Face (F2F) Service Officer escalation

Table 3

+-----------------------------------+-----------------------------------+
| Step                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Try to resolve the enquiry + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | A F2F Service Officer must try to |
|                                   | resolve the customer enquiry      |
|                                   | first by using:                   |
|                                   |                                   |
|                                   | -   Operational Blueprint         |
|                                   | -   staff messages such as Daily  |
|                                   |     Comms                         |
|                                   | -   comments in the Consumer      |
|                                   |     Directory Maintenance System  |
|                                   |     (CDMS)                        |
|                                   | -   a VEVO system check           |
+-----------------------------------+-----------------------------------+
| 2                                 | Contact Medicare LPS for help +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | If the F2F Service Officer is     |
|                                   | unable to resolve the customer’s  |
|                                   | enquiry call Medicare and Aged    |
|                                   | Care - Local Peer Support (LPS).  |
|                                   |                                   |
|                                   | If it is deemed not urgent, the   |
|                                   | F2F Service Officer escalates the |
|                                   | enquiry to LPS. To do this the    |
|                                   | F2F Service Officer will:         |
|                                   |                                   |
|                                   | Scan all of the enrolment         |
|                                   | documents                         |
|                                   |                                   |
|                                   | In the CDMS:                      |
|                                   |                                   |
|                                   | -   Add a Comment at the Group    |
|                                   |     level (if the customer has a  |
|                                   |     Medicare number). For         |
|                                   |     example, the enrolment has    |
|                                   |     been escalated                |
|                                   | -   Add the LPS login ID details  |
|                                   |     in the comment                |
|                                   |                                   |
|                                   | This work will be finalised by a  |
|                                   | Medicare Eligibility and          |
|                                   | Enrolment trained Service Officer |
|                                   | or LPS.                           |
|                                   |                                   |
|                                   | If the enquiry is urgent, go to   |
|                                   | Step 3.                           |
+-----------------------------------+-----------------------------------+
| 3                                 | Escalate when the enquiry is      |
|                                   | urgent + Read more ...            |
|                                   |                                   |
|                                   | If an enquiry requires an         |
|                                   | immediate response, LPS will call |
|                                   | the Medicare Consumers Program    |
|                                   | Support team for advice.          |
|                                   |                                   |
|                                   | Note: if the Medicare Consumers   |
|                                   | Program Support team cannot       |
|                                   | resolve the enquiry over the      |
|                                   | phone, refer to Escalations from  |
|                                   | LPS to Medicare Consumers Program |
|                                   | Support.                          |

Escalations from LPS to Medicare Consumers Program Support

Table 4

+-----------------------------------+-----------------------------------+
| Step                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Escalate to Medicare Consumers    |
|                                   | Program Support + Read more ...   |
|                                   |                                   |
|                                   | If LPS is unable to resolve the   |
|                                   | enquiry, LPS will:                |
|                                   |                                   |
|                                   | escalate the enquiry via PaNDA    |
|                                   | using work type Medicare          |
|                                   | Eligibility Nat Office            |
|                                   | Escalations, or                   |
|                                   |                                   |
|                                   | email Medicare Consumers Program  |
|                                   | Support - Medicare Consumers      |
|                                   | Section - Medicare Branch         |
|                                   | including:                        |
|                                   |                                   |
|                                   | -   all enquiry details, and      |
|                                   | -   any forms and documents       |
|                                   |                                   |
|                                   | The Medicare Consumers Program    |
|                                   | Support will send a response      |
|                                   | within 10 business days.          |
+-----------------------------------+-----------------------------------+
| 2                                 | Escalate an urgent enquiry + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | If an Eligibility LPS needs help  |
|                                   | with an urgent query, the         |
|                                   | Eligibility LPS can call Medicare |
|                                   | Consumers Program Support.        |
|                                   |                                   |
|                                   | LPS will explain that the query   |
|                                   | is urgent and give Medicare       |
|                                   | Consumers Program Support all of  |
|                                   | the details.                      |
|                                   |                                   |
|                                   | The Medicare Consumers Program    |
|                                   | Support will:                     |
|                                   |                                   |
|                                   | -   tell them how to resolve the  |
|                                   |     enquiry, or                   |
|                                   | -   ask them to send all of the   |
|                                   |     details                       |
|                                   |                                   |
|                                   | Email requests                    |
|                                   |                                   |
|                                   | The Medicare Consumers Program    |
|                                   | Support will respond to urgent    |
|                                   | email requests within 2 business  |
|                                   | days. The team will assess urgent |
|                                   | requests and respond to them in   |
|                                   | order of priority.                |
|                                   |                                   |
|                                   | The response time may be longer   |
|                                   | if the case needs external        |
|                                   | stakeholder advice.               |
|                                   |                                   |
|                                   | PaNDA requests:                   |
|                                   |                                   |
|                                   | -   Applications escalated via    |
|                                   |     PaNDA must be escalated using |
|                                   |     work type Medicare            |
|                                   |     Eligibility and enrolment,    |
|                                   |     Nat.Office Escalations        |
|                                   | -   Requests to remove children   |
|                                   |     under 15 from a person        |
|                                   |     Medicare card escalate using  |
|                                   |     work type Nat.Office          |
|                                   |     Escalation DCP                |

Escalate a Department of Child Protection (DCP) or a prisoner enquiry

Table 5

+-----------------------------------+-----------------------------------+
| Step                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | How to handle a F2F DCP or        |
|                                   | prisoner enquiry + Read more ...  |
|                                   |                                   |
|                                   | F2F Service Officer must:         |
|                                   |                                   |
|                                   | -   escalate the query to a       |
|                                   |     Medicare and Aged Care -      |
|                                   |     Local Peer Support (LPS)      |
|                                   | -   if processing is required     |
|                                   |     scan all of the documents and |
|                                   |     forms to PaNDA                |
|                                   |                                   |
|                                   | The DCP or a non-government       |
|                                   | organisation may make a verbal    |
|                                   | request for a child’s Medicare    |
|                                   | card details. Do not give out     |
|                                   | these details. The DCP or         |
|                                   | non-government organisation needs |
|                                   | to call a Medicare Eligibility    |
|                                   | trained Service Officer and give  |
|                                   | their password and agency ID.     |
|                                   |                                   |
|                                   | Any request from a non-government |
|                                   | body must not be actioned, for    |
|                                   | example, Anglicare or Wesley      |
|                                   | Mission. Call the Medicare        |
|                                   | Eligibility LPS team for help.    |
+-----------------------------------+-----------------------------------+
| 2                                 | DCP and prisoner queries for a    |
|                                   | Service Officer + Read more ...   |
|                                   |                                   |
|                                   | If a Service Officer needs help,  |
|                                   | call Eligibility LPS.             |
|                                   |                                   |
|                                   | If LPS asks the Service Officer   |
|                                   | to send them the enquiry, they    |
|                                   | must:                             |
|                                   |                                   |
|                                   | Forward the details to the LPS    |
|                                   | via the digitised workflow        |
|                                   |                                   |
|                                   | Details of the enquiry may        |
|                                   | include:                          |
|                                   |                                   |
|                                   | -   The file name Work Item ID    |
|                                   | -   A Medicare card number        |
|                                   | -   The applicant’s name          |
+-----------------------------------+-----------------------------------+
| 3                                 | DCP and prisoner queries for      |
|                                   | Medicare Consumers Program        |
|                                   | Support + Read more ...           |
|                                   |                                   |
|                                   | If LPS needs help:                |
|                                   |                                   |
|                                   | -   Call Medicare Consumers       |
|                                   |     Program Support, or           |
|                                   | -   escalate via PaNDA using work |
|                                   |     type Medicare Eligibility     |
|                                   |     Nat.Office Escalations DCP    |
|                                   |                                   |
|                                   | The Medicare Consumers Program    |
|                                   | Support will respond to the LPS.  |
|                                   | A LPS or a Service Officer will   |
|                                   | complete the enrolment.           |

Escalate a third Medicare card request

Table 6

+-----------------------------------+-----------------------------------+
| Step                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | A third card request + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | F2F or Service Officers must      |
|                                   | escalate all third card requests  |
|                                   | to the Medicare Eligibility LPS   |
|                                   | team via PaNDA. LPS do not decide |
|                                   | and grant third card approvals    |
|                                   | verbally.                         |
|                                   |                                   |
|                                   | Service Officers must ensure a    |
|                                   | completed application and all     |
|                                   | relevant documentation has been   |
|                                   | provided by the applicant.        |
|                                   |                                   |
|                                   | Service Officers will escalate to |
|                                   | a Medicare Eligibility LPS via    |
|                                   | PaNDA under the work type:        |
|                                   |                                   |
|                                   | -   DCP 3rd card request to       |
|                                   |     Medicare Eligibility DCP      |
|                                   |     Escalations.                  |
|                                   | -   Other 3rd card requests to    |
|                                   |     Medicare Eligibility          |
|                                   |     Escalations                   |
|                                   |                                   |
|                                   | The LPS will:                     |
|                                   |                                   |
|                                   | -   assess and confirm that the   |
|                                   |     request is valid              |
|                                   | -   include approval/rejection in |
|                                   |     PaNDA work item notes         |
|                                   | -   re-assign the third card      |
|                                   |     request back to the           |
|                                   |     processing queue for the next |
|                                   |     Medicare Eligibility trained  |
|                                   |     Service Officer to complete   |
|                                   |                                   |
|                                   | All 4th card (or more) must be    |
|                                   | escalated to the Medicare         |
|                                   | Consumers Program Support team.   |
|                                   | The LPS must send the request via |
|                                   | PaNDA using work type Medicare    |
|                                   | Eligibility Nat.Office            |
|                                   | Escalations DCP.                  |
