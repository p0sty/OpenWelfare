1-SCP number error message

Table 1: this table describes how to resolve error message 1-SCP#. This
displays when a specimen collection point (SCP) number has been entered
incorrectly or in the wrong format.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Examine account documents + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Examine the account documents for |
|                                   | an SCP number.                    |
+-----------------------------------+-----------------------------------+
| 2                                 | Locate the SCP + Read more ...    |
|                                   |                                   |
|                                   | Is the SCP found?                 |
|                                   |                                   |
|                                   | -   Yes, key the SCP (preceded by |
|                                   |     an S qualifier) in the FEE    |
|                                   |     field against all pathology   |
|                                   |     items within the claim. For   |
|                                   |     example, a SCP of 01A is      |
|                                   |     keyed as S01A and press       |
|                                   |     [Enter]                       |
|                                   | -   No, reject all pathology      |
|                                   |     items within the claim with   |
|                                   |     RSN 551 (SCP is incorrect or  |
|                                   |     not supplied) and press       |
|                                   |     [Enter]                       |
+-----------------------------------+-----------------------------------+

4-PATH RQ error message

Table 4: this table describes how to resolve 4-PATH RQ error message.
This displays when a pathology request (PA) line or processing indicator
(PI) 11 has not been keyed for a pathology item from Group P1 to Group
P8.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Examine account documents + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Examine the account documents for |
|                                   | a request indication.             |
+-----------------------------------+-----------------------------------+
| 2                                 | Action request details + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | -   If request details are found, |
|                                   |     key a pathology request (PA)  |
|                                   |     line containing details of    |
|                                   |     the request and press [Enter] |
|                                   | -   If the service was            |
|                                   |     self-determined, key          |
|                                   |     processing indicator (PI) 11  |
|                                   |     (pathology service has been   |
|                                   |     self-determined) in the PI    |
|                                   |     field against the items       |
|                                   |     involved and press [Enter]    |
|                                   | -   If request details are not on |
|                                   |     the account and the services  |
|                                   |     are not self-determined,      |
|                                   |     reject the item involved with |
|                                   |     RSN 137 (details of           |
|                                   |     requesting provider not shown |
|                                   |     on account or receipt) and    |
|                                   |     press [Enter]                 |

4-CHK RQST error message

Table 5: this table describes how to resolve 4-CHK RQST error message.
This displays when a pathology request (PA) line is keyed without an
associated pathology item or does not require a PA line.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Determine if there are associated |
|                                   | pathology items + Read more ...   |
|                                   |                                   |
|                                   | Examine the details keyed on the  |
|                                   | screen and those shown on the     |
|                                   | account documents to determine    |
|                                   | whether there are associated      |
|                                   | pathology items.                  |
+-----------------------------------+-----------------------------------+
| 2                                 | Action pathology items + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | If the associated pathology items |
|                                   | from Group P1 to Group P8 are on  |
|                                   | the screen:                       |
|                                   |                                   |
|                                   | -   rekey details of the          |
|                                   |     pathology request (PA) line   |
|                                   |     immediately after the         |
|                                   |     associated pathology items    |
|                                   |     involved                      |
|                                   | -   delete the original PA line   |
|                                   |                                   |
|                                   | If the associated pathology items |
|                                   | from Group P1 to Group P8 are on  |
|                                   | the account and have not been     |
|                                   | keyed, insert item lines          |
|                                   | containing the pathology items    |
|                                   | immediately before the PA line    |
|                                   | involved                          |
|                                   |                                   |
|                                   | If there are no associated        |
|                                   | pathology items from Group P1 to  |
|                                   | Group P8 on the screen or on the  |
|                                   | account, delete the PA line       |
|                                   | involved                          |
|                                   |                                   |
|                                   | If the associated pathology item  |
|                                   | lines contains processing         |
|                                   | indicator 11 (pathology service   |
|                                   | has been self-determined):        |
|                                   |                                   |
|                                   | -   erase processing indicator    |
|                                   |     (PI) 11 from the associated   |
|                                   |     item lines, or                |
|                                   | -   delete the PA line involved   |
|                                   |                                   |
|                                   | If processing an online claim,    |
|                                   | follow the error message and      |
|                                   | reject the claim                  |
|                                   |                                   |
|                                   | Press [Enter].                    |

6-CMP error message

Table 6: this table describes how to resolve 6-CMP error message. This
displays when a composite situation is detected during processing. The
composite item that must be paid shows in the message and the individual
items are highlighted. This table also describes circumstances where the
pathology account contains the notation Rule 3 Exemption or S4B(3).

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Claiming channel + Read more ...  |
|                                   |                                   |
|                                   | -   Online claims, see Medicare   |
|                                   |     online claiming – 6 CMP       |
|                                   | -   Manual claims, go to Step 2   |
+-----------------------------------+-----------------------------------+
| 2                                 | Examine account documents + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Examine the account documents for |
|                                   | any indication that the items     |
|                                   | involved can be paid              |
|                                   | individually.                     |
+-----------------------------------+-----------------------------------+
| 3                                 | Determine pathology service       |
|                                   | type + Read more ...              |
|                                   |                                   |
|                                   | Determine if the services are:    |
|                                   |                                   |
|                                   | Composite services, go to Step 4. |
|                                   | Note: pathology services are      |
|                                   | considered to be composite when   |
|                                   | the account documents:            |
|                                   |                                   |
|                                   | -   do not contain the notation   |
|                                   |     ‘Rule 3 Exemption’ or         |
|                                   |     ‘S4B(3)’ which allows the     |
|                                   |     items to be paid individually |
|                                   | -   contain the notation ‘Rule 3  |
|                                   |     Exemption’ but the items      |
|                                   |     claimed and circumstances     |
|                                   |     involved do not allow an      |
|                                   |     exemption to be granted       |
|                                   | -   contain the notation ‘S4B(3)’ |
|                                   |     but the delegate determines   |
|                                   |     that the items are not to be  |
|                                   |     paid individually             |
|                                   |                                   |
|                                   | Non-composite services, go to     |
|                                   | Step 5. Note: pathology services  |
|                                   | are considered to be              |
|                                   | non-composite when there is a     |
|                                   | notation on the account documents |
|                                   | which allows the items to be paid |
|                                   | individually (that is, ‘Rule 3    |
|                                   | Exemption’ or ‘S4B(3)’)           |
+-----------------------------------+-----------------------------------+
| 4                                 | Action composite services + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | -   Write down the composite item |
|                                   |     number displayed in the       |
|                                   |     message                       |
|                                   | -   Identify the items that are   |
|                                   |     to be rejected (highlighted   |
|                                   |     by the system)                |
|                                   | -   Reject these items with RSN   |
|                                   |     370 (benefit paid on item     |
|                                   |     number other than that        |
|                                   |     claimed)                      |
|                                   | -   Insert an item line before    |
|                                   |     the associated pathology      |
|                                   |     request (PA) line involved    |
|                                   | -   Key details of the composite  |
|                                   |     item on the blank item line   |
|                                   |     with the charge being the     |
|                                   |     total of all the amounts      |
|                                   |     claimed for the items being   |
|                                   |     rejected                      |
|                                   |                                   |
|                                   | Press [Enter].                    |
+-----------------------------------+-----------------------------------+
| 5                                 | Action non-composite services +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | For items with notation ‘Rule 3   |
|                                   | Exemption’ or ‘R3EX’ key:         |
|                                   |                                   |
|                                   | -   processing indicator (PI) 41  |
|                                   |     in the PI field               |
|                                   | -   RSN 366 (benefit paid for     |
|                                   |     Rule 3 Exemption item) (in    |
|                                   |     the RSN field)                |
|                                   |                                   |
|                                   | Press [Enter].                    |
|                                   |                                   |
|                                   | Forward ‘S4B(3)’ claims to a Team |
|                                   | leader.                           |
|                                   |                                   |
|                                   | After the S4B(3) delegate         |
|                                   | determination process, determine  |
|                                   | whether the items are payable. If |
|                                   | payable key:                      |
|                                   |                                   |
|                                   | -   processing indicator (PI) 41  |
|                                   |     in the PI field               |
|                                   | -   RSN 365 (benefit paid for     |
|                                   |     S4B(3))                       |
|                                   |                                   |
|                                   | Press [Enter]                     |
