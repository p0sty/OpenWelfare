|                                   | the last 12 months, an SU1 will   |
|                                   | be required for each employer.    |
|                                   | However, there are certain        |
|                                   | circumstances, where an SU1 may   |
|                                   | not be required. The following    |
|                                   | circumstances determine when an   |
|                                   | SU1 may not be required:          |
|                                   |                                   |
|                                   | Single Touch Payroll (STP)        |
|                                   |                                   |
|                                   | STP cessation data may pre-fill   |
|                                   | during statement reporting or     |
|                                   | within an online claim. If the    |
|                                   | cessation details are confirmed   |
|                                   | by the customer, an SU1 is not    |
|                                   | required for that employer,       |
|                                   | unless changes have been made to  |
|                                   | amounts paid or there is any      |
|                                   | doubt about the number of days    |
|                                   | for leave or termination          |
|                                   | payments.                         |
|                                   |                                   |
|                                   | Note: STP data does not prefill   |
|                                   | the number of days or period      |
|                                   | covered for these payments.       |
|                                   | Therefore it is important to      |
|                                   | review the days provided by the   |
|                                   | customer is reasonable for the    |
|                                   | amount paid, example: $3,000 for  |
|                                   | one day's leave is not reasonable |
|                                   | and would need to be verified.    |
|                                   |                                   |
|                                   | Casual or Part-Time Employees     |
|                                   |                                   |
|                                   | If a customer is still employed   |
|                                   | but has had the number of working |
|                                   | hours reduced, an SU1 is not      |
|                                   | required as their employment has  |
|                                   | not been terminated. If the       |
|                                   | customer was a casual or          |
|                                   | part-time employee but is no      |
|                                   | longer employed an SU1 is         |
|                                   | required.                         |
|                                   |                                   |
|                                   | Seasonal, Contract or             |
|                                   | Intermittent Workers              |
|                                   |                                   |
|                                   | If the customer advises as part   |
|                                   | of an online claim they are a     |
|                                   | Seasonal, Contract or             |
|                                   | Intermittent Worker, the          |
|                                   | information required will be      |
|                                   | collected as part of the claim.   |
|                                   |                                   |
|                                   | If the customer claims via other  |
|                                   | means (or it is only identified   |
|                                   | after the online claim has been   |
|                                   | submitted that the customer was a |
|                                   | Seasonal, Contract or             |
|                                   | Intermittent Worker), an SU496    |
|                                   | must be completed. For more       |
|                                   | information, refer to Seasonal    |
|                                   | Work Preclusion Period (SWPP).    |
|                                   |                                   |
|                                   | Self-Employment                   |
|                                   |                                   |

|                                   | self-employed, a Module F -       |

|                                   | required instead of an SU1.       |

Creating a Q267 letter

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Confirm Q267 required + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | The Q267 letter is a notice under |
|                                   | Social Security Law that provides |
|                                   | the employer with 14 days to      |
|                                   | supply the required information.  |
|                                   | This letter also includes         |
|                                   | information on the options        |
|                                   | available for the employer to     |
|                                   | provide the SU1.                  |
|                                   |                                   |

|                                   |                                   |

|                                   |     classified, and               |
|                                   | -   other evidence (for example,  |
|                                   |     verbal SU1s) are clearly      |
|                                   |     documented on the customer's  |
|                                   |     record - including details of |
|                                   |     how this evidence was sourced |
|                                   |                                   |
|                                   | Employment cessation details,     |
|                                   | unused leave and/or redundancy    |
|                                   | payments reported by employers    |

|                                   | pre-filled in online claims or    |
|                                   | Assisted Customer Claims (ACC).   |
|                                   | Staff can view this information   |
|                                   | within the Claim Summary in       |
|                                   | Process Direct. The source will   |
|                                   | be recorded in the claim as       |
|                                   | 'Employer Reported'.              |
|                                   |                                   |

|                                   | information contained in the SU1  |
|                                   | to determine the job seeker's     |
|                                   | eligibility and payability for    |
|                                   | the income support payment.       |
|                                   |                                   |

|                                   | the number of days or period      |
|                                   | covered for these payments.       |
|                                   | Therefore, it is important to     |
|                                   | review the days provided by the   |
|                                   | customer is reasonable for the    |
|                                   | amount paid, example: $3,000 for  |
|                                   | one day's leave is not reasonable |
|                                   | and would need to be verified.    |

|                                   | Waiting periods                   |

|                                   |     Period                        |
