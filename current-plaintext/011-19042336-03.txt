This page contains the processes Service Officers follow when an FA081
and MS004 form is lodged by a customer at a service centre.

On this page:

Processing Newborn Child Declaration (FA081) form lodged at a service
centre

Processing Medicare enrolment application (MS004) form at a service
centre

Processing Newborn Child Declaration (FA081) form lodged at a service centre

Table 1: This table describes how Service Officers process FA081 forms
for Medicare purposes when a customer attends a service centre and does
not accept the digital lodgement option.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Customer requests face-to-face    |
|                                   | interview or immediate issue of   |
|                                   | Medicare card + Read more ...     |
|                                   |                                   |
|                                   | Does the customer require a       |
|                                   | Medicare card for the newborn     |
|                                   | child due to immediate medical    |
|                                   | care?                             |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, go to Step 3              |
+-----------------------------------+-----------------------------------+
| 2                                 | Does the customer want to enrol   |
|                                   | the newborn in Medicare only? +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, if the customer also      |
|                                   |     wants to lodge an online      |
|                                   |     claim for family assistance,  |
|                                   |     for example, for Family Tax   |
|                                   |     Benefit, Parental Leave Pay   |
|                                   |     or Dad and Partner Pay,       |
|                                   |     advise the customer [] to     |
|                                   |     lodge via myGov (unless there |
|                                   |     is an immediate medical need  |
|                                   |     for Medicare card, go to Step |
|                                   |     7)                            |
+-----------------------------------+-----------------------------------+
| 3                                 | Sight completed FA081 and proof   |
|                                   | of birth + Read more ...          |
|                                   |                                   |
|                                   | Sight the form and check the      |
|                                   | following:                        |
|                                   |                                   |
|                                   | -   both the applicant and        |
|                                   |     partner (if applicable) have  |
|                                   |     signed the form               |
|                                   | -   proof of birth section has    |
|                                   |     been completed by the doctor  |
|                                   |     or midwife                    |
|                                   | -   official hospital stamp is    |
|                                   |     present                       |
|                                   | -   applicantâ€™s Medicare number   |
|                                   |     is included                   |
|                                   |                                   |
|                                   | Are all sections on the form      |
|                                   | complete?                         |
|                                   |                                   |
|                                   | -   Yes, go to Step 6             |
|                                   | -   No, go to Step 4              |
|                                   |                                   |
|                                   | Note: if the customer is not the  |
|                                   | applicant, they must be the       |
|                                   | partner or legal guardian. If the |
|                                   | form is lodged by anyone else,    |
|                                   | Services Australia will accept    |
|                                   | the form but not conduct any      |
|                                   | face-to-face action or interview  |
|                                   | at this time. The form will be    |
|                                   | scanned to the Customer Reference |
|                                   | Number (CRN) to be processed as   |

|                                   | proof of birth on FA081 + Read    |

|                                   |                                   |

|                                   |                                   |

|                                   | -   No, go to Step 5              |

| 5                                 | Form still incomplete + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | If the form is still incomplete   |
|                                   | because it has:                   |
|                                   |                                   |
|                                   | no signature from the applicant   |
|                                   | and/or partner, give the form     |

|                                   | via [] digital lodgement option.  |
|                                   | Procedure ends here               |
|                                   |                                   |
|                                   | no proof of birth section         |

|                                   |     re-lodge via [] digital       |
|                                   |     lodgement option. Procedure   |

|                                   | Safety Net registration + Read    |
|                                   | more ...                          |

|                                   |                                   |
|                                   | Does the customer have an         |
|                                   | immediate medical need for the    |
|                                   | Medicare card?                    |

|                                   | Yes, go to Step 7                 |

|                                   |                                   |

|                                   | to existing Medicare card in CDMS |
|                                   | table.                            |
|                                   |                                   |

Processing Medicare enrolment application (MS004) form at a service centre

Table 2: This table describes how First Contact Officers and Service
Officers manage customers attending a service centre with an MS004 form
to enrol their newborn.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | First Contact Officer books       |
|                                   | customer into wait room + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Book the customer into the wait   |
|                                   | room for a face-to-face           |
|                                   | interview.                        |
|                                   |                                   |
|                                   | The customer waits to be called   |
|                                   | by the next available Service     |
|                                   | Officer. The Service Officer will |
|                                   | speak with the customer to enrol  |
|                                   | the newborn.                      |
|                                   |                                   |
|                                   | The Resources page has a link to  |
|                                   | the MS004.                        |
+-----------------------------------+-----------------------------------+
| 2                                 | Service Officer sights completed  |
|                                   | MS004 and proof of birth + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Sight the form and check the      |
|                                   | following.                        |
|                                   |                                   |
|                                   | Determine if the customer is the  |
|                                   | applicant:                        |
|                                   |                                   |
|                                   | -   customer is not the           |
|                                   |     applicant, return the form    |
|                                   |     and documentation to the      |
|                                   |     customer and advise that the  |
|                                   |     applicant is to attend for a  |
|                                   |     face-to-face interview        |
|                                   | -   customer is the applicant but |
|                                   |     not the birth mother or legal |
|                                   |     guardian, further questioning |
|                                   |     is required to determine who  |
|                                   |     the person is and why they    |
|                                   |     wish to enrol this newborn on |
|                                   |     their card. See Newborn child |
|                                   |     enrolment applicant is not    |
|                                   |     the legal parent table        |
|                                   |                                   |
|                                   | If there:                         |
|                                   |                                   |
|                                   | -   is no signature from the      |
|                                   |     applicant, return to          |
|                                   |     applicant and obtain          |
|                                   | -   are missing or incomplete     |
|                                   |     details on the form from the  |
|                                   |     applicant, return to          |
|                                   |     applicant for completion      |
|                                   |                                   |
|                                   | Accept the completed MS004 and    |
|                                   | proof of birth. See Processing    |
|                                   | newborn child enrolments in       |
|                                   | Medicare.                         |
+-----------------------------------+-----------------------------------+
| 3                                 | Complete Medicare enrolments and  |
|                                   | Safety Net registration + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | While the applicant is present    |
|                                   | complete the following enrolments |
|                                   | and registrations (if             |
|                                   | applicable), see Process newborn  |
|                                   | enrolment in CDMS table. See also |
|                                   | Register children (including      |
|                                   | children in care) for Medicare    |
|                                   | Safety Net.                       |
+-----------------------------------+-----------------------------------+
