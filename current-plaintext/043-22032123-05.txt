AWS quarterly cap examples

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Example 1 - BAC/CAC Quarterly cap |
|                                   |                                   |
|                                   | -   BAC claim period 1 January    |
|                                   |     2022 – 31 March 2022          |
|                                   | -   Employer lodges a claim for   |
|                                   |     BAC and declares a gross      |
|                                   |     amount of $15,000 was paid to |
|                                   |     the apprentice for that       |
|                                   |     quarter                       |
|                                   | -   While 50% of $15,000 is       |
|                                   |     $7,500, the employer is only  |
|                                   |     eligible to be paid $7,000    |
|                                   |     for that quarter as this is   |
|                                   |     the quarterly cap for BAC     |
+-----------------------------------+-----------------------------------+
| 2                                 | Example 2 - PWS Quarterly cap     |
|                                   |                                   |
|                                   | -   PWS claim period 24 October   |
|                                   |     2022 – 23 January 2023        |
|                                   | -   The apprenticeship commenced  |
|                                   |     on 24 October 2022            |
|                                   | -   Employer lodges a claim for   |
|                                   |     PWS and declares a gross      |
|                                   |     amount of $20,000 was paid to |
|                                   |     the apprentice for that       |
|                                   |     quarter                       |
|                                   | -   While 10% of $20,000 is       |
|                                   |     $2,000, the employer is only  |
|                                   |     eligible to be paid $1,500    |
|                                   |     for that quarter as this is   |
|                                   |     the quarterly cap for the     |
|                                   |     PWS, during the first 12      |
|                                   |     months of an apprenticeship   |
