This page contains the steps to help Disability Processing staff,
Authorised Review Officers and International Services staff check and
action a Disability Medical Assessment (DMA) report submitted by a
Government-contracted Doctor (GCD), and finalise the DSP claim. DMA
acceptance in Process Direct is only available for new claim processing.

For DSP claim processing, select the Process Direct subtab. For DSP
appeals , select Customer First.

| 3                                 | Compare Job Capacity Assessment   |
|                                   | (JCA) and Disability Medical      |
|                                   | Assessment (DMA) reports + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Compare the DMA report to the JCA |
|                                   | report. Ensure all medical        |
|                                   | conditions assessed in the JCA    |
|                                   | report are also considered and    |
|                                   | recorded in the DMA report.       |
|                                   | Conditions assessed as customer   |
|                                   | or recipient reported or other in |
|                                   | the JCA report do not have to be  |
|                                   | considered by the GCD, but may be |
|                                   | assessed if new medical evidence  |
|                                   | is provided or obtained during    |
|                                   | the course of the DMA.            |
|                                   |                                   |
|                                   | In some cases, the GCD may        |
|                                   |                                   |
|                                   | -   disagree with the JCA about   |
|                                   |     the presence of a medical     |
|                                   |     condition.                    |
|                                   | -   recommend the customer is     |
|                                   |     manifestly medically          |
|                                   |     eligible. In this case, the   |
|                                   |     GCD must record the manifest  |
|                                   |     indictor in the DMA report.   |
|                                   |     They also need to assess the  |
|                                   |     condition as verified by      |
|                                   |     medical evidence and          |
|                                   |     permanent. It is not          |
|                                   |     mandatory to mark the         |
|                                   |     condition as fully diagnosed, |
|                                   |     treated and stabilised (FDTS) |
|                                   |     or assign impairment rating/s |
|                                   |                                   |
|                                   | The GCD must outline this in the  |
|                                   | DMA report section titled Does    |
|                                   | your report differ from the JCA?  |
|                                   | In these cases, it is considered  |
|                                   | that the medical condition has    |
|                                   | been assessed.                    |
|                                   |                                   |
|                                   | The GCD:                          |
|                                   |                                   |
|                                   | -   must not provide an           |
|                                   |     alternative diagnosis to that |
|                                   |     documented in the medical     |
|                                   |     evidence, or a provisional    |
|                                   |     diagnosis of a medical        |
|                                   |     condition, and                |
|                                   | -   may determine the customer    |
|                                   |     has additional medical        |
|                                   |     conditions or the medical     |
|                                   |     condition/s selected by the   |
|                                   |     JCA are incorrect. Check this |
|                                   |     is clearly explained in the   |
|                                   |     DMA report                    |
|                                   |                                   |
|                                   | Medical conditions assessed       |
|                                   | separately in the JCA report must |
|                                   | be assessed separately in the DMA |
|                                   | report. If a GCD has chosen to    |
|                                   | combine or separate out medical   |
|                                   | conditions as documented in the   |
|                                   | JCA report, check this is clearly |
|                                   | explained in the DMA report.      |
|                                   |                                   |
|                                   | Contributing GCD assessments      |
|                                   |                                   |
|                                   | For some DMA referrals, multiple  |
|                                   | GCDs may be involved in assessing |
|                                   | the customer's medical            |
|                                   | conditions. The customer will     |
|                                   | have their DMA assessment         |
|                                   | conducted by the Primary GCD. A   |
|                                   | Contributing GCD completes the    |
|                                   | assessment of other conditions as |
|                                   | needed.                           |
|                                   |                                   |
|                                   | The Primary GCD is assigned based |
|                                   | on the medical condition with the |
|                                   | highest functional                |
|                                   | impact/combined functional impact |
|                                   | as indicated in the DMA referral. |
|                                   | If the nature of the condition    |
|                                   | is:                               |
|                                   |                                   |
|                                   | -   physical, a Medical           |
|                                   |     Practitioner is assigned      |
|                                   | -   psychological, a Clinical     |
|                                   |     Psychologist is assigned      |
|                                   | -   split between physical and    |
|                                   |     psychological, a Medical      |
|                                   |     Practitioner is assigned      |
|                                   |                                   |
|                                   | If the Primary GCD can assess the |
|                                   | secondary conditions they can do  |
|                                   | so, but a Contributing GCD is     |
|                                   | compulsory if the customer has 2  |
|                                   | primary conditions that attract   |
|                                   | 20 points each on different       |
|                                   | Impairment Tables. For example,   |
|                                   | 20 points on a physical table and |
|                                   | 20 points on a psychological      |
|                                   | table.                            |
|                                   |                                   |
|                                   | The Primary GCD may consider      |
|                                   | obtaining a Contributing GCD      |
|                                   | assessment if:                    |
|                                   |                                   |
|                                   | -   they see the secondary        |
|                                   |     conditions as out of scope of |
|                                   |     practice. For example, if a   |
|                                   |     DMA has primary physical      |
|                                   |     conditions and significant    |
|                                   |     psychological secondary       |
|                                   |     conditions such as            |
|                                   |     intellectual disability or    |
|                                   |     mental health conditions      |
|                                   | -   recommendations on FDTS or    |
|                                   |     impairment rating assessments |
|                                   |     for the primary conditions    |
|                                   |     are significantly different   |
|                                   |     to the JCA or SA471           |
|                                   |     (Disability Medical           |
|                                   |     Assessment during appeal)     |
|                                   |                                   |
|                                   | Where multiple GCDs have been     |
|                                   | involved in the DMA assessment    |
|                                   | (assessing different medical      |
|                                   | conditions), the PDF version of   |
|                                   | the DMA report contains           |
|                                   | information from each GCD. The    |
|                                   | Primary GCD's complete assessment |
|                                   | is followed by the Contributing   |
|                                   | GCD's complete assessment.        |
|                                   |                                   |
|                                   | The DMA report must be returned   |
|                                   | if a GCD indicates:               |
|                                   |                                   |
|                                   | -   they cannot assess some       |
|                                   |     medical conditions, even if   |
|                                   |     the primary medical condition |
|                                   |     attracts 20 points or more.   |
|                                   |     Return the report for the     |
|                                   |     additional conditions to be   |
|                                   |     assessed, before it is        |
|                                   |     accepted to process a claim   |
|                                   |     or undertake a review of      |
|                                   |     decision                      |
|                                   | -   that 'assessment of secondary |
|                                   |     medical conditions are not    |
|                                   |     relevant for the purposes of  |
|                                   |     this assessment' (or similar  |
|                                   |     wording), and there are       |
|                                   |     secondary medical conditions  |
|                                   |     which have not been assessed  |
|                                   | -   the customer meets manifest   |
|                                   |     criteria, but the relevant    |
|                                   |     indicator is not displayed in |
|                                   |     the DMA report                |
|                                   |                                   |
|                                   | Does the DMA report consider all  |
|                                   | medical conditions supported by   |
|                                   | medical evidence and assessed in  |
|                                   | the JCA report?                   |
|                                   |                                   |
|                                   | -   Yes, go to Step 4             |
|                                   | -   No, return the DMA report, go |
|                                   |     to Step 9                     |
|                                   |                                   |
|                                   | If unsure, go to Step 8 to hold   |
|                                   | DMA report and refer to SSO.      |
+-----------------------------------+-----------------------------------+
| 4                                 | Independent view of customer's    |
|                                   | conditions + Read more ...        |
|                                   |                                   |
|                                   | The GCD must show they have       |
|                                   | formed an independent view on all |
|                                   | medical conditions the customer   |
|                                   | has at the time of the            |
|                                   | assessment. This must include all |
|                                   | conditions assessed in the JCA    |
|                                   | report which are supported by     |
|                                   | medical evidence. The GCD must    |
|                                   | not rely solely on the JCA report |
|                                   | to form their opinion or advise   |
|                                   | they are not qualified to         |
|                                   | complete an assessment of         |
|                                   | particular medical conditions. If |
|                                   | text in the DMA report            |
|                                   | substantially relies on or copies |
|                                   | the text in the JCA report, the   |
|                                   | DMA report must be returned.      |
|                                   |                                   |
|                                   | To show an independent view of    |
|                                   | all medical conditions, the       |
|                                   | report should contain enough      |
|                                   | medical information and           |
|                                   | justification of recommendations  |
|                                   | for the Delegate to be confident  |
|                                   | the recommendations are           |
|                                   | evidence-based and comply with    |
|                                   | legislation. The Delegate should  |
|                                   | not have to refer to medical      |
|                                   | evidence (including the JCA or    |
|                                   | SA463) to understand DMA          |
|                                   | recommendations.                  |
|                                   |                                   |
|                                   | Has the GCD formed an independent |
|                                   | view of the customer's medical    |
|                                   | conditions?                       |
|                                   |                                   |
|                                   | -   Yes, go to Step 5             |
|                                   | -   No, return the DMA report, go |
|                                   |     to Step 9                     |
|                                   |                                   |
|                                   | If unsure, go to Step 8 to hold   |
|                                   | DMA report and refer to SSO.      |
+-----------------------------------+-----------------------------------+
| 5                                 | Check DMA report for double       |
|                                   | counting + Read more ...          |
|                                   |                                   |
|                                   | If a customer has multiple        |
|                                   | conditions rated on the same      |
|                                   | Impairment Table, the combined    |
|                                   | functional impact must be         |
|                                   | assigned against one of the       |
|                                   | medical conditions. The second or |
|                                   | subsequent time that Impairment   |
|                                   | Table is used, an impairment      |
|                                   | rating of zero must be applied.   |
|                                   |                                   |
|                                   | If a customer has 2 or more       |
|                                   | medical conditions with a common  |
|                                   | impairment, only one relevant     |
|                                   | table should be applied and a     |
|                                   | single impairment rating assigned |
|                                   | to reflect the combined           |
|                                   | impairment. It would be           |
|                                   | inappropriate to assign a         |
|                                   | separate impairment rating for    |
|                                   | each medical condition as this    |
|                                   | would result in the same          |
|                                   | impairment being assessed more    |
|                                   | than once (double counting).      |
|                                   |                                   |
|                                   | Does the same Impairment Table    |
|                                   | appear more than once in the DMA  |
|                                   | report and have ratings greater   |
|                                   | than zero been applied more than  |
|                                   | once against the medical          |
|                                   | conditions or has the same        |
|                                   | impairment being assessed more    |
|                                   | than once?                        |
|                                   |                                   |
|                                   | -   Yes, return the DMA report,   |
|                                   |     go to Step 9                  |
|                                   | -   No, go to Step 6              |
|                                   |                                   |
|                                   | If unsure, go to Step 8 to hold   |
|                                   | DMA report and refer to SSO.      |
+-----------------------------------+-----------------------------------+
| 6                                 | Treating Health Professional      |
|                                   | (THP) contacts + Read more ...    |
|                                   |                                   |
|                                   | Sometimes GCDs need to contact a  |
|                                   | customer's THP to confirm or      |
|                                   | clarify medical evidence. In      |
|                                   | limited cases, a DMA can be       |
|                                   | completed with the customer's     |
|                                   | THP, for example when the person  |
|                                   | is in prison, psychiatric         |
|                                   | confinement or deceased.          |
|                                   |                                   |
|                                   | If THP contact is needed, a GCD   |
|                                   | must document the following.      |
|                                   |                                   |
|                                   | For successful contacts:          |
|                                   |                                   |
|                                   | -   the date they contacted the   |
|                                   |     THP                           |
|                                   | -   the name of the person they   |
|                                   |     spoke with, and               |
|                                   | -   all relevant information      |
|                                   |     provided by the THP           |
|                                   |                                   |
|                                   | For unsuccessful contacts, in the |
|                                   | additional details section of the |
|                                   | DMA report:                       |
|                                   |                                   |
|                                   | -   the dates (2 different        |
|                                   |     business days) on which they  |
|                                   |     tried to contact              |
|                                   | -   which THP they tried to       |
|                                   |     contact, and                  |
|                                   | -   what information they needed  |
|                                   |     from the THP                  |
|                                   |                                   |
|                                   | If this information is not        |
|                                   | included, return the report for   |
|                                   | the information to be added.      |
|                                   |                                   |
|                                   | Refer all submitted DMA reports   |
|                                   | conducted without contact with    |
|                                   | the customer or the THP to the    |
|                                   | GCD Liaison Officer for advice    |
|                                   | before acceptance. In the         |
|                                   | referral, explain why the DMA     |
|                                   | report should or should not be    |
|                                   | accepted. The GCD Liaison Officer |
|                                   | will provide advice on how to     |
|                                   | proceed.                          |
|                                   |                                   |
|                                   | Note: in certain circumstances,   |
|                                   | DMA appointments are booked to be |
|                                   | completed with a THP as a         |
|                                   | directive from Services           |
|                                   | Australia. To identify these      |
|                                   | cases, check the referral note in |
|                                   | the DMA referral. If the GCD has  |
|                                   | been unable to contact the THP,   |
