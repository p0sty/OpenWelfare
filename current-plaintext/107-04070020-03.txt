| 2                                 | Debt actioned before the end of   |

|                                   |                                   |

|                                   | refund of PAYG withheld from      |
|                                   | either the employer or the ATO.   |

|                                   | The employer must refund the PAYG |
|                                   | amount to the employee before the |
|                                   | end of the financial year in      |

|                                   | the:                              |
|                                   |                                   |

|                                   |     the withholding amount        |

|                                   | amount to the employee even if    |
|                                   | they have paid this to the ATO.   |
|                                   | If this is the case, the employer |
|                                   | can:                              |
|                                   |                                   |
|                                   | -   recover the refunded amount   |
|                                   |     from the ATO as a debt, or    |
|                                   | -   offset or reduce another      |
|                                   |     payment to the ATO by the     |
|                                   |     refunded withholding amount   |
|                                   |                                   |
|                                   | The employer will need to include |
|                                   | adjustments in their payroll      |
|                                   | records.                          |
|                                   |                                   |
|                                   | On their income tax return, the   |
|                                   | employee will include:            |
