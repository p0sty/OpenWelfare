This page contains a table summarising which nominees can be issued a
BasicsCard and make allocations and sign forms and statements on behalf
of the customer. There is also a link to the form for authorising a
person or organisation to enquire or act on a customer's behalf.

Access to BasicsCard by nominees

Table 1: This table outlines which nominee can be issued a BasicsCard
and make allocations and sign forms and statements on behalf of the
customer.

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Issue of BasicsCard               |
|                                   |                                   |
|                                   | The BasicsCard can be issued to:  |
|                                   |                                   |
|                                   | -   Payment nominee               |
+-----------------------------------+-----------------------------------+
| 2                                 | Regular expenses allocations and  |
|                                   | allocations to the BasicsCard     |
|                                   |                                   |
|                                   | The following nominee types can   |
|                                   | make regular expense allocations  |
|                                   | as well as allocations to the     |
|                                   | BasicsCard:                       |
|                                   |                                   |
|                                   | -   Payment nominee               |
|                                   | -    Correspondence nominee -     |
|                                   |     they can act as if they were  |
|                                   |     the customer (principal) if   |
|                                   |     there is no payment nominee.  |
|                                   |     The best interests of the     |
|                                   |     customer and their dependants |
|                                   |     must be considered.           |
+-----------------------------------+-----------------------------------+
| 3                                 | Sign relevant forms and           |
|                                   | statements                        |
|                                   |                                   |
|                                   | The following nominee types are   |
|                                   | able to sign relevant forms and   |
|                                   | statements on behalf of the       |
|                                   | customer:                         |
|                                   |                                   |
|                                   | -   Correspondence nominee        |
+-----------------------------------+-----------------------------------+
