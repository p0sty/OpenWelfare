| 2                     | For (name)            | Individual the        |

| 3                     | Date of birth         | Individual date of    |
|                       |                       | birth, recorded by    |
|                       |                       | Medicare or the AIR   |
|                       |                       | for individuals not   |

+-----------------------+-----------------------+-----------------------+
| 4                     | NIP immunisation      | NIP Immunisation      |
|                       | status                | status:               |
|                       |                       |                       |
|                       | (only for individuals | -   up to date - if   |
|                       | under 20)             |     an individual is  |
|                       |                       |     not overdue for   |
|                       |                       |     any immunisations |
|                       |                       | -   up to date/not up |
|                       |                       |     to date -         |
|                       |                       |     Catch-up schedule |
|                       |                       |     active until      |
|                       |                       |     dd/mm/yyyy        |
|                       |                       | -   not up to date -  |
|                       |                       |     if an individual  |
|                       |                       |     is overdue for at |
|                       |                       |     least one         |
|                       |                       |     immunisation      |
+-----------------------+-----------------------+-----------------------+
| 5                     | Schedule              | Age at which the      |
|                       |                       | vaccines are listed   |
|                       | (only for individuals | on the NIP Schedule   |

|                       |                       | 6 months).            |
+-----------------------+-----------------------+-----------------------+
| 6                     | Immunisation          | Disease antigens      |
|                       |                       | immunised against     |
|                       |                       | (for example,         |
|                       |                       | diphtheria).          |
+-----------------------+-----------------------+-----------------------+
| 7                     | Date given            | Date the vaccine/s    |
|                       |                       | was given.            |
+-----------------------+-----------------------+-----------------------+
| 8                     | Brand name given      | Vaccination brand     |

|                       |                       | example,              |
|                       |                       | Infanrix-IPV). All    |
|                       |                       | recorded vaccination  |

|                       |                       | next. If there are    |
|                       |                       | multiple overdue      |
|                       |                       | dates, only single    |
|                       |                       | antigens will be      |
|                       |                       | listed (e.g.          |
|                       |                       | diphtheria, polio).   |
+-----------------------+-----------------------+-----------------------+
| 10                    | Date due              | Date when the next    |
|                       |                       | immunisations are or  |
|                       | (only for individuals | were due. This date   |
|                       | under 20)             | may be in future      |
|                       |                       | (due) or past         |
|                       |                       | (overdue).            |
+-----------------------+-----------------------+-----------------------+
| 11                    | Notice/s              | Notice/s appear under |
|                       |                       | specific              |
|                       |                       | circumstances.        |

FAQ for Service Officers

Table 4

+-----------------------------------+-----------------------------------+
| Item                              | Question and response             |
+-----------------------------------+-----------------------------------+
| 1                                 | Why doesn’t the immunisation      |
|                                   | history statement show all my or  |
|                                   | my child’s immunisations?         |
|                                   |                                   |
|                                   | If any immunisations are missing  |
|                                   | from your immunisation history    |
|                                   | statement, speak to your          |
|                                   | vaccination provider and ask them |
|                                   | to record them on the AIR. All    |
|                                   | immunisations recorded on the AIR |
|                                   | show on your immunisation history |
|                                   | statement.                        |
|                                   |                                   |
|                                   | The AIR does not record vaccines  |
|                                   | given before 1 January 1996.      |
|                                   |                                   |
|                                   | The AIR records National          |
|                                   | Immunisation Program (NIP)        |
|                                   | Schedule vaccines, private        |
|                                   | vaccines (such as for the         |
|                                   | influenza or travel), vaccines    |
|                                   | given through school programs and |
|                                   | vaccines given overseas.          |
+-----------------------------------+-----------------------------------+
| 2                                 | Why are my or my child's personal |
|                                   | details incorrect on the proof of |
|                                   | vaccination?                      |
|                                   |                                   |
|                                   | Personal details on the AIR come  |
|                                   | from your Medicare enrolment. You |
|                                   | can change the details with       |
|                                   | Medicare and the AIR will update  |
|                                   | overnight.                        |
|                                   |                                   |
|                                   | The next day, you can access      |
|                                   | proof of your vaccinations using  |
|                                   | your Express Plus Medicare mobile |
|                                   | app or through myGov using your   |
|                                   | Medicare online account, or My    |
|                                   | Health Record.                    |
|                                   |                                   |
|                                   | If you’re not enrolled in         |
|                                   | Medicare, your personal details   |
|                                   | come from:                        |
|                                   |                                   |
|                                   | -   your IHI record,              |
|                                   | -   or your vaccination provider  |
|                                   |     when they notified the AIR of |
|                                   |     your vaccinations             |
|                                   |                                   |
|                                   | You can update your personal      |
|                                   | details with the AIR and your     |
|                                   | immunisation record will be       |
|                                   | updated. You can then access      |
|                                   | proof of your vaccinations using  |
|                                   | the IHI service or My Health      |
|                                   | Record in myGov.                  |
+-----------------------------------+-----------------------------------+
| 3                                 | My or my child’s proof of         |
|                                   | vaccination has wrong information |
|                                   | on it. How do I get this fixed?   |
|                                   |                                   |
|                                   | If your or your child’s           |
|                                   | vaccination history is showing    |
|                                   | incorrect information, speak with |
|                                   | your vaccination provider and ask |
|                                   | them to update the AIR.           |
|                                   |                                   |
|                                   | You can take evidence of your     |
|                                   | immunisations to a vaccination    |
|                                   | provider and ask them to record   |
|                                   | them on the AIR.                  |
|                                   |                                   |
|                                   | For help to correct your          |
|                                   | vaccination history, contact the  |
|                                   | AIR public enquiry line.          |
+-----------------------------------+-----------------------------------+
| 4                                 | My child is 14 or older, can I    |
|                                   | get proof of their vaccinations?  |
|                                   |                                   |
|                                   | If your child is 14 or older, you |
|                                   | will not be able to access their  |
|                                   | vaccination history.              |
|                                   |                                   |
|                                   | Individuals 14 or older can get   |
|                                   | proof of their own vaccinations   |
|                                   | online. For contact details, see  |
|                                   | AIR enquiries.                    |
|                                   |                                   |
|                                   | An individual 14 years or older   |
|                                   | can also give consent for their   |
|                                   | parent or guardian to access      |
|                                   | their vaccination information.    |
|                                   | They can do this over the phone   |
|                                   | or in person at a service centre. |
|                                   | This consent is only valid for    |
|                                   | that instance.                    |
|                                   |                                   |
|                                   | If a:                             |
|                                   |                                   |
|                                   | -   Power of Attorney,            |
|                                   | -   guardianship order,           |
|                                   | -   administrative order, or      |
|                                   | -   other legal arrangement is in |
|                                   |     place, and                    |
|                                   | -   is registered with Medicare,  |
|                                   |     the parent or guardian has    |
|                                   |     legal authority to act on     |
|                                   |     behalf of the person          |
|                                   |                                   |
|                                   | Parents, guardians or carers of   |
|                                   | individuals 14 or older who       |
|                                   | cannot manage their own affairs   |
|                                   | can submit an:                    |
|                                   |                                   |
|                                   | -   Authorisation to act on an    |
|                                   |     incapacitated person’s behalf |
|                                   |     for Medicare purposes form    |
|                                   |     (MO050) if no legal           |
|                                   |     arrangement is in place       |
+-----------------------------------+-----------------------------------+
| 5                                 | Why can’t I get my 14 or older    |
|                                   | child’s vaccination information?  |
|                                   |                                   |
|                                   | The AIR is administered in        |
|                                   | accordance with the Australian    |
|                                   | Immunisation Register Act 2015,   |
|                                   | the Privacy Act 1988 and the      |
|                                   | Services Australia privacy        |
|                                   | policy.                           |
|                                   |                                   |
|                                   | Parents and guardians are able to |
|                                   | access their child’s vaccination  |
|                                   | information until the child turns |
|                                   | 14. When an individual is 14 or   |
|                                   | older, Services Australia         |
|                                   | requires their consent to provide |
|                                   | their vaccination information.    |
|                                   | This includes requests for proof  |
|                                   | of vaccination.                   |
