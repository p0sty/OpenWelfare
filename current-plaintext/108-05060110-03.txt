Actioning a customer request to commute an ATE income stream

This table describes the process for actioning a customer request to
commute an asset test exempt (ATE) income stream due to extreme
financial hardship.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Extreme financial hardship        |
|                                   | rules + Read more ...             |
|                                   |                                   |
|                                   | The Social Security Act requires  |
|                                   | a customer who wishes to commute  |
|                                   | all or part of an asset-test      |
|                                   | exempt (ATE) income stream on the |
|                                   | grounds of extreme financial      |
|                                   | hardship to make an application   |
|                                   | in writing to Centrelink for      |
|                                   | approval. Centrelink will assess  |
|                                   | whether the customer meets the    |
|                                   | extreme financial hardship rules. |
|                                   |                                   |
|                                   | The applications are processed by |
|                                   | the Financial Industry and        |
|                                   | Network Support (FINS) Helpdesk.  |
|                                   |                                   |
|                                   | -   If the customer is asking for |
|                                   |     a letter from Centrelink to   |
|                                   |     commute their ATE income      |
|                                   |     stream due to extreme         |
|                                   |     financial hardship, go to     |
|                                   |     Step 2.                       |
|                                   | -   If the customer is enquiring  |
|                                   |     about their application       |
|                                   |     already lodged to commute all |
|                                   |     or part of an ATE income      |
|                                   |     stream under extreme          |
|                                   |     financial hardship, go to     |
|                                   |     Step 3.                       |
|                                   | -   Is the customer advising they |
|                                   |     have commuted their ATE       |
|                                   |     income stream and had prior   |
|                                   |     approval, see Commutation of  |
|                                   |     asset-test exempt (ATE)       |
|                                   |     income streams Procedure ends |
|                                   |     here.                         |
|                                   | -   If the customer is advising   |
|                                   |     they have commuted their ATE  |
|                                   |     income stream due to extreme  |
|                                   |     financial hardship and did    |
|                                   |     not obtain Centrelink         |
|                                   |     approval first, see           |
|                                   |     Commutation of asset-test     |
|                                   |     exempt (ATE) income streams.  |
|                                   |     Procedure ends here.          |
|                                   | -   If a response from FINS       |
|                                   |     Helpdesk has been received as |
|                                   |     a result of the application   |
|                                   |     being sent to them            |
|                                   |     previously, go to Step 4.     |
+-----------------------------------+-----------------------------------+
| 2                                 | Letter from Centrelink + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | The customer is to:               |
|                                   |                                   |
|                                   | -   complete the form             |
|                                   |     'Application to be allowed to |
|                                   |     commute all or part of an     |
|                                   |     asset-test exempt income      |
|                                   |     stream because of extreme     |
|                                   |     financial hardship', and      |
|                                   | -   supply any supporting         |
|                                   |     evidence to verify their      |
|                                   |     financial hardship and        |
|                                   |     unavoidable expenditure       |
|                                   |                                   |
|                                   | The Service Officer is to:        |
|                                   |                                   |
|                                   | -   Complete the form 'Referral   |
|                                   |     and the recommendation        |
|                                   |     application to be allowed to  |
|                                   |     commute all or part of an     |
|                                   |     asset-test exempt income      |
|                                   |     stream because of extreme     |
|                                   |     financial hardship'           |
|                                   | -   Upload the form and any       |
|                                   |     supporting documents to the   |
|                                   |     customer's record             |
|                                   | -   Submit a Level 2 Policy       |
|                                   |     Helpdesk enquiry to notify    |
|                                   |     FINS Helpdesk of intended     |
|                                   |     commutation request           |
|                                   | -   Record details on a DOC.      |
|                                   |                                   |
|                                   | See the Resources page for a copy |
|                                   | of the forms.                     |
|                                   |                                   |
|                                   | The FINS Helpdesk will determine  |
|                                   | if the customer's proposed        |
|                                   | commutation meets extreme         |
|                                   | financial hardship rules and they |
|                                   | have unavoidable expenditure.     |
|                                   |                                   |
|                                   | Procedure ends here until the     |
|                                   | recommendation is returned from   |
|                                   | the FINS Helpdesk. When the       |
|                                   | recommendation is received, go to |
|                                   | Step 4.                           |
+-----------------------------------+-----------------------------------+
| 3                                 | Enquiring about application       |
|                                   | already lodged + Read more ...    |
|                                   |                                   |
|                                   | When the customer is enquiring    |
|                                   | about the progress of their       |
|                                   | application:                      |
|                                   |                                   |
|                                   | Check the customer's record for a |
|                                   | DOC regarding the application:    |
|                                   |                                   |
|                                   | If the FINS Helpdesk has made a   |
|                                   | decision, go to Step 4            |
|                                   |                                   |
|                                   | If there is no DOC on the         |
|                                   | customer's record with            |
|                                   | information on the application,   |
|                                   | follow up with the FINS Helpdesk  |
|                                   | as to the progress of the         |
|                                   | application and advise the        |
|                                   | customer                          |
|                                   |                                   |
|                                   | -   Record details on a DOC       |
|                                   | -   Procedure ends here           |
+-----------------------------------+-----------------------------------+
| 4                                 | Response to application received  |
|                                   | from FINS Helpdesk + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | [                                 |
|                                   | \\INTERNAL.DEPT.LOCAL\Shared\NAT\ |
|                                   | SERDELEXCEL\WORKPRODIMP\Operation |
|                                   | Blueprint Migration\RDT Release   |
|                                   | Icons\32w\icon-face-to-face.png]  |
|                                   |                                   |
|                                   | Advise the customer of the        |
|                                   | outcome of their application to   |
|                                   | commute all or part of their ATE  |
|                                   | income stream due to extreme      |
|                                   | financial hardship by either:     |
|                                   |                                   |
|                                   | -   completing a Q999 letter in   |
|                                   |     the customer's record using   |
|                                   |     the instructions included in  |
|                                   |     the response from the FINS    |
|                                   |     Helpdesk, or                  |
|                                   | -   contacting the customer via   |
|                                   |     phone to advise of the        |
|                                   |     outcome as well as sending a  |
|                                   |     Q999                          |
|                                   |                                   |
|                                   | Record details on a DOC           |
|                                   |                                   |
|                                   | For actioning the commutation,    |
|                                   | see Commutation of asset-test     |
|                                   | exempt (ATE) income streams       |
+-----------------------------------+-----------------------------------+
