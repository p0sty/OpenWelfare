This page contains information on the required process when an enquiry
is received from an Income Managed customer regarding the appointment or
cancellation of a payment nominee arrangement.

On this page:

Payment nominee appointment/cancellation enquiry

Principal and/or payment nominee contact

Payment nominee arrangements for specific IM measures

Commencing, updating or ceasing Nominee Income Management

Issuing BasicsCard and related correspondence

Payment nominee appointment/cancellation enquiry

Portions of this process must only be completed by the Income Management
Payment and Contact Team (IMPACT) or Income Management Policy Helpdesk.

|                                   | purposes + Read more ...          |
|                                   |                                   |
|                                   | If a payment nominee's payments   |

|                                   | Management measure on their own.  |

|                                   |                                   |

|                                   | action is required.               |
