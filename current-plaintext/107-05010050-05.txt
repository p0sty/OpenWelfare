Contract/Labour Hire staff role in requesting Online Search Facility (OLS) access

Table 1

+-----------------------------------+-----------------------------------+
| Item                              | Contract/Labour Hire staff        |
|                                   | responsibilities                  |
+-----------------------------------+-----------------------------------+
| 1                                 | Ensure access is for the          |
|                                   | permitted purposes in accordance  |
|                                   | with Schedule 1, Item 18 of the   |
|                                   | Electoral and Referendum          |
|                                   | Regulation 2016 (Cth) – excerpt   |
|                                   | below:                            |
|                                   |                                   |
|                                   | -   preventing and detecting      |
|                                   |     fraud relating to identity or |
|                                   |     incorrect payments, or        |
|                                   | -   identifying or locating       |
|                                   |     clients for debt management,  |
|                                   |     determination or correction   |
|                                   |     purposes, or                  |
|                                   | -   undertaking compliance,       |
|                                   |     investigation or enforcement  |
|                                   |     activities relating to the    |
|                                   |     Chief Executive Medicare's    |
|                                   |     functions under section 5 of  |
|                                   |     the Human Services (Medicare) |
|                                   |     Act 1973, or                  |
|                                   | -   recovering overpayments or    |
|                                   |     other amounts due to the      |
|                                   |     Commonwealth in relation to   |
|                                   |     the performance of the Chief  |
|                                   |     Executive Medicare's          |
|                                   |     functions under section 5 of  |
|                                   |     the Human Services (Medicare) |
|                                   |     Act 1973                      |
+-----------------------------------+-----------------------------------+
| 2                                 | Request a copy of the Deed Poll   |
|                                   | from your Line Manager. Complete, |
|                                   | sign and save the Deed Poll in    |
|                                   | .pdf format 'Deed Poll –          |
|                                   | NAME_Logon.pdf' and forward for   |
|                                   | Line Manager signature.           |
+-----------------------------------+-----------------------------------+
| 3                                 | Await further instructions before |
|                                   | submitting an OLS 3 (AEC)         |
|                                   | resource request in ISP.          |
+-----------------------------------+-----------------------------------+
| 4                                 | A system generated email will be  |
|                                   | received notifying the outcome of |
|                                   | the ISP resource request, that    |
|                                   | is, approved or denied.           |
+-----------------------------------+-----------------------------------+
| 5                                 | End date the resource in ISP      |
|                                   | immediately when ceasing to       |
|                                   | perform duties that require       |
|                                   | access to Elector Information for |
|                                   | the permitted purposes.           |
+-----------------------------------+-----------------------------------+

Line Manager role in provisioning Online Search Facility (OLS) access

Table 2

+-----------------------------------+-----------------------------------+
| Item                              | Line Manager responsibilities     |
+-----------------------------------+-----------------------------------+
| 1                                 | Ensure staff require the access   |
|                                   | for the permitted purposes in     |
|                                   | accordance with Schedule 1, Item  |
|                                   | 18 of the Electoral and           |
|                                   | Referendum Regulation 2016 (Cth)  |
|                                   | – excerpt below.                  |
|                                   |                                   |
|                                   | -   preventing and detecting      |
|                                   |     fraud relating to identity or |
|                                   |     incorrect payments, or        |
|                                   | -   identifying or locating       |
|                                   |     clients for debt management,  |
|                                   |     determination or correction   |
|                                   |     purposes, or                  |
|                                   | -   undertaking compliance,       |
|                                   |     investigation or enforcement  |
|                                   |     activities relating to the    |
|                                   |     Chief Executive Medicare's    |
|                                   |     functions under section 5 of  |
|                                   |     the Human Services (Medicare) |
|                                   |     Act 1973, or                  |
|                                   | -   recovering overpayments or    |
|                                   |     other amounts due to the      |
|                                   |     Commonwealth in relation to   |
|                                   |     the performance of the Chief  |
|                                   |     Executive Medicare's          |
|                                   |     functions under section 5 of  |
|                                   |     the Human Services (Medicare) |
|                                   |     Act 1973                      |
+-----------------------------------+-----------------------------------+
| 2                                 | Provide staff with a Deed Poll –  |
|                                   | request form from the Strategic   |
|                                   | Support and Coordination          |
|                                   | Positional Mailbox.               |
+-----------------------------------+-----------------------------------+
| 3                                 | Save completed deed poll(s) in    |
|                                   | .pdf format 'Deed Poll –          |
|                                   | NAME_Logon.pdf' and email to      |
|                                   | Strategic Support and             |
|                                   | Coordination Positional Mailbox.  |
+-----------------------------------+-----------------------------------+
| 4                                 | Await further instructions before |
|                                   | supporting the staff member to    |
|                                   | submit an OLS 3 (AEC) resource    |
|                                   | request in ISP.                   |
+-----------------------------------+-----------------------------------+
| 5                                 | Requests submitted in ISP for OLS |
|                                   | (3) AEC will be rejected where    |
|                                   | there is no record of AEC's       |
|                                   | receipt of the deed poll.         |
+-----------------------------------+-----------------------------------+
| 6                                 | End date the resource in ISP      |
|                                   | immediately when a staff member   |
|                                   | ceases to perform duties          |
|                                   | requiring access to Elector       |
|                                   | Information for the permitted     |
|                                   | purposes.                         |
