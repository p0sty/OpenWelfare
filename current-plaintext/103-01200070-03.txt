This page contains the procedure to be used for Service Officers, who
are conducting a Disbursement interview for customers exiting Supporting
People at Risk (SPaR) Income Management. (This procedure does not cover
the action required after the death of a customer.)

On this page:

|                                   | When conducting a Disbursement    |

|                                   | is likely to have their payments  |
|                                   | income managed within the next 60 |
|                                   | days. Service Officers should     |
|                                   | consider:                         |
|                                   |                                   |
|                                   | Whether the customer is likely to |
|                                   | be eligible for another measure   |
|                                   | of Income Management when the     |
|                                   | current measure ends (by checking |
|                                   | the Income Management summary     |
|                                   | screen)                           |
|                                   |                                   |
|                                   | Where Income Management is ending |
|                                   | due to their eligible payment     |
|                                   | cancelling, consider the reason   |

|                                   |     reason is unknown, it is      |
|                                   |     likely the customer will be   |
|                                   |     income managed within 60 days |
|                                   | -   CAN/CLR- check for details of |
|                                   |     why the customer requested    |
|                                   |     cancellation of their         |
|                                   |     payment. If the reason is     |

|                                   |     customer will be Income       |
|                                   |     Management again within 60    |

|                                   | -   CAN/DNL or CAN/FRP - it is    |
|                                   |     likely Income Management will |
|                                   |     be reinstated within 60 days  |
|                                   |     once any outstanding report   |
|                                   |     is lodged. However, if there  |
|                                   |     is information on the         |
|                                   |     customer's record that        |
|                                   |     indicates they no longer wish |
|                                   |     to receive income support     |
|                                   |     payments, it is not likely    |
|                                   |     they will be income managed   |
|                                   |     within 60 days                |

|                                   | being revoked or the customer has |
|                                   | an excluded payment nominee for   |
|                                   | Income Management purposes.       |
|                                   |                                   |

|                                   |                                   |

|                                   | received:                         |

|                                   | -   In the Supporting People at   |

|                                   | IM Decision/Exemption screen:     |
