+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Record of interview + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | Is it appropriate to conduct a    |
|                                   | Record of Interview (ROI)?        |
|                                   |                                   |
|                                   | -   How serious is the offence    |
|                                   |     and are there reasonable      |
|                                   |     prospects of a prosecution?   |
|                                   | -   Is it in the public interest  |
|                                   |     to seek a criminal            |
|                                   |     prosecution, will the         |
|                                   |     customer be fit to stand      |
|                                   |     trial?                        |
|                                   | -   Will alerting the customer to |
|                                   |     the investigation jeopardise  |
|                                   |     the collection of further     |
|                                   |     evidence?                     |
|                                   | -   Are there any customer        |
|                                   |     vulnerabilities or are there  |
|                                   |     risks to the safety of the    |
|                                   |     customer, investigators, or   |
|                                   |     other person?                 |
|                                   |                                   |
|                                   | It is not necessary to have a     |
|                                   | full case against the customer,   |
|                                   | but there must be reasonable      |
|                                   | grounds for suspecting them of    |
|                                   | criminal conduct.                 |
|                                   |                                   |
|                                   | It is not appropriate if a        |
|                                   | decision has been made to         |
|                                   | terminate the investigation and   |
|                                   | seek only an administrative       |
|                                   | action.                           |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, conduct an enquiry        |
|                                   |     interview if only seeking     |
|                                   |     administrative action.        |
|                                   |     Procedure ends here           |
+-----------------------------------+-----------------------------------+
| 2                                 | Contacting the customer + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Before contacting the customer,   |
|                                   | consider:                         |
|                                   |                                   |
|                                   | -   Social worker? If the         |
|                                   |     customer is at risk,          |
|                                   |     vulnerable, in crisis or has  |
|                                   |     a mental illness, consult     |
|                                   |     with a Payments and Integrity |
|                                   |     social worker. They will      |
|                                   |     provide advice on the best    |
|                                   |     ways to deal with the         |
|                                   |     customer. If they recommend   |
|                                   |     not conducting a ROI, discuss |
|                                   |     appropriate action with the   |
|                                   |     Senior Investigator/Approver  |
|                                   |     of Investigations             |
|                                   | -   AFP? If the customer poses a  |
|                                   |     safety risk, for example,     |
|                                   |     known violent offender,       |
|                                   |     consult the Australian        |
|                                   |     Federal Police (AFP) out      |
|                                   |     posted officer. They will     |
|                                   |     provide advice on the best    |
|                                   |     way to conduct the interview, |
|                                   |     which may include having an   |
|                                   |     AFP officer present at the    |
|                                   |     interview. They may recommend |
|                                   |     referring the case to the AFP |
|                                   | -   Nominee? If the customer has  |
|                                   |     a nominee it may be           |
|                                   |     appropriate to ask the        |
|                                   |     nominee to attend the         |
|                                   |     interview and/or to send them |
|                                   |     a copy of the interview       |
|                                   |     invitation. See Obligations   |
|                                   |     of nominees                   |
|                                   | -   Interpreter? For criminal     |
|                                   |     prosecutions they must be     |
|                                   |     appropriately qualified       |
|                                   | -   Where? ROIs are generally     |
|                                   |     conducted at the customer's   |
|                                   |     closest Service Centre or     |
|                                   |     other agency offices within   |
|                                   |     reasonable access for the     |
|                                   |     customer. Conducting it       |
|                                   |     anywhere else must be         |
|                                   |     approved by the Senior        |
|                                   |     Investigator/Approver of      |
|                                   |     Investigations, considering   |
|                                   |     possible safety issues and    |
|                                   |     risks. A home visit may be    |
|                                   |     appropriate only if there are |
|                                   |     no other options              |
+-----------------------------------+-----------------------------------+
| 3                                 | Send invitation + Read more ...   |
|                                   |                                   |
|                                   | Send Offer of ROI letter to the   |
|                                   | customer. The letter advises the  |
|                                   | interview is voluntary and        |
|                                   | includes information about the    |
|                                   | interview and the customer's      |
|                                   | rights.                           |
|                                   |                                   |
|                                   | Record details on investigation   |
|                                   | record.                           |
|                                   |                                   |
|                                   | If the customer does not respond, |
|                                   | update the investigation record.  |
|                                   | Continue investigation.           |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 4                                 | Customer responds + Read more ... |
|                                   |                                   |
|                                   | If the customer contacts to       |
|                                   | discuss the ROI, advise:          |
|                                   |                                   |
|                                   | -   reason for the interview,     |
|                                   |     nature of the alleged offence |
|                                   | -   attendance is voluntary and   |
|                                   |     the interview can be stopped  |
|                                   |     at any time                   |
|                                   | -   they may seek legal advice    |
|                                   |     before deciding to attend     |
|                                   | -   they may have a lawyer or     |
|                                   |     friend attend with them       |
|                                   | -   the interview will be         |
|                                   |     recorded and may be used as   |
|                                   |     evidence (they will receive a |
|                                   |     copy of the recording at the  |
|                                   |     end of the interview)         |
|                                   | -   if they are under 18 they     |
|                                   |     should be accompanied by a    |
|                                   |     parent or guardian            |
|                                   | -   if they are Indigenous        |
|                                   |     (check), advise legal         |
|                                   |     assistance available and that |
|                                   |     they can have an interview    |
|                                   |     friend present                |
|                                   |                                   |
|                                   | If the customer asks about the    |
|                                   | likely outcome of prosecution,    |
|                                   | they must be advised to seek      |
|                                   | legal advice. At no time give any |
|                                   | advice about likely penalties     |
|                                   | that the Court may or may not     |
|                                   | impose.                           |
|                                   |                                   |
|                                   | Take notes of the conversation    |
|                                   | and record details on the         |
|                                   | investigation record.             |
+-----------------------------------+-----------------------------------+
| 5                                 | Participation in a ROI + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Does the customer want to         |
|                                   | participate in a ROI?             |
|                                   |                                   |
|                                   | Yes,                              |
|                                   |                                   |
|                                   | -   Advise the date, time and     |
|                                   |     location of the interview in  |
|                                   |     writing. The ROI confirmation |
|                                   |     letter can be issued. It      |
|                                   |     reminds the customer of their |
|                                   |     rights and that the interview |
|                                   |     is voluntary                  |
|                                   | -   Book interpreter if required  |
|                                   | -   Record a DOC on the           |
|                                   |     customer's record to ensure   |
|                                   |     that the appointment details  |
|                                   |     are easy to access if the     |
|                                   |     customer attends a service    |
|                                   |     centre. Include the           |
|                                   |     appointment date, time and    |
|                                   |     location, and the             |
|                                   |     investigator's contact        |
|                                   |     details                       |
|                                   | -   Update investigation record   |
|                                   |                                   |
|                                   | No, record details on             |
|                                   | investigation record. Continue    |
|                                   | investigation. Procedure ends     |
|                                   | here                              |
+-----------------------------------+-----------------------------------+
| 6                                 | Complete interview plan + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Purpose: To ensure the ROI is     |
|                                   | conducted according to the        |
|                                   | requirements of Part 1C of the    |
|                                   | Crimes Act 1914. See the          |
|                                   | References page for links to      |
|                                   | relevant Legislation.             |
|                                   |                                   |
|                                   | Prepare a record of interview     |
|                                   | plan, including:                  |
|                                   |                                   |
|                                   | -   Details of specific           |
|                                   |     information to advise the     |
|                                   |     customer - the caution, their |
|                                   |     rights, the interview process |
|                                   | -   The allegation, in terms the  |
|                                   |     customer can understand. It   |
|                                   |     should be clear and concise   |
|                                   |     and state the exact reason    |
|                                   |     for the interview. It should  |
|                                   |     give context to the caution   |
|                                   |     and enable the customer to    |
|                                   |     make an informed decision on  |
|                                   |     whether to participate in the |
|                                   |     interview                     |
|                                   | -   The objectives of the         |
|                                   |     interview - the information   |
|                                   |     being sought from the         |
|                                   |     customer through questioning. |
|                                   |     Prepare questions for each    |
|                                   |     objective, but relying on a   |
|                                   |     prescribed list or script can |
|                                   |     lead to not listening to the  |
|                                   |     customer's answers and        |
|                                   |     missing appropriate follow-up |
|                                   |     questions                     |
|                                   | -   Evidence and documentation to |
|                                   |     be presented, for the         |
|                                   |     customer to be asked to       |
|                                   |     comment on. Ensure it is      |
|                                   |     available, and organised in a |
|                                   |     logical way so it can be      |
|                                   |     readily obtained and          |
|                                   |     presented during the          |
|                                   |     interview                     |

Conducting a Record of interview (ROI)

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Conducting a ROI + Read more ...  |
|                                   |                                   |
|                                   | -   Two interviewers are          |
|                                   |     required:                     |
|                                   |                                   |
|                                   | Two people must conduct the ROI,  |
|                                   | a lead interviewer and an         |
|                                   | interviewer whose role is to      |
|                                   | corroborate.                      |
|                                   |                                   |
|                                   | They should have obtained at      |
|                                   | least Certificate IV in           |
|                                   | Government (Investigation) or be  |
|                                   | supervised by an experienced      |
|                                   | investigator with that            |
|                                   | qualification.                    |
|                                   |                                   |
|                                   | One should be the same sex as the |
|                                   | person being interviewed. If this |
|                                   | is not possible, document an      |
|                                   | assessment in consultation with   |
|                                   | the Senior Investigator/Approver  |
|                                   | of Investigations.                |
|                                   |                                   |
|                                   | Role of lead interviewer          |
|                                   |                                   |
|                                   | -   prepare interview plan        |
|                                   | -   outline the interview         |
|                                   |     process, including the        |
|                                   |     intention to electronically   |
|                                   |     record the interview          |
|                                   | -   explain the reasons for the   |
|                                   |     interview                     |
|                                   | -   issue the caution and advise  |
|                                   |     rights                        |
|                                   | -   ask most of the questions     |
|                                   | -   ensure exhibits are available |
|                                   | -   ensure the corroborator is    |
|                                   |     briefed on the investigation  |
|                                   |     and the interview plan        |
|                                   |                                   |
|                                   | Role of corroborator              |
|                                   |                                   |
|                                   | To assist the lead interviewer.   |
|                                   | They are a potential witness in   |
|                                   | court proceedings. They may ask   |
|                                   | questions when relevant, but      |
|                                   | normally at the end of the        |
|                                   | interview or if prompted by the   |
|                                   | lead interviewer. They may also:  |
|                                   |                                   |
|                                   | -   take notes during the         |
|                                   |     interview (using these as a   |
|                                   |     basis for questions if        |
|                                   |     relevant)                     |
|                                   | -   record the start and finish   |
|                                   |     times and times of all        |
|                                   |     suspensions and resumptions   |
|                                   | -   record Proof of               |
|                                   |     Identification supplied by    |
|                                   |     the customer                  |
|                                   | -   note the exhibit number of    |
|                                   |     all exhibits shown to the     |
|                                   |     customer during the interview |
|                                   | -   set up and test recording     |
|                                   |     equipment before the customer |
|                                   |     arrives                       |
|                                   | -   describe and record any       |
|                                   |     evidence or documents         |
|                                   |     supplied by the customer      |
|                                   |     during the interview          |
|                                   | -   monitor the interview         |
|                                   |     questions, considering the    |
|                                   |     proofs of the offence and     |
|                                   |     ensure any missed questions   |
|                                   |     or rights are brought to the  |
|                                   |     attention of the lead         |
|                                   |     interviewer                   |
|                                   |                                   |
|                                   | Purpose built verifiable          |
|                                   | equipment such as TPR recording   |
|                                   | devices must be used. Most        |
|                                   | equipment used by the agency      |
|                                   | records audio and video           |
|                                   | simultaneously. Triple deck tape  |
|                                   | recorders may also be used.       |
|                                   |                                   |
|                                   | The media being used for the      |
|                                   | recording should be removed from  |
|                                   | its packaging in the presence of  |
|                                   | the customer.                     |
|                                   |                                   |
|                                   | If using an interpreter, explain  |
|                                   | interview purpose and process     |
|                                   | before the customer arrives,      |
|                                   | introduce them to the customer to |
|                                   | check for communication problems  |
|                                   | and their independence, ensure    |
|                                   | they translate in first person.   |
|                                   |                                   |
|                                   | The interpreter must be           |
|                                   | independent of the matter under   |
|                                   | investigation (do not use agency  |
|                                   | staff or a person connected to    |
|                                   | the customer, for example, family |
|                                   | or friend).                       |
|                                   |                                   |
|                                   | If the need for an interpreter is |
|                                   | not identified until the          |
|                                   | interview, defer questioning      |
|                                   | until interpreter is present.     |
|                                   |                                   |
|                                   | Speak with the interpreter before |
|                                   | the customer arrives:             |
|                                   |                                   |
|                                   | -   briefly outline the case      |
|                                   | -   advise the purpose of the     |
|                                   |     interview                     |
|                                   | -   advise the process to be used |
|                                   |     during the interview          |
|                                   | -   check they have experience in |
|                                   |     interpreting in the           |
|                                   |     circumstances proposed        |
|                                   | -   advise that they will be      |
|                                   |     required to supply a          |
|                                   |     statement at the completion   |
|                                   |     of the interview and          |
|                                   |     potentially be called as a    |
|                                   |     witness in subsequent court   |
|                                   |     proceedings                   |
|                                   |                                   |
|                                   | Introduce the interpreter to the  |
|                                   | customer and allow them to talk   |
|                                   | privately in the customer's       |
|                                   | preferred language. This is to    |
|                                   | establish:                        |
|                                   |                                   |
|                                   | -   If there are any              |
|                                   |     communication problems        |
|                                   | -   That the interpreter is       |
|                                   |     competent to interpret in the |
|                                   |     language                      |
|                                   | -   If they have any prior or     |
|                                   |     current relationship. If so,  |
|                                   |     check the nature of the       |
|                                   |     relationship. If they are not |
|                                   |     considered to be independent, |
|                                   |     arrange a different           |
|                                   |     interpreter                   |
|                                   | -   If there is any social,       |
|                                   |     cultural or political         |
|                                   |     affiliation, or animosity     |
|                                   |     between them. If so, check    |
|                                   |     the nature of the             |
|                                   |     relationship. If they are not |
|                                   |     considered to be independent, |
|                                   |     arrange a different           |
|                                   |     interpreter                   |
|                                   |                                   |
|                                   | During the interview:             |
|                                   |                                   |
|                                   | -   position the interpreter      |
|                                   |     between the interviewer and   |
|                                   |     the customer                  |
|                                   | -   ensure translation is in the  |
|                                   |     first person, for example, 'I |
|                                   |     didn't fill in the form', not |
|                                   |     'he says he didn't fill in    |
|                                   |     the form'. Interpreters must  |
|                                   |     not answer on behalf of the   |
|                                   |     customer, but merely          |
|                                   |     interpret what the customer   |
|                                   |     is saying                     |
|                                   |                                   |
|                                   | Check the customer's health       |
|                                   | before and during the interview   |
|                                   |                                   |
|                                   | An existing medical condition     |
|                                   | should have been considered when  |
|                                   | deciding if the interview was     |
|                                   | appropriate. If advice was        |
|                                   | provided that it was appropriate, |
|                                   | continue to gauge the customer's  |
|                                   | state of health.                  |
|                                   |                                   |
|                                   | When the customer arrives, check  |
|                                   | that they do not appear to be:    |
|                                   |                                   |
|                                   | -   suffering from an illness or  |
|                                   |     fatigued, or                  |
|                                   | -   under the influence of        |
|                                   |     alcohol or drugs              |
|                                   |     (prescription or illicit)     |
|                                   |                                   |
|                                   | An interview conducted in the     |
|                                   | above circumstances may be ruled  |
|                                   | as inadmissible. If the customer  |
|                                   | insists on participating in the   |
|                                   | interview, advise them that due   |
|                                   | to agency policy the interview    |
|                                   | will not proceed at this time and |
|                                   | an alternative date will be       |
|                                   | arranged.                         |
|                                   |                                   |
|                                   | If interviewing:                  |
|                                   |                                   |
|                                   | -   a customer with an            |
|                                   |     intellectual disability or    |
|                                   |     mental illness, ensure they   |
|                                   |     are capable of being          |
|                                   |     interviewed                   |
|                                   |                                   |
|                                   | People with mental illness may    |
|                                   | not exhibit any signs and the     |
|                                   | agency's records may not always   |
|                                   | indicate that the person is ill.  |
|                                   | Consider:                         |
|                                   |                                   |
|                                   | -   Is the customer capable of    |
|                                   |     understanding the caution?    |
|                                   | -   Are they aware of their       |
|                                   |     surroundings?                 |
|                                   | -   Are they behaving in a        |
|                                   |     rational manner?              |
|                                   | -   Are they capable of           |
|                                   |     understanding and responding  |
|                                   |     to the questions?             |
|                                   |                                   |
|                                   | If it is determined that the      |
|                                   | customer:                         |
|                                   |                                   |
|                                   | is not capable of being           |
|                                   | interviewed, consider this when   |
|                                   | deciding if they had intended to  |
|                                   | commit an offence                 |
|                                   |                                   |
|                                   | is capable of being interviewed:  |
|                                   |                                   |
|                                   | -   Ensure they understand the    |
|                                   |     caution and their rights. Ask |
|                                   |     them to repeat the caution in |
|                                   |     their own words               |
|                                   | -   Communicate in a way they can |
|                                   |     understand and check that     |
|                                   |     they do understand            |
|                                   |                                   |
|                                   | Offer to have their interview     |
|                                   | friend, parent or guardian        |
|                                   | present at the interview. If they |
|                                   | cannot name an interview friend,  |
|                                   | seek advice/help of social        |
|                                   | worker.                           |
|                                   |                                   |
|                                   | The Court may exclude an          |
|                                   | admission by the defendant in a   |
|                                   | record of interview if the Court  |
|                                   | believes that:                    |
|                                   |                                   |
|                                   | -   the admission is not          |
|                                   |     reliable, or                  |
|                                   | -   it would be unfair to allow   |
|                                   |     the admission into evidence   |
|                                   |                                   |
|                                   | The fact that the customer has an |
|                                   | intellectual disability or a      |
|                                   | mental illness could be a factor  |
|                                   | the Court considers in            |
|                                   | determining the above.            |
|                                   |                                   |
|                                   | -   An Indigenous customer,       |
|                                   |     consider the Anunga rules     |
|                                   |                                   |
|                                   | These special guidelines apply to |
|                                   | Aboriginal persons and Torres     |
|                                   | Strait Islanders, particularly    |
|                                   | those living the tradition        |
|                                   | lifestyle. The law recognises     |
|                                   | that they may be at a             |
|                                   | disadvantage in deciding whether  |
|                                   | or not to exercise their right to |
|                                   | remain silent.                    |
|                                   |                                   |
|                                   | -   An interpreter who speaks the |
|                                   |     customer's dialect should be  |
|                                   |     present unless the customer   |
|                                   |     is fluent in English          |
|                                   | -   Where practical, an interview |
|                                   |     friend should be present.     |
|                                   |     They should be someone        |
|                                   |     independent of the agency who |
|                                   |     will not feel intimidated by  |
|                                   |     the interview. They should    |
|                                   |     not be agency employees such  |
|                                   |     as an Indigenous Specialist   |
|                                   |     Officer, a co-offender (such  |
|                                   |     as a partner), or a witness   |
|                                   |     to the proceedings. If the    |
|                                   |     customer waives this right it |
|                                   |     should be clear it is being   |
|                                   |     done voluntarily              |
|                                   | -   The caution should be given   |
|                                   |     with special care to ensure   |
|                                   |     proper understanding. Phrase  |
|                                   |     it in simple terms and ask    |
|                                   |     the customer to repeat it in  |
|                                   |     their own words, phrase by    |
|                                   |     phrase. Do not proceed unless |
|                                   |     satisfied that the customer   |
|                                   |     understands the right to      |
|                                   |     remain silent                 |
|                                   | -   Avoid questions that suggest  |
|                                   |     the answer (leading           |
|                                   |     questions)                    |
|                                   | -   Make every effort to put the  |
|                                   |     customer at ease, as they are |
|                                   |     often nervous or              |
|                                   |     uncomfortable in the presence |
|                                   |     of authority figures          |
|                                   |                                   |
|                                   | Legal assistance: If sought, all  |
|                                   | reasonable steps should be taken  |
|                                   | to enable the customer to obtain  |
|                                   | legal assistance. This may        |
|                                   | include helping them contact an   |
|                                   | Aboriginal legal aid              |
|                                   | organisation.                     |
|                                   |                                   |
|                                   | -   A customer aged under 18, a   |
|                                   |     parent, guardian or friend    |
|                                   |     must be present and their     |
|                                   |     name, address and             |
|                                   |     relationship to the customer  |
|                                   |     included in the interview     |
|                                   |                                   |
|                                   | Most jurisdictions impose special |
|                                   | obligations on interviewers if    |
|                                   | the person is under 18. In        |
|                                   | particular, the presence of a     |
|                                   | parent, guardian or friend is     |
|                                   | obligatory.                       |
|                                   |                                   |
|                                   | Advice may need to be sought from |
|                                   | the CDPP.                         |
|                                   |                                   |
|                                   | Courts are particularly vigilant  |
|                                   | when assessing whether a          |
|                                   | confession or admission made by a |
|                                   | child was voluntary. They         |
|                                   | recognise that a child may have a |
|                                   | reduced capacity to understand    |
|                                   | their rights or to look after     |
|                                   | their own interests.              |
|                                   |                                   |
|                                   | The role of a parent/guardian or  |
|                                   | other adult is to support and     |
|                                   | help the child in deciding        |
|                                   | whether or not to answer          |
|                                   | questions.                        |
|                                   |                                   |
|                                   | Explain to the customer, clearly  |
|                                   | and precisely, why they are being |
|                                   | interviewed in the presence of    |
|                                   | the friend, parent or guardian.   |
|                                   |                                   |
|                                   | If the customer:                  |
|                                   |                                   |
|                                   | -   is unable or unwilling to     |
|                                   |     provide the name of a person  |
|                                   |     to attend the interview, seek |
|                                   |     advice/help of social worker  |
|                                   | -   insists on being interviewed  |
|                                   |     without a third person        |
|                                   |     present, explain why the      |
|                                   |     presence of an adult is       |
|                                   |     required                      |
|                                   |                                   |
|                                   | If a third party is present,      |
|                                   | obtain customer permission,       |
|                                   | explain purpose of interview and  |
|                                   | their role.                       |
|                                   |                                   |
|                                   | Third parties are normally        |
|                                   | present at the interview to       |
|                                   | support the customer.             |
|                                   |                                   |
|                                   | Co-offenders should not be        |
|                                   | present.                          |
|                                   |                                   |
|                                   | -   Explain the purpose of the    |
|                                   |     interview and obtain the      |
|                                   |     customer's permission for the |
|                                   |     third person to be present    |
|                                   | -   Advise the interview friend   |
|                                   |     they are there to act in an   |
|                                   |     advisory role and to help the |
|                                   |     customer understand the       |
|                                   |     matters discussed. They       |
|                                   |     should not answer a question  |
|                                   |     on behalf of the customer     |
|                                   |                                   |
|                                   | (The only admissible evidence     |
|                                   | from an interview is what the     |
|                                   | customer says, not what the       |
|                                   | interview says, even if the       |
|                                   | customer agrees to what is said.) |
|                                   |                                   |
|                                   | Advise the customer and interview |
|                                   | friend that the friend is not     |
|                                   | permitted to interrupt or         |
|                                   | interfere with the interview      |
|                                   |                                   |
|                                   | -   If they interrupt during the  |
|                                   |     interview, politely tell them |
|                                   |     that the customer is the      |
|                                   |     person being interviewed, and |
|                                   |     that they will be given an    |
|                                   |     opportunity later in the      |
|                                   |     interview to make a statement |
|                                   | -   If they continue to disrupt   |
|                                   |     the interview, politely       |
|                                   |     request that they stop        |
|                                   |     interrupting or leave         |
|                                   |                                   |
|                                   | (An interview with constant       |
|                                   | interruptions from a third party  |
|                                   | may be of little evidentiary      |
|                                   | value. If they continually        |
|                                   | interrupt, the interviewer may    |
|                                   | decide not to continue with the   |
|                                   | interview unless they leave. The  |
|                                   | customer will need to decide      |
|                                   | whether they continue without the |
|                                   | third party or end the            |
|                                   | interview.)                       |
|                                   |                                   |
|                                   | If a solicitor is present, allow  |
|                                   | them to speak with the customer   |
|                                   | before the interview, explain     |
|                                   | interview process.                |
|                                   |                                   |
|                                   | It is the right of a customer to  |
|                                   | seek legal advice before and      |
|                                   | during the interview. The         |
|                                   | customer also has the right to    |
|                                   | have legal representation present |
|                                   | during questioning.               |
|                                   |                                   |
|                                   | A solicitor representing the      |
|                                   | customer should be given the      |
|                                   | opportunity to talk with their    |
|                                   | client in private before the      |
|                                   | interview.                        |
|                                   |                                   |
|                                   | When the solicitor wishes to be   |
|                                   | present, explain:                 |
|                                   |                                   |
|                                   | -   how the interview will be     |
|                                   |     conducted and the reason for  |
|                                   |     the interview                 |
|                                   | -   the interview is between the  |
|                                   |     customer and the interviewer  |
|                                   |     and the solicitor cannot      |
|                                   |     answer questions on behalf of |
|                                   |     the client                    |
|                                   | -   the solicitor may become a    |
|                                   |     witness to the things said or |
|                                   |     done during the interview     |
|                                   | -   any decision about referring  |
|                                   |     the customer to the CDPP will |
|                                   |     be made at a time after the   |
|                                   |     interview                     |
|                                   |                                   |
|                                   | Seating: If possible, position    |
|                                   | the solicitor where they are      |
|                                   | unable to provide advice to their |
|                                   | client through body language. Any |
|                                   | advice they wish to give must     |
|                                   | then be given verbally and this   |
|                                   | will be recorded.                 |
+-----------------------------------+-----------------------------------+
| 2                                 | The interview + Read more ...     |
|                                   |                                   |
|                                   | Treat the customer with respect   |
|                                   | and courtesy. Answer any          |
|                                   | questions honestly and clearly.   |
|                                   | Build rapport with the customer   |
|                                   | so they do not feel intimidated   |
|                                   | by the process.                   |
|                                   |                                   |
|                                   | Preliminary discussion            |
|                                   |                                   |
|                                   | -   When the customer arrives,    |
|                                   |     interviewers are to introduce |
|                                   |     themselves and show           |
|                                   |     identification                |
|                                   | -   Remind customer that          |
|                                   |     participation is voluntary,   |
|                                   |     they can have a break or stop |
|                                   |     the interview at any time     |
|                                   | -   Show the customer the         |
|                                   |     recording equipment, explain  |
|                                   |     how it works, and that they   |
|                                   |     will get a copy of the audio  |
|                                   |     recording at the end of the   |
|                                   |     interview. If they do not     |
|                                   |     wish the interview to be      |
|                                   |     recorded electronically, a    |
|                                   |     written record may be offered |
|                                   |                                   |
|                                   | A written record of interview     |
|                                   | should be recorded in the         |
|                                   | interviewer's official notebook.  |
|                                   | If this is not available, a note  |
|                                   | pad can be used. The notes are to |
|                                   | be kept in the investigation file |
|                                   | for inclusion in the brief of     |
|                                   | evidence.                         |
|                                   |                                   |
|                                   | Note the names of those present   |
|                                   | and the time the interview        |
|                                   | started and ended.                |
|                                   |                                   |
|                                   | At the end of the interview ask   |
|                                   | the customer:                     |
|                                   |                                   |
|                                   | -   To read over the interview    |
|                                   |     and point out any errors      |
|                                   | -   If they are prepared to sign  |
|                                   |     the record of interview. If   |
|                                   |     they are, ask them to sign or |
|                                   |     initial the record on each    |
|                                   |     page and to initial any       |
|                                   |     alterations that they have    |
|                                   |     made (in the presence of the  |
|                                   |     interviewer)                  |
|                                   |                                   |
|                                   | If the customer does not sign the |
|                                   | record of interview, the          |
|                                   | prosecution will need to argue    |
|                                   | its probative value, which may or |
|                                   | may not be admitted into          |
|                                   | evidence.                         |
|                                   |                                   |
|                                   | Provide the customer a copy of    |
|                                   | the notes within seven days of    |
|                                   | the interview.                    |
|                                   |                                   |
|                                   | -   Ask the customer if they are  |
|                                   |     happy to begin. If so,        |
|                                   |     commence the interview        |
|                                   |                                   |
|                                   | Start recording                   |
|                                   |                                   |
|                                   | Formally identify all persons     |
|                                   | present, and the relationship of  |
|                                   | any third parties to the          |
|                                   | customer.                         |
|                                   |                                   |
|                                   | State the reason for the          |
|                                   | interview and the allegation of   |
|                                   | criminal conduct under            |
|                                   | investigation.                    |
|                                   |                                   |
|                                   | The caution                       |
|                                   |                                   |
|                                   | An appropriate form of caution    |
|                                   | is:                               |
|                                   |                                   |
|                                   | Investigator: "I must advise you  |
|                                   | that you do not have to do or say |
|                                   | anything unless you wish to do    |
|                                   | so, but anything you do say or do |
|                                   | will be recorded and may be used  |
|                                   | in evidence. Do you understand    |
|                                   | the caution?"                     |
|                                   |                                   |
|                                   | Ask the customer to repeat the    |
|                                   | caution in their words.           |
|                                   |                                   |
|                                   | It is vital that the customer     |
|                                   | understands the caution. If there |
|                                   | is any doubt, ask further         |
|                                   | questions like:                   |
|                                   |                                   |
|                                   | Investigator: "Do you have to     |
|                                   | talk to me today? Can you stop    |
|                                   | the interview at any time and     |
|                                   | leave?"                           |
|                                   |                                   |
|                                   | The caution must be given before  |
|                                   | any admission. Re-caution the     |
|                                   | customer after any break in the   |
|                                   | interview.                        |
|                                   |                                   |
|                                   | The purpose of cautioning a       |
|                                   | person in relation to an alleged  |
|                                   | offence is to advise:             |
|                                   |                                   |
|                                   | -   their right to remain silent, |
|                                   |     and                           |
|                                   | -   the use to which their        |
|                                   |     answers may be put            |
|                                   |                                   |
|                                   | The caution assists the           |
|                                   | prosecution in proving that any   |
|                                   | answers given after the caution   |
|                                   | were made voluntarily.            |
|                                   |                                   |
|                                   | Re-caution the customer after any |
|                                   | break in the interview, after     |
|                                   | changing CDs or after a rest      |
|                                   | break.                            |
|                                   |                                   |
|                                   | If an admission is made before a  |
|                                   | caution is given, advise the      |
|                                   | customer to cease, give the       |
|                                   | caution, then ask the customer to |
|                                   | restate their admission under     |
|                                   | caution.                          |
|                                   |                                   |
|                                   | The right to remain silent is not |
|                                   | affected by the customer          |
|                                   | consenting to be interviewed.     |
|                                   | Therefore, the customer may       |
|                                   | refuse to answer questions during |
|                                   | the interview and may at any time |
|                                   | request that the interview end.   |
|                                   | This right must be respected and  |
|                                   | if requested by the customer the  |
|                                   | interview should be immediately   |
|                                   | terminated.                       |
|                                   |                                   |
|                                   | Any admissions must be voluntary. |
|                                   | There must be no inducements.     |
|                                   |                                   |
|                                   | An admission is a statement that  |
|                                   | goes towards establishing an      |
|                                   | element of the offence. It is not |
|                                   | the purpose of a ROI to obtain an |
|                                   | admission.                        |
|                                   |                                   |
|                                   | If an admission is made before a  |
|                                   | caution is given, advise the      |
|                                   | customer to cease, give the       |
|                                   | caution, and then ask them to     |
|                                   | restate their admission under     |
|                                   | caution.                          |
|                                   |                                   |
|                                   | Inducements: Admission will not   |
|                                   | be accepted by the Court if there |
|                                   | is any kind of inducement made by |
|                                   | the interviewers, usually threats |
|                                   | or promises, for example:         |
|                                   |                                   |
|                                   | Threats against the person who    |
|                                   | makes the admission or another    |
|                                   | person                            |
|                                   |                                   |
|                                   | Statements that suggest the       |
|                                   | outcome of a confession might be  |
|                                   | to bring some beneficial result   |
|                                   | in connection with the            |
|                                   | prosecution. These statements may |
|                                   | not always be explicit. For       |
|                                   | example it would be inappropriate |
|                                   | to say:                           |
|                                   |                                   |
|                                   | -   'You had better tell the      |
|                                   |     truth', or                    |
|                                   | -   'If you don't tell me you may |
|                                   |     get yourself into trouble and |
|                                   |     it will be worse for you'     |
|                                   |                                   |
|                                   | Admissions made because of        |
|                                   | physical coercion and oppression  |
|                                   |                                   |
|                                   | Always phrase questions in a      |
|                                   | non-threatening way.              |

| 3                                 | The interview (continued) + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Questioning the customer          |
|                                   |                                   |
|                                   | Generally, it is important to     |
|                                   | remain courteous, keep an open    |
|                                   | mind, remain objective, and pay   |
|                                   | attention to what the customer is |
|                                   | saying.                           |
|                                   |                                   |
|                                   | Relying too much on prepared      |
|                                   | questions can result in missing   |
|                                   | important information or leads.   |
|                                   |                                   |
|                                   | Do not badger the customer if     |
|                                   | they do not answer or state they  |
|                                   | cannot remember.                  |
|                                   |                                   |
|                                   | Discrepancies in the customer's   |
|                                   | answers can be clarified in a     |
|                                   | courteous and respectful way.     |
|                                   |                                   |
|                                   | If it is thought that the         |
|                                   | customer is not telling the       |
|                                   | truth, avoid any expressions of   |
|                                   | disbelief. Do not seek to         |
|                                   | directly challenge or discredit   |
|                                   | the customer by questioning the   |
|                                   | accuracy, credibility or value of |
|                                   | their answers.                    |
|                                   |                                   |
|                                   | Avoid:                            |
|                                   |                                   |
|                                   | -   leading questions or          |
|                                   |     questions that suggest an     |
|                                   |     answer, especially with       |
|                                   |     vulnerable customers          |
|                                   | -   multiple questions within a   |
|                                   |     question, as it can be        |
|                                   |     unclear which question the    |
|                                   |     customer was answering        |
|                                   | -   asking the customer to        |
|                                   |     speculate or guess            |
|                                   | -   using acronyms and jargon (it |
|                                   |     is best to ask the customer   |
|                                   |     about their payment, and then |
|                                   |     refer to it as they know it)  |
|                                   |                                   |
|                                   | Introducing forms, business       |
|                                   | records and exhibits              |
|                                   |                                   |
|                                   | Fully describe any document when  |
|                                   | introducing it to the customer.   |
|                                   | Include reference to identifying  |
|                                   | numbers, date stamps, names or    |
|                                   | exhibit numbers.                  |
|                                   |                                   |
|                                   | Question the customer about it,   |
|                                   | for example, Do they recognise    |
|                                   | it? Can they say who completed    |
|                                   | it? Is it their handwriting? Is   |
|                                   | it their signature?               |
|                                   |                                   |
|                                   | Interview breaks                  |
|                                   |                                   |
|                                   | Do not discuss the case with the  |
|                                   | customer while the interview is   |
|                                   | suspended                         |
|                                   |                                   |
|                                   | When the interview is resumed:    |
|                                   |                                   |
|                                   | -   re-caution the customer       |
|                                   | -   ask them to confirm on record |
|                                   |     that there was no discussion  |
|                                   |     about the case during the     |
|                                   |     break (if there was, state    |
|                                   |     that for the record)          |
|                                   |                                   |
|                                   | A consideration break near the    |
|                                   | end of the interview can allow    |
|                                   | the customer to consider anything |
|                                   | else they may wish to say, and    |
|                                   | the interviewers to consider any  |
|                                   | further questions.                |

|                                   | recording to the customer and ask |
|                                   | them to sign a receipt stating    |
|                                   | that they have been given a copy  |
|                                   | of the audio recording.           |
|                                   |                                   |
|                                   | Each copy of the recording should |

|                                   | customer (and interpreter), in    |
|                                   | the presence of each other. If    |
|                                   | the customer is under 18, the     |
|                                   | copy of the recording should also |
|                                   | be signed by the adult present.   |
|                                   |                                   |
|                                   | If the customer admitted that any |
|                                   | documents presented relate to     |
|                                   | them, ask them to initial and     |
|                                   | date the records shown to confirm |
|                                   | that they were the ones discussed |
|                                   | during the interview. The         |
|                                   | customer may refuse to do so.     |
