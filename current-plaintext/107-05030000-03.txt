|                                   | Did the customer request a review |
|                                   | of their existing recovery        |
|                                   | arrangement?                      |
|                                   |                                   |
|                                   | -   Yes, a review is not          |
|                                   |     required. Renegotiate a new   |
|                                   |     arrangement. See Payment      |
|                                   |     arrangements                  |
|                                   | -   No, go to Step 2              |
+-----------------------------------+-----------------------------------+
| 2                                 | Broken Arrangement + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | Go to the Payment Arrangement     |

|                                   |                                   |
|                                   | Is the arrangement Broken?        |
|                                   |                                   |
|                                   | -   Yes, a review is not          |
|                                   |     required. See Payment         |
|                                   |     arrangements                  |
|                                   | -   No, go to Step 3              |
+-----------------------------------+-----------------------------------+
| 3                                 | Capacity to pay + Read more ...   |
|                                   |                                   |
|                                   | Check the customer’s record to    |
|                                   | determine if they have failed to  |
|                                   | disclose their true capacity to   |
|                                   | pay at the time they made the     |
|                                   | arrangement. Ensure the           |
|                                   | information is accurate, up to    |
|                                   | date and reliable. For example, a |
|                                   | recently returned Q608/QSS40 is   |
|                                   | more up to date and reliable than |
|                                   | old savings on the Savings        |
|                                   | Summary (SVS) screen. Check:      |
|                                   |                                   |
|                                   | -   Income and Assets Task        |
|                                   |     Selector (ITS) for            |
|                                   |     information on savings and    |
|                                   |     investments                   |
|                                   | -   Document tools, for Q608 and  |
|                                   |     QSS40 notices                 |
|                                   | -   Document List (DL) for the    |
|                                   |     arrangement DOC and any       |
|                                   |     discussions about their       |
|                                   |     capacity to pay               |
|                                   | -   OPAL for the start date of    |
|                                   |     the arrangement               |
|                                   |                                   |
|                                   | Has the customer failed to        |
|                                   | disclose information about their  |
|                                   | true capacity to repay their      |
|                                   | debt?                             |
|                                   |                                   |
|                                   | -   Yes, go to Table 2, Step 4    |
|                                   | -   No, go to Step 4              |
+-----------------------------------+-----------------------------------+
| 4                                 | Less than 3 months + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | Go to the OPAL screen. Check the  |
|                                   | ADD date field.                   |
|                                   |                                   |
|                                   | Is the arrangement less than 3    |
|                                   | months old?                       |
|                                   |                                   |
|                                   | -   Yes, a review is not          |
|                                   |     required. Procedure ends      |
|                                   |     here.                         |
|                                   | -   No, go to Step 5              |
+-----------------------------------+-----------------------------------+

|                                   | more ...                          |
|                                   |                                   |
|                                   | On the OPAL screen check the      |

|                                   | arrangement details.              |
|                                   |                                   |
|                                   | Does the arrangement meet or      |
|                                   | exceed the acceptable cash level? |
|                                   |                                   |
|                                   | -   Yes, a review is not          |
|                                   |     required. Procedure ends      |
|                                   |     here.                         |
|                                   | -   No go to Step 6               |
+-----------------------------------+-----------------------------------+
| 6                                 | Agreement + Read more ...         |
|                                   |                                   |
|                                   | Go to the DL. Check the           |
|                                   | Arrangement DOC for any           |

|                                   | decision, a reduced rate for 12   |
|                                   | months etc.                       |
|                                   |                                   |
|                                   | Is an agreement in place?         |
|                                   |                                   |
|                                   | -   Yes, a review is not          |
|                                   |     required. Procedure ends      |
|                                   |     here.                         |

Reviewing recovery arrangements

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Contact the customer + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | Make reasonable effort to contact |
|                                   | the customer to discuss their     |
|                                   | capacity to pay and reason for    |
|                                   | the review.                       |
|                                   |                                   |
|                                   | Was the customer contacted?       |
|                                   |                                   |
|                                   | Yes, go to Step 2                 |
|                                   |                                   |
|                                   | No:                               |
|                                   |                                   |
|                                   | -   to end an arrangement for     |
|                                   |     failure to disclose           |
|                                   |     information about their       |
|                                   |     capacity to pay, go to Step 4 |
|                                   | -   for other reviews go to Step  |
|                                   |     3                             |
+-----------------------------------+-----------------------------------+
| 2                                 | Agree to a review + Read more ... |
|                                   |                                   |
|                                   | It is a requirement for the       |
|                                   | agency to provide the customer    |
|                                   | with 28 days’ notice before       |
|                                   | altering or terminating an        |
|                                   | arrangement. If the customer      |
|                                   | agrees to a review, negotiations  |
|                                   | can commence without the 28 days’ |
|                                   | notice.                           |
|                                   |                                   |
|                                   | Did the customer agree to review  |
|                                   | their existing arrangement?       |
|                                   |                                   |
|                                   | Yes, see Payment arrangements     |
|                                   |                                   |
|                                   | No, advise the customer:          |
|                                   |                                   |
|                                   | -   their arrangement will cease  |
|                                   |     in 28 days                    |
|                                   | -   they can continue to pay      |
|                                   |     their current amount for the  |
|                                   |     next 28 days                  |
|                                   | -   they will need to enter into  |
|                                   |     a new arrangement after 28    |
|                                   |     days                          |
|                                   | -   if the arrangement breaks     |
|                                   |     before 28 days, the           |
|                                   |     arrangement will cease        |
|                                   | -   of the recovery actions that  |
|                                   |     may take place under the cost |
|                                   |     effectiveness guidelines      |
|                                   | -   DOC the details of your       |
|                                   |     conversation                  |
|                                   | -   go to Step 3                  |
+-----------------------------------+-----------------------------------+
| 3                                 | 28 days' notice + Read more ...   |
|                                   |                                   |
|                                   | Provide the customer with 28      |
|                                   | days’ notice.                     |
|                                   |                                   |
|                                   | Send a Q312 letter with a         |
|                                   | Financial Information (SS245)     |
|                                   | form to the customer. The letter  |
|                                   | will advise:                      |
|                                   |                                   |
|                                   | -   their current arrangement is  |
|                                   |     no longer acceptable          |
|                                   | -   they have a 14 day grace      |
|                                   |     period before the new         |
|                                   |     arrangement takes effect      |
|                                   |                                   |
|                                   | Wait for the customer to respond  |
|                                   | or for 28 days to pass.           |
|                                   |                                   |
|                                   | Did the customer respond to the   |
|                                   | Q312 letter?                      |
|                                   |                                   |
|                                   | -   Yes, explain the reason for   |
|                                   |     the review and renegotiate    |
|                                   |     the arrangement. See Payment  |
|                                   |     arrangements                  |
|                                   | -   No, go to Step 4              |
+-----------------------------------+-----------------------------------+
| 4                                 | End the arrangement + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | Only end the arrangement:         |
|                                   |                                   |
|                                   | -   after the customer is given   |
|                                   |     28 days' notice, or           |
|                                   | -   there is sufficient evidence  |
|                                   |     to show customer has failed   |
|                                   |     to disclose information about |
|                                   |     their true capacity to repay  |
|                                   |     their debt. For example a     |
|                                   |     recently returned Q608 or     |
|                                   |     QSS40                         |
|                                   |                                   |
|                                   | Cease the arrangement:            |
|                                   |                                   |
|                                   | on the Payment Arrangement List   |
|                                   | (OPAL):                           |
|                                   |                                   |
|                                   | -   'S'elect the arrangement      |
|                                   | -   press [Enter]                 |
|                                   |                                   |
|                                   | enter 'T' for today in the Cease  |
|                                   | date field                        |
|                                   |                                   |
|                                   | enter any comments                |
|                                   |                                   |
|                                   | press [Enter]                     |
|                                   |                                   |
|                                   | the arrangement will show as CAN  |
|                                   |                                   |
|                                   | record a DOC with the following:  |
|                                   |                                   |
|                                   | -   CRN balance                   |
|                                   | -   Debt Ids                      |
|                                   | -   Reason for ceasing the        |
|                                   |     arrangement                   |
|                                   | -   Section of the Act            |
|                                   | -   28 days’ notice given to the  |
|                                   |     customer (if applicable)      |
