General information

Social Security Agreement between Australia and Italy

This table describes general information relating to the Agreement with
Italy including the social security system in Italy, history of the
Agreement, exchange of information and contact details and explains the
role of the Italian Patronati.

| Italian Social Security System    | Italian Social Security System +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | In Italy, the pension insurance   |
|                                   | legislation requires the          |
|                                   | compulsory payment of social      |
|                                   | security contributions on behalf  |
|                                   | of employees, usually through     |
|                                   | deductions from paid employment.  |
|                                   |                                   |
|                                   | Contributions insure employees    |
|                                   | against disability and            |
|                                   | unemployment and provides for     |
|                                   | income in the form of a pension   |
|                                   | once the person reaches           |
|                                   | retirement age.                   |
|                                   |                                   |
|                                   | The legislation is the            |
|                                   | responsibility of the Italian     |
|                                   | federal government but the        |
|                                   | collection of contributions and   |
|                                   | payment of pensions is the        |
|                                   | responsibility of the various     |
|                                   | pension funds. These are broadly  |
|                                   | arranged to cover the different   |
|                                   | industries.                       |
|                                   |                                   |
|                                   | In addition to the contribution   |
|                                   | based system, there is also a     |
|                                   | welfare system which gives        |
|                                   | benefits to people who are not    |
|                                   | covered by the contributory       |
|                                   | system. Payments under the        |
|                                   | welfare system are income tested  |
|                                   | and generally paid at a lower     |
|                                   | rate than those paid under the    |
|                                   | contributory scheme.              |

| Italian Patronati                 | Italian Patronti + Read more ...  |
|                                   |                                   |
|                                   | Patronati are assigned by the     |
|                                   | Italian government to help a      |
|                                   | person who has been an Italian    |
|                                   | resident or is of Italian descent |
|                                   | regarding their welfare and       |
|                                   | rights. Any assistance given by a |
|                                   | Patronato is free of charge.      |
|                                   |                                   |
|                                   | Patronati offices are set up to   |
|                                   | assist customers to claim and     |
|                                   | maintain an Italian pension and   |
|                                   | are experienced in the complex    |
|                                   | requirements. Patronati may also  |
|                                   | assist their customers to claim a |
|                                   | pension from another country,     |
|                                   | e.g. Switzerland, Germany etc.    |
|                                   |                                   |
|                                   | Note: foreign pension claims      |
|                                   | lodged through a Patronato office |
|                                   | are not to be coded as sent       |
|                                   | overseas direct as the Patronato  |
|                                   | office will send the claim to     |
|                                   | International Services (CIS). If  |
|                                   | a customer indicates they have    |
|                                   | lodged a foreign pension claim    |
|                                   | with a Patronato office, transfer |
|                                   | the case to CIS. See Assisting    |
|                                   | customers to claim a foreign      |
|                                   | pension.                          |
|                                   |                                   |
|                                   | Customers should be encouraged to |
|                                   | lodge Italian pension claims      |
|                                   | through a Patronato office and    |
|                                   | seek assistance for ongoing       |
|                                   | issues, such as:                  |
|                                   |                                   |
|                                   | -   Life Certificates. See the    |
|                                   |     Life certificates on the      |
|                                   |     Italian payments tab          |
|                                   | -   The Mod Red Campaign. See the |
|                                   |     Mod Red Campaign on the       |
|                                   |     Italian payments tab          |
|                                   |                                   |
|                                   | Contact details for Patronati are |
|                                   | available in the CODES facility.  |
|                                   |                                   |
|                                   | Note: Services Australia must not |
|                                   | advertise or recommend any        |
|                                   | individual Patronato office, see  |
|                                   | below.                            |

|                                   |                                   |
|                                   | Authority to deal with            |
|                                   | Patronati + Read more ...         |
|                                   |                                   |
|                                   | Since customers need to give a    |
|                                   | mandate for assistance, the       |
|                                   | Patronato takes on the function   |
|                                   | of Customer Agent. Services       |
|                                   | Australia can deal with the agent |
|                                   | as if they are the customer and   |
|                                   | release information to the agent  |
|                                   | (see part 5.3060 Customer         |
|                                   | Authority for Release - Express   |
|                                   | Authority in the Privacy intranet |
|                                   | page).                            |
|                                   |                                   |
|                                   | However, if this form is not      |
|                                   | signed, Patronati can also be     |
|                                   | seen as a member for a 'helping'  |
|                                   | profession. Member of the         |
|                                   | 'helping' profession are seen as  |
|                                   | acting in the interest of a       |
|                                   | customer and an implied authority |
|                                   | can be taken to be in place. See  |
|                                   | the Privacy intranet page).       |
|                                   |                                   |
|                                   | Issues when dealing with          |
|                                   | Patronati + Read more ...         |
|                                   |                                   |
|                                   | -   Patronati are commercial      |
|                                   |     enterprises                   |
|                                   | -   They represent different      |
|                                   |     Italian organisations (about  |
|                                   |     eight in Australia) and are   |
|                                   |     in competition with each      |
|                                   |     other for 'processing points' |
|                                   |     that is, they are funded      |
|                                   |     according to the number of    |
|                                   |     claims they assist with       |
|                                   | -   Services Australia must not   |
|                                   |     advertise or recommend any    |
|                                   |     individual Patronato office   |
|                                   | -   If a Patronato Office, other  |
|                                   |     than that appearing on the    |
|                                   |     customer's Foreign Claim      |
|                                   |     Details (FGD) screen,         |
|                                   |     contacts Services Australia   |
|                                   |     the enquiring Patronato must  |
|                                   |     provide a current customer    |
|                                   |     authorisation, evidencing     |
|                                   |     that the Patronato is now     |
|                                   |     administering the customer's  |
|                                   |     affairs. Generally a          |
|                                   |     Patronato would have a        |
|                                   |     Revocation Mandate Order if   |
|                                   |     they are taking over mandate  |
|                                   |     from another Patronato as the |
|                                   |     client's agent. Once evidence |
|                                   |     has been provided the new     |
|                                   |     Patronato Office must be      |
|                                   |     recorded on the FGD screen    |
|                                   | -   Details given to a Patronato  |
|                                   |     should be limited to data     |
|                                   |     generated by the customer's   |
|                                   |     Services Australia            |
|                                   |     entitlement (for example,     |
|                                   |     Customer Reference Number     |
|                                   |     (CRN), rate details etc). A   |
|                                   |     Patronato should not need to  |
|                                   |     ask basic information about   |
|                                   |     our mutual customers, for     |
|                                   |     example, their address,       |
|                                   |     marital status or Italian     |
|                                   |     pension reference. This is    |
|                                   |     information the Patronato can |
|                                   |     obtain from the customer and  |
|                                   |     a Service Officer must not    |
|                                   |     provide it to them if they    |
|                                   |     ask                           |
|                                   | -   Patronato requests for bulk   |
|                                   |     information in relation to    |
|                                   |     multiple customers are        |
|                                   |     limited to the annual Mod Red |
|                                   |     Campaign which is             |
|                                   |     administered by the           |
|                                   |     International Program. See    |
|                                   |     the Mod Red Campaign on the   |
|                                   |     Italian payments tab          |
