For the Income Management Exemption and Assessment Team (IMEAT) only

This page contains processing information relating to determining
financial vulnerability as part of an assessment for an exemption from
Income Management for customers with dependent children.

| 2                                 | IM Financial Vulnerability        |
|                                   | Assessment Decision screen + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | []                                |
|                                   |                                   |
|                                   | A Financial Vulnerability         |
|                                   | Assessment decision must be       |
|                                   | recorded using the following      |
|                                   | criteria:                         |
|                                   |                                   |
|                                   | -   Meeting priority needs - The  |
|                                   |     customer has been applying    |
|                                   |     appropriate resources to meet |
|                                   |     the priority needs of         |
|                                   |     themselves and their family   |
|                                   | -   Financial exploitation - The  |
|                                   |     customer has control over     |
|                                   |     their money and is not        |
|                                   |     subject to financial          |
|                                   |     exploitation                  |
|                                   | -   Budgeting and payment         |
|                                   |     patterns - The customer did   |
|                                   |     not regularly require urgent  |
|                                   |     funds to pay for foreseeable  |
|                                   |     costs; did not regularly      |
|                                   |     change their EPED and had     |
|                                   |     valid reasons for applying    |
|                                   |     for urgent payments           |
|                                   | -   Financial stability - The     |
|                                   |     customer has a stable pattern |
|                                   |     of payments and budgeting     |
|                                   |     practices to meet their       |
|                                   |     priority needs from their     |
|                                   |     income support payment and    |
|                                   |     family assistance payment     |
|                                   |                                   |
|                                   | Financial Vulnerability           |
|                                   | Assessment decision overview      |
|                                   |                                   |
|                                   | A decision cannot be recorded     |
|                                   | until each criteria is addressed  |
|                                   | as indicated by a green tick      |
|                                   | (indicating a 'Yes') or a red     |
|                                   | cross (indicating a 'No') under   |
|                                   | the 'Decision'.                   |
|                                   |                                   |
|                                   | Financial Vulnerability Decision  |
|                                   | details                           |
|                                   |                                   |
|                                   | Has the customer been applying    |
|                                   | appropriate resources to meet     |
|                                   | priority needs? Select Yes or No  |
|                                   | as appropriate.                   |
|                                   |                                   |
|                                   | The reasons for the decision must |
|                                   | be recorded in the Free text:     |
|                                   | field.                            |
|                                   |                                   |
|                                   | Financial Vulnerability           |
|                                   | Assessment result                 |
|                                   |                                   |
|                                   | Based on the results recorded for |
|                                   | each interview factor and         |
|                                   | decision, select the Overall      |
