|                                   | -   For CP, see Carer Payment     |

| 3                                 | Earnings not declared or          |
|                                   | incorrectly declared + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | Update the customer's record to   |
|                                   | reflect the correct details.      |
|                                   |                                   |
|                                   | For example, the customer has     |
|                                   | declared the net earnings instead |

|                                   |     income details and to call    |
|                                   |     back once they have all the   |
|                                   |     details needed                |
|                                   | -   Do not ask the customer to    |
|                                   |     provide paper verification    |

|                                   |     over the phone                |
|                                   | -   Use questioning skills to     |
|                                   |     obtain as much information as |
|                                   |     possible from the customer    |
|                                   |     over the phone to determine   |
|                                   |     when thy worked and what they |
|                                   |     were paid                     |
|                                   |                                   |

|                                   |     Commencing or returning to    |
|                                   |     work or self-employment       |
|                                   |     Disability Support Pension    |
|                                   |     (DSP)                         |
|                                   | -   For CP customers, see Carer   |
|                                   |     Payment (CP) and Carer        |
|                                   |     Allowance (CA) carer          |
|                                   |     undertakes paid employment,   |
|                                   |     self-employment, voluntary    |
|                                   |     work, education or training   |
|                                   |     activities                    |
|                                   | -   For CP customers where the    |
|                                   |     care receiver commences or    |
|                                   |     returns to work               |
|                                   |                                   |

|                                   |                                   |
|                                   | -   There may be times when the   |

|                                   |     correct employment income     |
|                                   |     details in the initial phone  |
|                                   |     discussion                    |
|                                   | -   Allowing the customer to call |

|                                   |     source the correct            |
