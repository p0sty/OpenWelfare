FAQs

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Question: Does the first          |
|                                   | continuing have to be after 12    |
|                                   | weeks?                            |
|                                   |                                   |
|                                   | Answer: Yes, the patient needs to |
|                                   | have at least 12 weeks of         |
|                                   | treatment to demonstrate a        |
|                                   | response to treatment. Measure 12 |
|                                   | weeks from approval.              |
|                                   |                                   |
|                                   | Note: the date of test results    |
|                                   | cannot be greater than 4 weeks    |
|                                   | from the end of the most recent   |
|                                   | course of treatment.              |
+-----------------------------------+-----------------------------------+
| 2                                 | Question: The patient did not     |
|                                   | provide an ESR at baseline but is |
|                                   | passing on ESR now – is this ok?  |
|                                   |                                   |
|                                   | Answer: No, a response can only   |
|                                   | be against the parameters         |
|                                   | provided at baseline.             |
+-----------------------------------+-----------------------------------+
| 3                                 | Question: Patient provided both   |
|                                   | ESR and CRP at baseline but is    |
|                                   | only responding on CRP currently  |
|                                   | – is this ok?                     |
|                                   |                                   |
|                                   | Answer: Yes. As long as both      |
|                                   | bloods were qualifying for entry  |
|                                   | at baseline, only responding on   |
|                                   | one of them currently is ok.      |
