|                                   |                                   |

|                                   | Before finalising activity, check |
|                                   | if the customer is eligible to    |

Coding self-employment and finalising assessment

Table 2: this table describes the coding required relating to the
customer's employment status and the coding of self-employment details.
Portions of this process are for Complex Assessment Officers (CAOs)
only.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Not unemployed + Read more ...    |
|                                   |                                   |
|                                   | Take the following action:        |
|                                   |                                   |
|                                   | -   Advise the customer of their  |
|                                   |     mutual obligation             |
|                                   |     requirements. See Applying    |
|                                   |     mutual obligation             |
|                                   |     requirements                  |
|                                   | -   Cancel the customer's JSP or  |
|                                   |     YA, as they are not           |
|                                   |     satisfying their mutual       |
|                                   |     obligation requirements. See  |
|                                   |     Coding a cancellation         |
|                                   | -   Advise customer of their      |
|                                   |     continued entitlement to a    |
|                                   |     Health Care Card (HCC) for    |
|                                   |     six months (if they have been |
|                                   |     in receipt of payments for 12 |
|                                   |     months)                       |
|                                   | -   Advise customer to contact    |
|                                   |     Services Australia as soon as |
|                                   |     possible for advice on        |
|                                   |     reclaiming if the business    |
|                                   |     does not succeed              |
|                                   |                                   |
|                                   | If the customer:                  |
|                                   |                                   |
|                                   | -   has a partner on income       |
|                                   |     support, reassess partner's   |
|                                   |     entitlement, then go to Step  |
|                                   |     2                             |
|                                   | -   does not have a partner on    |
|                                   |     income support, record        |
|                                   |     details on a DOC. Procedure   |
|                                   |     ends here                     |
+-----------------------------------+-----------------------------------+
| 2                                 | Unemployed + Read more ...        |
|                                   |                                   |
|                                   | Complete the following tasks:     |
|                                   |                                   |
|                                   | -   Advise customer how income    |
|                                   |     from self-employment affects  |
|                                   |     their payments. Staff can     |
|                                   |     access the Combined Rate      |
|                                   |     Estimator in Customer First.  |
|                                   |     For help, see Online          |
|                                   |     estimator options             |
|                                   | -   Advise customer how to        |
|                                   |     declare income from           |
|                                   |     self-employment. The customer |
|                                   |     must still fulfil their usual |
|                                   |     reporting requirements if     |
|                                   |     they also have employment     |
|                                   |     income                        |
|                                   | -   If in receipt of JSP or YA    |
|                                   |     (job seeker) advise the       |
|                                   |     customer that they must       |
|                                   |     satisfy their mutual          |
|                                   |     obligation requirements. If   |
|                                   |     in receipt of Youth Allowance |
|                                   |     (student) or Austudy, remind  |
|                                   |     the customer of their         |
|                                   |     continued obligation to       |
|                                   |     satisfy their respective      |
|                                   |     Activity Tests                |
|                                   | -   Advise the customer that if   |
|                                   |     they are in receipt of JSP/YA |
|                                   |     (job seeker) with a partial   |
|                                   |     capacity to work, they may    |
|                                   |     satisfy their mutual          |
|                                   |     obligation requirements if    |
|                                   |     they are working a minimum of |
|                                   |     15 hours per week and         |
|                                   |     declaring at least the        |
|                                   |     minimum wages for this        |
|                                   |     period. Note: their work      |
|                                   |     capacity may be reviewed if   |
|                                   |     they work more than 60 hours  |
|                                   |     per fortnight over a period   |
|                                   |     of three consecutive          |
|                                   |     fortnights                    |
|                                   | -   If the customer is the        |
|                                   |     principal carer of a          |
|                                   |     dependent child, they may     |
|                                   |     satisfy their compulsory      |
|                                   |     participation requirements if |
|                                   |     they are working a minimum of |
|                                   |     15 hours per week (or more    |
|                                   |     than 30 hours per fortnight)  |
|                                   |     and declaring at least        |
|                                   |     minimum wages for this period |
|                                   | -   Code a review to follow-up    |
|                                   |     profit and loss statements.   |
|                                   |     See Setting up Manual Reviews |
|                                   | -   If the customer has a partner |
|                                   |     on income support, reassess   |
|                                   |     partner's entitlement         |
+-----------------------------------+-----------------------------------+
| 3                                 | Code income from                  |
|                                   | self-employment + Read more ...   |
|                                   |                                   |
|                                   | Key the sole trader/partnership   |
|                                   | details in either:                |
|                                   |                                   |
|                                   | Process Direct:                   |
|                                   |                                   |
|                                   | -   key Start into the Super Key  |
|                                   |     and select the                |
|                                   |     Self-Employment workflow from |
|                                   |     the Task Selector             |
|                                   |                                   |
|                                   | Customer Record:                  |
|                                   |                                   |
|                                   | -   use the Self Employment       |
|                                   |     script to update income and   |
|                                   |     assets                        |
|                                   |                                   |
|                                   | Manually code the details if the  |
|                                   | workflow and script are not       |
|                                   | available.                        |
|                                   |                                   |
|                                   | For trusts and companies:         |
|                                   |                                   |
|                                   | -   The case must be referred to  |
|                                   |     a Complex Assessment Officer  |
|                                   |     (CAO) to assess the           |
|                                   |     customer's involvement in the |
|                                   |     business                      |
|                                   | -   Ensure the relevant documents |
|                                   |     and MOD forms are scanned for |
|                                   |     processing                    |
|                                   | -   Use the CAO Referral guided   |
|                                   |     procedure and forward it to   |
|                                   |     region code CAO               |
|                                   | -   Procedure ends here           |
