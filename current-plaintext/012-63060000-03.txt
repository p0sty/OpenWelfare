Clean up report

This table outlines how to run a clean-up report from the Complex drugs.
Clean-up reports must be empty on the Service Officer's last working day
of the month.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Click on the Clean-Up Button +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Click on the Clean-Up button on   |
|                                   | the Complex Drugs portal.         |
+-----------------------------------+-----------------------------------+
| 2                                 | Select the processing team + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Select the team that have had     |
|                                   | processing work completed in the  |
|                                   | calendar month.                   |
|                                   |                                   |
|                                   | This will generate a report which |
|                                   | will list the program and the     |
|                                   | potential clean-up reason.        |
+-----------------------------------+-----------------------------------+
| 3                                 | Investigation + Read more ...     |
|                                   |                                   |
|                                   | Investigate the program/s record. |
|                                   | If unable to fix the clean-up,    |
|                                   | escalate to a Local Peer Support  |
|                                   | (LPS).                            |
+-----------------------------------+-----------------------------------+
| 4                                 | Close out of the database + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Close out of the programs         |
|                                   | database and clean-up report.     |
+-----------------------------------+-----------------------------------+
| 5                                 | Clean-up report empty + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | Once the clean-up report is empty |
|                                   | it will appear blank or a message |
|                                   | will pop-up.                      |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
