|                                   | A person in Australia who is      |
|                                   | claiming a payment from an        |
|                                   | agreement country will usually    |
|                                   | receive an official claim form to |
|                                   | complete and lodge with           |
|                                   | Centrelink. For most other        |
|                                   | countries, the notice advises the |
|                                   | customer to complete a standard   |
|                                   | proforma (Request for Claim       |
|                                   | Papers) requesting the foreign    |
|                                   | pension authority to send them    |
|                                   | the necessary claim forms.        |
|                                   |                                   |
|                                   | For the standard proforma,        |
|                                   | customers should complete all the |
|                                   | necessary details and sign the    |
|                                   | document.                         |
|                                   |                                   |
|                                   | For official claim forms, the     |
|                                   | main points to note are:          |
|                                   |                                   |
|                                   | -   Signature: it is critical     |
|                                   |     that claim forms are signed   |
|                                   |     by the customer in the        |
|                                   |     appropriate sections of forms |
|                                   | -   Identity: it is important     |
|                                   |     that all names the customer   |
|                                   |     has been known by, for        |
|                                   |     example, maiden name and name |
|                                   |     at birth are entered into the |
|                                   |     relevant section of the form  |
|                                   |     and as much supporting        |
|                                   |     evidence as possible is       |
|                                   |     provided                      |
|                                   |                                   |
|                                   | For claims from an agreement      |
|                                   | country, customers who are        |
|                                   | currently in receipt of           |
|                                   | Australian income support         |
|                                   | payments have already verified    |
|                                   | their identity. Non-customers     |
|                                   | should provide identity           |
|                                   | documentation that is normally    |
|                                   | required for Australian income    |
|                                   | support payments. See Identity    |
|                                   | Confirmation for further          |
|                                   | information.                      |
|                                   |                                   |
|                                   | For claims from a non-agreement   |
|                                   | country, staff should copy and    |
|                                   | certify whatever identity         |
|                                   | documents the customer provides.  |
|                                   |                                   |
|                                   | -   Periods of Coverage: most     |
|                                   |     foreign pension claims are    |
|                                   |     based on proving periods of   |
|                                   |     coverage in that country's    |
|                                   |     social security system, for   |
|                                   |     example, paid social security |
|                                   |     contributions through         |
|                                   |     employment. It is important   |
|                                   |     that customers give as much   |
|                                   |     detail as possible in the     |
|                                   |     relevant section of the form  |
|                                   |     and provide as much           |
|                                   |     supporting evidence as        |
|                                   |     possible                      |
|                                   | -   Personal Details: some claims |
|                                   |     are reliant on proof of       |
|                                   |     personal details, for         |
|                                   |     example, survivor or widow    |
|                                   |     pension (SUR) claims          |
|                                   |     generally require evidence of |
|                                   |     legal status of the           |
|                                   |     relationship or widowhood,    |
|                                   |     such as a Marriage            |
|                                   |     Certificate or Death          |
|                                   |     Certificate                   |
|                                   | -   Australian residence periods: |
|                                   |     For claims from an agreement  |
|                                   |     country, proof of Australian  |
|                                   |     periods of coverage is        |
|                                   |     normally required. This is    |
|                                   |     usually Australian Working    |
|                                   |     Life Residence (AWLR) but     |
|                                   |     does vary for some            |
|                                   |     agreements. Forms are         |
|                                   |     provided in agreement country |
|                                   |     claim packages. For           |
|                                   |     information, see the relevant |
|                                   |     country file in International |
|                                   |     Social Security Agreements    |
|                                   |                                   |
|                                   | For information on verifying      |
|                                   | Australian residence, see Coding  |
|                                   | the Country of Residence (CRES)   |
|                                   | screen.                           |
|                                   |                                   |

Foreign Pension System (FPS) notices

Table 2:

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Customer asks about a CFP notice  |
|                                   | they have received + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | CFP notices are Batch Laser       |
|                                   | Advice (BLA) letters issued via   |
|                                   | the FPS. FPS processing includes  |
|                                   | reminder letters and automatic    |
|                                   | suspension of payments.           |
|                                   |                                   |
|                                   | If a customer has been suspended  |
|                                   | for failing to take reasonable    |
|                                   | steps to make a claim (SUS-CFP),  |
|                                   | the case must be reviewed by      |
|                                   | Centrelink International Services |
|                                   | (CIS). Go to Step 2.              |
|                                   |                                   |
|                                   | Navigate to the Foreign Claim     |
|                                   | Summary (FGS) screen and check    |
|                                   | the Status: field for the         |
|                                   | relevant country and foreign      |
|                                   | pension type.                     |
|                                   |                                   |
|                                   | If the Status: field is:          |
|                                   |                                   |
|                                   | ISS-VOL Issued - voluntary claim, |
|                                   | advise the customer that:         |
|                                   |                                   |
|                                   | -   This notice provides          |
|                                   |     information about claiming    |
|                                   |     the relevant foreign pension  |
|                                   | -   It is up to the customer      |
|                                   |     whether they decide to make a |
|                                   |     claim but Centrelink will     |
|                                   |     assist them in the manner     |
|                                   |     indicated in the notice       |
|                                   | -   Procedure ends here           |
|                                   |                                   |
|                                   | ISS Issued - blank (compulsory),  |
|                                   | go to Step 2                      |
|                                   |                                   |
|                                   | OVS Claim sent o/s, advise the    |
|                                   | customer that:                    |
|                                   |                                   |
|                                   | -   The claim form for the        |
|                                   |     foreign pension has been sent |
|                                   |     to the foreign pension        |
|                                   |     authority but no result has   |
|                                   |     been advised                  |
|                                   | -   They will need to follow up   |
|                                   |     the progress with the foreign |
|                                   |     pension authority directly.   |
|                                   |     Provide the customer with     |
|                                   |     known contact details for the |
|                                   |     foreign pension authority     |
|                                   | -   Advise the customer of the    |
|                                   |     FPS review process including  |
|                                   |     the timing of further letters |
|                                   |     and possible suspension of    |
|                                   |     payments. See Foreign Pension |
|                                   |     System (FPS) statuses and     |
|                                   |     reviews for further           |
|                                   |     information                   |
|                                   | -   Procedure ends here           |
|                                   |                                   |
|                                   | FIN-NOM Finalised - Not           |
|                                   | entitled - other or FIN-EXE       |
|                                   | Finalised - Exempt, advise the    |
|                                   | customer that:                    |
|                                   |                                   |
|                                   | -   This notice confirms that     |
|                                   |     they either do not qualify    |
|                                   |     (NOM) or that they are exempt |
|                                   |     from the requirement to claim |
|                                   |     the foreign pension (EXE)     |
|                                   | -   No further action will be     |
|                                   |     taken in relation to a claim  |
|                                   |     for that particular foreign   |
|                                   |     pension                       |
|                                   | -   If the customer wishes to     |
|                                   |     make a claim, re-code         |
|                                   |     relevant FGD screen as        |
|                                   |     ISS-VOL                       |
|                                   | -   Procedure ends here           |
|                                   |                                   |
|                                   | GEP Granted - Embargo/OVP pending |
|                                   | or FIN-GRA Finalised - Claim      |
|                                   | granted, go to Step 2             |
