This process should only be completed by Families and Child care Smart
Centre Staff.

This page contains information for Families and Child care Smart Centre
Staff to process Paid Parental Leave scheme claims outside of PPL and
DAP claim lodgement timeframes or when an Additional Primary Claimant
has lodged a claim.

On this page:

Assessing and processing of PPL and DAP claims

Recording a claim lodgement or exceptional circumstances override - for
Families Override Contacts only

Assessing and processing of PPL and DAP claims

Table 1: this table describes how to process PPL and DAP claims that are
outside of claim lodgement timeframes, have exceptional circumstances
and where a manual override is required.

| 2                                 | Processing PPL/DAP claims outside |
|                                   | of claim lodgement timeframes +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | In most cases PPL and DAP claims  |
|                                   | must be lodged within 12 months   |
|                                   | of the date the child was born or |
|                                   | came into care as part of the     |
|                                   | adoption process.                 |
|                                   |                                   |
|                                   | Additionally, for children that   |
|                                   | are born or adopted on or after 1 |
|                                   | July 2020:                        |
|                                   |                                   |
|                                   | PPL claims can be lodged by a     |
|                                   | secondary or tertiary claimants   |
|                                   | up to 24 months after the date    |
|                                   | the child was born or came into   |
|                                   | care as part of the adoption      |
|                                   | process                           |
|                                   |                                   |
|                                   | Flexible PPL days can be accessed |
|                                   | up to 24 months after the date    |
|                                   | the child was born or came into   |
|                                   | care as part of the adoption      |
|                                   | process. However to have access   |
|                                   | to a balance of Flexible PPL      |
|                                   | days, the primary claimant must   |
|                                   | claim PPL:                        |
|                                   |                                   |
|                                   | -   within 12 months of the date  |
|                                   |     the child was born or came    |
|                                   |     into care as part of the      |
|                                   |     adoption process, and         |
|                                   | -   be eligible for Flexible PPL  |
|                                   |     days                          |
|                                   |                                   |
|                                   | In limited circumstances, a claim |
|                                   | for PPL or DAP may need to be     |
|                                   | processed outside of claim        |
|                                   | lodgement timeframes.             |
|                                   |                                   |
|                                   | -   A review of decision has      |
|                                   |     resulted in a favourable      |
|                                   |     decision for the customer     |
|                                   | -   A Ministerial decision which  |
|                                   |     has resulted in a changed     |
|                                   |     outcome                       |
|                                   | -   Act of Grace ruling which has |
|                                   |     resulted in a changed         |
|                                   |     outcome, or                   |
|                                   | -   The customer lodged a claim   |
|                                   |     inside of the claim lodgement |
|                                   |     timeframes, however a system  |
|                                   |     issue was identified that     |
|                                   |     resulted in the claim not     |
|                                   |     able to be granted            |
|                                   | -   Correction to incorrect data, |
|                                   |     resulting in an inappropriate |
|                                   |     rejection or non-effective    |
|                                   |     claim                         |
|                                   |                                   |
|                                   | Generally this will be following  |
|                                   | a review of decision of a         |
|                                   | rejected or not effective claim.  |
|                                   |                                   |
|                                   | Does the claim meet the criteria  |
|                                   | for a PPL or DAP claim to be      |
|                                   | processed outside of claim        |
|                                   | lodgement timeframes?             |
|                                   |                                   |
|                                   | Yes, go to Step 3.                |
|                                   |                                   |
|                                   | No,                               |
|                                   |                                   |
|                                   | -   Override referral is not      |
|                                   |     required. Continue claim      |
|                                   |     processing. See               |
|                                   |     Pre-processing checks for     |
|                                   |     standalone and combined       |
|                                   |     claims for family assistance  |
|                                   |     and/or Parental Leave         |
|                                   |     Payments                      |
|                                   | -   Procedure ends here           |

|                                   | e.g. customer returned to work or |

|                                   | original PPL period               |

|                                   | be made and PPL will be delivered |

|                                   | cases                             |
|                                   |                                   |
|                                   | For children born on or after 1   |
|                                   | July 2020, do not include any     |
|                                   | connected or Flexible PPL days in |
|                                   | the new claim. This will result   |
|                                   | in an error when attempting to    |
|                                   | process the override information  |
|                                   |                                   |
|                                   | After all information has been    |
|                                   | completed and all claim           |

|                                   | ‘E913P1 - Claim for child over 12 |

|                                   | (ELD) screen.                     |

| 5                                 | Processing PPL claims under       |
|                                   | exceptional circumstances for an  |
|                                   | Additional Primary Claimant +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | The primary claimant will most    |
|                                   | often be the birth mother or the  |
|                                   | initial primary carer of an       |
|                                   | adopted or surrogate child.       |
|                                   |                                   |
|                                   | Sometimes, PPL may be payable to  |
|                                   | a new primary claimant after the  |
|                                   | birth mother/primary carer has    |
|                                   | received PPL payments.            |
|                                   |                                   |
|                                   | Example                           |
|                                   |                                   |
|                                   | -   The birth mother or initial   |
|                                   |     primary carer is no longer    |
|                                   |     able to care for the child,   |
|                                   |     and                           |
|                                   | -   additional circumstances have |
|                                   |     been met, and                 |
|                                   | -   evidence has been provided by |
|                                   |     a new primary carer, and      |
|                                   | -   all other PPL eligibility is  |
|                                   |     met                           |
|                                   |                                   |
|                                   | Does the claimant meet the        |
|                                   | exceptional circumstances         |
|                                   | criteria?                         |
|                                   |                                   |
|                                   | Yes, ensure that updates have     |
|                                   | been made to the Child in Primary |
|                                   | Care screen and the Flexible PPL  |
|                                   | days for each carer:              |
|                                   |                                   |
|                                   | -   For the previous primary      |
|                                   |     carer check or code the Child |
|                                   |     in Primary Care screen with   |
|                                   |     Primary care ‘No’ and Reason  |
|                                   |     ‘OTH’ from the date the child |
|                                   |     left their primary care. This |
|                                   |     should be done in a separate  |
|                                   |     activity, see Customer        |
|                                   |     notifies a change of          |
|                                   |     circumstances for Parental    |
|                                   |     Leave Pay (PPL)               |
|                                   | -   For the previous primary      |
|                                   |     claimant and/or secondary     |
|                                   |     claimants, check or code the  |
|                                   |     Flexible PPL days as          |
|                                   |     withdrawn with reason ‘OTH’   |
|                                   |     using the PPL circumstances   |
|                                   |     guided procedure, see         |
|                                   |     Customer notifies a change of |
|                                   |     circumstances for Parental    |
|                                   |     Leave Pay (PPL)               |
|                                   | -   For the new primary carer     |
|                                   |     code ‘Yes’ from the date the  |
|                                   |     child entered their primary   |
|                                   |     care within the PPL claim     |
|                                   | -   Go to Step 6                  |
|                                   |                                   |
|                                   | No:                               |
|                                   |                                   |

| 7                                 | Override referral + Read more ... |
|                                   |                                   |
|                                   | Authority to record an override   |
|                                   | is restricted to a Families       |
|                                   | Override Officer. The override    |
|                                   | must be recorded before the claim |
|                                   | can be finalised. There are a     |
|                                   | number of different override      |
|                                   | options available:                |
|                                   |                                   |
|                                   | -   O1Y (Over 1 Year) for PPL     |
|                                   |     claims (children born or      |
|                                   |     adopted before 1 July 2020    |
|                                   |     only)                         |
|                                   | -   FLO (Flexible Override) for   |
|                                   |     PPL claims (children born or  |
|                                   |     adopted on or after 1 July    |
|                                   |     2020 only)                    |
|                                   | -   APC (Additional Primary       |
|                                   |     Claimant) for PPL claims      |
|                                   |     (children born or adopted on  |
|                                   |     or after 1 July 2020 only)    |
|                                   | -   OTH (Other) when no other     |
|                                   |     code can be applied for DAP   |
|                                   |     or PPL (children born or      |
|                                   |     adopted before 1 July 2020    |
|                                   |     only)                         |
|                                   |     Note: OTH is only available   |
|                                   |     in Process Direct in limited  |
|                                   |     circumstances                 |
|                                   |                                   |
|                                   | Service Officers in:              |
|                                   |                                   |
|                                   | -   face to face, complete a      |
|                                   |     Families Override Request     |
|                                   | -   smart centres, complete a     |
|                                   |     Direct referral to SSO        |
|                                   |                                   |
|                                   | See the Resources page for a link |
|                                   | to the Families Override Contacts |
|                                   | Program webform(s), to create a   |
|                                   | referral.                         |
|                                   |                                   |
|                                   | Include the following information |
|                                   | in the referral to the Families   |
|                                   | Override Contact:                 |
|                                   |                                   |
|                                   | -   type of override required     |
|                                   |     (O1Y/FLO/APC/OTH)             |
|                                   | -   reason for the override (e.g. |
|                                   |     review of decision)           |
|                                   | -   PPL/DAP start and end date.   |
|                                   |     It is essential the correct   |
|                                   |     dates are recorded on this    |
|                                   |     screen. This will calculate   |
|                                   |     the rate of payment to be     |
|                                   |     issued to the customer. Note: |
|                                   |     if the customer is claiming   |
|                                   |     any connected Flexible PPL    |
|                                   |     days these can be within the  |
|                                   |     total start and end dates     |
|                                   | -   all dates/range of dates      |
|                                   |     nominated by customer for all |
|                                   |     Flexible PPL days (not        |
|                                   |     connected) to the PPL period  |
|                                   |                                   |
|                                   | Note: where more than one         |
|                                   | override needs to be recorded for |
|                                   | the same child e.g. transfer      |
|                                   | portion of PPL to the customer's  |
|                                   | partner, record an override on    |
|                                   | both PPL records. Start dates and |
|                                   | end dates will need to be entered |
|                                   | for each customer, and must meet  |
|                                   | eligibility rules, i.e. no break  |
|                                   | in payment for the PPL period.    |
|                                   |                                   |
|                                   | DOC the record with details of    |
|                                   | the claim and override decision,  |
|                                   | and the above information.        |
|                                   |                                   |
|                                   | Note: any inappropriate referrals |
|                                   | will be returned to the           |
|                                   | initiating Service Officer.       |
|                                   |                                   |
|                                   | Once the Families Override        |
|                                   | Officer has coded the override,   |
|                                   | go to Step 8.                     |
