Medical practitioner prescriber number

Table 1: this table describes how to issue a prescriber number for a
medical practitioner when an Application for a Medicare provider number
and, or prescriber number for a medical practitioner form (HW019) is
received.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Assess application details + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Check the application to make     |
|                                   | sure the following fields have    |
|                                   | been completed:                   |
|                                   |                                   |
|                                   | -   First name and surname        |
|                                   | -   Date of birth                 |
|                                   | -   Existing provider number (if  |
|                                   |     applicable)                   |
|                                   | -   Indicated prescriber number   |
|                                   |     is needed                     |
|                                   | -   Declaration must be signed    |
+-----------------------------------+-----------------------------------+
| 2                                 | Check eligibility + Read more ... |
|                                   |                                   |
|                                   | Check that the medical            |
|                                   | practitioner:                     |
|                                   |                                   |
|                                   | -   Has a provider number (or is  |
|                                   |     applying for a provider       |
|                                   |     number)                       |
|                                   | -   Has current medical           |
|                                   |     registration with Ahpra that  |
|                                   |     does not have restrictions to |
|                                   |     prescribe medications         |
|                                   |                                   |
|                                   | If an application cannot be       |
|                                   | processed or if the request is    |
|                                   | incomplete, see Table 6.          |
+-----------------------------------+-----------------------------------+
| 3                                 | Allocate a prescriber number +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Allocate a prescriber number in   |
|                                   | the Provider Directory System     |
|                                   | (PDS).                            |
|                                   |                                   |
|                                   | Select Qualification tab          |
|                                   |                                   |
|                                   | In the Prescriber Status field    |
|                                   | select Prescriber                 |
|                                   |                                   |
|                                   | -   Select Save                   |
+-----------------------------------+-----------------------------------+
| 4                                 | Issue confirmation letter and     |
|                                   | finalise + Read more ...          |
|                                   |                                   |
|                                   | Select the appropriate letter     |
|                                   | template                          |
|                                   |                                   |
|                                   | -   Z2701 for a prescriber number |
|                                   |     issued in conjunction with a  |
|                                   |     new provider number or        |
|                                   | -   Z2193 for a prescriber number |
|                                   |     only                          |
|                                   | -   complete appropriate fields   |
|                                   |     of letter template            |
|                                   | -   save letter as a PDF          |
|                                   | -   upload letter to PaNDA        |
|                                   | -   make sure that the Surname    |
|                                   |     First Name, Provider Number   |
|                                   |     and Case ID fields are        |
|                                   |     completed and correct         |
|                                   | -   print the letter and send to  |
|                                   |     the medical practitioner      |
|                                   | -   add any appropriate notes to  |
|                                   |     the PaNDA file                |
|                                   | -   select Complete               |
|                                   | -   Select Request Approved       |
|                                   | -   Select Save or Save and Next, |
|                                   |     as appropriate                |
|                                   |                                   |
|                                   | Note: start date of the           |
|                                   | prescriber number on the letter   |
|                                   | is date of issue or first         |
|                                   | location start date whichever is  |
|                                   | later.                            |
+-----------------------------------+-----------------------------------+

Dentist prescriber number

Table 2: this table describes how to issue a prescriber number for a
dentist when an Application for a Medicare provider number and, or
prescriber number for allied health and non-medical health professionals
form (HW093) is received.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Assess application details + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Check the application to make     |
|                                   | sure the following fields have    |
|                                   | been completed:                   |
|                                   |                                   |
|                                   | -   First name and surname        |
|                                   | -   Date of birth                 |
|                                   | -   Existing provider number (if  |
|                                   |     applicable)                   |
|                                   | -   Indicated prescriber number   |
|                                   |     is needed                     |
|                                   | -   Declaration must be signed    |
+-----------------------------------+-----------------------------------+
| 2                                 | Check eligibility + Read more ... |
|                                   |                                   |
|                                   | Check that the dentist:           |
|                                   |                                   |
|                                   | -   Has a provider number (or is  |
|                                   |     applying for a provider       |
|                                   |     number)                       |
|                                   | -   Holds registration with Ahpra |
|                                   |     as a dentist, and does not    |
|                                   |     have restrictions to          |
|                                   |     prescribe medications         |
|                                   |                                   |
|                                   | Go to the Ahpra website and take  |
|                                   | a snip of the Ahpra registration  |
|                                   | details. Copy the image into a    |
|                                   | Word document, convert to a PDF   |
|                                   | and upload to PaNDA (even if a    |
|                                   | copy of the Ahpra registration    |
|                                   | has been supplied with the        |
|                                   | application).                     |
|                                   |                                   |
|                                   | Note: Representative Public       |
|                                   | Dentists (RPDs) are not eligible  |
|                                   | for a prescriber number.          |
|                                   |                                   |
|                                   | If an application cannot be       |
|                                   | processed or if the request is    |
|                                   | incomplete, see Table 6.          |
+-----------------------------------+-----------------------------------+
| 3                                 | Forward application in PANDA for  |
|                                   | Delegate approval + Read more ... |
|                                   |                                   |
|                                   | In PaNDA                          |
|                                   |                                   |
|                                   | -   Make sure the Surname, First  |
|                                   |     Name. Provider Number and     |
|                                   |     Case ID fields are complete   |
|                                   |     and correct                   |
|                                   | -   Select Processing Notes       |
|                                   | -   Select Add Processing Note    |
|                                   | -   Add a processing note stating |
|                                   |     ‘delegate approval required   |
|                                   |     for prescriber number’        |
|                                   | -   Select Confirm                |
|                                   | -   Select Details                |
|                                   | -   Select Send for Delegation    |
|                                   |                                   |
|                                   | When the application is Approved, |
|                                   | go to Step 4.                     |
+-----------------------------------+-----------------------------------+
| 4                                 | Allocate a prescriber number +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Allocate a prescriber number in   |
|                                   | the Provider Directory System     |
|                                   | (PDS).                            |
|                                   |                                   |
|                                   | -   Select Qualification tab      |
|                                   | -   in the Prescriber Status      |
|                                   |     field select Prescriber       |
|                                   | -   Select Save                   |
+-----------------------------------+-----------------------------------+
| 5                                 | Issue confirmation letter and     |
|                                   | finalise + Read more ...          |
|                                   |                                   |
|                                   | Select the appropriate letter     |
|                                   | template:                         |
|                                   |                                   |
|                                   | -   Z2599 for a prescriber number |
|                                   |     issued in conjunction with a  |
|                                   |     new provider number or        |
|                                   | -   Z2193 for a prescriber number |
|                                   |     only                          |
|                                   |                                   |
|                                   | -   Complete appropriate fields   |
|                                   |     of letter template            |
|                                   | -   Save letter as a PDF          |
|                                   | -   Upload letter to PaNDA        |
|                                   | -   Make sure that the Surname    |
|                                   |     First Name, Provider Number   |
|                                   |     and Case ID fields are        |
|                                   |     completed and correct         |
|                                   | -   Print the letter and send to  |
|                                   |     the medical practitioner      |
|                                   | -   Add any appropriate notes to  |
|                                   |     the PaNDA file                |
|                                   | -   Select Complete               |
|                                   | -   Select Request Approved       |
|                                   | -   Select Save or Save and Next, |
|                                   |     as appropriate                |
|                                   |                                   |
|                                   | Note: start date of the           |
|                                   | prescriber number on the letter   |
|                                   | is date of issue or first         |
|                                   | location start date whichever is  |
|                                   | later.                            |

| Step                              | Action                            |

| 1                                 | Assess application details + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Check the application to make     |
|                                   | sure the following fields have    |
|                                   | been completed:                   |

|                                   |     registration as an            |
|                                   |     optometrist                   |

|                                   |     qualified to administer,      |
|                                   |     obtain, possess, prescribe,   |
|                                   |     supply or use topical         |
|                                   |     schedule 2, 3 or 4 medicines  |
|                                   |     for the purposes of the       |
|                                   |     practice of optometry         |

|                                   | When the application is Approved, |
|                                   | go to Step 4.                     |
+-----------------------------------+-----------------------------------+
| 4                                 | Allocate a prescriber number +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Allocate a prescriber number in   |
|                                   | Provider Directory System (PDS).  |
|                                   |                                   |
|                                   | Select Qualification tab          |
|                                   |                                   |
|                                   | in the Prescriber Status field    |
|                                   | select Prescriber                 |
|                                   |                                   |
|                                   | Select Specialty tab              |
|                                   |                                   |
|                                   | -   Select Add Specialty Code     |
|                                   | -   Key 551 into Specialty Code   |
|                                   |     Description                   |

|                                   | for a Medicare provider number    |
|                                   | and/or PBS prescriber number for  |
|                                   | a midwife or nurse practitioner   |
|                                   | form (HW088) has been completed.  |
+-----------------------------------+-----------------------------------+
| 2                                 | Check nurse practitioner's        |
|                                   | details + Read more ...           |
|                                   |                                   |
|                                   | Check that the nurse              |
|                                   | practitioner:                     |
|                                   |                                   |
|                                   | -   Has a provider number (or is  |
|                                   |     applying for a provider       |
|                                   |     number)                       |
|                                   | -   Holds current Ahpra           |

|                                   | -   Has 'Nurse Practitioner'      |

|                                   |     registration                  |

|                                   |     the image into a Word         |
|                                   |     document, convert to a PDF    |

|                                   | If an application cannot be       |
|                                   | processed or if the request is    |
|                                   | incomplete, see Table 6           |
+-----------------------------------+-----------------------------------+
| 3                                 | Move application to the National  |
|                                   | Office Escalation folder in PaNDA |
|                                   | for Delegate approval + Read more |
|                                   | ...                               |
|                                   |                                   |

|                                   | -   Add a processing note stating |
|                                   |     “delegate approval required   |
|                                   |     for prescriber number”        |
|                                   | -   Select Confirm                |
|                                   | -   Select Details                |
|                                   | -   Select Unassign               |
|                                   | -   Select Recategorised in       |
|                                   |     Reason field                  |
|                                   | -   Select Provider Registration  |
|                                   |     in Work Type Program          |
|                                   | -   Select PROV_NAT_OFFICE_ESC as |
|                                   |     Work Type                     |
|                                   | -   Select Submit                 |
|                                   |                                   |
|                                   | When the application is Approved, |
|                                   | go to Step 4.                     |
+-----------------------------------+-----------------------------------+
| 4                                 | Determine the location nurse      |
|                                   | practitioner works in + Read more |
|                                   | ...                               |
