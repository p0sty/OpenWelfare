How to determine the date a claim is 'made' and start day under an agreement

Table 1

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Determining the date a claim is   |
|                                   | 'made' and start day              |
|                                   |                                   |
|                                   | To be paid an Australian social   |
|                                   | security payment, a person must   |
|                                   | generally make a claim. A claim   |
|                                   | is 'made' by being 'lodged' (that |
|                                   | is, received) but may be deemed   |
|                                   | or 'taken to be made' on a        |
|                                   | different date, for example,      |
|                                   | early claims.                     |
|                                   |                                   |
|                                   | For claims made after 1 July      |
|                                   | 2018, the 'intent to claim'       |
|                                   | provisions apply only to          |
|                                   | customers in vulnerable           |
|                                   | circumstances. See Intent to      |
|                                   | claim and vulnerable customers.   |
|                                   |                                   |
|                                   | Policy advice is that a person    |
|                                   | may meet the residence rules for  |
|                                   | claims on either the date the     |
|                                   | claim is 'made' or the date the   |
|                                   | claim is 'taken to be made'       |
|                                   | taking into account CCD, DOCs or  |
|                                   | early claim, as well as any other |
|                                   | backdated Start Day rules. For    |
|                                   | example, lodged within 14 days of |
|                                   | their partner or within 4 weeks   |
|                                   | of childbirth.                    |
|                                   |                                   |
|                                   | Note: subsection 16(7) of the     |
|                                   | Social Security (Administration)  |
|                                   | Act 1999 allows a delegate at the |
|                                   | appropriate level to determine    |
|                                   | that a person may make a claim in |
|                                   | another way.                      |
|                                   |                                   |
|                                   | In addition:                      |
|                                   |                                   |
|                                   | -   All agreements allow the date |
|                                   |     a claim is lodged with the    |
|                                   |     agreement authority in the    |
|                                   |     agreement country to be used  |
|                                   |     as the date the claim is      |
|                                   |     lodged with Services          |
|                                   |     Australia                     |
|                                   | -   Some agreements allow the     |
|                                   |     date of a claim for the other |
|                                   |     country's pension to be used  |
|                                   |     as the date of lodgement of a |
|                                   |     claim for the corresponding   |
|                                   |     Australian payment, subject   |
|                                   |     to meeting specific criteria  |
|                                   |                                   |
|                                   | Accordingly, for claims under     |
|                                   | agreements, the date the claim is |
|                                   | 'made' (that is, lodged/received) |
|                                   | may be any of the following:      |
|                                   |                                   |
|                                   | -   The date the corresponding    |
|                                   |     foreign pension claim is      |
|                                   |     lodged in the agreement       |
|                                   |     country                       |
|                                   |     Note: only for claims under   |
|                                   |     those agreements with the     |
|                                   |     relevant provision            |
|                                   | -   The date the Australian claim |
|                                   |     is lodged with agreement      |
|                                   |     country authority in the      |
|                                   |     agreement country - Overseas  |
|                                   |     Lodgement Date (DOL)          |
|                                   | -   The date the Australian claim |
|                                   |     is lodged (received) in       |
|                                   |     Australia - Date of Receipt   |
|                                   |     (DOR)                         |
|                                   |                                   |
|                                   | If the customer needs to use an   |
|                                   | agreement to be paid, the date    |
|                                   | cannot be earlier than the date   |
|                                   | the agreement started.            |
|                                   |                                   |
|                                   | Staff should first check if the   |
|                                   | customer meets the residence      |
|                                   | rules for claims under the        |
|                                   | agreement on the earliest         |
|                                   | possible date the claim was       |
|                                   | 'made' under the agreement        |
|                                   | according to the points above:    |
|                                   |                                   |
|                                   | -   Check if the claim can be     |
|                                   |     'taken to be made' on a       |
|                                   |     different day from the date   |
|                                   |     the claim was actually 'made' |
|                                   |     (as above), including early   |
|                                   |     claim, CCD, claim after       |
|                                   |     partner's claim, claim after  |
|                                   |     giving birth                  |
|                                   |     For claims made after 1 July  |
|                                   |     2018, the 'intent to claim'   |
|                                   |     provisions apply only to      |
|                                   |     customers in vulnerable       |
|                                   |     circumstances. See Intent to  |
|                                   |     claim and vulnerable          |
|                                   |     customers                     |
|                                   | -   Check if the customer meets   |
|                                   |     the residence rules for       |
|                                   |     claims under the agreement on |
|                                   |     each date in chronological    |
|                                   |     order and either was          |
|                                   |     qualified and payable on that |
|                                   |     date or became qualified and  |
|                                   |     payable within 13 weeks of    |
|                                   |     that date (early claim)       |
|                                   |                                   |
|                                   | Users must code the DOL in the    |
|                                   | agreement country and DOR in      |
|                                   | Australia correctly as part of    |
|                                   | the claim. Users can manually     |
|                                   | shift the Start Date on the       |
|                                   | Pensions Assessment (PNA) screen  |
|                                   | if needed for a deemed claim or   |
|                                   | to apply the 'claim for one =     |
|                                   | claim for other' provision of an  |
|                                   | agreement. If the Start Date is   |
|                                   | moved from DOL/DOR, the Date of   |
|                                   | Event for agreement coding on the |
|                                   | Additional Residence Detail (ARD) |
|                                   | screen also be moved to match the |
|                                   | Start Date.                       |
|                                   |                                   |
|                                   | System processing will then       |
|                                   | determine if the customer meets   |
|                                   | all the requirements for payment  |
|                                   | under the agreement in sequence   |
|                                   | and grant the claim from the      |
|                                   | earliest date the person became   |
|                                   | qualified.                        |
|                                   |                                   |
|                                   | The customer may simply not meet  |
|                                   | the all the requirements for      |
|                                   | payment and should be rejected.   |
