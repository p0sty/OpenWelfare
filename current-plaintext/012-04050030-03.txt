Checking the Network Organisation registration application

Table 1: This table describes how Service Officers check a My Health
Record system Provider Registration Application assigned to them.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Locate, open and save My Health   |
|                                   | Record registration application + |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Open and save the most recent My  |
|                                   | Health Record registration        |
|                                   | application assigned to Service   |
|                                   | Officer in their individual       |
|                                   | folder.                           |
|                                   |                                   |
|                                   | Applications are assigned to      |
|                                   | Service Officers by the Team      |
|                                   | Leaders (TL5) in the individual   |
|                                   | Service Officer's folder. Service |
|                                   | Officers check their individual   |
|                                   | folders each morning to locate    |
|                                   | assigned work.                    |
+-----------------------------------+-----------------------------------+
| 2                                 | Check all questions completed +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Has applicant completed all       |
|                                   | questions on page 3?              |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, contact the applicant,    |
|                                   |     obtain missing information.   |
|                                   |     When form is complete, go to  |
|                                   |     Step 3                        |
+-----------------------------------+-----------------------------------+
| 3                                 | Question 2 - Check if applicant   |
|                                   | is RO or OMO + Read more ...      |
|                                   |                                   |
|                                   | Is the applicant the Seed         |
|                                   | Organisation's Responsible        |
|                                   | Officer (RO) or Organisation      |
|                                   | Maintenance Officer (OMO) as      |
|                                   | recorded in the Provider          |
|                                   | Participation Register (PPR)?     |
|                                   |                                   |
|                                   | -   Yes, go to Step 4             |
|                                   | -   No, contact the applicant,    |
|                                   |     advise they are not listed as |
|                                   |     RO/OMO for the Seed           |
|                                   |     Organisation. The form must   |
|                                   |     be completed by the RO/OMO.   |
|                                   |     Procedure ends here           |
+-----------------------------------+-----------------------------------+
| 4                                 | Question 3 and 4 - Check personal |
|                                   | details match PPR + Read more ... |
|                                   |                                   |
|                                   | Are the RO/OMO's personal details |
|                                   | (first name, last name, date of   |
|                                   | birth etc.) the same as those     |
|                                   | recorded in the Provider          |
|                                   | Participation Register (PPR)?     |
|                                   |                                   |
|                                   | -   Yes, go to Step 5             |
|                                   | -   No, contact the applicant,    |
|                                   |     confirm their details over    |
|                                   |     the phone, go to Step 5       |
+-----------------------------------+-----------------------------------+
| 5                                 | Question 6 and 7 - Check Seed     |
|                                   | Organisation registered in PPR +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Is the Seed Organisation          |
|                                   | registered in PPR?                |
|                                   |                                   |
|                                   | -   Yes, go to Step 6             |
|                                   | -   No, contact the applicant to  |
|                                   |     confirm details or advise     |
|                                   |     process for registration.     |
|                                   |     Procedure ends here           |
+-----------------------------------+-----------------------------------+
| 6                                 | Check Declaration + Read more ... |
|                                   |                                   |
|                                   | Has the applicant signed and      |
|                                   | dated the declaration on page 4?  |
|                                   |                                   |
|                                   | -   Yes, go to Step 7             |
|                                   | -   No, contact the applicant,    |
|                                   |     advise they must sign and     |
|                                   |     date the declaration.         |
|                                   |     Procedure ends here           |
+-----------------------------------+-----------------------------------+
| 7                                 | Check Attachment A and/or         |
|                                   | Attachment B + Read more ...      |
|                                   |                                   |
|                                   | Has the applicant completed       |
|                                   | Attachment A and/or Attachment B? |
|                                   |                                   |
|                                   | -   Yes, go to Step 8             |
|                                   | -   No, contact the applicant,    |
|                                   |     advise to sign and complete   |
|                                   |     Attachment A and/or           |
|                                   |     Attachment B. Procedure ends  |
|                                   |     here                          |
+-----------------------------------+-----------------------------------+
| 8                                 | Check table A and/or B + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | In table A and/or B is the        |
|                                   | Network Organisation(s) part of   |
|                                   | the Seed Hierarchy in PPR?        |
|                                   |                                   |
|                                   | -   Yes, go to Complete checking  |
|                                   |     the Network Organisation      |
|                                   |     registration application      |
|                                   |     table                         |
|                                   | -   No, contact the applicant,    |
|                                   |     advise the listed network     |
|                                   |     organisation is not           |
|                                   |     registered in the Health      |
|                                   |     Identifiers Services as part  |
|                                   |     of the Seed Organisation      |
|                                   |     Hierarchy                     |
