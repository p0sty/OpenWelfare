+-----------------------------------+-----------------------------------+
| 1                                 | Access to the Child Relationships |
|                                   | page + Read more ...              |
|                                   |                                   |
|                                   | The Child Relationships page is   |
|                                   | accessed after selecting the      |
|                                   | relevant child from the:          |
|                                   |                                   |
|                                   | -   Child Selection (CHS) page in |
|                                   |     Customer First, or            |
|                                   | -   Relations menu in Process     |
|                                   |     Direct                        |
|                                   |                                   |
|                                   | The Child Relationships page can  |
|                                   | be accessed from the Child Task   |
|                                   | Selector (CHTS) or by keying:     |
|                                   |                                   |
|                                   | -   '!CHRE' in Nxt: field after   |
|                                   |     selecting the child, or       |
|                                   | -   CHRE in Super Key in Process  |
|                                   |     Direct                        |
|                                   |                                   |
|                                   | It is included in the Online and  |
|                                   | Families Claim workflow and the   |
|                                   | screen flow for all families      |
|                                   | claims and add child activities.  |
|                                   |                                   |
|                                   | If the child is located in a      |
|                                   | different environment, see        |
|                                   | Linking a child to a customer's   |
|                                   | record.                           |
|                                   |                                   |
|                                   | E186FY - Relationship between     |
|                                   | Customer and xxxx must be         |
|                                   | recorded                          |
|                                   |                                   |
|                                   | This edit can occur during a      |
|                                   | range of activities including:    |
|                                   |                                   |
|                                   | -   Family Tax Benefit (FTB),     |
|                                   |     Parental Leave Pay (PPL) and  |
|                                   |     Dad and Partner Pay (DAP)     |
|                                   |     claims,                       |
|                                   | -   Parenting Payment Single      |
|                                   |     (PPS) to Parenting Payment    |
|                                   |     Partnered (PPP) Benefit       |
|                                   |     Transfers (BTR), and          |
|                                   | -   change of care/custody coding |
|                                   |                                   |
|                                   | The edit presents when one or     |
|                                   | more children do not have a       |
|                                   | relationship to the customer      |
|                                   | and/or partner/step-parent        |
|                                   | recorded. The relationship of the |
|                                   | parent or step-parent to the      |
|                                   | child must be recorded on the     |
|                                   | Child Relationship page.          |

| 2                                 | Adding details in Customer        |
|                                   | First + Read more ...             |
|                                   |                                   |
|                                   | Go to the Child Relationships     |
|                                   | page, see Item 1 for details.     |
|                                   |                                   |
|                                   | To add a row (person):            |
|                                   |                                   |
|                                   | Select Add row button             |
|                                   |                                   |
|                                   | Select the person or their        |
|                                   | Customer Reference Number (CRN).  |
|                                   | Note: only the person’s first     |
|                                   | name and middle name will show    |
|                                   |                                   |
|                                   | In Relationship: field, select    |
|                                   | the person's relationship to the  |
|                                   | child from the drop down box:     |
|                                   |                                   |
|                                   | -   parent (NAT) - the person is  |
|                                   |     the child's biological parent |
|                                   |     or their relationship parent  |
|                                   | -   adoptive parent (ADT)         |
|                                   | -   grandparent (GRP)             |
|                                   | -   step parent (STEP)            |
|                                   | -   foster parent (FOS)           |
|                                   | -   other (OTH)                   |
|                                   |                                   |
|                                   | In Date of Event: field, code the |
|                                   | relationship start date (this     |
|                                   | depends on the person's           |
|                                   | relationship to the child)        |
|                                   |                                   |
|                                   | Code Source: and Receipt Date:    |
|                                   | fields                            |
|                                   |                                   |
|                                   | Select Continue button - this     |
|                                   | will auto-populate the CRN if the |
|                                   | person's name was coded, or the   |
|                                   | person's name if their CRN was    |
|                                   | coded (if the person's name and   |
|                                   | CRN are coded and they do not     |
|                                   | match, an error message will      |
|                                   | show)                             |
|                                   |                                   |
|                                   | Finalise activity on the          |
|                                   | Assessments Results (AR) screen.  |

|                                   | Relationship. Drop down box to    |
|                                   | select the person's relationship  |
|                                   | to the child                      |
|                                   |                                   |
|                                   | -   parent (NAT) - the person is  |
|                                   |     the child's biological parent |
|                                   |     or their relationship parent  |
|                                   | -   adoptive parent (ADT)         |
|                                   | -   grandparent (GRP)             |
|                                   | -   step parent (STEP)            |
|                                   | -   foster parent (FOS)           |
|                                   | -   other (OTH)                   |
|                                   |                                   |
|                                   | Date of Event. Code the           |
|                                   | relationship start date (this     |
|                                   | depends on the person's           |
|                                   | relationship to the child)        |
|                                   |                                   |
|                                   | Currently in Care - Display only  |

|                                   | the current customer/claimant or  |

|                                   |                                   |
|                                   | -   Source - Display only:        |

|                                   |     from Child Miscellaneous      |

|                                   |     on 20 June 2011)              |
|                                   |                                   |
|                                   | Source: and Receipt Date: fields  |

| 9                                 | Claim processing + Read more ...  |
|                                   |                                   |

|                                   | Benefit (FTB), Parental Leave Pay |
|                                   | (PPL), Dad and Partner Pay (DAP)  |
|                                   | and the Child Relationships page  |
|                                   | must be coded for each child the  |

|                                   |                                   |
|                                   | The Online and Families Claim     |
|                                   | workflow will upload child        |

|                                   | this data to be input when coding |
|                                   | a paper claim.                    |

|                                   | (!CHRE/CHRE) page is included in  |

|                                   | Selector (CHTS) for all the       |

|                                   |                                   |
|                                   | -   NCL (New Claim)               |
|                                   | -   CCL (Combined Claim)          |
