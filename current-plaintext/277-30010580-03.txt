Steps for viewing or updating a notice

Table 2:

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Start point - View or update a    |
|                                   | notice + Read more ...            |
|                                   |                                   |
|                                   | Start point: Collection window.   |
+-----------------------------------+-----------------------------------+
| 2                                 | Collection action record + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Select the relevant Collection    |
|                                   | Action record in the Collection   |
|                                   | Action list box and select [Open] |
|                                   | or the Go button to open the      |
|                                   | Issue Section 72 A/B Notice       |
|                                   | window.                           |
|                                   |                                   |
|                                   | -   To approve a pended notice,   |
|                                   |     go to Step 3                  |
|                                   | -   To update a sent notice that  |
|                                   |     has been complied with, go to |
|                                   |     Step 5                        |
|                                   | -   To view a response or         |
|                                   |     reported payment, go to Step  |
|                                   |     7                             |
+-----------------------------------+-----------------------------------+
| 3                                 | Update the notice status + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Select Accepted from the Notice   |
|                                   | Status dropdown menu.             |
+-----------------------------------+-----------------------------------+
| 4                                 | Authorise notice + Read more ...  |
|                                   |                                   |
|                                   | -   Select the About Notice       |
|                                   |     button                        |
|                                   | -   Key the authorisation text    |
|                                   |     into the Reason for Issue     |
|                                   |     group box and select the Ok   |
|                                   |     button. Go to Step 8 to save  |
|                                   |     changes made                  |
+-----------------------------------+-----------------------------------+
| 5                                 | Record if the payment has been    |
|                                   | received + Read more ...          |
|                                   |                                   |
|                                   | -   Select the In Full or Part    |
|                                   |     Payment Received tick box in  |
|                                   |     the Complied with group box   |
|                                   | -   Select the About Notice       |
|                                   |     button to open the About      |
|                                   |     Notice dialogue box and       |
|                                   |     select which radio button     |
|                                   |     applies in the Funds          |
|                                   |     Information group box         |
+-----------------------------------+-----------------------------------+
| 6                                 | Update the additional             |
|                                   | information + Read more ...       |
|                                   |                                   |
|                                   | Key any details of the payment in |
|                                   | the Additional Information text   |
|                                   | box. Select the OK button to      |
|                                   | return to the Issue Section 72    |
|                                   | A/B Notice window. Go to Step 8   |
|                                   | to save changes made.             |
+-----------------------------------+-----------------------------------+
| 7                                 | Record the notice responses +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Select the Responses button which |
|                                   | opens the Response and Reported   |
|                                   | Payments dialogue box.            |
+-----------------------------------+-----------------------------------+
| 8                                 | Save changes + Read more ...      |
|                                   |                                   |
|                                   | Select [Save] to save any changes |
|                                   | made.                             |
