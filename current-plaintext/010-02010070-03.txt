This page contains a table with explanatory notes to assist with
completing the Claim for ABSTUDY Schooling A (SY018).

Explanatory notes for the SY018

This table describes information for completing the Claim for ABSTUDY
Schooling A (SY018). The form is divided into the sections. These are
listed in the table. Within each section, those questions of a more
complex nature have been identified with explanatory notes.

+-----------------------------------+-----------------------------------+
| Step                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Current record available + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Customers who already have a      |
|                                   | Centrelink record (including a    |
|                                   | CHIld record) may be able to      |
|                                   | lodge a claim using the ABSTUDY   |
|                                   | Claiming Tool instead of a paper  |
|                                   | claim.                            |
|                                   |                                   |
|                                   | Customers completing a claim      |
|                                   | through the ABSTUDY Claiming Tool |
|                                   | can be offered a verbal           |
|                                   | declaration. Customers who do not |
|                                   | accept a verbal declaration are   |
|                                   | required to return a signed CDF.  |
|                                   |                                   |
|                                   | A customer’s acceptance to the    |
|                                   | verbal declaration is recorded in |
|                                   | the Verbal Declaration DOC, and   |
|                                   | in the Progress of ABSTUDY claim  |
|                                   | DOC.                              |
|                                   |                                   |
|                                   | Note: on 28 November 2016, a      |
|                                   | verbal declaration replaced the   |
|                                   | requirement for the customer to   |
|                                   | return a signed Customer          |
|                                   | Declaration Form (CDF). Customers |
|                                   | who completed a claim through the |
|                                   | ABSTUDY Claiming Tool prior to 28 |
|                                   | November 2016 are may still be    |
|                                   | required to return their signed   |
|                                   | CDF. See ABSTUDY Claiming Tool    |
|                                   | process for more information.     |
+-----------------------------------+-----------------------------------+
| 2                                 | Student details (Q1 to Q9) + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Notes for questions 1, 2, 8 and 9 |
|                                   |                                   |
|                                   | Q1 Aboriginality                  |
|                                   |                                   |
|                                   | For ABSTUDY, an Australian        |
|                                   | Aboriginal or Torres Strait       |
|                                   | Islander is someone who:          |
|                                   |                                   |
|                                   | -   is of Australian Aboriginal   |
|                                   |     or Torres Strait Islander     |
|                                   |     descent                       |
|                                   | -   identifies themselves as an   |
|                                   |     Australian Aboriginal or      |
|                                   |     Torres Strait Islander, and   |
|                                   | -   is accepted as such by the    |
|                                   |     community in which they live, |
|                                   |     or have previously lived      |
|                                   |                                   |
|                                   | Normally, the declaration on the  |
|                                   | claim form will suffice, however  |
|                                   | where doubt exists, the customer  |
|                                   | will need to provide evidence of  |
|                                   | their Aboriginality.              |
|                                   |                                   |
|                                   | Q2 Australian Citizen             |
|                                   |                                   |
|                                   | -   Students born overseas may    |
|                                   |     still qualify for ABSTUDY if  |
|                                   |     they are an Australian        |
|                                   |     citizen. Note: ABSTUDY        |
|                                   |     assistance is not available   |
|                                   |     to permanent residents        |
|                                   | -   Proof of citizenship is       |
|                                   |     required for claims for       |
|                                   |     students who were born        |
|                                   |     overseas only where such      |
|                                   |     proof has not been previously |
|                                   |     provided to Services          |
|                                   |     Australia. Acceptable proof   |
|                                   |     may be provided by way of:    |
|                                   | -   passport, or                  |
|                                   | -   citizenship certificate       |
|                                   | -   The Newly Arrived Resident's  |
|                                   |     Waiting Period (NARWP) does   |
|                                   |     not apply to ABSTUDY          |
|                                   |                                   |
|                                   | Q8 - Date student came into care  |
|                                   |                                   |
|                                   | If the student entered the        |
|                                   | claimant's care after the 1       |
|                                   | January in the current year of    |
|                                   | study, the date care commenced is |
|                                   | used to determine the correct     |
|                                   | School Term Allowance that is     |
|                                   | payable.                          |
|                                   |                                   |
|                                   | Q9 - Other Commonwealth           |
|                                   | government assistance             |
|                                   |                                   |
|                                   | ABSTUDY is not payable to         |
|                                   | customers who are already in      |
|                                   | receipt of a Commonwealth         |
|                                   | government education scheme such  |
|                                   | as the Assistance for Isolated    |
|                                   | Children (AIC) Scheme.            |
+-----------------------------------+-----------------------------------+
| 3                                 | School details (Q10 to Q12) +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Q12 School Fees Allowance         |
|                                   |                                   |
|                                   | Normally, School Fees Allowance   |
|                                   | is paid directly to the school    |
|                                   | listed in Question 11.            |
|                                   |                                   |
|                                   | The applicant may, however to     |
|                                   | choose to pay the school and      |
|                                   | claim the allowance after         |
|                                   | providing Services Australia with |
|                                   | proof of payment to the school.   |
|                                   |                                   |
|                                   | Reimbursement will be made for    |
|                                   | the lesser of the amount of       |
|                                   | school fees paid and the maximum  |
|                                   | School Fees Allowance entitlement |
|                                   | for the student.                  |
+-----------------------------------+-----------------------------------+
| 4                                 | Section C - Carer and Carer’s     |
|                                   | partner details (Q13 to Q31) +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Q21 - Relationship to the student |
|                                   |                                   |
|                                   | The carer is the person who       |
|                                   | normally looks after the student. |
|                                   |                                   |
|                                   | For ABSTUDY, a foster parent is   |
|                                   | the carer of a student who has    |
|                                   | been placed in substitute care    |
|                                   | through a State or Territory      |
|                                   | welfare authority or legal        |
|                                   | process.                          |
+-----------------------------------+-----------------------------------+
| 5                                 | Payment Details Q31 to Q33 + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Q33 - Payment options             |
|                                   |                                   |
|                                   | Customers may choose to have      |
|                                   | ABSTUDY paid into a bank account  |
|                                   | that a current payment is made    |
|                                   | into, or into a different bank    |
|                                   | account.                          |
|                                   |                                   |
|                                   | The applicant is the correct      |
|                                   | payee. Payment of Schooling A     |
|                                   | allowances can only be made to    |
|                                   | the applicant. Note: School Fees  |
|                                   | Allowance is normally paid direct |
|                                   | to the school.                    |
+-----------------------------------+-----------------------------------+
| 6                                 | Section E - Information on this   |
|                                   | form (Q35) + Read more ...        |
|                                   |                                   |
|                                   | Q35 - Express authority           |
|                                   |                                   |
|                                   | The applicant may choose to allow |
|                                   | their partner or another person   |
|                                   | to make enquiries to Services     |
|                                   | Australia on their behalf.        |
|                                   |                                   |
|                                   | The applicant may choose to alter |
|                                   | who they allow to have access to  |
|                                   | their record at any time.         |
+-----------------------------------+-----------------------------------+
| 7                                 | Tax File Number details (Q36) +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Q36 - Supplying a Tax File Number |
|                                   | (TFN)                             |
|                                   |                                   |
|                                   | A Tax File Number (TFN)           |
|                                   | declaration must be made by both  |
|                                   | the applicant and the applicant's |
|                                   | partner (where applicable)        |
|                                   |                                   |
|                                   | Applicants and/or their partners  |
|                                   | who do not have, or do not know   |
|                                   | what their TFN is, must be given  |
|                                   | a Tax File Number application or  |
|                                   | enquiry for an Aboriginal or      |
|                                   | Torres Strait Islander form       |
|                                   | (NAT1589) for completion and      |
|                                   | lodgement within 14 days of       |
|                                   | lodging their claim               |
|                                   |                                   |
|                                   | If the TFN has not been           |
|                                   | previously recorded, the customer |
|                                   | must supply a TFN if they do not  |
|                                   | meet an exemption reason outlined |
|                                   | below, or a TFN application form, |
|                                   | prior to grant of ABSTUDY. Refer  |
|                                   | to Tax File Number (TFN) for      |
|                                   | ABSTUDY for further information   |
|                                   |                                   |
|                                   | Customers must also supply        |
|                                   | acceptable identity documentation |
|                                   | when lodging a completed NAT1589  |
|                                   | form                              |
|                                   |                                   |
|                                   | The Australian Taxation Office    |
|                                   | (ATO) will notify both Services   |
|                                   | Australia and the customer of     |
|                                   | their TFN within 28 days of       |
|                                   | lodgement                         |
|                                   |                                   |
|                                   | Customers who do not have to      |
|                                   | supply their TFN details include: |
|                                   |                                   |
|                                   | -   students under 16             |
|                                   | -   recipients of a social        |
|                                   |     security pension or benefit,  |
|                                   |     or a Department of Veterans'  |
|                                   |     Affairs (DVA) pension         |
|                                   | -   a person who is temporarily   |
|                                   |     overseas                      |
|                                   | -   a person who is attending a   |
|                                   |     traditional ceremony when     |
|                                   |     their form is lodged          |
|                                   | -   a person who is at risk from  |
|                                   |     disclosing their TFN details  |
|                                   | -   a person who has lost all of  |
|                                   |     their records from fire or    |
|                                   |     flood damage within six       |
|                                   |     months of claiming ABSTUDY    |
+-----------------------------------+-----------------------------------+
| 8                                 | Declaration (Q40) + Read more ... |
|                                   |                                   |
|                                   | Q40 - Customer declaration        |
|                                   |                                   |
|                                   | All customers must sign the       |
|                                   | declaration in order for their    |
|                                   | claim to be accepted and          |
|                                   | determined. For Schooling A       |
|                                   | claims, the applicant/s are       |
|                                   | required to sign the declaration. |
|                                   | The student does not need to      |
|                                   | sign.                             |
|                                   |                                   |
|                                   | Note: in certain circumstances    |
|                                   | only one parent/guardian          |
|                                   | signature is necessary to assess  |
|                                   | a student's entitlement.          |
+-----------------------------------+-----------------------------------+

|                                   | Australia field officer.          |
