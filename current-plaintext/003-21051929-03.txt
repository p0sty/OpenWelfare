| 1                                 | Check if grant can be reversed +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | The decision to grant Crisis      |
|                                   | Payment (CrP) can be reversed for |
|                                   | the following reasons:            |
|                                   |                                   |
|                                   | a CrP-PRI pre-release customer    |
|                                   | was released after their expected |
|                                   | release date                      |
|                                   |                                   |
|                                   | the customer advises they did not |
|                                   | make the claim for CrP. Someone   |
|                                   | else has claimed using the        |
|                                   | customer’s details                |
|                                   |                                   |
|                                   | staff incorrectly granted a CrP   |
|                                   |                                   |
|                                   | -   for example, CrP was granted  |
|                                   |     on the wrong customer record  |
|                                   |     or the customer is clearly    |
|                                   |     not entitled                  |
|                                   |                                   |
|                                   | an internal review determines the |
|                                   | customer was not entitled to      |
|                                   | receive CrP                       |
|                                   |                                   |
|                                   | Does one of the above reasons     |
|                                   | apply?                            |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, procedure ends here       |
+-----------------------------------+-----------------------------------+
| 2                                 | CrP entitlement to be reversed +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Access to the Crisis Payment      |
|                                   | function in Process Direct and    |
|                                   | the Crisis Payment CRIPAY screen  |
|                                   | in Customer First is limited to:  |
|                                   |                                   |
|                                   | -   service centre and Smart      |
|                                   |     Centre staff at the APS 5     |
|                                   |     level or above                |
|                                   | -   ICT Service Desk staff        |
|                                   | -   payment helpdesk staff        |
|                                   | -   Services Officers who have    |
|                                   |     been given access             |
|                                   |                                   |
|                                   | Does the Service Officer have     |
|                                   | access to the CRIPAY              |
|                                   | screen/function?                  |
|                                   |                                   |
|                                   | Yes, go to Step 4                 |
|                                   |                                   |
|                                   | No, and the Service Officer:      |
|                                   |                                   |
|                                   | -   needs ongoing access, go to   |
|                                   |     Step 3                        |
|                                   | -   does not need ongoing access  |
|                                   |     - DOC the record why the      |
|                                   |     payment must be reversed      |
|                                   |     - escalate following usual    |
|                                   |     procedures. For service       |
|                                   |     centre and Smart Centre       |
|                                   |     staff, this can be to a staff |
|                                   |     member at the APS 5 level or  |
|                                   |     above, for example, Service   |
|                                   |     Support Officer. For Smart    |
|                                   |     Centre staff, see Service     |
|                                   |     Officer role when seeking     |
|                                   |     Technical Peer Support (TPS)  |
|                                   |     in Smart Centres. Procedure   |
|                                   |     ends here                     |
+-----------------------------------+-----------------------------------+
| 3                                 | Access required + Read more ...   |
|                                   |                                   |
|                                   | Send an email to                  |
|                                   | CRISIS.PAYMENT.EMEP.PROGRAM.      |
|                                   |                                   |
|                                   | Subject line: Access required for |
|                                   | CRIPAY screen                     |

|                                   | -   staff logon                   |
|                                   | -   role                          |

|                                   | -   approval from Line Manager    |
|                                   | -   the period access is required |
|                                   |                                   |
|                                   | Access can take several days to   |
|                                   | be granted. If urgent CRIPAY      |
|                                   | action is needed, escalate to     |
|                                   | relevant staff, for example,      |
|                                   | another Prison Liaison Officer    |
|                                   | with access or Roxy Triage staff. |
|                                   |                                   |
|                                   | Procedure ends here until access  |
|                                   | has been granted.                 |

|                                   | In Process Direct:                |
|                                   |                                   |

|                                   |                                   |

|                                   | -   ERR (processing error)        |
|                                   | -   REV (review)                  |
|                                   | -   FRD (fraud)                   |
|                                   |                                   |
|                                   | Select Save                       |
|                                   |                                   |
|                                   | Enter the date and channel and    |
|                                   | select Save                       |
|                                   |                                   |

|                                   |     screen                        |

|                                   | Go to the Assessment Consequences |

|                                   | Go to Step 7.                     |

Pay top up Crisis Payment

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Check if the customer is eligible |
|                                   | for top up payment + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | A customer is eligible for a top  |
|                                   | up payment if they were on a      |
|                                   | partnered rate when assessed for  |
|                                   | Crisis Payment, but are now on a  |
|                                   | single rate on the date of event. |
|                                   |                                   |
|                                   | Was the customer paid at the      |
|                                   | partnered rate of Crisis Payment? |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, procedure ends here       |
+-----------------------------------+-----------------------------------+
| 2                                 | Escalate to ICT + Read more ...   |
|                                   |                                   |
|                                   | The top up payment is made by     |
|                                   | ICT.                              |
|                                   |                                   |
|                                   | Staff need to escalate this via   |
|                                   | Roxy.                             |
|                                   |                                   |
|                                   | Roxy needs the following details: |
|                                   |                                   |
|                                   | -   Date of marital status change |
|                                   | -   Date of Crisis Payment grant  |
|                                   | -   Date of event of extreme      |
|                                   |     circumstance                  |
|                                   | -   Crisis Payment Type           |
|                                   | -   If the Crisis Payment was     |
|                                   |     EXV, the reason for grant     |
|                                   |     (left home, remained in home  |
|                                   |     or perpetrator)               |
|                                   | -   Adjustment amount needed      |
|                                   |     (including the calculations   |
|                                   |     at how this amount was        |
|                                   |     arrived at)                   |
|                                   |                                   |
|                                   | Procedure ends here.              |
