| 1                                 | When do I get paid?               |
|                                   |                                   |
|                                   | A: There are a number of factors  |
|                                   | that may affect the start date of |
|                                   | payments such as waiting and      |
|                                   | exclusion periods. The start date |
|                                   | of payments are assessed on a     |
|                                   | case by case basis.               |
|                                   |                                   |
|                                   | As a general rule job seekers     |
|                                   | must supply all supporting        |
|                                   | documentation and meet the        |
|                                   | qualification and eligibility     |
|                                   | criteria for payment.             |
|                                   |                                   |
|                                   | Job seekers subject to            |
|                                   | RapidConnect requirements         |
|                                   |                                   |
|                                   | Generally job seekers are payable |
|                                   | once they have met their          |
|                                   | RapidConnect requirement by       |
|                                   | either:                           |
|                                   |                                   |
|                                   | -   acknowledge your Job Plan     |
|                                   |     online (if referred to the    |
|                                   |     Workforce Australia Online),  |
|                                   | -   or attend an appointment (if  |
|                                   |     referred to an Employment     |
|                                   |     Service Provider)             |
|                                   |                                   |
|                                   | Failure to meet RapidConnect      |
|                                   | requirements within 28 days of    |
|                                   | the Participation Interview, the  |
|                                   | income support claim is rejected  |
|                                   | or cancelled, (reason code FCJ    |
|                                   | 'Failed to connect with           |
|                                   | Employment Providers').           |
|                                   |                                   |
|                                   | Job seekers not subject to        |
|                                   | RapidConnect requirements         |
|                                   |                                   |
|                                   | Generally, when a job seeker is   |
|                                   | exempt from RapidConnect          |
|                                   | requirements the start day is the |
|                                   | date the claim was submitted with |
|                                   | all supporting documentation if   |
|                                   | they meet the qualification and   |
|                                   | eligibility for payment.          |
|                                   |                                   |
|                                   | For more information, see:        |
|                                   |                                   |
|                                   | -   Calculating the start day     |
|                                   |     general rule                  |
|                                   | -   RapidConnect                  |

Types of mutual obligation requirements - FAQ

Table 5

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | What are mutual obligation        |
|                                   | requirements?                     |
|                                   |                                   |
|                                   | Mutual obligation requirements    |
|                                   | are designed to ensure that       |
|                                   | unemployed people receiving       |
|                                   | activity tested income support    |
|                                   | payments are actively looking for |
|                                   | work and are participating in     |
|                                   | activities that will help them    |
|                                   | into employment.                  |
|                                   |                                   |
|                                   | When claiming a payment, job      |
|                                   | seekers are generally required to |
|                                   | engage with employment services   |
|                                   | before receiving payment. Job     |
|                                   | seekers will then be required to  |
|                                   | enter into a Job Plan which       |
|                                   | outlines all of a job seeker's    |
|                                   | mutual obligation requirements.   |
|                                   | These requirements will generally |
|                                   | include looking for and accepting |
|                                   | suitable work, attending          |
|                                   | appointments with their           |
|                                   | employment services provider,     |
|                                   | attending all job interviews and  |
|                                   | undertaking other suitable        |
|                                   | activities.                       |
|                                   |                                   |
|                                   | Job seekers' mutual obligation    |
|                                   | requirements are generally        |
|                                   | determined by the following       |
|                                   | factors:                          |
|                                   |                                   |
|                                   | -   age                           |
|                                   | -   assessed work capacity        |
|                                   | -   the primary responsibility    |
|                                   |     for the care of a child.      |
|                                   |                                   |
|                                   | For more information, see Mutual  |
|                                   | obligation requirements.          |
+-----------------------------------+-----------------------------------+
| 2                                 | How many hours am I required to   |
|                                   | do each week?                     |
|                                   |                                   |
|                                   | Job seekers with full-time (30+   |
|                                   | hours a week) mutual obligation   |
|                                   | requirements should be:           |
|                                   |                                   |
|                                   | -   looking for work on a         |
|                                   |     full-time basis, and          |
|                                   | -   actively addressing any       |
|                                   |     circumstances that may affect |
|                                   |     their capacity to undertake   |
|                                   |     paid work                     |
|                                   |                                   |
|                                   | The following job seekers have    |
|                                   | part-time (15 hours a week)       |
|                                   | mutual obligation requirements:   |
|                                   |                                   |
|                                   | -   Principal carer parents, once |
|                                   |     the youngest child in their   |
|                                   |     care turns 6 years of age     |
|                                   | -   Job seekers with a partial    |
|                                   |     capacity to work (PCW) of 15  |
|                                   |     to 29 hours per week          |
|                                   | -   Mature age job seekers, 55    |
|                                   |     years and over                |
|                                   |                                   |
|                                   | For more information, see         |
|                                   | Applying mutual obligation        |
|                                   | requirements.                     |

|                                   | allowed, in recognition of their  |
|                                   | family care responsibilities.     |
|                                   |                                   |
|                                   | This means mutual obligations,    |
|                                   | appointments and other            |
|                                   | requirements should be scheduled  |
|                                   | to occur during school hours only |
|                                   | (generally between 9 am and 3 pm  |
|                                   | during school terms), unless      |
|                                   | otherwise requested or agreed to  |
|                                   | by the principal carer. This      |
|                                   | includes participation in         |
|                                   | employment services and           |
|                                   | requirements negotiated by        |
|                                   | providers.                        |
|                                   |                                   |
|                                   | For more information, see Mutual  |
|                                   | obligation requirements for       |
|                                   | principal carers.                 |
+-----------------------------------+-----------------------------------+
| 2                                 | What do I do in school holidays?  |
|                                   |                                   |
|                                   | Principal carer parents are not   |
|                                   | required to meet any requirements |
|                                   | for the fortnight during which    |
|                                   | the Christmas Day public holiday  |
|                                   | falls. However, they are still    |
|                                   | required to lodge their report to |
|                                   | Services Australia for this       |
|                                   | period in order to report any     |
|                                   | earnings and stimulate their      |
|                                   | payment.                          |
|                                   |                                   |
|                                   | During school term holidays, job  |
|                                   | seekers are to discuss their      |
|                                   | individual circumstances with     |
|                                   | their provider.                   |
|                                   |                                   |
|                                   | For more information, see Mutual  |
|                                   | obligation requirements for       |
|                                   | principal carers.                 |

|                                   |                                   |
|                                   | -   Job seekers will be required  |
|                                   |     to look for work, or          |

Job Plans - FAQ

Table 11

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | What's a Job Plan?                |
|                                   |                                   |
|                                   | A Job Plan is an agreement you    |
|                                   | enter into, it includes all the   |
|                                   | mutual obligation requirements    |
|                                   | you agree to do in order to       |
|                                   | receive payments. A Job Plan      |
|                                   | records the requirements          |
|                                   | negotiated between a job seeker   |
|                                   | and their Employment Services     |
|                                   | Provider or Services Australia    |
|                                   | that will satisfy their           |
|                                   | obligation requirements.          |
|                                   |                                   |
|                                   | For more information, see Job     |
|                                   | Plans.                            |
+-----------------------------------+-----------------------------------+
| 2                                 | How do I complete a Job plan?     |
|                                   |                                   |
|                                   | Generally job seekers can         |
|                                   | complete their Job Plan online.   |
|                                   |                                   |
|                                   | Provider Managed job seekers      |
|                                   |                                   |
|                                   | Job seekers will negotiate their  |
|                                   | Job Plan with their Employment    |
|                                   | Services Provider if they are     |
|                                   | provider managed. For more        |
|                                   | information, see How to use the   |
|                                   | Workforce Australia website.      |
|                                   |                                   |
|                                   | Centrelink managed job seekers    |
|                                   |                                   |
|                                   | Job Plans may be negotiated by    |
|                                   | suitably skilled Service Officers |
|                                   | either in person or over the      |
|                                   | phone where all necessary         |
|                                   | documentation and/or information  |
|                                   | is available. Job seekers must    |
|                                   | formally agree with the Job Plan  |
|                                   | either by signing a printed copy, |
|                                   | or by using their online account  |
|                                   | (not including the Centrelink     |
|                                   | app) to provide agreement. Job    |
|                                   | Plans cannot be finalised by      |
|                                   | verbal agreement.                 |
|                                   |                                   |
|                                   | For more information, see         |
|                                   | Negotiating Job Plans.            |
+-----------------------------------+-----------------------------------+
| 3                                 | What sort of activities are in a  |
|                                   | Job Plan?                         |
|                                   |                                   |
|                                   | To meet mutual obligation         |
|                                   | requirements, job seekers must    |
|                                   | participate in a range of         |
|                                   | activities aimed at improving     |
|                                   | their employment prospects and    |
|                                   | targeted towards a job seeker's   |
|                                   | individual needs. Compulsory and  |
|                                   | voluntary activities, or a        |
|                                   | combination of both, can be       |
|                                   | included in the Job Plan.         |
|                                   |                                   |
|                                   | Some examples of types of         |
|                                   | activities in a Job Plan include: |
|                                   |                                   |
|                                   | -   Attend Appointments           |
|                                   | -   Undertake Paid Work           |
|                                   | -   Accept referrals for jobs     |
|                                   | -   Self Employment               |
|                                   | -   Adult Migrant English         |
|                                   |     Programme                     |
|                                   | -   Undertake studies             |
|                                   | -   Provide a copy of jobs        |
|                                   |     applied for each month        |
+-----------------------------------+-----------------------------------+
| 4                                 | If I have questions about my Job  |
|                                   | Plan, who do I speak to?          |
|                                   |                                   |
|                                   | If a provider managed/Workforce   |
|                                   | Australia Online job seeker       |
|                                   | enquires about the mutual         |
|                                   | obligation requirements contained |
|                                   | in their Job Plan or they require |
|                                   | a new Job Plan, refer them to     |
|                                   | their Employment Services         |
|                                   | Provider or the Digital Services  |
|                                   | Contact Centre for Workforce      |
|                                   | Australia Online job seekers.     |
+-----------------------------------+-----------------------------------+
| 5                                 | If I don't agree with the         |
|                                   | requirements in my Job Plan what  |
|                                   | do I need to do?                  |

|                                   | If a provider managed job seeker  |
|                                   | wishes to lodge a complaint       |
|                                   | regarding their provider or their |

|                                   | refer the job seeker to their     |
|                                   | Employment Services Provider.     |

Reporting - FAQ

Table 12

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Why do I have to report?          |
|                                   |                                   |
|                                   | Job seekers are required to       |
|                                   | report so the correct entitlement |
|                                   | can be determined based on their  |
|                                   | and/or their partner's employment |
|                                   | income. Reporting captures any    |
|                                   | changes to a customer's           |
|                                   | circumstances (such as change of  |
|                                   | address, phone number etc.).      |
|                                   |                                   |
|                                   | Regular reporting also confirms   |
|                                   | job seekers are continuing to     |
|                                   | meet their mutual obligation      |
|                                   | requirements and remain eligible  |
|                                   | for payments.                     |
+-----------------------------------+-----------------------------------+
| 2                                 | When do I have to report?         |
|                                   |                                   |
|                                   | Most job seekers claiming for or  |
|                                   | receiving JSP or YA (job seeker)  |
|                                   | with mutual obligation            |
|                                   | requirements are placed on two    |
|                                   | weekly reporting frequency.       |
|                                   |                                   |
|                                   | Job seekers getting payments and  |
|                                   | have provided their mobile number |
|                                   | or email address, are sent a      |
|                                   | reminder via email or Short       |
|                                   | Message Service (SMS) the day     |
|                                   | before they are due to report.    |
+-----------------------------------+-----------------------------------+
| 3                                 | How do I report?                  |
|                                   |                                   |
|                                   | Job seekers can report through    |
|                                   | self service (online accounts,    |
|                                   | Express Plus Centrelink mobile    |
|                                   | app and phone). Job seekers can:  |
|                                   |                                   |
|                                   | -   view a summary of their due   |
|                                   |     to report dates and reporting |
|                                   |     periods in their online       |
|                                   |     account. This displays all    |
|                                   |     reporting periods within 12   |
|                                   |     weeks of the date viewed      |
|                                   | -   access this summary by        |
|                                   |     selecting Income and Assets > |
|                                   |     Employment Income > Next      |
|                                   |     reporting periods from the    |
|                                   |     menu                          |
+-----------------------------------+-----------------------------------+
| 4                                 | What if I start working what do I |
|                                   | report?                           |
|                                   |                                   |
|                                   | Job seekers need to report the    |
|                                   | amount, hours and the period for  |
|                                   | which they were paid. They can do |
|                                   | this via their self service       |
|                                   | (online accounts, Express Plus    |
|                                   | Centrelink mobile app and phone)  |
|                                   | on their reporting day.           |
|                                   |                                   |
|                                   | For more information, see         |
|                                   | Reporting requirements for job    |
|                                   | seekers.                          |
