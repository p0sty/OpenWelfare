|                                   |     screen if needed              |

|                                   |                                   |

|                                   | -   select the Validate button    |

|                                   |     Step 3                        |
|                                   |                                   |

|                                   | -   contact the applicant to      |

|                                   | -   copy and paste and update the |
|                                   |     text ‘No organisation in EHP  |
|                                   |     matches organisation details  |
|                                   |     in application’ into the      |
|                                   |     Comments box                  |
|                                   | -   procedure ends here           |

| 3                                 | Organisation Updates tab + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | If there are issues to process,   |
|                                   | cross check any issues using the  |
|                                   | second screen.                    |
|                                   |                                   |
|                                   | Evidence of authority to commit   |
|                                   | and change of ownership:          |
|                                   |                                   |
|                                   | The applicant must provide both   |
|                                   | evidence of authority to commit   |
|                                   | and evidence of change of         |
|                                   | ownership.                        |
|                                   |                                   |
|                                   | For evidence of authority to      |
|                                   | commit, the online form does an   |
|                                   | upfront check to see if the       |
|                                   | applicant is on the               |
|                                   | organisation’s ABR record as a    |
|                                   | person with authority to commit.  |
|                                   |                                   |
|                                   | If they are:                      |
|                                   |                                   |
|                                   | -   able to commit, the applicant |
|                                   |     will only need to provide     |
|                                   |     document(s) to support the    |
|                                   |     evidence of change of         |
|                                   |     ownership                     |
|                                   | -   not able to commit, the       |
|                                   |     applicant will need to        |
|                                   |     provide both document(s) to   |
|                                   |     support the evidence of       |
|                                   |     change of ownership and       |
|                                   |     evidence of authority to      |
|                                   |     commit                        |
|                                   |                                   |
|                                   | If either of the above evidence   |
|                                   | has not been provided or is       |
|                                   | incorrect:                        |
|                                   |                                   |
|                                   | -   contact the applicant and     |
|                                   |     request the applicant to      |
|                                   |     upload additional evidence of |
|                                   |     authority to commit           |
|                                   |     documentation and/or change   |
|                                   |     of ownership documentation    |
|                                   |     for the new ABN/ACN           |
|                                   | -   copy and paste and update the |
|                                   |     text ‘Applicant needs to      |
|                                   |     provide evidence of authority |
|                                   |     to commit documentation       |
|                                   |     and/or change of ownership    |
|                                   |     documentation for the new     |
|                                   |     ABN/ACN’ into the Comments    |
|                                   |     box                           |
|                                   | -   place application on hold,    |
|                                   |     select the Hold radio button  |
|                                   | -   procedure ends here           |
|                                   |                                   |
|                                   | If correct evidence of authority  |
|                                   | to commit documentation and/or    |
|                                   | change of ownership documentation |
|                                   | has been supplied, go to Step 4.  |

|                                   |     updating all of the networked |
