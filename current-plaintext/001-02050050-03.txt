| 4                                 | Educational Attainment + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | This factor recognises the job    |
|                                   | seeker's highest level of         |
|                                   | educational attainment. Job       |
|                                   | seekers with greater levels of    |
|                                   | educational attainment generally  |
|                                   | have a greater advantage in the   |
|                                   | labour market.                    |
|                                   |                                   |
|                                   | Weightings allocated for          |
|                                   | Education Attainment are based on |
|                                   | the responses to the questions    |
|                                   | asked on the Education page in    |
|                                   | the Online Participation Workflow |
|                                   | (OPW) and the Looking For Work    |
|                                   | workflow.                         |
|                                   |                                   |
|                                   | The level of educational          |
|                                   | attainment is established through |
|                                   | 2 questions relating to:          |
|                                   |                                   |
|                                   | -   level of primary or secondary |
|                                   |     school education reached and  |
|                                   | -   further education such as     |
|                                   |     accredited certificates,      |
|                                   |     diplomas, degrees and post    |
|                                   |     graduate study                |
|                                   |                                   |
|                                   | The available responses to select |
|                                   | from in the drop down box are:    |
|                                   |                                   |
|                                   | -   Year 12/13 or equivalent (for |
|                                   |     example, Form 6 or            |
|                                   |     Matriculation)                |
|                                   | -   Year 11 or equivalent (for    |
|                                   |     example, Form 5)              |
|                                   | -   Year 10 or equivalent (for    |
|                                   |     example, Form 4)              |
|                                   | -   completed primary school but  |
|                                   |     less than Year 10 or          |
|                                   |     equivalent                    |
|                                   | -   special school/special        |
|                                   |     support unit in school or     |
|                                   |     equivalent                    |
|                                   | -   primary school or less or     |
|                                   |     equivalent                    |
|                                   | -   did not go to school          |
|                                   |                                   |
|                                   | The next question is have you     |
|                                   | completed any other               |
|                                   | qualification(s)? If the answer   |
|                                   | is 'Yes', a further question is   |
|                                   | asked:                            |
|                                   |                                   |
|                                   | What are they? What               |
|                                   | qualification(s) have you         |
|                                   | completed?                        |
|                                   |                                   |
|                                   | -   Doctoral Degree or equivalent |
|                                   | -   Masters Degree or equivalent  |
|                                   | -   Bachelor Degree or equivalent |
|                                   | -   Associate Diploma, Advanced   |
|                                   |     Diploma, Associate Degree or  |
|                                   |     equivalent                    |
|                                   | -   Tradesperson's qualifications |
|                                   |     (Certificate III or IV or     |
|                                   |     equivalent)                   |
|                                   | -   Other non-trade Vocational    |
|                                   |     Education and Training        |
|                                   |     (Certificate II)              |
|                                   | -   Other non-trade Vocational    |
|                                   |     Education and Training        |
|                                   |     Certificate III or IV         |
|                                   | -   Vocational Education and      |
|                                   |     Training Centre Certificate I |
|                                   |     or other Industry-specific    |
|                                   |     licence or ticket             |
|                                   | -   Course run by private or      |
|                                   |     community organisation        |
|                                   | -   TAFE qualifications at        |
|                                   |     certificate level include     |
|                                   |     qualifications recognised at  |
|                                   |     Certificates I, II, III or IV |
|                                   |     in the Australian             |
|                                   |     Qualifications Framework      |
|                                   |     (AQF) such as Vocational      |
|                                   |     Education and Training        |
|                                   |     Certificate I completed at    |
|                                   |     school or since leaving       |
|                                   |     school                        |
|                                   |                                   |
|                                   | Associate Diplomas, Diplomas and  |
|                                   | Advanced Diplomas recognised in   |
|                                   | the AQF. Generally a minimum of 2 |
|                                   | years full-time study is required |
|                                   | to complete these qualifications, |
|                                   | however it is not always the      |
|                                   | case.                             |
|                                   |                                   |
|                                   | Introduction of the AQF was       |
|                                   | phased in from 1995. Job seekers  |
|                                   | who completed a qualification     |
|                                   | prior to that date may need to    |
|                                   | match their qualification to 1 of |
|                                   | the responses listed.             |
|                                   |                                   |
|                                   | Responses should include          |
|                                   | education completed in Australia  |
|                                   | or the Australian equivalent of   |
|                                   | educational qualifications gained |
|                                   | overseas.                         |

| 5                                 | Vocational Qualification + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | This factor recognises the        |
|                                   | significant difference in         |
|                                   | employability between job seekers |
|                                   | with work qualifications and      |
|                                   | those without.                    |
|                                   |                                   |
|                                   | Points allocated for vocational   |
|                                   | qualifications are based on       |
|                                   | responses to questions on the     |
|                                   | Education page in the Online      |
|                                   | Participation Workflow (OPW) and  |
|                                   | the Looking for Work workflow.    |
|                                   |                                   |
|                                   | These questions gather            |
|                                   | information about the job         |
|                                   | seeker's work qualifications and  |
|                                   | whether these qualifications are  |
|                                   | relevant in the labour market.    |
|                                   | They can also relate to a formal  |
|                                   | qualification which allows a      |
|                                   | person to work in a specific      |
|                                   | occupation.                       |
|                                   |                                   |
|                                   | The purpose of these questions is |
|                                   | to identify recognised            |
|                                   | qualifications which offer an     |
|                                   | advantage to the job seeker in    |
|                                   | obtaining employment. Where a job |
|                                   | seeker has 2 or more              |
|                                   | qualifications they should        |
|                                   | respond to the question with      |
|                                   | details of the qualification      |
|                                   | which provides the greatest       |
|                                   | employment advantage.             |
|                                   |                                   |
|                                   | What is considered vocational?    |
|                                   |                                   |
|                                   | Services Australia staff should   |
|                                   | generally accept the job seeker's |
|                                   | own judgement about the           |
|                                   | vocational nature of their        |
|                                   | qualification, as they generally  |
|                                   | know if their qualification is    |
|                                   | required for a specific job. If   |
|                                   | they are unsure about the         |
|                                   | vocational nature of a particular |
|                                   | qualification, the following      |
|                                   | examples should be used as a      |
|                                   | guide:                            |
|                                   |                                   |
|                                   | -   the added value of            |
|                                   |     educational qualifications    |
|                                   |     which have a vocational       |
|                                   |     orientation (for example, a   |
|                                   |     nursing or architecture       |
|                                   |     degree compared to a general  |
|                                   |     arts degree or a degree in    |
|                                   |     philosophy)                   |
|                                   | -   trade qualifications required |
|                                   |     for particular occupations    |
|                                   |     (for example, plumbing and    |
|                                   |     electrical trade              |
|                                   |     certificates)                 |
|                                   | -   other non-educational         |
|                                   |     qualifications required for   |
|                                   |     particular occupations for    |
|                                   |     example:                      |
|                                   | -   special licences (including   |
|                                   |     bus, truck and fork-lift),    |
|                                   |     this does not include a       |
|                                   |     standard car or motorcycle    |
|                                   |     driver's licence as a formal  |
|                                   |     qualification                 |
|                                   | -   tickets (for example,         |
|                                   |     seaman's ticket) and other    |
|                                   |     technical qualifications      |
|                                   | -   Occupational Health and       |
|                                   |     Safety (OH&S), tickets should |
|                                   |     not be classified as          |
|                                   |     vocational or work            |
|                                   |     qualifications                |
|                                   | -   short courses that are        |
|                                   |     formally accredited or        |
|                                   |     generally recognised by       |
|                                   |     employers and constitute the  |
|                                   |     basic prerequisites for entry |
|                                   |     to a particular occupation    |
|                                   |                                   |
|                                   | Unusable Qualifications           |
|                                   |                                   |
|                                   | If the job seeker answers 'Yes'   |
|                                   | to having a vocational            |
|                                   | qualification, they are next      |
|                                   | asked if there is some reason     |
|                                   | which prevents them using the     |
|                                   | qualification or working in the   |
|                                   | industry which recognises it.     |
|                                   |                                   |
|                                   | There are 5 responses to why a    |
|                                   | job seeker cannot still use their |
|                                   | qualification. Points are         |
|                                   | allocated if the job seeker falls |
|                                   | into any of these categories.     |
|                                   | Again, Services Australia staff   |
|                                   | should rely on the job seeker's   |
|                                   | own understanding of their        |
|                                   | circumstances. If they require    |
|                                   | any advice, the Service Officer   |
|                                   | may need to make a judgement      |
|                                   | based on a discussion with the    |
|                                   | job seeker and their knowledge of |
|                                   | local employment conditions.      |
|                                   |                                   |
|                                   | Note: keep in mind that job       |
|                                   | seekers with overseas             |
|                                   | qualifications may require        |
|                                   | additional assistance as some     |
|                                   | overseas qualifications are not   |
|                                   | recognised in Australia.          |

| 1                                 | English Proficiency + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | English Proficiency factor        |
|                                   | recognises the problems that may  |
|                                   | be faced by job seekers who have  |
|                                   | difficulty with spoken and/or     |
|                                   | written English.                  |
|                                   |                                   |
|                                   | Services Australia does not       |
|                                   | purchase professional assessments |
|                                   | for job seekers who exhibit       |
|                                   | difficulty speaking, reading or   |
|                                   | writing English. For the JSCI, a  |
|                                   | job seeker's English proficiency  |
|                                   | is based on their own assessment  |
|                                   | of language and literacy skills   |
|                                   | in conjunction with the           |
|                                   | observations of the interviewing  |
|                                   | officer.                          |
|                                   |                                   |
|                                   | The JSCI uses a combination of    |
|                                   | the information recorded to       |
|                                   | allocate points for English       |
|                                   | proficiency levels.               |
|                                   |                                   |
|                                   | The level of English skill        |
|                                   | recorded for the JSCI should be   |
|                                   | discussed with the job seeker.    |
|                                   | The job seeker must agree with    |
|                                   | any assessment of their English   |
|                                   | skill before it is recorded. If   |
|                                   | the job seeker claims good        |
|                                   | literacy, but the interviewing    |
|                                   | officer believes the person has   |
|                                   | minimal literacy skills, the job  |
|                                   | seeker's opinion must be          |
|                                   | recorded, unless an alternative   |
|                                   | is agreed. Where the Service      |
|                                   | Officer considers that the job    |
|                                   | seeker may benefit from Skills    |
|                                   | for Education and Employment      |
|                                   | (SEE) despite their responses to  |
|                                   | the English proficiency           |
|                                   | questions, the Service Officer    |
|                                   | may refer the job seeker to SEE   |
|                                   | after discussion of the benefits  |
|                                   | with the job seeker.              |
|                                   |                                   |
|                                   | Job seekers who are to be         |
|                                   | referred to SEE must be referred  |
|                                   | for an ESAt at the same time.     |
|                                   |                                   |
|                                   | Language and literacy             |
|                                   | difficulties for sight impaired   |
|                                   | job seekers                       |
|                                   |                                   |
|                                   | Some sign language users and      |
|                                   | sight impaired job seekers may    |
|                                   | have language and literacy        |
|                                   | difficulties, especially in the   |
|                                   | work place. Where these job       |
|                                   | seekers self-identify as having   |
|                                   | language and literacy             |
|                                   | difficulties, points are          |
|                                   | allocated for their response to   |
|                                   | this question.                    |
|                                   |                                   |
|                                   | The JSCI is also populated with   |
|                                   | information on whether the job    |
|                                   | seeker has attended any English   |
|                                   | language courses within the last  |
|                                   | 6 months.                         |
|                                   |                                   |
|                                   | Remote job seekers                |
|                                   |                                   |
|                                   | Job seekers living in remote      |
|                                   | locations may lodge forms by mail |
|                                   | or register over the phone. This  |
|                                   | is the least reliable situation   |
|                                   | to judge a job seeker's English   |
|                                   | language skills. Telephone        |
|                                   | interviews should only be used    |
|                                   | when it is impractical for the    |
|                                   | job seeker to attend an           |
|                                   | interview.                        |
|                                   |                                   |
|                                   | Language and literacy levels      |
|                                   | should be reassessed at the first |
|                                   | opportunity for a face-to-face    |
|                                   | interview. In a telephone         |
|                                   | interview, the job seeker's       |
|                                   | ability to communicate            |
|                                   | effectively over the phone may    |
|                                   | provide the interviewing officer  |
|                                   | with an opinion of their ability  |
|                                   | to speak and understand English.  |
|                                   |                                   |
|                                   | Record the job seeker's Read      |
|                                   | English and Write English         |
|                                   | responses as 'Very Limited' if    |
|                                   | the job seeker responds 'yes' to  |
|                                   | any question about requiring help |
|                                   | to complete the form or having    |
|                                   | difficulty with reading or        |
|                                   | writing.                          |
|                                   |                                   |
|                                   | JSCI interviews should occur      |
|                                   | face-to-face wherever possible.   |
|                                   |                                   |
|                                   | This increases the chance of      |
|                                   | Services Australia staff          |
|                                   | detecting a language or literacy  |
|                                   | barrier which could otherwise     |
|                                   | remain hidden. People with        |
|                                   | literacy barriers frequently make |
|                                   | excuses when called upon to read  |
|                                   | or fill in a form. Typical        |
|                                   | excuses are: 'I forgot my         |
|                                   | glasses', 'my wrist is sore', 'I  |
|                                   | forgot my pen' or 'I was in a     |
|                                   | hurry'.                           |
|                                   |                                   |
|                                   | By observing and listening        |
|                                   | carefully to the job seeker       |
|                                   | during the interview, the         |
|                                   | interviewing officer can form an  |
|                                   | opinion on the job seeker's       |
|                                   | command of spoken English. See    |
|                                   | Eligibility for Skills for        |
|                                   | Education and Employment (SEE).   |

| 2                                 | Country of Birth + Read more ...  |
|                                   |                                   |
|                                   | Information for this factor is    |
|                                   | populated from information held   |
|                                   | in the Centrelink system. No      |
|                                   | questions are needed to be asked  |
|                                   | during the JSCI interview for     |
|                                   | this factor.                      |
|                                   |                                   |
|                                   | When explaining the job seeker's  |
|                                   | option to provide this            |
|                                   | information, it should be         |
|                                   | explained that the more           |
|                                   | information the job seeker        |
|                                   | provides, the better it is to     |
|                                   | determine the job seeker's        |
|                                   | employment needs.                 |
|                                   |                                   |
|                                   | People coming to Australia may    |
|                                   | face any number of difficulties   |
|                                   | entering the job market,          |
|                                   | including:                        |
|                                   |                                   |
|                                   | -   lack of knowledge about, and  |
|                                   |     attachment to, the Australian |
|                                   |     labour market                 |
|                                   | -   lack of networks and/or       |
|                                   |     contacts within the           |
|                                   |     Australian labour market      |
|                                   | -   cultural differences between  |
|                                   |     their country of origin and   |
|                                   |     Australia                     |
|                                   |                                   |
|                                   | The unemployment rates, in        |
|                                   | Australia, of job seekers from    |
|                                   | various countries have been       |
|                                   | examined to see which countries   |
|                                   | of origin result in the greatest  |
|                                   | disadvantage in the labour        |
|                                   | market. The points allocated      |
|                                   | reflect the employment experience |
|                                   | of migrants from different        |
|                                   | countries in the Australian       |
|                                   | labour market.                    |

| 3                                 | Indigenous Status + Read more ... |
|                                   |                                   |
|                                   | The question relating to this     |
|                                   | factor is on the Personal Details |
|                                   | page in the Online Participation  |
|                                   | Workflow (OPW) and the Looking    |
|                                   | For Work workflow. This question  |
|                                   | is voluntary.                     |
|                                   |                                   |
|                                   | Weightings allocated to           |
|                                   | Indigenous status recognise the   |
|                                   | employment disadvantage faced by  |
|                                   | Indigenous Australians.           |
|                                   |                                   |
|                                   | Australian born job seekers only  |
|                                   | are asked whether they identify   |
|                                   | as an Aboriginal Australian or a  |
|                                   | Torres Strait Islander, and if    |
|                                   | they do, are they Aboriginal or a |
|                                   | Torres Strait Islander. Job       |
|                                   | seekers can select more than 1    |
|                                   | answer or they can choose not to  |
|                                   | answer.                           |
|                                   |                                   |
|                                   | The Privacy Protocol regarding    |
|                                   | the collection of personal        |
|                                   | information of Aboriginal and     |
|                                   | Torres Strait Islander people     |
|                                   | issued by the Information         |
|                                   | Commissioner clarifies the rules  |
|                                   | relating to the collection of     |
|                                   | personal information, set out in  |
|                                   | the Privacy ACT 1988, regarding   |
|                                   | Indigenous status. Culturally     |
|                                   | sensitive information should only |
|                                   | be asked for on a voluntary basis |
|                                   | unless it is absolutely essential |
|                                   | for agencies statutory functions. |
|                                   | Questions about a person's        |
|                                   | Indigenous status should only be  |
|                                   | asked where the job seeker has    |
|                                   | the option of not responding. The |
|                                   | option, 'do not wish to answer'   |
|                                   | is included to ensure a job       |
|                                   | seeker does not provide           |
|                                   | information which they are not    |
|                                   | required to or do not wish to     |
|                                   | provide.                          |
|                                   |                                   |
|                                   | Aboriginal and Torres Strait      |
|                                   | Islander people                   |
|                                   |                                   |
|                                   | Disadvantages faced are related   |
|                                   | to:                               |
|                                   |                                   |
|                                   | -   standards of health:          |
|                                   |     Aboriginal and Torres Strait  |
|                                   |     Islander people have a        |
|                                   |     considerably lower standard   |
|                                   |     of general health compared    |
|                                   |     with other Australians        |
|                                   | -   cultural requirements: this   |
|                                   |     includes elements such as     |
|                                   |     observance of cultural        |
|                                   |     occasions and practices and   |
|                                   |     the need for absences from    |
|                                   |     employment to attend          |
|                                   |     culturally significant        |
|                                   |     occasions. These requirements |
|                                   |     can vary depending on         |
|                                   |     lifestyle and may be more     |
|                                   |     significant to Aboriginal and |
|                                   |     Torres Strait Islander people |
|                                   |     living traditional or         |
|                                   |     semi-traditional lifestyles   |

|                                   | conviction provisions of the      |
|                                   | Crimes Act 1914, as to when this  |
|                                   | information should be collected   |
|                                   | and how it should be collected.   |
|                                   | It is advisable that questions of |
|                                   | this nature are only asked where  |
|                                   | the job seeker has the option of  |
|                                   | not responding.                   |
|                                   |                                   |
|                                   | The option 'Do not wish to        |
|                                   | answer' is included to ensure a   |
|                                   | job seeker does not provide       |
|                                   | information which they are not    |
|                                   | required or do not want to        |
|                                   | provide.                          |
|                                   |                                   |
|                                   | It is important to inform the job |
|                                   | seeker that Services Australia is |
|                                   | not seeking information about the |
|                                   | nature of their offence, but only |
|                                   | if they have a criminal           |
|                                   | conviction, and whether a         |
|                                   | custodial sentence was imposed.   |
|                                   |                                   |
|                                   | The job seeker must be advised    |
|                                   | that the more information they    |
|                                   | provide about any possible        |
|                                   | barrier to employment, the better |
|                                   | Services Australia is able to     |
|                                   | determine the most appropriate    |
|                                   | assistance for the job seeker.    |
|                                   | The job seeker should be assured  |
|                                   | that this information is          |
|                                   | confidential. It is not released  |
|                                   | to employers, or anybody else,    |
|                                   | without the job seeker's express  |
|                                   | permission. While job seekers     |
|                                   | should understand why this        |
|                                   | question is asked, they should    |
|                                   | not feel pressured to release     |
|                                   | this information or to discuss    |
|                                   | it.                               |
|                                   |                                   |
|                                   | Custodial Sentences               |
|                                   |                                   |
|                                   | There are a variety of sentencing |
|                                   | options available to the courts   |
|                                   | which involve a restriction on    |
|                                   | the individual's freedom. Job     |
|                                   | seekers may be unsure if the      |
|                                   | sentence they have served         |
|                                   | represents a custodial sentence.  |
|                                   | View the guide to identifying     |
|                                   | custodial sentences.              |
|                                   |                                   |
|                                   | Non-Custodial Sentences           |
|                                   |                                   |
|                                   | These convictions should be       |
|                                   | recorded as criminal offence with |
|                                   | no custodial sentence served.     |
|                                   |                                   |
|                                   | However Judges and magistrates    |
|                                   | can, in some circumstances,       |
|                                   | convict and fine an individual,   |
|                                   | and order that the conviction not |
|                                   | be recorded against the           |
|                                   | individual. If the job seeker     |
|                                   | reports that this is the case, a  |
|                                   | conviction should not be recorded |
|                                   | for the job seeker for this       |
|                                   | question.                         |
|                                   |                                   |
|                                   | A person placed on a good         |
|                                   | behaviour bond should only be     |
|                                   | recorded as a response if a       |
|                                   | conviction was recorded.          |
