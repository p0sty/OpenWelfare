Seed becomes a network of another seed or network moves to a different hierarchy (complex change)

Table 2: This table describes the required actions of the involved

organisation's hierarchy, or when a network of one hierarchy is bought
by an organisation under a different hierarchy.

| Role                              | Action                            |
+-----------------------------------+-----------------------------------+

|                                   | HPOS. This will include providing |

|                                   | evidence to support the new       |
|                                   | ownership arrangements            |
|                                   |                                   |
|                                   | a letter stating:                 |
|                                   |                                   |
|                                   | -   the gaining RO has the        |
|                                   |     authority to act on behalf of |
|                                   |     the incoming organisation     |
|                                   | -   the name and Healthcare       |
|                                   |     Provider Identifier -         |
|                                   |     Organisation (HPI-O) of where |
|                                   |     the incoming organisation is  |
|                                   |     moving to                     |
|                                   | -   the names and HPI-Os of any   |
|                                   |     networks of the incoming      |
|                                   |     organisation, and how the     |
|                                   |     hierarchy should be           |
|                                   |     structured                    |
+-----------------------------------+-----------------------------------+
| Where there is a new Organisation |                                   |
| Maintenance Officer (OMO) for the |                                   |
| incoming organisation, either:    |                                   |
|                                   |                                   |

| -   an OMO higher in the          |                                   |
|     hierarchy to the incoming     |                                   |
|     organisation                  |                                   |

Network becomes a seed organisation (complex change)

Table 3: This table describes the required actions of the new seed
organisation, losing seed organisation and HI Service Officer when a
network is bought out by a company and it becomes a seed in its own
right.

+-----------------------------------+-----------------------------------+
| Role                              | Action                            |
+-----------------------------------+-----------------------------------+
| New seed organisation             | Apply online using the Healthcare |
|                                   | Identifiers – Register Seed       |
|                                   | Organisation application process  |
|                                   | in HPOS. This will include        |
|                                   | providing relevant supporting     |
|                                   | documentation, including:         |
|                                   |                                   |
|                                   | a letter stating:                 |
|                                   |                                   |
|                                   | -   the names and Healthcare      |
|                                   |     Provider                      |
|                                   |     Identifier-Organisations      |
|                                   |     (HPI-O) of any networks that  |
|                                   |     may move with the new seed    |
|                                   | -   any changes in the hierarchy  |
|                                   |     structure                     |
+-----------------------------------+-----------------------------------+
| Losing seed organisation          | Provide a letter agreeing to the  |
|                                   | loss of the networks moving to    |
|                                   | the new seed organisation.        |
+-----------------------------------+-----------------------------------+
| Service Officer                   | Where an applicant uses the       |
|                                   | online Healthcare Identifiers -   |
|                                   | Register Seed Organisation form   |
|                                   | in HPOS:                          |
|                                   |                                   |
|                                   | -   Follow the Process online     |
|                                   |     registrations of HPI-O        |
|                                   |     records for seed              |
|                                   |     organisations                 |
|                                   |                                   |
|                                   | Where a manual Application to     |
|                                   | register a Seed Organisation form |
|                                   | (HW018) is provided:              |
|                                   |                                   |
|                                   | -   Ensure all necessary          |
|                                   |     documentation has been        |
|                                   |     provided and follow up with   |
|                                   |     the applicant if it hasn’t.   |
|                                   |     Note: if any of the networks  |
|                                   |     are not transferring with the |
|                                   |     organisation, contact Tier 3  |
|                                   |     for advice before processing  |
|                                   | -   Follow the Restructure        |
|                                   |     Healthcare Provider           |
|                                   |     Identifier - Organisation     |
|                                   |     (HPI-O) in Healthcare         |
|                                   |     Identifiers (HI) Service to   |
|                                   |     transfer the incoming         |
|                                   |     organisation to the gaining   |
|                                   |     seed organisation, and to     |
|                                   |     update any network hierarchy  |
|                                   |     changes once the incoming     |
|                                   |     organisation has been         |
|                                   |     transferred                   |
|                                   | -   Notify the applicant when     |
|                                   |     processing is complete, and   |
