Scenarios - Calculating daily rate of PPL for ISP

Table 6

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Calculating PPL income for ISP    |
|                                   | customers                         |
|                                   |                                   |
|                                   | PPL period (and any connected     |
|                                   | days)                             |
|                                   |                                   |
|                                   | Average Daily Rate (for ISP       |
|                                   | Purposes)                         |
|                                   |                                   |
|                                   | To determine the rate of PPL      |
|                                   | affecting a customer and/or their |
|                                   | partners rate of income support   |
|                                   | payment (ISP) the below           |
|                                   | calculation needs to be applied.  |
|                                   |                                   |
|                                   | ((Daily Rate of the Period)       |
|                                   | multiplied by (number of weekday  |
|                                   | in the period)) divided by        |
|                                   | (number of calendar days in the   |
|                                   | Period) = Daily rate (for ISP     |
|                                   | purposes)                         |
|                                   |                                   |
|                                   | -   Daily rate of the period =    |
|                                   |     the total number of weekdays  |
|                                   |     in the PPL period plus        |
|                                   |     connected PPL days            |
|                                   | -   Calendar days in the period = |
|                                   |     all days in the PPL period,   |
|                                   |     including weekends            |
|                                   | -   Daily rate = daily rate of    |
|                                   |     the period x number of        |
|                                   |     weekdays in the period /      |
|                                   |     number of calendar days in    |
|                                   |     the period                    |
|                                   |                                   |
|                                   | Note: the NMW is indexed on the   |
|                                   | 1st July each year. When the PPL  |
|                                   | period spans across financial     |
|                                   | years the daily rate of PPL will  |
|                                   | change.                           |
|                                   |                                   |
|                                   | Flexible PPL (not connected)      |
|                                   |                                   |
|                                   | NMW daily rate is applied on all  |
|                                   | calendar day and includes         |
|                                   | weekends                          |
|                                   |                                   |
|                                   | When Flexible PPL days aren’t     |
|                                   | connected to the end of the PPL   |
|                                   | period, the NMW will be used as   |
|                                   | the daily rate for the customer   |
|                                   | and/or their partners income      |
|                                   | support payment (ISP).            |
|                                   |                                   |
|                                   | The NMW daily rate will be used   |
|                                   | for every individual day within a |
|                                   | block of Flexible PPL days.       |
|                                   |                                   |
|                                   | For Flexible PPL days not         |
|                                   | connected to the PPL period the   |
|                                   | NMW daily rate                    |
|                                   |                                   |
|                                   | -   used for a block of days      |
|                                   |     applies on any calendar day   |
|                                   |     and includes weekends         |
|                                   | -   is used for individual        |
|                                   |     claimed Flexible PPL days     |
+-----------------------------------+-----------------------------------+
| 2                                 | Example: PPL Period only          |
|                                   |                                   |
|                                   | Lauren is receiving Parenting     |
|                                   | Payment and is also receiving PPL |
|                                   | for the same period. Laurens PPL  |
|                                   | income for the ISP period is      |
|                                   | calculated on a daily ISP rate of |
|                                   | $110.3642. The average Daily Rate |
|                                   | for ISP Purposes is used.         |
|                                   |                                   |
