| 4                                 | Voluntary work as an approved     |

|                                   |                                   |
|                                   | To be included in the Job Plan as |
|                                   | an approved activity, voluntary   |

|                                   | -   occur without payment other   |
|                                   |     than for out of pocket        |

|                                   |     seeker's own home             |
|                                   | -   not primarily promote a       |
|                                   |     particular religious or       |
|                                   |     political view                |

|                                   |     job seeker's or a family      |
|                                   |     member of the job seeker's    |
|                                   |     own organisation              |
|                                   | -   not involve violence towards  |
|                                   |     people or damage to property  |

|                                   |     would normally be done by a   |

|                                   |     casual or part-time paid      |

|                                   |     reduce overtime of an         |
|                                   |     existing worker               |

Mutual obligation requirements not met

Table 2: this table outlines the process when eligible job seekers
choose to use voluntary work to satisfy their mutual obligation
requirements, but do not meet their requirements.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Reasonable excuse for not meeting |
|                                   | requirements + Read more ...      |
|                                   |                                   |
|                                   | Note: where the job seeker has a  |
|                                   | previously approved voluntary     |
|                                   | work activity coded on the        |
|                                   | Activity and Exemption Summary    |
|                                   | (AEX) screen and the voluntary    |
|                                   | work organisation has contacted   |
|                                   | to advise the job seeker is not   |
|                                   | meeting their voluntary work      |
|                                   | duties, go to Step 5.             |
|                                   |                                   |
|                                   | Reasonable excuses                |
|                                   |                                   |
|                                   | The only excuses accepted as      |
|                                   | reasonable are:                   |
|                                   |                                   |
|                                   | -   customer ill or injured       |
|                                   | -   sick children                 |
|                                   | -   a cancelled shift             |
|                                   | -   short term reduction in hours |
|                                   |     due to public holiday         |
|                                   | -   pupil-free days               |
|                                   | -   failure of special disability |
|                                   |     related transport             |
|                                   |                                   |
|                                   | A reasonable excuse is only       |
|                                   | acceptable where there is an      |
|                                   | expectation that working hours    |
|                                   | will return to normal in the next |
|                                   | fortnight.                        |
|                                   |                                   |
|                                   | Does the job seeker have a        |
|                                   | reasonable excuse for not meeting |
|                                   | their requirements?               |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, go to Step 4              |
+-----------------------------------+-----------------------------------+
| 2                                 | Third consecutive reasonable      |
|                                   | excuse + Read more ...            |
|                                   |                                   |
|                                   | Is this the third consecutive     |
|                                   | reasonable excuse?                |
|                                   |                                   |
|                                   | -   Yes, the job seeker can no    |
|                                   |     longer meet their mutual      |
|                                   |     obligation requirements with  |
|                                   |     voluntary work, go to Step 4  |
|                                   | -   No, go to Step 3              |
+-----------------------------------+-----------------------------------+
| 3                                 | First or second consecutive       |
|                                   | reasonable excuse + Read more ... |
|                                   |                                   |
|                                   | If this is the first or second    |
|                                   | consecutive reasonable excuse,    |
|                                   | the Service Officer should code   |
|                                   | the reason in the Earnings and    |
|                                   | Reporting workflow and process    |
|                                   | the job seeker's payment as       |
|                                   | usual. Payment will not be        |
|                                   | affected.                         |
|                                   |                                   |
|                                   | Revert their reporting            |
|                                   | requirements to fortnightly if    |
|                                   | they are a mature age 55+ job     |
|                                   | seeker and had previously been    |
|                                   | placed on variable reporting.     |
|                                   |                                   |
|                                   | The Service Officer should advise |
|                                   | the job seeker that a maximum of  |
|                                   | 3 consecutive reasonable excuses  |
|                                   | may be given. However, on the     |
|                                   | third fortnight the job seeker    |
|                                   | must choose a new activity and be |
|                                   | referred to, or reconnected with  |
|                                   | an Employment Services Provider.  |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 4                                 | Job seeker does not have a        |
|                                   | reasonable excuse + Read more ... |
|                                   |                                   |
|                                   | If the job seeker does not have a |
|                                   | reasonable excuse, or has already |
|                                   | provided 3 consecutive excuses    |
|                                   | for failing to meet their         |
|                                   | requirements, the job seeker must |
|                                   | choose another activity type, as  |
|                                   | they can no longer meet their     |
|                                   | obligations with voluntary work.  |
|                                   |                                   |
|                                   | Investigate other approved        |
|                                   | activities for the job seeker. If |
|                                   | the job seeker is not doing       |
|                                   | another activity or combination   |
|                                   | of activities that can be         |
|                                   | approved by Centrelink as fully   |
|                                   | meeting requirements, end date    |
|                                   | the voluntary work activity       |
|                                   | coding on the AEX screen          |
|                                   |                                   |
|                                   | If the customer is a mature age   |
|                                   | 55+ job seeker, revert their      |
|                                   | reporting requirements to         |
|                                   | fortnightly if they had           |
|                                   | previously been on variable       |
|                                   | reporting. This shows the         |
|                                   | provider the job seeker is not    |
|                                   | meeting requirements through an   |
|                                   | approved activity and ensures the |
|                                   | job seeker will be engaged by the |
|                                   | provider in a timely manner. The  |
|                                   | job seeker's record will display  |
|                                   | as provider managed when the      |
|                                   | Department of Education, Skills   |
|                                   | and Employment send an advice     |
|                                   | that the job seeker has started   |
|                                   | or restarted in Employment        |
|                                   | Provider Services                 |
|                                   |                                   |
|                                   | Provider managed job seekers:     |
|                                   |                                   |
|                                   | -   must be reconnected           |
|                                   |     immediately to their          |
|                                   |     Employment Services Provider  |
|                                   |     or another appropriate        |
|                                   |     provider (if not already      |
|                                   |     active with a provider) to    |
|                                   |     update their Job Plan.        |
|                                   |     Procedure ends here           |
|                                   |                                   |
|                                   | For Centrelink managed job        |
|                                   | seekers:                          |
|                                   |                                   |
|                                   | -   if they are and will remain   |
|                                   |     Centrelink managed, the       |
|                                   |     Service Officer must          |
|                                   |     renegotiate the Job Plan to   |
|                                   |     remove voluntary work as a    |
|                                   |     compulsory activity and add   |
|                                   |     appropriate activities        |
|                                   | -   process the outstanding       |
|                                   |     report. Procedure ends here   |
+-----------------------------------+-----------------------------------+
| 5                                 | Voluntary work organisation       |
|                                   | notifies that the job seeker      |
|                                   | failed to meet agreed voluntary   |
|                                   | work requirements + Read more ... |
|                                   |                                   |
|                                   | []                                |
|                                   |                                   |
|                                   | In cases where a job seeker is    |
|                                   | not meeting their requirements    |
|                                   | with voluntary work, the          |
|                                   | voluntary work organisation may   |
|                                   | contact Services Australia of a   |
|                                   | job seeker's failure to meet      |
|                                   | agreed voluntary work activity    |
|                                   | requirements via:                 |
|                                   |                                   |
|                                   | -   their local service centre    |
|                                   | -   the Services Australia        |
|                                   |     website                       |

Reviewing voluntary work as an approved activity

Table 3:

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Annual review of voluntary work   |
|                                   | as an approved activity + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Where the Activity and Exemption  |
|                                   | Summary (AEX) screen is updated   |
|                                   | to record the voluntary work type |
|                                   | of VWK or VPE coded as an         |
|                                   | approved activity for 52 weeks, a |
|                                   | Voluntary Work 12 Monthly Review  |
|                                   | (VWR) is automatically generated  |
|                                   | for 48 weeks in the future to     |
|                                   | alert Service Officers to review  |
|                                   | the job seeker's voluntary work.  |
|                                   |                                   |
|                                   | Once the date of the review is    |
|                                   | reached, an appointment will be   |
|                                   | automatically booked and a letter |
|                                   | sent to the job seeker advising   |
|                                   | details of the appointment and of |
|                                   | the need to provide details of    |
|                                   | the voluntary work organisation   |
|                                   | at the interview.                 |
|                                   |                                   |
|                                   | If a job seeker fails to attend   |
|                                   | the interview, make 2 genuine     |
|                                   | attempts to contact them and if   |
|                                   | they cannot be contacted, their   |
|                                   | payment should be suspended with  |
|                                   | the reason 'Failed to Attend      |
|                                   | Interview' (FAI).                 |
|                                   |                                   |
|                                   | Following successful contact with |
|                                   | Services Australia, reschedule    |
|                                   | the interview and when the job    |
|                                   | seeker attends their appointment, |
|                                   | payment can be restored. Service  |
|                                   | Officers must record a DOC with   |
|                                   | reasons for non-attendance.       |

|                                   | If an appointment cannot be       |
