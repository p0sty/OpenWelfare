Initial assessment of voluntary work as an approved activity

Table 1: portions of this process can be completed by Participation
Solutions Team (PST) staff only.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Job seeker proposing to           |
|                                   | participate in voluntary work (or |
|                                   | a combination of voluntary work   |
|                                   | and paid employment) for at least |
|                                   | 30 hours per fortnight + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Is the job seeker proposing to    |
|                                   | participate in voluntary work     |
|                                   | that fully meets their mutual     |
|                                   | obligation requirements?          |
|                                   |                                   |
|                                   | Yes, go to Step 2                 |
|                                   |                                   |
|                                   | No, the job seeker may need a     |
|                                   | referral to an Employment         |
|                                   | Services Provider if they are not |
|                                   | already connected. Providers are  |
|                                   | responsible for negotiating and   |
|                                   | maintaining the Job Plan for      |
|                                   | provider managed job seekers. If  |
|                                   | Centrelink managed, the job       |
|                                   | seeker either needs to select a   |
|                                   | complementary activity to         |
|                                   | supplement the hours (e.g.        |
|                                   | part-time work or                 |
|                                   | self-employment) or choose        |
|                                   | another activity to meet their    |
|                                   | requirements. If voluntary work   |
|                                   | is not approved, DOC the job      |
|                                   | seeker's record with the reason   |
|                                   | it was not approved. Ensure the   |
|                                   | job seeker is verbally told of    |
|                                   | the decision:                     |
|                                   |                                   |
|                                   | -   If no supplementary activity  |
|                                   |     is chosen, their current      |
|                                   |     activity type will remain.    |
|                                   |     Procedure ends here           |
|                                   | -   If a new activity is chosen,  |
|                                   |     follow the procedure for that |
|                                   |     Approved Activity. Procedure  |
|                                   |     ends here                     |
+-----------------------------------+-----------------------------------+
| 2                                 | Organisation approved for         |
|                                   | voluntary work + Read more ...    |
|                                   |                                   |
|                                   | Organisations already assessed by |
|                                   | Services Australia are in the     |
|                                   | Local Activities Database (LAD)   |
|                                   | available via the Intranet. See   |
|                                   | the Resources page for a link.    |
|                                   |                                   |
|                                   | Is the organisation an approved   |
|                                   | voluntary work organisation?      |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, give the job seeker a     |
|                                   |     Voluntary Work - Request for  |
|                                   |     Organisation Approval (SU461) |
|                                   |     form for the organisation to  |
|                                   |     complete. Once the form has   |
|                                   |     been returned with the        |
|                                   |     appropriate documentation     |
|                                   |     (including evidence of the    |
|                                   |     organisation's not for profit |
|                                   |     status and appropriate        |

|                                   |     organisation can be approved. |
|                                   |     Procedure ends here           |
|                                   |                                   |
|                                   | Note: approved organisations      |
|                                   | listed on the LAD do not need an  |
|                                   | SU461 to be completed. If the     |
|                                   | organisation is on the LAD as     |
|                                   | 'not approved', a new SU461 with  |
|                                   | additional information for        |
|                                   | assessment can be provided so the |

|                                   | participation in voluntary work + |

|                                   | doing voluntary work using the    |
|                                   | Verification of Voluntary Work    |
|                                   | (SU462) form. This form must only |
|                                   | be issued to the job seeker once  |
|                                   | the organisation has been         |
|                                   | approved. It is completed by both |
|                                   | the job seeker and voluntary work |

|                                   |                                   |

|                                   |                                   |
|                                   | -   Yes, go to Step 4             |
|                                   | -   No, voluntary work cannot be  |
|                                   |     an approved activity.         |
|                                   |     Procedure ends here           |
|                                   |                                   |
|                                   | Note: where the organisation has  |
|                                   | not been approved (or is pending  |

| 4                                 | Voluntary work as an approved     |
|                                   | activity + Read more ...          |
|                                   |                                   |
|                                   | To be an approved activity,       |
|                                   | voluntary work must:              |
|                                   |                                   |
|                                   | -   occur without payment other   |
|                                   |     than for out of pocket        |
|                                   |     expenses                      |
|                                   | -   have a community focus        |
|                                   | -   not be undertaken in the job  |
|                                   |     seeker's own home             |
|                                   | -   not primarily promote a       |
|                                   |     particular religious or       |
|                                   |     political view                |
|                                   | -   not involve working for the   |
|                                   |     job seeker's or a family      |
|                                   |     member of the job seeker's    |
|                                   |     own organisation              |
|                                   | -   not involve violence towards  |
|                                   |     people or damage to property  |
|                                   | -   not undertake tasks which     |
|                                   |     would normally be done by a   |
|                                   |     paid employee, including a    |
|                                   |     casual or part-time paid      |
|                                   |     employee                      |
|                                   | -   not reduce hours usually      |
|                                   |     worked by a paid employee or  |
|                                   |     reduce overtime of an         |
|                                   |     existing worker               |
|                                   |                                   |
|                                   | When assessing voluntary work as  |
|                                   | an approved activity for job      |
|                                   | seekers aged 55 and over, the     |
|                                   | above points must be considered,  |
|                                   | as well as whether the            |
|                                   | organisation is or can be         |
|                                   | approved for voluntary work.      |
|                                   |                                   |

|                                   | attend a 'base' or like place,    |
|                                   | and is participating in a         |
|                                   | work-like activity for the        |
|                                   | duration of their time on call,   |
|                                   | these hours may count towards     |
|                                   | their voluntary work hours.       |
|                                   |                                   |
|                                   | If the job seeker is going about  |
|                                   | their daily life, without         |
|                                   | participating in any work-like    |
|                                   | activity while on call, these     |
|                                   | hours cannot be counted towards   |
|                                   | their voluntary work hours.       |
|                                   |                                   |
|                                   | If the voluntary work placement   |
|                                   | is not approved, the job seeker's |
|                                   | record should include a DOC to    |
|                                   | advise why the voluntary work     |
|                                   | activity has not been approved.   |
|                                   | Ensure the job seeker is verbally |
|                                   | told of the decision.             |

|                                   | number/LAD number)                |
|                                   |                                   |
|                                   | For Disability Employment         |
|                                   | Services (DES) provider managed   |
|                                   | job seekers, go to Step 5         |
|                                   |                                   |
|                                   | In Process Direct, key 'AEX' in   |
|                                   | the Super Key, select [Enter]     |

|                                   | The Activity and Exemption        |
|                                   | Summary (AEX) screen will display |

|                                   | Code the AEX screen with the      |
|                                   | required approved activity, start |
|                                   | and end date                      |
|                                   |                                   |
|                                   | -   Activity type: 'VWK'          |
|                                   |     (voluntary work) or 'VPE'     |
|                                   |     (voluntary and part-time work |
|                                   |     combined). If the job seeker  |
|                                   |     is doing a combination of     |
|                                   |     voluntary work and            |
|                                   |     self-employment, use both     |
|                                   |     codes 'SEL' and 'VWK'         |
|                                   | -   Start date/end date: period   |

|                                   |     duration                      |
|                                   |                                   |
|                                   | Determine if a new Job Plan is    |
|                                   | required. Do not update the Job   |
|                                   | Plan if the customer is a DES     |
|                                   | participant                       |
|                                   |                                   |

|                                   | Exemption (AEX) screen, do not    |

|                                   | within Process Direct. A new Job  |

|                                   |                                   |

|                                   | Viewing Centrelink customers'     |
|                                   | digital images                    |

Mutual obligation requirements not met

Table 2: this table outlines the process when eligible job seekers
choose to use voluntary work to satisfy their mutual obligation
requirements, but do not meet their requirements.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Reasonable excuse for not meeting |
|                                   | requirements + Read more ...      |
|                                   |                                   |
|                                   | Note: where the job seeker has a  |
|                                   | previously approved voluntary     |
|                                   | work activity coded on the        |
|                                   | Activity and Exemption Summary    |
|                                   | (AEX) screen and the voluntary    |
|                                   | work activity has contacted to    |
|                                   | advise the job seeker is not      |
|                                   | meeting their voluntary work      |
|                                   | duties, go to Step 5.             |
|                                   |                                   |
|                                   | Reasonable excuses                |
|                                   |                                   |
|                                   | The only excuses accepted as      |
|                                   | reasonable are:                   |
|                                   |                                   |
|                                   | -   customer ill or injured       |
|                                   | -   sick children                 |
|                                   | -   a cancelled shift             |
|                                   | -   short term reduction in hours |
|                                   |     due to public holiday         |
|                                   | -   pupil-free days               |
|                                   | -   failure of special disability |
|                                   |     related transport             |
|                                   |                                   |
|                                   | A reasonable excuse is only       |
|                                   | acceptable where there is an      |
|                                   | expectation that working hours    |
|                                   | will return to normal in the next |
|                                   | fortnight.                        |
|                                   |                                   |
|                                   | Does the job seeker have a        |
|                                   | reasonable excuse for not meeting |
|                                   | their requirements?               |
|                                   |                                   |
|                                   | -   Yes, go to Step 2             |
|                                   | -   No, go to Step 4              |
+-----------------------------------+-----------------------------------+
| 2                                 | Third consecutive reasonable      |
|                                   | excuse + Read more ...            |
|                                   |                                   |
|                                   | Is this the third consecutive     |
|                                   | reasonable excuse?                |
|                                   |                                   |
|                                   | -   Yes, the job seeker can no    |
|                                   |     longer meet their mutual      |
|                                   |     obligation requirements with  |
|                                   |     voluntary work, go to Step 4  |
|                                   | -   No, go to Step 3              |
+-----------------------------------+-----------------------------------+
| 3                                 | First or second consecutive       |
|                                   | reasonable excuse + Read more ... |
|                                   |                                   |
|                                   | If this is the first or second    |
|                                   | consecutive reasonable excuse,    |
|                                   | the Service Officer should code   |
|                                   | the reason in the Earnings and    |
|                                   | Reporting workflow and process    |
|                                   | the job seeker's payment as       |
|                                   | usual. Payment will not be        |
|                                   | affected.                         |
|                                   |                                   |
|                                   | Revert their reporting            |
|                                   | requirements to fortnightly if    |
|                                   | they are a mature age 55+ job     |
|                                   | seeker and had previously been    |
|                                   | placed on variable reporting.     |
|                                   |                                   |
|                                   | The Service Officer should advise |
|                                   | the job seeker that a maximum of  |
|                                   | 3 consecutive reasonable excuses  |
|                                   | may be given. However, on the     |
|                                   | third fortnight the job seeker    |
|                                   | must choose a new activity and be |
|                                   | referred to, or reconnected with  |
|                                   | an Employment Services Provider.  |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 4                                 | Job seeker does not have a        |
|                                   | reasonable excuse + Read more ... |
|                                   |                                   |
|                                   | If the job seeker does not have a |
|                                   | reasonable excuse, or has already |
|                                   | provided 3 consecutive excuses    |
|                                   | for failing to meet their         |
|                                   | requirements, the job seeker must |
|                                   | choose another activity type, as  |
|                                   | they can no longer meet their     |
|                                   | obligations with voluntary work.  |
|                                   |                                   |
|                                   | Investigate other approved        |
|                                   | activities for the job seeker. If |
|                                   | the job seeker is not doing       |
|                                   | another activity or combination   |
|                                   | of activities that can be         |
|                                   | approved by Centrelink as fully   |
|                                   | meeting requirements, end date    |
|                                   | the voluntary work activity       |
|                                   | coding on the AEX screen          |
|                                   |                                   |
|                                   | If the customer is a mature age   |
|                                   | 55+ job seeker, revert their      |
|                                   | reporting requirements to         |
|                                   | fortnightly if they had           |
|                                   | previously been on variable       |
|                                   | reporting. This shows the         |
|                                   | provider the job seeker is not    |
|                                   | meeting requirements through an   |
|                                   | approved activity and ensures the |
|                                   | job seeker will be engaged by the |
|                                   | provider in a timely manner. The  |
|                                   | job seeker's record will display  |
|                                   | as provider managed when the      |
|                                   | Department of Employment and      |
|                                   | Workplace Relations send an       |
|                                   | advice that the job seeker has    |
|                                   | started or restarted in           |
|                                   | Employment Provider Services      |
|                                   |                                   |
|                                   | Provider managed job seekers:     |
|                                   |                                   |
|                                   | -   must be reconnected           |
|                                   |     immediately to their          |
|                                   |     Employment Services Provider  |
|                                   |     or another appropriate        |
|                                   |     provider (if not already      |
|                                   |     active with a provider) to    |
|                                   |     update their Job Plan.        |
|                                   |     Procedure ends here           |
|                                   |                                   |
|                                   | For Centrelink managed job        |
|                                   | seekers:                          |
|                                   |                                   |
|                                   | -   if they are and will remain   |
|                                   |     Centrelink managed, the       |
|                                   |     Service Officer must remove   |
|                                   |     voluntary work as a           |
|                                   |     compulsory activity on the    |
|                                   |     AEX screen and add an         |
|                                   |     appropriate activity          |
|                                   | -   process the outstanding       |
|                                   |     report. Procedure ends here   |
+-----------------------------------+-----------------------------------+
| 5                                 | Voluntary work organisation       |
|                                   | notifies that the job seeker      |
|                                   | failed to meet agreed voluntary   |
|                                   | work activity + Read more ...     |
|                                   |                                   |
|                                   | []                                |
|                                   |                                   |
|                                   | In cases where a job seeker is    |
|                                   | not attending their voluntary     |
|                                   | work activity, the voluntary work |
|                                   | organisation may contact Services |
|                                   | Australia to notify of a job      |
|                                   | seeker's failure to meet agreed   |
|                                   | voluntary work requirements via:  |
|                                   |                                   |
|                                   | -   their local service centre    |
|                                   | -   the Services Australia        |
|                                   |     website                       |
|                                   | -   telephone 131 524             |
|                                   | -   mail: Centrelink Reply Paid   |
|                                   |     7800 Canberra BC ACT 3610     |
|                                   |                                   |
|                                   | See Report Suspected Fraud and    |
|                                   | Corruption.                       |

Approved Activity Review Job Plan Contact

contact in relation to an Approved Activity Review or a staff member is
allocated an Approved Activity Review work item.

|                                   | ongoing participation in the      |

|                                   | exists before coding the          |
|                                   | extension.                        |
|                                   |                                   |
|                                   | Is customer contacting about      |
|                                   | online task or letter in relation |
|                                   | to reviewing their Job Plan?      |
|                                   |                                   |
|                                   | -   Yes, advise customer they     |
|                                   |     need to complete the review   |
|                                   |     if they still meet the        |
|                                   |     approved activity criteria    |
|                                   |     and they will have the        |
|                                   |     activity extended. See        |
|                                   |     Approved Activity Review Job  |
|                                   |     Plan Contact. Procedure ends  |
|                                   |     here                          |
|                                   | -   No, go to Step 2              |
+-----------------------------------+-----------------------------------+
| 2                                 | Job Plan or AEX Review work       |
|                                   | item + Read more ...              |

|                                   | Online-MOR and Job Plan or AEX    |

|                                   | work item?                        |

|                                   |                                   |
|                                   | No:                               |
|                                   |                                   |
|                                   | -   once the job seeker completes |
|                                   |     an approved program or        |
|                                   |     activity which attracts the   |
|                                   |     Approved Activity Review      |
|                                   |     contact, the job seeker will  |
|                                   |     receive an online task or     |
|                                   |     letter depending on whether   |
|                                   |     they are signed up for self   |
|                                   |     service                       |
|                                   | -   the system will create online |

|                                   |     start and they will be        |
|                                   |     referred to an employment     |
|                                   |     services provider. Procedure  |
|                                   |     ends here                     |
