+-----------------------------------+-----------------------------------+
| 1                                 | Assessing eligible activities +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | The Farm Household Allowance      |
|                                   | Voucher for activity supplement   |
|                                   | (A2555) is issued to the customer |
|                                   | once it is determined they have   |
|                                   | an approved eligible activity in  |
|                                   | their signed Financial            |
|                                   | Improvement Agreement (FIA). The  |
|                                   | initial FIA will auto issue the   |
|                                   | A2555 voucher if it contains an   |
|                                   | activity that is eligible for the |
|                                   | supplement. In all other cases,   |
|                                   | the A2555 must be manually        |
|                                   | printed from Online forms when    |
|                                   | the activity is due to be         |
|                                   | completed.                        |
|                                   |                                   |
|                                   | If the customer has a specific    |
|                                   | course/training activity in mind, |
|                                   | they should be advised to provide |
|                                   | the details to their FHCO who     |
|                                   | will assess whether the activity  |
|                                   | supplement can be used to pay for |
|                                   | the activity.                     |
|                                   |                                   |
|                                   | The FHCO must record the details  |
|                                   | of the activity, including the    |
|                                   | cost and qualifications of the    |
|                                   | provider in Case Notes.           |
|                                   |                                   |
|                                   | If the activity is not already    |
|                                   | recorded as 'eligible for         |
|                                   | activity supplement' in the FIA,  |
|                                   | the FIA will have to be updated.  |
|                                   |                                   |
|                                   | Make sure specific details        |
|                                   | relevant to the training or       |
|                                   | advice being undertaken by the    |
|                                   | customer (such as training course |
|                                   | name or the aims to be achieved   |
|                                   | from obtaining advice) are        |
|                                   | reflected in the FIA before it is |
|                                   | agreed. See Financial Improvement |
|                                   | Agreements.                       |
|                                   |                                   |
|                                   | The A2555 voucher can be used to  |
|                                   | pay for reasonable travel and     |
|                                   | accommodation costs associated    |
|                                   | with an agreed activity in the    |
|                                   | FIA, including if the activity is |
|                                   | free.                             |
|                                   |                                   |
|                                   | Note: the activity supplement is  |
|                                   | not to be used to cover the cost  |
|                                   | of activities considered Business |
|                                   | as Usual (BAU). While an activity |
|                                   | may be considered BAU for one     |
|                                   | customer based on their           |
|                                   | circumstances and industry, it    |
|                                   | may not be considered BAU for     |
|                                   | another customer.                 |
|                                   |                                   |
|                                   | If the activity is:               |
|                                   |                                   |
|                                   | -   advice, go to Step 2          |
|                                   | -   study or training, go to Step |
|                                   |     3                             |
|                                   | -   counselling, go to Step 4     |

|                                   | Read more ...                     |
|                                   |                                   |

|                                   |                                   |
|                                   | -   in a field related to the     |
|                                   |     subject of the advice being   |
|                                   |     provided, and                 |
|                                   | -   recognised by a professional  |
|                                   |     institution                   |
|                                   |                                   |
|                                   | Are both requirements met?        |
|                                   |                                   |
|                                   | Yes:                              |
|                                   |                                   |

|                                   |     activity. Go to Step 5        |
|                                   |                                   |
|                                   | No:                               |
|                                   |                                   |
|                                   | -   the activity is not eligible  |
|                                   |     for the Activity Supplement   |
|                                   | -   make genuine attempts to      |

|                                   | -   tell the customer they can    |

|                                   |     training as a valid FIA       |
