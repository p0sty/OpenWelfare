Calculating the effective life age of equipment

Note: the date from which the age of equipment is worked out is:

the date that the equipment was first installed in Australia; or

if the equipment was imported as used equipment — the date of
manufacture of the oldest component

+-----------------------+-----------------------+-----------------------+
| Modality              | Effective life age    | Maximum extended life |
|                       |                       | age                   |
|                       |                       |                       |
|                       |                       | The maximum extended  |
|                       |                       | life age for          |
|                       |                       | diagnostic imaging    |
|                       |                       | equipment applies     |
|                       |                       | where equipment has   |
|                       |                       | been upgraded on or   |
|                       |                       | before its effective  |
|                       |                       | life age, that is, 10 |
|                       |                       | years for ultrasound, |
|                       |                       | 15 years for X-ray.   |
+-----------------------+-----------------------+-----------------------+
| Ultrasound            | 10 years              | 15 years              |
+-----------------------+-----------------------+-----------------------+
| Mammography           | 10 years              | 15 years              |
+-----------------------+-----------------------+-----------------------+
| Angiography           | 10 years              | 15 years              |
+-----------------------+-----------------------+-----------------------+
| Nuclear medicine      | 10 years              | 15 years              |
| (excluding PET)       |                       |                       |
+-----------------------+-----------------------+-----------------------+
| Fluoroscopy, X-ray,   | 15 years              | 20 years              |
| OPG                   |                       |                       |
+-----------------------+-----------------------+-----------------------+
| MRI                   | 10 years              | 20 years              |

Forms

Application to register or amend a diagnostic imaging or radiation
oncology practice form (HW061)

Note: this form and any supporting documentation can be uploaded and
submitted through Health Professional Online Services (HPOS), see
Services Australia's website. If the form has been received via HPOS
then correspondence must be delivered via HPOS. Forms received via HPOS
can be identified, as the naming convention will contain the word
'HPOS'. For example, YYYYMMDD.HPOS12345678_ WorkID.WorkType.

For more details, see the Process page in Health Professional Online
Services Support Mail Centre – Managing emails from health
professionals.
