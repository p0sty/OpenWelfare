| 1                                 | Identifying eligible students and |
|                                   | whether they need to lodge a      |
|                                   | claim + Read more ...             |
|                                   |                                   |
|                                   | []                                |
|                                   |                                   |
|                                   | A claim for Additional Assistance |
|                                   | will not be required in all       |
|                                   | cases.                            |
|                                   |                                   |
|                                   | Students meeting one of the       |
|                                   | special independence criteria for |
|                                   | students without parental support |
|                                   | (listed in Step 1 Table 1), will  |
|                                   | be identified by the ABSTUDY      |
|                                   | Smart Centre. They will not need  |
|                                   | to lodge a claim. Confirmation    |
|                                   | that they have no access to       |
|                                   | financial support will be         |
|                                   | established by Centrelink records |
|                                   | showing they meet one of those    |
|                                   | independence criteria and their   |
|                                   | ABSTUDY entitlements are paid     |
|                                   | direct to the board provider,     |
|                                   | including during school           |
|                                   | vacations.                        |
|                                   |                                   |
|                                   | Note: students who continue to    |
|                                   | meet one of the special           |
|                                   | independence criteria without     |
|                                   | parental support can qualify for  |
|                                   | Additional Assistance during      |
|                                   | school vacations (subject to      |
|                                   | personal income test) while       |
|                                   | ABSTUDY Living Allowance and      |
|                                   | other boarding related payments   |
|                                   | are paid directly to their board  |
|                                   | provider. A social worker         |
|                                   | assessing continued eligibility   |
|                                   | of a student under the UTLAH      |
|                                   | criteria will look at any         |
|                                   | financial support they receive.   |
|                                   | If that support or assistance is  |
|                                   | determined as not continuous,     |
|                                   | reliable or meaningful and does   |
|                                   | not affect eligibility under the  |
|                                   | UTLAH criteria, the student's     |
|                                   | qualification for Additional      |
|                                   | Assistance is not affected        |
|                                   | assuming the other requirements   |
|                                   | are met.                          |
|                                   |                                   |
|                                   | Single students eligible for      |
|                                   | ABSTUDY under other independence  |
|                                   | criteria or those who were in     |
|                                   | State Care (but no longer in      |
|                                   | State Care only because of their  |
|                                   | age) will need to lodge a claim   |
|                                   | each year.                        |
|                                   |                                   |
|                                   | -   The claim needs to include    |
|                                   |     confirmation they have no     |
|                                   |     access to financial support   |
|                                   |     from parent(s)/guardian(s),   |
|                                   |     other family members or other |
|                                   |     sources during school         |
|                                   |     vacations                     |
|                                   | -   For students formerly in      |
|                                   |     State Care, their claim will  |
|                                   |     need to show they have no     |
|                                   |     access to financial support   |
|                                   |     and are not in the care or    |
|                                   |     custody of a parent or other  |
|                                   |     person during school          |
|                                   |     vacations                     |

|                                   | ABSTUDY.                          |
|                                   |                                   |
|                                   | Additional Assistance rate is     |
|                                   | paid for relevant vacation period |
|                                   | (s) during the calendar year they |
|                                   | are not residing at the boarding  |
|                                   | school or hostel.                 |
|                                   |                                   |
|                                   | The payment period of Additional  |
|                                   | Assistance starts when the        |
|                                   | student has left the boarding     |
|                                   | provider's care, normally it is   |
|                                   | within the last few days of the   |
|                                   | term. The period will end on the  |
|                                   | day the student returns to the    |
|                                   | boarding provider. A beneficial   |
|                                   | interpretation is used in         |
|                                   | determining the period the        |
|                                   | student is not in the boarding    |
|                                   | provider’s care.                  |
|                                   |                                   |
|                                   | The rate of Additional Assistance |
|                                   | would be subject to the personal  |
|                                   | income test, and income bank      |
|                                   | applicable to the Living          |
|                                   | Allowance, and any Rent           |
|                                   | Assistance, Remote Area Allowance |
|                                   | and any other income affected     |
|                                   | boarding related components being |
|                                   | paid to the board provider. See   |
|                                   | Personal Income Test and Income   |
|                                   | Bank for ABSTUDY, Austudy and     |
|                                   | Youth Allowance (YA) students and |
|                                   | Australian Apprentices.           |
|                                   |                                   |
|                                   | Any residual payment of these     |
|                                   | allowances that is made to the    |

|                                   | Assistance payable.               |
