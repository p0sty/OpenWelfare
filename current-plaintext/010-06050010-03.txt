| 5                                 | Parental income test for rural    |
|                                   | and remote areas + Read more ...  |
|                                   |                                   |
|                                   | The parental income cut-off for   |
|                                   | self-supporting independence      |
|                                   | purposes is $160,000, plus        |
|                                   | $10,000 for each eligible         |
|                                   | additional children in their      |
|                                   | Family Unit Regional (FUR). The   |

|                                   | include the customer.             |

|                                   | calculations as the Parental      |
|                                   | Income Test.                      |

|                                   | 1 of 3 financial years where it   |
|                                   | is more beneficial for them. This |
|                                   | may be the financial year:        |

|                                   |     (pre-gap tax year)            |

|                                   |     year), or                     |
|                                   | -   following their gap year      |
