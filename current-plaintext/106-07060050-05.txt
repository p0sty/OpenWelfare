Completing the Q267 letter fields

Table 1

+-----------------------------------+-----------------------------------+
| Field                             | Action                            |
+-----------------------------------+-----------------------------------+
| PHN008                            | Select '2' for this field to not  |
|                                   | include the 008 number.           |
+-----------------------------------+-----------------------------------+
| SECRQ                             | Select '1' to confirm this is the |
|                                   | first written request for an      |
|                                   | Employment Separation             |
|                                   | Certificate.                      |
|                                   |                                   |
|                                   | Note: only one Q267 letter per    |
|                                   | employer is appropriate before it |
|                                   | is reasonable to accept           |
|                                   | alternative evidence from the     |
|                                   | customer.                         |
+-----------------------------------+-----------------------------------+
| SMTS                              | Select '1', even if the name of   |
|                                   | the employer/supervisor is known. |
+-----------------------------------+-----------------------------------+
| SSALF                             | Select '1' to reflect the request |
|                                   | is being made under s192 of the   |
|                                   | Social Security (Administration)  |
|                                   | Act 1999.                         |
+-----------------------------------+-----------------------------------+
| CHAT                              | Select '2' to include the phone   |
|                                   | number relevant to the customer's |
|                                   | payment type.                     |
|                                   |                                   |
|                                   | Note: the Business Gateway        |
|                                   | contact number is manually        |
|                                   | inserted, see                     |
|                                   | GEN.ENQ.PHONE.INIT.OFF in Table   |
|                                   | 3.                                |
+-----------------------------------+-----------------------------------+
| RVCHG                             | Select '2' to ensure the reverse  |
|                                   | call option is not presented to   |
|                                   | the employer.                     |
+-----------------------------------+-----------------------------------+

Q267 variable data options

Table 2

+-----------------------------------+-----------------------------------+
| Field                             | Example text                      |
+-----------------------------------+-----------------------------------+
| GEN.ENQ.PHONE.INIT.OFF            | The system will generally include |
|                                   | the phone number that is relevant |
|                                   | to the customer's payment type.   |
|                                   | It is more appropriate that       |
|                                   | employers be provided with the    |
|                                   | contact number for the National   |
|                                   | Business Gateway (NBG).           |
|                                   |                                   |
|                                   | Replace the number defaulted by   |
|                                   | the system with the number 131    |
|                                   | 158.                              |
+-----------------------------------+-----------------------------------+
| THIRD PARTY ADDRESSEE             | Insert the name of the            |
|                                   | business/organisation the         |
|                                   | customer was working for here.    |
+-----------------------------------+-----------------------------------+
| THIRD PARTY ADDRESS               | Insert the address for the        |
|                                   | business/organisation the         |
|                                   | customer was working for here.    |
+-----------------------------------+-----------------------------------+
| CLIENT.FULL.NAME                  | By default, the system will       |
|                                   | include the name of the customer  |
|                                   | using information available on    |
|                                   | their record.                     |
|                                   |                                   |
|                                   | No changes are required to this   |
|                                   | field unless the customer advises |
|                                   | they were known to their employer |
|                                   | by another name.                  |
+-----------------------------------+-----------------------------------+
| DTE OF CESSATION OF EMPLOYMENT    | Record the date the customer      |
|                                   | advised they ceased employment.   |
|                                   |                                   |
|                                   | Note: dates entered in the format |
|                                   | D/M/YYYY will be converted        |
|                                   | automatically by the system in    |
|                                   | the final version of the letter   |
|                                   | (for example, '1/1/2016' will be  |
|                                   | converted to 1 January 2016).     |
+-----------------------------------+-----------------------------------+
