| 1                                 | Estimate or valuation returned +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Estimation or valuation request   |
|                                   | has failed.                       |
|                                   |                                   |
|                                   | -   A Manual follow up (MFU) has  |
|                                   |     been created.                 |
|                                   | -   The MFU will have a keyword   |
|                                   |     which will provide            |
|                                   |     information about why the     |
|                                   |     request failed and what       |
|                                   |     action is needed.             |
|                                   | -   Go to Step 4                  |
|                                   |                                   |
|                                   | Estimation or valuation data      |
|                                   | returned.                         |
|                                   |                                   |
|                                   | Estimation                        |
|                                   |                                   |
|                                   | If the estimation verification    |
|                                   | has been returned and the         |
|                                   | estimate is acceptable the        |
|                                   | Valuation Source code on the Real |
|                                   | Estate Detail (RE) screen will be |
|                                   | automatically updated to EST -    |
|                                   | Estimate Verified                 |
|                                   |                                   |
|                                   | Valuation                         |
|                                   |                                   |
|                                   | The returned valuation data       |
|                                   | automatically uploads with source |
|                                   | code VAL or ONS.                  |
|                                   |                                   |
|                                   | If there:                         |
|                                   |                                   |
|                                   | -   is a claim still to be        |
|                                   |     finalised, go to Step 2       |
|                                   | -   was a change to the rate of   |
|                                   |     payment, go to Step 3         |
+-----------------------------------+-----------------------------------+
| 2                                 | Claim not finalised: + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | When the assessed value is        |
|                                   | returned, the claim activity will |
|                                   | be updated:                       |
|                                   |                                   |
|                                   | on the RE screen:                 |
|                                   |                                   |
|                                   | -   the Current Market Value,     |
|                                   |     Valuers Job Number and Valuer |
|                                   |     Property Number fields will   |
|                                   |     be coded                      |
|                                   | -   Asset Valuation Source field  |
|                                   |     will be VAL or ONS            |
|                                   |                                   |
|                                   | a keyword of VALRTN will be added |
|                                   | to the new claim activity         |
|                                   |                                   |
|                                   | status of claim will change from  |
|                                   | Hold (HLD) to Started (STA)       |
|                                   |                                   |
|                                   | a Real Estate update DOC will be  |
|                                   | added to the record               |
|                                   |                                   |
|                                   | the claim activity is ready for   |
|                                   | finalisation                      |
|                                   |                                   |
|                                   | Following claim finalisation      |
|                                   | automatic risk profiling will     |
|                                   | occur for property types house,   |
|                                   | unit and townhouse                |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 3                                 | Valuation resulted in rate        |
|                                   | change + Read more ...            |
|                                   |                                   |
|                                   | If a valuation has resulted in a  |
|                                   | change to the customer's rate     |
|                                   | they will be sent a rate change   |
|                                   | letter. The letter will say there |
|                                   | has been a change to the value of |
|                                   | real estate assets.               |
|                                   |                                   |
|                                   | The customer can request an       |
|                                   | explanation or apply for a formal |
|                                   | review.                           |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 4                                 | Follow up required- MFU has been  |
|                                   | returned + Read more ...          |
|                                   |                                   |
|                                   | An MFU activity will have an      |
|                                   | error keyword.                    |
|                                   |                                   |
|                                   | Keyword REBEXP- Valuation or      |
|                                   | estimation data not received      |
|                                   |                                   |
|                                   | -   The valuation or estimation   |
|                                   |     data has not been received    |
|                                   |     within required timeframes    |
|                                   | -   Contact the Valuation         |
|                                   |     Helpdesk for further advice   |
|                                   | -   The Valuation Helpdesk will   |
|                                   |     contact the valuation         |
|                                   |     provider to confirm the       |
|                                   |     expected date of receipt      |
|                                   | -   If the helpdesk advises the   |
|                                   |     valuation amount, update the  |
|                                   |     record and finalise the MFU   |
|                                   | -   If the data is received       |
|                                   |     automatically after creation  |
|                                   |     of the MFU, the system will   |
|                                   |     automatically upload the      |
|                                   |     valuation amount and finalise |
|                                   |     the MFU                       |
|                                   |                                   |
|                                   | Keyword ERRSLD - Property sold    |
|                                   | follow up required                |
|                                   |                                   |
|                                   | -   The valuation provider has    |
|                                   |     advised the property has been |
|                                   |     sold                          |
|                                   | -   Contact the customer and      |
|                                   |     update the customer's record  |
|                                   |     as needed                     |
|                                   |                                   |
|                                   | Keyword - REJISI - Insufficient   |
|                                   | information follow up required    |
|                                   |                                   |
|                                   | -   There has not been enough     |
|                                   |     information provided to       |
|                                   |     enable the valuation provider |
|                                   |     to complete this valuation    |
|                                   | -   Contact the Valuation         |
|                                   |     Helpdesk. The helpdesk will   |
|                                   |     confirm what additional       |
|                                   |     information is required       |
|                                   | -   The REVI screen on the record |
|                                   |     must be updated with the      |
|                                   |     required information (the     |
|                                   |     customer may have to be       |
|                                   |     contacted to request/confirm  |
|                                   |     required details)             |
|                                   | -   Once the record has been      |
|                                   |     updated, a manual VAL request |
|                                   |     must be re-submitted by an    |
|                                   |     APS 5 or above                |
|                                   |                                   |
|                                   | Keyword REJPMM - Property         |
|                                   | mismatch follow up required -     |
|                                   | returned by provider              |
|                                   |                                   |
|                                   | -   Contact the Valuation         |
|                                   |     Helpdesk for more             |
|                                   |     information.                  |
|                                   |                                   |
|                                   | Keyword REJNAP - Property error   |
|                                   |                                   |
|                                   | -   Contact the Valuation         |
|                                   |     Helpdesk for more             |
|                                   |     information.                  |
|                                   |                                   |
|                                   | Keyword REBIMM - Internal         |
|                                   | mismatch                          |
|                                   |                                   |
|                                   | -   A property mismatch can occur |
|                                   |     when the Pensions system      |
|                                   |     tries to apply the returned   |
|                                   |     estimation or valuation data  |
|                                   | -   An activity will be created   |
|                                   |     on the Activity List (AL)     |
|                                   |     screen advising the           |
|                                   |     following: Contact the        |
|                                   |     Valuation Helpdesk for more   |
|                                   |     information                   |
|                                   |                                   |
|                                   | Keyword REBUVR - Unsolicited VAL, |
|                                   | ONS or EST response               |
|                                   |                                   |
|                                   | -   A valuation response has been |
|                                   |     received with no request      |
|                                   |     outstanding                   |
|                                   | -   Contact the Valuation         |
|                                   |     Helpdesk                      |
|                                   | -   If a customer's relationship  |
|                                   |     status changes while a        |
|                                   |     valuation is in progress,     |
|                                   |     only the customer's record    |
|                                   |     will be updated. A new        |
|                                   |     request must be submitted for |
|                                   |     the ex-partner or the         |
|                                   |     surviving partner             |
