Effect on updating address types in CDMS

Table 1: this table describes the effect of updating address types.

+-----------------------+-----------------------+-----------------------+
| Address type          | Level                 | Effect on updating    |
+-----------------------+-----------------------+-----------------------+
| Mailing address       | Group                 | Changes personal      |
|                       |                       | mailing address for   |
|                       |                       | all members on card.  |
+-----------------------+-----------------------+-----------------------+
| Customer mailing      | Personal              | Does not change       |
| address               |                       | mailing address at    |
|                       |                       | group level or        |
|                       |                       | customer residential  |
|                       |                       | address.              |
+-----------------------+-----------------------+-----------------------+
| Customer residential  |                       | Does not change       |
| address               |                       | either the customer   |
|                       |                       | mailing address or    |
|                       |                       | group address to      |
|                       |                       | change.               |
|                       |                       |                       |
|                       |                       | Note: to change all   |
|                       |                       | residential addresses |
|                       |                       | on card, select the   |
|                       |                       | tick box against      |
|                       |                       | 'Change for all'      |
|                       |                       | under the address     |
|                       |                       | field.                |
+-----------------------+-----------------------+-----------------------+

Effect of address changes in CDMS to other programmes

Table 2: this table details the effect of changing an address in CDMS to
other programmes.

+-----------------------------------+-----------------------------------+
| Program                           | Change in CDMS                    |
+-----------------------------------+-----------------------------------+
| Pharmaceutical Benefits Scheme    | When a group address is changed   |
| (PBS)                             | in CDMS, the change is            |
|                                   | automatically updated in PBS      |
|                                   | records during overnight          |
|                                   | processing.                       |
+-----------------------------------+-----------------------------------+
| Australian Immunisation Register  | Medicare card number is known:    |
| (AIR)                             |                                   |
|                                   | -   When a customer residential   |
|                                   |     or mailing address is changed |
|                                   |     in CDMS, there is no effect   |
|                                   |     on AIR records for Personal   |
|                                   |     Identification Number (PIN)   |
|                                   |     records                       |
|                                   | -   When a group address is       |
|                                   |     changed in CDMS, the change   |
|                                   |     is automatically updated in   |
|                                   |     AIR records for PIN records   |
|                                   |     during overnight processing   |
|                                   |                                   |
|                                   | Note: if advice is received over  |
|                                   | the phone that an AIR address is  |
|                                   | incorrect, the AIR Service        |
|                                   | Officer updates the group address |
|                                   | via CDMS - which will then be     |
|                                   | reflected in AIR after overnight  |
|                                   | processing.                       |
|                                   |                                   |
|                                   | Medicare card number is not known |
|                                   | or the individual does not have a |
|                                   | Medicare card number:             |
|                                   |                                   |
|                                   | For Supplementary Identification  |
|                                   | Number (SIN) records, the reverse |
|                                   | process more or less occurs where |
|                                   | the Consumer Directory (the       |
|                                   | back-end of the CDMS) is updated  |
|                                   | with AIR records.                 |
|                                   |                                   |
|                                   | For a SIN record, where a         |
|                                   | registration:                     |
|                                   |                                   |
|                                   | -   does not exist in the         |
|                                   |     Consumer Directory (CD), an   |
|                                   |     automatic weekly process      |
|                                   |     causes the CD to be updated   |
|                                   |     with AIR details              |
|                                   | -   exists in the CD, address     |
|                                   |     updates are passed from AIR   |
|                                   |     to the CD within five minutes |
|                                   |     of the AIR update             |
+-----------------------------------+-----------------------------------+
| Australian Organ Donor Register   | AODR processing takes place in    |
| (AODR)                            | CDMS so any address changes are   |
|                                   | immediately available.            |
+-----------------------------------+-----------------------------------+
| Compensation Recovery             | When an address is changed in     |
|                                   | CDMS, it does not cause the       |
|                                   | address to change in the          |
|                                   | Compensation Case Management      |
|                                   | System.                           |
|                                   |                                   |
|                                   | Note: Compensation Recovery       |
|                                   | Service Officers follow manual    |
|                                   | procedures to ensure the          |
|                                   | compensable person's address is   |
|                                   | up to date.                       |

A service office may need to refer to these abbreviations when there is
an address conflict in CDMS and they need to override/manually enter an
address and for those who are claiming via mainframe and update address
details within the same system.
