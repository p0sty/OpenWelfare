Fax transmission errors

Where an error message is received which is not listed below, a
mySupport form should be completed with full details of the error
message. The following information must be included to enable
appropriate investigation:

-   Fax number used
-   Details of the document(s)/form(s) attempting to be sent. (e.g.
    FAO012, system generated SU19, manual SU19 etc.)
-   Whether an attempt has been made to re-send the fax using Desktop
    Faxing, and the outcome
-   Whether an attempt to fax manually has been made, and the outcome
-   Forward any conformation or failure emails (transmission reports)
    received from Message.Manager
-   If known, the model of the fax machine, and/or whether the fax
    machine is a dual phone/fax machine or a dual fax/photocopier
    machine

Table 2: this table describes common errors when an error message is

+-----------------------------------+-----------------------------------+
| Term                              | Description                       |

| 1                                 | Dial tone detected                |
|                                   |                                   |

|                                   |                                   |

|                                   |                                   |
|                                   | Check the fax number is complete  |
|                                   | and try sending the fax again. If |
|                                   | it fails a second time, manually  |

|                                   | problem with the Services         |
|                                   | Australia fax gateway, report the |
|                                   | issue using mySupport.            |
+-----------------------------------+-----------------------------------+
| 2                                 | Digital Command Signal (DCS) sent |
|                                   | three times without response      |
|                                   |                                   |
|                                   | The sending fax machine is trying |
|                                   | to communicate the fax            |
|                                   | transmission protocol with the    |
|                                   | receiving fax machine. It usually |
|                                   | indicates a hardware              |
|                                   | incompatibility between the fax   |
|                                   | machines.                         |
|                                   |                                   |
|                                   | Solution:                         |
|                                   |                                   |

|                                   | the fax again. If it fails a      |

|                                   | required documents.               |

|                                   |                                   |

|                                   | protocol with the receiving fax   |
|                                   | machine.                          |
|                                   |                                   |
|                                   | Solution:                         |
|                                   |                                   |

|                                   | receive fax.                      |

|                                   |                                   |

|                                   | Attempt sending fax again. If     |
