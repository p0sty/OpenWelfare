| Scenario                          | Details                           |
+-----------------------------------+-----------------------------------+
| Income and assets not in aged     | -   Care recipient entered        |
| care systems from date of entry   |     residential care 13 April     |
|                                   |     2018                          |
|                                   | -   MND from date of entry        |
|                                   | -   Means assessment completed 15 |
|                                   |     August 2019 but income and    |
|                                   |     asset data not updated in     |
|                                   |     ACPS from date of entry       |
|                                   | -   Income and assets applied     |
|                                   |     from 1 July 2019              |
|                                   | -   Refund calculated based on $0 |
|                                   |     income and assets from 13     |
|                                   |     April 2018 - 30 June 2019     |
|                                   |     paid in Sep-19 QR             |
|                                   | -   Income and assets updated to  |
|                                   |     ACPS from 13 April 2018       |
|                                   | -   If the difference between the |
|                                   |     correct refund and initial    |
|                                   |     refund is less than $5,000,   |
|                                   |     refund is released            |
|                                   | -   If the difference between the |
|                                   |     correct refund and initial    |
|                                   |     refund is more than $5,000,   |
|                                   |     QR holds the refund until     |
|                                   |     after the next QR             |
|                                   | -   ACPS completes further        |
|                                   |     adjustments in the Jan-20 QR  |
|                                   |     and offsets negative          |
|                                   |     adjustments (if applicable)   |
+-----------------------------------+-----------------------------------+
| Manual adjustment calculation for | Include home care fees in a       |
| a home care and residential care  | manual adjustment calculation for |
| recipient who entered residential | care recipients who entered       |
| care prior to 25 November 2018    | residential care prior to         |
|                                   | November 2018:                    |
|                                   |                                   |
|                                   | Care recipient entered respite    |
|                                   | care on 2 December 2014 and       |
|                                   | departed care on 20 December 2014 |
|                                   |                                   |
|                                   | Care recipient then commenced     |
|                                   | home care on 2 February 2015 and  |
|                                   | departed on 3 May 2016            |
|                                   |                                   |
|                                   | Care recipient then entered       |
|                                   | permanent residential care on 3   |
|                                   | May 2016                          |
|                                   |                                   |
|                                   | Respite care is not included in   |
|                                   | the manual adjustment             |
|                                   | calculation, as BDF only          |
|                                   |                                   |
|                                   | The manual adjustment calculation |
|                                   | start date is 2 February 2015     |
|                                   |                                   |
|                                   | Include home care fees in the     |
|                                   | calculations:                     |
|                                   |                                   |
|                                   | -   home care fees paid from      |
|                                   |     2/2/15 - 1/2/16 = $5,000      |
|                                   | -   home care fees paid from      |
|                                   |     2/2/16 - 2/5/16 = $1,500      |
|                                   |                                   |
|                                   | Home care fees affect the         |
|                                   | calculation of an annual cap when |
|                                   | the care recipient commences      |
|                                   | residential care                  |
|                                   |                                   |
|                                   | Residential care fees calculated: |
|                                   |                                   |
|                                   | Residential care fees paid from   |
|                                   | 3/5/16 - 15/10/16 = $25,939.62    |
|                                   | (annual cap applied)              |
|                                   |                                   |
|                                   | As the home care anniversary date |
|                                   | is 2/2/16, the care recipient is  |
|                                   | not liable to pay more than the   |
|                                   | annual cap in fees in a care year |
|                                   |                                   |
|                                   | Only include home care fees paid  |
|                                   | since the anniversary date in     |
|                                   | annual cap assessment             |
|                                   |                                   |
|                                   | For the transition year,          |
|                                   | calculate the annual cap as       |
|                                   | follows:                          |
|                                   |                                   |
|                                   | Home care fees 2/2/16 - 2/5/16 =  |
|                                   | $1,500                            |
|                                   |                                   |
|                                   | Residential care fees commenced   |
|                                   | from 3/5/16 and the care          |
|                                   | recipient reached $25,939.62      |
|                                   | (annual cap) on 1/10/16           |
|                                   |                                   |
|                                   | The annual cap is applied from    |
|                                   | 1/10/16 and any refund is paid    |
|                                   | from that date                    |
|                                   |                                   |
|                                   | Due to aged care system           |
|                                   | limitations, after the            |
|                                   | application of the annual cap,    |
|                                   | the anniversary date changes to 3 |
|                                   | May                               |
|                                   |                                   |
|                                   | In this transition year, the      |
|                                   | annual cap is applied and the     |
|                                   | fees will reset more than 12      |
|                                   | months from the start date        |
|                                   |                                   |
|                                   | Home care information does not    |
|                                   | transfer to SPARC. A manual       |
|                                   | review is created based on the    |
|                                   | date the care recipient is        |
|                                   | expected to reach their lifetime  |
|                                   | cap                               |
|                                   |                                   |
|                                   | When the care recipient reaches   |
|                                   | their lifetime cap, all of their  |
|                                   | previous home care fees paid are  |
|                                   | included in a manual adjustment   |
|                                   | calculation                       |
