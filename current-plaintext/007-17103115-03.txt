| 11                                | Activity Test result + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | The hours of subsidised care a    |
|                                   | customer is entitled to is based  |
|                                   | on their hours of recognised      |
|                                   | activity. This is referred to as  |
|                                   | the Activity Test result.         |
|                                   |                                   |
|                                   | For partnered customers, the      |
|                                   | subsidised hours will be based on |
|                                   | the member of the couple with the |
|                                   | lowest Activity Test result, even |
|                                   | when an exemption applies.        |
|                                   |                                   |
|                                   | If both members of a couple have  |
