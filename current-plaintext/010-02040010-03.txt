Coding ABSTUDY independence when a customer is currently or previously married, or in a registered relationship

Table 2: For ABSTUDY Smart Centre Processing staff only.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Assessing eligibility + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | Check the evidence provided to    |
|                                   | determine if the student or       |
|                                   | Australian Apprentice meets       |
|                                   | eligibility for independence due  |
|                                   | to them being currently or        |
|                                   | previously married or in a        |
|                                   | registered relationship.          |
|                                   |                                   |
|                                   | If the Marital Status (MS) screen |
|                                   | is coded as currently Married     |
|                                   | (MAR) or Registered Relationship  |
|                                   | (REG), they are automatically     |
|                                   | assessed as independent. The      |
|                                   | Independent/Homeless/Away from    |
|                                   | Home Details (NIH) screen does    |
|                                   | not need to be coded with the     |
|                                   | independence details.             |
|                                   |                                   |
|                                   | Is the student currently married  |
|                                   | or in a registered relationship?  |
|                                   |                                   |
|                                   | -   Yes: Link the student or      |
|                                   |     Australian Apprentice and     |
|                                   |     their partner to ensure the   |
|                                   |     correct ABSTUDY rate is paid. |
|                                   |     See Change in relationship    |
|                                   |     status from single to         |
|                                   |     partnered. Once linked there  |
|                                   |     is no further action          |
|                                   |     required. The system will     |
|                                   |     grant independence            |
|                                   |     automatically from the MS     |
|                                   |     coding. Procedure ends here.  |
+-----------------------------------+-----------------------------------+
| 2                                 | Previous circumstances + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Was the student previously        |
|                                   | married or in a registered        |
|                                   | relationship?                     |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, the student or Australian |
|                                   |     Apprentice is not eligible    |
|                                   |     for independence under the    |
|                                   |     married or registered         |
|                                   |     relationship criteria. Record |
|                                   |     the details of this decision  |
|                                   |     in a DOC. Procedure ends here |
+-----------------------------------+-----------------------------------+
| 3                                 | Coding independence details +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Student or Australian Apprentice  |
|                                   | meets eligibility for             |
|                                   | independence due to previously    |
|                                   | being married, or in a registered |
|                                   | relationship.                     |
|                                   |                                   |
|                                   | Complete the following fields on  |
|                                   | the NIH screen:                   |
|                                   |                                   |
|                                   | Action, key 'IND'                 |
|                                   |                                   |
|                                   | Start/Reject Reason, key 'PBM'    |
|                                   |                                   |
|                                   | Start Date, key the later of:     |
|                                   |                                   |
|                                   | -   ABSTUDY start date            |
|                                   | -   date the student or           |
|                                   |     Australian Apprentice became  |
|                                   |     independent                   |
|                                   |                                   |
|                                   | Record the details of the         |
|                                   | decision in a DOC.                |
|                                   |                                   |
|                                   | Go to Step 4.                     |
+-----------------------------------+-----------------------------------+
| 4                                 | Determine if a current Child      |
|                                   | Support assessment exists + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | To check if an assessment exists  |
|                                   | for the student or Australian     |
|                                   | Apprentice, see Maintenance       |
|                                   | Income Test (MIT) for ABSTUDY and |
|                                   | Youth Allowance (YA).             |
|                                   |                                   |
|                                   | Is there a current Child Support  |
|                                   | assessment for the student or     |
|                                   | Australian Apprentice?            |
|                                   |                                   |
|                                   | -   Yes, go to Step 5             |
|                                   | -   No, procedure ends here       |
+-----------------------------------+-----------------------------------+
| 5                                 | Terminating event for Child       |
|                                   | Support send ACTDOC to CSJ + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Send an ABSTUDY Child Support     |
|                                   | Update Fast Note. Select Auto     |
|                                   | text, use Students, ABSTUDY and   |
|                                   | Apprentices > Update > ABSTUDY    |
|                                   | Child Support Update              |
|                                   |                                   |
|                                   | Complete the details within the   |
|                                   | Fast Note and select Confirm,     |
|                                   | ensuring the Fast Note remains    |
|                                   | open.                             |
