Coding ABSTUDY independence when a customer is currently or previously married, in a de facto relationship of at least 6 months, or in a registered relationship

Table 2: For ABSTUDY Smart Centre Processing staff only.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Assessing eligibility + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | Check the evidence provided to    |
|                                   | determine if the student or       |
|                                   | Australian Apprentice meets       |
|                                   | eligibility for independence due  |
|                                   | to currently or previously        |
|                                   | married, in a de facto            |
|                                   | relationship of at least 6        |
|                                   | months, or in a registered        |
|                                   | relationship.                     |
|                                   |                                   |
|                                   | If the Marital Status (MS) screen |
|                                   | is coded as Married (MAR) or      |
|                                   | Registered Relationship (REG),    |
|                                   | they are automatically assessed   |
|                                   | as independent. The               |
|                                   | Independent/Homeless/Away from    |
|                                   | Home Details (NIH) screen does    |
|                                   | not need to be coded with the     |
|                                   | independence details.             |
|                                   |                                   |
|                                   | Is there enough evidence to       |
|                                   | support granting independence due |
|                                   | to currently or previously        |
|                                   | married, in a de facto            |
|                                   | relationship of at least 6        |
|                                   | months, or in a registered        |
|                                   | relationship?                     |
|                                   |                                   |
|                                   | Yes, if processed in:             |
|                                   |                                   |
|                                   | -   Customer First, go to Step 4  |
|                                   | -   Process Direct, go to Step 2  |
|                                   |                                   |
|                                   | No, if processed in:              |
|                                   |                                   |
|                                   | -   Customer First, go to Step 5  |
|                                   | -   Process Direct, go to Step 3  |
+-----------------------------------+-----------------------------------+
| 2                                 | Coding independence details in    |
|                                   | Process Direct + Read more ...    |
|                                   |                                   |
|                                   | The independence data will        |
|                                   | pre-fill the NIH screen from      |
|                                   | information in the claim.         |
|                                   |                                   |
|                                   | Check the NIH screen via Super    |
|                                   | Key or by selecting the My Circs  |
|                                   | icon.                             |
|                                   |                                   |
|                                   | If the information has not        |
|                                   | pre-filled or needs to be         |
|                                   | changed, code manually. For       |
|                                   | example, an incorrect start date  |
|                                   | has been recorded.                |
|                                   |                                   |
|                                   | If manual coding is required      |
|                                   | complete the following fields:    |
|                                   |                                   |
|                                   | Action, key 'IND'                 |
|                                   |                                   |
|                                   | Start/Reject Reason, key 'PBM'    |
|                                   |                                   |
|                                   | Start Date, key the later of:     |
|                                   |                                   |
|                                   | -   ABSTUDY start date            |
|                                   | -   date the student or           |
|                                   |     Australian Apprentice became  |
|                                   |     independent                   |
|                                   |                                   |
|                                   | Link the student or Australian    |
|                                   | Apprentice and their partner to   |
|                                   | ensure the correct ABSTUDY rate   |
|                                   | is paid. See Change in            |
|                                   | relationship status from single   |
|                                   | to partnered.                     |
|                                   |                                   |
|                                   | Go to Step 6.                     |
+-----------------------------------+-----------------------------------+
| 3                                 | Deleting independence details in  |
|                                   | Process Direct + Read more ...    |
|                                   |                                   |
|                                   | If the student or Australian      |
|                                   | Apprentice does not meet the      |
|                                   | eligibility for independence due  |
|                                   | to currently or previously        |
|                                   | married, in a de facto            |
|                                   | relationship of at least 6        |
|                                   | months, or in a registered        |
|                                   | relationship, the pre-filled      |
|                                   | information must be deleted from  |
|                                   | the NIH screen.                   |
|                                   |                                   |
|                                   | To delete an entry:               |
|                                   |                                   |
|                                   | -   go to the NIH screen via      |
|                                   |     Super Key or by selecting the |
|                                   |     My Circs icon                 |
|                                   | -   select the Delete icon to the |
|                                   |     left of the line where        |
|                                   |     start/reject reason PBM is    |
|                                   |     coded                         |
|                                   | -   select Yes to confirm         |
|                                   |     deletion                      |
|                                   |                                   |
|                                   | Go to Step 6.                     |
+-----------------------------------+-----------------------------------+
| 4                                 | Coding independence details in    |
|                                   | Customer First + Read more ...    |
|                                   |                                   |
|                                   | Student or Australian Apprentice  |
|                                   | meets eligibility for             |
|                                   | independence due to currently or  |
|                                   | previously married, in a de facto |
|                                   | relationship of at least 6        |
|                                   | months, or in a registered        |
|                                   | relationship.                     |
|                                   |                                   |
|                                   | Complete the following fields on  |
|                                   | the NIH screen:                   |
|                                   |                                   |
|                                   | Action, key 'IND'                 |
|                                   |                                   |
|                                   | Start/Reject Reason, key 'PBM'    |
|                                   |                                   |
|                                   | Start Date, key the later of:     |
|                                   |                                   |
|                                   | -   ABSTUDY start date            |
|                                   | -   date the student or           |
|                                   |     Australian Apprentice became  |
|                                   |     independent                   |
|                                   |                                   |
|                                   | Link the student or Australian    |
|                                   | Apprentice and their partner to   |
|                                   | ensure the correct ABSTUDY rate   |
|                                   | is paid. See Change in            |
|                                   | relationship status from single   |
|                                   | to partnered.                     |
+-----------------------------------+-----------------------------------+
| 5                                 | Record decision in Customer       |
|                                   | First + Read more ...             |
|                                   |                                   |
|                                   | Record details of the decision in |
|                                   | a DOC. The information must       |
|                                   | indicate the grounds on which the |

|                                   | granted or rejected, and if       |
|                                   | rejected, the reasons for the     |
|                                   | rejection.                        |
|                                   |                                   |
|                                   | Has the claim for independence    |
|                                   | been granted?                     |
|                                   |                                   |
|                                   | -   Yes, go to Step 7             |
|                                   | -   No, procedure ends here       |
+-----------------------------------+-----------------------------------+

|                                   | Direct + Read more ...            |
|                                   |                                   |
|                                   | Record details of the decision in |
|                                   | Notes or Document annotations.    |
|                                   | The information must indicate the |
|                                   | grounds on which the claim for    |
|                                   | independence has been granted or  |
|                                   | rejected, and if rejected, the    |
|                                   | reasons for the rejection.        |
|                                   |                                   |
|                                   | Has the claim for independence    |
|                                   | been granted?                     |
|                                   |                                   |
|                                   | -   Yes, go to Step 7             |
|                                   | -   No, procedure ends here       |
+-----------------------------------+-----------------------------------+

|                                   | Support assessment exists + Read  |
