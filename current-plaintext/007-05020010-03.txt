PPL/DAP details for customers with children born or entering care before
1 July 2023 only.

This page contains a summary of how to process a claim for Dad and
Partner Pay (DAP).

All FCC processing staff, select the Process Direct tab.

The Customer First tab is only for staff where there is written
directive to handover the claim to Customer First.

| 6                                 | Special circumstances - family    |
|                                   | and domestic violence and the     |
|                                   | work test + Read more ...         |
|                                   |                                   |
|                                   | Customers who do not meet the DAP |
|                                   | work test are asked in their      |
|                                   | online claim if there were any    |
|                                   | special circumstances that        |
|                                   | affected their ability to meet    |
|                                   | the work test. If a customer      |

|                                   | priority keyword (PPLFDV) is      |
|                                   | automatically added to the claim  |
|                                   | work item.                        |
|                                   |                                   |
|                                   | The customer must provide         |
|                                   | evidence of employment confirming |
|                                   | they would have worked during the |
|                                   | work test period. The customer's  |
|                                   | employer may not know why the     |
|                                   | customer was unable to work, and  |
|                                   | the customer is not expected to   |
|                                   | tell their employer.              |
|                                   |                                   |
|                                   | A social worker referral needs to |
|                                   | be completed when allocated a     |
|                                   | family and domestic violence      |
|                                   | claim.                            |
|                                   |                                   |
|                                   | Before sending a social working   |
|                                   | referral, check the customer's    |
|                                   | employment evidence lodged with   |
|                                   | their claim to assess if          |
|                                   | sufficient. For evidence          |
|                                   | requirements, see Paid Parental   |
|                                   | Leave scheme Work Test.           |
|                                   |                                   |
|                                   | If not sufficient, attempt to     |
|                                   | contact the customer by phone to  |
|                                   | request the missing claim         |
|                                   | information. Issue a written      |
