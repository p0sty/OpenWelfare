This page contains a summary of how to process a claim for Dad and
Partner Pay (DAP).

All FCC processing staff, select the Process Direct tab.

The Customer First tab is only for:

-   Bereavement claims trained processing staff, or
-   where there is a written directive to handover the claim to Customer
    First

|                                   | Special circumstances - family    |
|                                   | and domestic violence             |
|                                   |                                   |
|                                   | Customers who do not meet the DAP |
