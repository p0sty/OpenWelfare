+-----------------------------------+-----------------------------------+

|                                   | automatically generated each time |

|                                   | by 4 numeric values. The claim ID |

|                                   | any time. The claim ID will be    |
|                                   | the same.                         |
+-----------------------------------+-----------------------------------+
| 3                                 | Question: What happens if invalid |
|                                   | data is entered into DVA          |
|                                   | Webclaim? + Read more ...         |
|                                   |                                   |
|                                   | Answer: Claims are assessed in    |
|                                   | real-time. When invalid data is   |
|                                   | entered, DVA Webclaim displays an |
|                                   | error message at the top of the   |
|                                   | page which includes instructions  |
|                                   | on how to resolve the error.      |

|                                   | services and vouchers can be      |

|                                   | ...                               |

|                                   | -   Each voucher can have a       |
|                                   |     maximum of 6 items            |

|                                   |     have the same date-of-service |
|                                   |     and treatment location        |

|                                   | medical schedule items can be     |
|                                   | claimed by via DVA Webclaim. Only |
|                                   | private hospital admissions       |
|                                   | community nursing and nursing     |
|                                   | home contributions are excluded.  |
+-----------------------------------+-----------------------------------+
| 6                                 | Question: Can customers submit    |

Allied health FAQs

FAQs for allied health professionals

Table 6: This table describes answers to allied health professional
queries.

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Question: Do customers have to    |
|                                   | submit a separate item for        |
|                                   | loadings? + Read more ...         |
|                                   |                                   |
|                                   | Answer: No, any item that         |
|                                   | attracts a loading will           |
|                                   | automatically add the additional  |
|                                   | amount to the benefit.            |
+-----------------------------------+-----------------------------------+
| 2                                 | Question: How do customers claim  |
|                                   | dental step-down or jaw indicator |
|                                   | items? + Read more ...            |
|                                   |                                   |
|                                   | Answer:                           |
|                                   |                                   |
|                                   | -   DVA Webclaim will identify    |
|                                   |     any item that requires the    |
|                                   |     customer to specify if the    |
|                                   |     service was performed on the  |
|                                   |     upper or lower jaw. A         |
|                                   |     drop-down list will appear    |
|                                   |     against these items and the   |
|                                   |     customer must select either   |
|                                   |     UPR (upper) or LWR (lower).   |
|                                   | -   Items that have a two-tiered  |
|                                   |     fee structure require the     |
|                                   |     customer to enter the treated |
|                                   |     tooth number.                 |
+-----------------------------------+-----------------------------------+
| 3                                 | Question: How do customers claim  |
|                                   | Remote Area Allowance item        |
|                                   | OT80? + Read more ...             |
|                                   |                                   |
|                                   | Answer: The customer needs to key |
|                                   | OT80 in the line immediately      |
|                                   | after the associated consultation |
|                                   | item. The Distance Travelled      |
|                                   | field will open and the customer  |
|                                   | must enter the distance travelled |
|                                   | on the same voucher.              |
+-----------------------------------+-----------------------------------+
| 4                                 | Question: How do customers claim  |
|                                   | optical lens items when there is  |
|                                   | a valid reason for exceeding      |
|                                   | items provided within the past 2  |
|                                   | years? + Read more ...            |
|                                   |                                   |
|                                   | Answer: When an optical lens or   |
|                                   | frame item is claimed, DVA        |
|                                   | Webclaim will automatically       |
|                                   | display the Replacement Reason    |
|                                   | field if the veteran has received |
|                                   | glasses in the past 2 years.      |
|                                   | Customers must select from:       |
|                                   |                                   |
|                                   | -   LS - Lost                     |
|                                   | -   BK - Broken                   |
|                                   | -   RC - Significant Refraction   |
|                                   |     Change                        |
|                                   |                                   |
|                                   | The customer does not have to     |
|                                   | submit the veteran's signed       |
|                                   | declaration but the health        |
|                                   | professional is required to keep  |
|                                   | it for their own records.         |
+-----------------------------------+-----------------------------------+
| 5                                 | Question: How should a customer   |
|                                   | claim travel (kilometres)? + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Answers: Kilometres are claimed   |
|                                   | by entering KM in the Item field  |
|                                   | below the associated consultation |
|                                   | item.                             |
|                                   |                                   |
|                                   | -   Enter the number of           |
|                                   |     kilometres travelled in the   |
|                                   |     Distance Travelled field      |
|                                   | -   A minimum of 10 and a maximum |
|                                   |     of 999 KM can be entered      |
