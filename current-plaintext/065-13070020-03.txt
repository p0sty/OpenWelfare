Processing advice regarding a service under administration

Table 3: This table describes how Team Leaders and Subject Matter
Experts process a service under administration. This process is similar
to transferring a service.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Email advice received by Services |
|                                   | Australia + Read more ...         |
|                                   |                                   |
|                                   | The Department of Health notifies |
|                                   | Services Australia of aged care   |
|                                   | services that are under           |
|                                   | administration.                   |
|                                   |                                   |
|                                   | Email notification is sent by the |
|                                   | Department of Health to the Aged  |
|                                   | Care liaison inbox.               |
+-----------------------------------+-----------------------------------+
| 2                                 | Checking the claim history + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Service Officers must check the   |
|                                   | claiming history of the aged care |
|                                   | service that is under             |
|                                   | administration. This service must |
|                                   | be up to date with their claims.  |
|                                   |                                   |
|                                   | For example, if the closure date  |
|                                   | is 1 June 2019, the aged care     |
|                                   | service is up to date with their  |
|                                   | claiming when the service has     |
|                                   | submitted the May 2019 claim.     |
|                                   |                                   |
|                                   | Is the aged care service up to    |
|                                   | date with their claims?           |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, contact the aged care     |
|                                   |     service and ask to lodge the  |
|                                   |     claims. Procedure ends here   |
+-----------------------------------+-----------------------------------+
| 3                                 | Process the last monthly claim +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | To process the last monthly       |
|                                   | claim:                            |
|                                   |                                   |
|                                   | -   ACSP – see Step 7 in the      |
|                                   |     Using the Aged Care Staff     |
|                                   |     Portal home page table        |
|                                   | -   ACMPS - see Process claim     |
|                                   |     summary table                 |
+-----------------------------------+-----------------------------------+
| 4                                 | Suspend monthly advance + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Suspend the future monthly        |
|                                   | advance for the service that is   |
|                                   | under administration. This        |
|                                   | ensures the correct payment of    |
|                                   | subsidies to the approved         |
|                                   | provider who is operating the     |
|                                   | service.                          |
|                                   |                                   |
|                                   | To suspend advances in:           |
|                                   |                                   |
|                                   | -    ACSP – see Step 11 in the    |
|                                   |     Using the Aged Care Staff     |
|                                   |     Portal (ACSP) to manage       |
|                                   |     advances for residential aged |
|                                   |     care services (Service in     |
|                                   |     Context) table                |
|                                   | -   ACMPS – see the Suspend and   |
|                                   |     unsuspend advances table      |
+-----------------------------------+-----------------------------------+
| 5                                 | Update bank account details +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Update bank details received via  |
|                                   | PaNDA (email, post and fax).      |
|                                   |                                   |
|                                   | To update the bank account        |
|                                   | details in:                       |
|                                   |                                   |
|                                   | -   ACSP – see the Using the Aged |
|                                   |     Care Staff Portal for aged    |
|                                   |     care services' bank account   |
|                                   |     details (Service in Context)  |
|                                   |     table                         |
|                                   | -   ACMPS – see the Manage bank   |
|                                   |     account details in ACMPS      |
|                                   |     table                         |
+-----------------------------------+-----------------------------------+
| 6                                 | Lift suspension + Read more ...   |
|                                   |                                   |
|                                   | Lift the suspension (e.g.         |
|                                   | unsuspend the advance suspension) |
|                                   | on the advance and pay any        |
|                                   | outstanding advances.             |
|                                   |                                   |
|                                   | To unsuspend advances in:         |
|                                   |                                   |
|                                   | -   ACSP – see Step 13 in the     |
|                                   |     Using the Aged Care Staff     |
|                                   |     Portal (ACSP) to manage       |
|                                   |     advances for residential aged |
|                                   |     care services (Service in     |
|                                   |     Context) table                |
|                                   | -   ACMPS - see Suspend and       |
|                                   |     unsuspend advances table      |
|                                   |                                   |
|                                   | To pay any outstanding advances   |
|                                   | in:                               |
|                                   |                                   |
|                                   | -   ACSP – see Step 9 in the      |
|                                   |     Using the Aged Care Staff     |
|                                   |     Portal (ACSP) to manage       |
|                                   |     advances for residential aged |
|                                   |     care services (Service in     |
|                                   |     Context) table                |
|                                   | -   ACMPS - see Verify and        |
|                                   |     approve an advance table      |
+-----------------------------------+-----------------------------------+
| 7                                 | Reply to Department of Health     |
|                                   | when processed + Read more ...    |
|                                   |                                   |
|                                   | Reply to the Department of Health |
|                                   | when all actions have been        |
|                                   | processed.                        |
