|                                   | frequency                         |

|                                   | 1WE or 2WE frequency. If coding   |
|                                   | income using IOP or LOP, leave    |

|                                   | correcting employment income      |
|                                   | details for more information      |
|                                   |                                   |
|                                   | Income support payments will      |
|                                   | cancel automatically when the     |
|                                   | customer's Working Credit has     |
