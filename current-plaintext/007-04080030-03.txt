| 3                                 | Did Not Attend (DNA) process +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | If the customer fails to attend   |
|                                   | the appointment their payment     |

|                                   | -   any recent events (as         |
|                                   |     documented) that may impact   |
|                                   |     on the customers capacity to  |
|                                   |     comply, or                    |

|                                   |     unable to comply, or had a    |

|                                   |     doing so                      |

|                                   | customer's record.                |

|                                   | type) DOV.                        |

+-----------------------------------+-----------------------------------+
| 6                                 | PP compulsory participation +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | It is the Service Officer's       |
|                                   | responsibility to provide all     |
|                                   | relevant information throughout   |
|                                   | the interview to the customer.    |
|                                   |                                   |
|                                   | Confirm if the customer is        |
|                                   | participating in any of the       |
|                                   | following activities at this      |
|                                   | time:                             |
|                                   |                                   |
|                                   | -   Self-employment suitable paid |
|                                   |     work, for at least 30 hours   |
|                                   |     per fortnight (the income     |
|                                   |     declared must meet the        |
|                                   |     national minimum wage). For   |
|                                   |     self-employed customers the   |
|                                   |     Net Business Income needs to  |
|                                   |     be the equivalent or more of  |
|                                   |     30 hours per fortnight at the |
|                                   |     national minimum wage         |
|                                   | -   Studying for at least 30      |
|                                   |     hours per fortnight in an     |
|                                   |     approved course               |
|                                   | -   Combination of paid work      |
|                                   |     (including self-employment)   |
|                                   |     and study of at least 30      |
|                                   |     hours per fortnight           |
|                                   |                                   |
|                                   | If the customer is self-employed  |
|                                   | check how recent the last         |
|                                   | business income assessment is and |
|                                   | determine if a new assessment is  |
|                                   | required.                         |
|                                   |                                   |
|                                   | It is also necessary to determine |
|                                   | if the customer may be eligible   |
|                                   | for a long term exemption from    |
|                                   | their mutual obligation           |
|                                   | requirements based on their       |
|                                   | current circumstances. This       |
|                                   | includes exemptions due to:       |
|                                   |                                   |
|                                   | -   caring for a large family     |
|                                   |     (where the customer is the    |
|                                   |     principal carer of 4 or more  |
|                                   |     dependent children under 16   |
|                                   |     years of age, or a secondary  |
|                                   |     pupil child from 16-19 years  |
|                                   |     of age)                       |
|                                   | -   having a dependent child      |
|                                   |     home-schooled                 |
|                                   | -   supervising distance          |
|                                   |     education of a dependent      |
|                                   |     child                         |
|                                   | -   being a registered and active |
|                                   |     Foster Carer                  |
|                                   | -   caring for a sick or disabled |
|                                   |     child but not qualify for     |
|                                   |     Carer Payment (CP)            |
|                                   | -   being a relative carer (other |
|                                   |     than a parent) with a         |
|                                   |     parenting or protection order |
|                                   |     under the Family Law Act      |
|                                   | -   being a family or community   |
|                                   |     member (not the parent)       |
|                                   |     caring for a child under a    |
|                                   |     care plan supported by the    |
|                                   |     state/territory child         |
|                                   |     protection authority          |
|                                   |                                   |
|                                   | Evidence may be required from the |
|                                   | customer to support these         |
|                                   | exemptions.                       |
|                                   |                                   |
|                                   | Go to Step 7.                     |
+-----------------------------------+-----------------------------------+
| 7                                 | Information to discuss with the   |
|                                   | customer + Read more ...          |
|                                   |                                   |
|                                   | -   Discuss options for           |
|                                   |     satisfying their mutual       |
|                                   |     obligation requirements. If   |
|                                   |     the customer is not already   |
|                                   |     satisfying the requirements,  |
|                                   |     offer some examples of how to |
|                                   |     do so.                        |
|                                   | -   Explain the customer's        |
|                                   |     obligations in relation to    |
|                                   |     finding and accepting         |
|                                   |     suitable child care, being    |
|                                   |     clear that approved child     |
|                                   |     care is considered to be      |
|                                   |     suitable, and must be         |
|                                   |     accepted if it is available.  |
|                                   |     If approved child care is not |
|                                   |     available, it is up to the    |
|                                   |     customer to determine whether |
|                                   |     other child care options are  |
|                                   |     suitable. Consider using the  |
|                                   |     Child Care Availability       |
|                                   |     System to help the customer   |
|                                   |     locate child care services.   |
|                                   |     Discuss Child Care Subsidy    |
|                                   |     (CCS) and Additional Child    |
|                                   |     Care Subsidy (ACCS)           |
|                                   | -   Discuss additional            |
|                                   |     allowances, Pension Education |
|                                   |     Supplement and Education      |
|                                   |     Entry Payment (EdEP) for      |
|                                   |     full-time study               |
|                                   | -   Discuss available programs    |
|                                   |     and services, Adult Migrant   |
|                                   |     English Program (AMEP),       |
|                                   |     Skills for Education and      |
|                                   |     Employment (SEE), Financial   |
|                                   |     Counsellors, 'We Speak Your   |
|                                   |     Language' brochures, etc.     |
|                                   | -   Mention that the customer may |
|                                   |     be required to report         |
|                                   |     personally, on a regular      |
|                                   |     basis, even if an exemption   |
|                                   |     is granted. Clarify that this |
|                                   |     type of reporting is          |
|                                   |     different from how they may   |
|                                   |     be currently reporting their  |
|                                   |     income ([] online, phone, in  |
|                                   |     person or in writing by       |
|                                   |     fax/post/third party). If     |
|                                   |     personal lodgement of SU19    |
|                                   |     forms will be required        |
|                                   |     (customer is not eligible for |
|                                   |     self service reporting),      |
|                                   |     discuss whether a change in   |
|                                   |     payday code may make it       |
|                                   |     easier for them to attend on  |
|                                   |     a particular day, and give    |
|                                   |     them the option to change     |
|                                   |     either now or in the future.  |
|                                   |     Explain that customers may    |
|                                   |     discuss their needs with the  |
|                                   |     agency at any time            |
|                                   | -   Explain that the customer's   |
|                                   |     reporting options will be     |
|                                   |     confirmed at the end of the   |
|                                   |     interview, based on the       |
|                                   |     assessment that is made       |
|                                   |     during the interview          |
|                                   | -   Explain the compliance        |
|                                   |     framework, and the possible   |
|                                   |     penalties which can be        |
|                                   |     applied once compulsory       |
|                                   |     participation requirements    |
|                                   |     start                         |
|                                   |                                   |
|                                   | See the Resources page for a link |
|                                   | to mutual obligation requirements |
|                                   | and exemptions for principal      |
|                                   | carers information on the         |
|                                   | Services Australia website.       |

|                                   | -   SEL for customers fully       |

|                                   | -   PFT for customers fully       |

|                                   |     satisfying using a            |

|                                   |     satisfying using a            |
