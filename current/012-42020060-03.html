Assess a request to recognise additional training in FPS

Table 1

Step

Action

1

Notice of additional training in FPS received + Read more ...

Determine the organisation the notification of training in focussed psychological strategies was received from:

  • GPMHSC - notification to register are received via email from the GPMHSC with the subject line 'GPMHSC application to register as a GP provider of FPS’
  • RACGP - notification to register are received via email from the RACGP on behalf of GPMHSC with the subject line 'GPMHSC application to register as a GP provider of FPS’

If the request comes from any other source, then re-categorise the application in PaNDA to National Office Escalation.

2

Confirm Provider details on Provider Directory System (PDS) + Read more ...

  • Open PDS:
    • Select Provider
    • Select View
  • Search Detailed Search tab, by name
  • In the Family name field, key the first 3 letters of the last name, followed by a %
  • In the Given name field, key the first 3 letters of the first name, followed by a %
  • Select Medical Practitioner from the Profession drop down menu
  • Select Search to display a list of health professionals with a matching name and profession

Note: Detailed Search will ensure all records for the medical practitioner are located.

Does the medical practitioner's name and provider number (if provided) match the request?

  • Yes, go to Step 3
  • No, the information does not match contact a LPS for assistance

If an exact match is found and multiple stems exist then select one of the provider numbers. Update all unsynchronised linked stems individually and contact a LPS who will escalate the details to a PO for synchronising.

Does the PDS stem have the 540 Specialist General Practitioner code open and current?

  • No, go to Step 4

Update Specialty Code + Read more ...

  • Specialty Code - 665 (Mental Health Focussed Psychological Strategies)
  • Start Date:
    • If the date on the report is after the start date of the relevant specialty code in Table 1 Step 5 then key the start date as advised in the report
