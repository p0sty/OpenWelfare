NURIMM

Authorised Nurse Immuniser

Ancillary organisation

Enrolled Nurses and Registered Nurse/Midwifes who hold the relevant qualifications for administering immunisations. Each State and Territory have legislation that determines who meets the requirements to be an Authorised Nurse Immuniser.

For this provider type to report vaccinations to the AIR:

  • each authorised nurse immuniser must have an active ABN to complete the PRODA organisation registration process, or
  • plans on administering vaccinations independently, or
  • administers vaccines at a location that does not already have an AIR provider number

A Nurse Immuniser working at a location that already has an AIR provider number (Council, Commercial provider, medical practice) must not apply.

Troubleshooting error messages in PDS

Table 5

Message

Action

Green – information

An information message displays to advise that the action has been successful. An information message may also appear to advise that data has/has not been found, or advise of other data entry issues.

Orange – warning

A warning message displays to advise there may be a problem with the information entered. The message will indicate what the warning relates to. Check the data, make any necessary changes and then select Save to continue processing if the information is correct.

Red – error

An error message displays to advise there has been a problem processing the information entered. The message will indicate what the error relates to. The error will need to be fixed before the data can be saved.

Pop-up

Pop-ups alert Service Officers to important information for example, navigating away from a page without saving. Service Officers must acknowledge the pop-up by selecting OK or Cancel to continue.
