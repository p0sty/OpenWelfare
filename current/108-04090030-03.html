7

Locating the managed investment + Read more ...

Check the Investment Summary (MIS). Expand the Managed Investments (MIP) and Unregistered Managed investments (MIUS) sections

Only enquiries, revaluations and updates for managed investments can be completed over the phone.

Documents are not required. Only request further contact if the customer cannot provide all information.

If there are multiple updates, update the details where information can be given. When details are unable to be finished, handoff or request information.

If the customer wants to update their investments over the phone, they need to give details of all managed investments before coding updates.

If the customer advises a change of ownership, this can be coded at the Smart Centre.

  • For example, to correct a managed investment from 50% to 100%:
  • Update the managed investment to zero value, as ownership % cannot be changed, then:
  • add a new managed investment to code correct details

All new acquisitions of managed investments are to be completed with documents. This ensures that customers and staff select the correct investment.

If the investment is a margin loan coded using the Managed Investment Product List (MIP) screen, an MFU review will not generate for March and September bulk updates. Code a manual review for the March or September bulk updates. Request the latest statement from the lender showing the current market value of the managed investments and the current loan balance amount.

Follow a workaround for MIS/SIS updates just before 20 March or 20 September. If the bulk update for 20 March or 20 September has been actioned, the event date will show as a future date when updating a managed investment.

  • Go to Managed Investments (MIP)
  • Select Add on the MIP line to access the Create Managed Investment (MIP) options
  • Search for the managed investment by coding the details for the managed investment in 1 or more of the following fields:
    • Fund Manager: must have at least 2 characters, the first must be a letter
    • APIR Code: enter the Asia Pacific Investment Register (APIR) code. This code must be exact to obtain a match. A valid code has 9 characters. The first 3 are letters, the next 4 are numbers, and the final 2 are 'AU'. If coding this field do not code the Fund Manager, Product Group or Product Option

    • Product Option: multiple keywords can be entered, separated by a comma. Fund Manager must also be coded

Managed investment not on the MIN system + Read more ...

Select the investment from MIUS by expanding the investment.

Update the details of the investment. A Manual Follow-up (MFU) review will generate in March and September. This is to check if the investment can be moved to MIP if it is on the MIN database, and to update the investment value(s).

Select Add icon on the Unregistered Managed Investment History item line to access the Create Unregistered Managed Investment Entry options.

If the investment is a unit-based product, contact the MIN/SIN Helpdesk for an updated unit price.

If the investment is an account-based product, confirm with the customer the current account dollar value.

Once the changes are completed, the system will update the value of all shares and managed investments.

2

Updating Unregistered Managed Investments (MIUS) screen + Read more ...

Select Edit in the header details.

Update Item Details:

  • Event Date: the date of the change
  • Funeral Exemption: select Yes if the investment is an exempt funeral investment
  • Rural Exemption Code: select ASS - Asset Value to be exempt if the customer is claiming a rural payment and the investment is exempt from the assets test. For example, for superannuation if the fund is in the accumulation phase
  • If the investment is unit based:
    • check with the customer the number of units held and key the Number of Units: field. Whole numbers only. If the customer owns a partial number of units, round down to the nearest whole number
    • update the value of each unit in the Unit Value Amount: field
  • Asset Ded Amt $: If applicable, key the loan balance amount which is deductible from the investment value. In dollars and cents
  • If the investment is account based, confirm with the customer the current dollar value and key the account balance in the Total Asset Value: field. In dollars and cents. Note: for margin loans, key the gross current market value
  • Select Save
  • Update Receipt Date and Channel and select Save
  • Select Assess
  • Check the outcome on Entitlements (ELD) and select Finish
  • Finalisation Notes pre-fill with details of the income and assets update. Update to record any extra details
  • Select Finalise to complete the transaction

Notes from the update will copy to Document List (DL) in all systems.

Go to Step 7.

Updating Managed Investment (MIP) for a customer + Read more ...

If the bulk update for 20 March or 20 September has been actioned the event date will show as a future date. Follow the workaround for MIS/SIS updates before 20 March or 20 September to complete the update.

Update Item Details:

  • Event Date: the date of the change
  • Policy number: key the policy/account/ certificate number for the investment if it has not previously been recorded (Note: If this number exceeds 10 characters, key the last 10 only)
  • Funeral Exemption: select Yes if the investment is an exempt funeral investment
  • Rural Exemption Code: select ASS - Asset Value to be exempt if the customer is claiming a rural payment and the investment is exempt from the assets test. For example, for superannuation if the fund is in the accumulation phase
  • If applicable, update the loan amount deducted for assets test purposes in the Asset Deduction Amount $: field. In dollars and cents. Note: the amount recorded is deducted from the investment amount for the assets test
  • If the investment is unit-based, confirm the number of units held by the customer and key in the Number of Units: field. Whole numbers only. If the customer owns a partial number of units, round down to the nearest whole number
  • If the investment is account based, confirm the dollar value with the customer and update the amount (in dollars and cents) in the Account Amount: field. Note: for margin loans, key the gross current market value
  • Select Save
  • Select Assess
  • Check the outcome on Entitlements (ELD) and select Finish
  • Finalisation Notes pre-fill with details of the income and assets update. Update to record any extra details
  • Select Finalise to complete the transaction

Withdrawal from a Managed Investment + Read more ...

  • If the customer has withdrawn all funds from the investment, zero the number of units or balance. Do not delete the investment
  • Event Date: key the date of the change
  • If the investment is unit based, confirm with the customer the number of units held and key the Number of Units: field. Whole numbers only. If the customer owns a partial number of units, round down to the nearest whole number
  • If the investment is account based, confirm with the customer the current dollar value and key the account balance in the Total Asset Value: field. In dollars and cents. Note: for margin loans, key the gross current market value
  • Asset Ded Amt $: If applicable, key the loan balance amount in dollars and cents
  • Select Save
  • Update Receipt Date and Channel and select Save
  • Select Assess
  • Check the outcome on Entitlements (ELD) and select Finish
  • Finalisation Notes pre-fill with details of the income and assets update. Update to record any extra details
  • Select Finalise to complete the transaction

Notes from the update will copy to Document List (DL) in all systems.

Finalising the managed investment update + Read more ...

Advise the customer:

  • there is a bulk update of the value of all listed share and unitised managed investments each March and September
  • customers can ask for a review of the value of their investments at any time

Record details on a Note using the Add note function if required.

If the new investment is a margin loan, an MFU review will generate for March and September bulk updates each year. This is to request the latest statement from the lender showing the current market value and the current loan balance amount. Code a manual review outside March or September only if required.

Withdrawal from a Managed Investment + Read more ...

Before beginning this step, check the MIS screen to see if the managed investment(s) had already been recorded. Follow up details and record any balance not previously recorded.

  • While in the customer's record, go to the MI screen by selecting the appropriate investment from the MIS screen
  • If the customer has withdrawn all funds from the investment, zero the number of units or balance. (Note: do not 'd'elete or 'c'orrect the investment.)
  • If the investment is unit-based, confirm the number of units held by the customer and update the new number of units in the Units: field. Note: the system will not allow the coding of decimals. If the customer owns a partial number of units, round down to the nearest whole number
  • If the investment is account-based, confirm the account dollar value with the customer and update the new account balance (in dollar and cents) in the $: field. Note: for margin loans, key the gross current market value
  • On the MIUS screen zero out the Total Asset Value if the unit numbers and unit value have been coded. These field will also need to be blanked out
  • Key the date from which the change in the investment circumstances occurred in the Event Date: field
  • If applicable, key the loan balance amount (in dollars and cents) in the Asset Test Deduction Amt: field
  • Key the source of the information in the Source: field
  • Key the date of receipt of the information in the DOR: field
  • Key 'I' in the Action: field
  • Press [Enter]

Complete the activity on the AR screen.

Ensure FAL screen is checked and any review for the return of the documentation completed.

7

Finalising the managed investments + Read more ...

Advise the customer that there is a bulk update of the value of all listed share and unitised managed investments each March and September. Also advise that customers can ask for a review of the value of their investments at any time.

If the new investment is a margin loan, a Manual Follow-up (MFU) review will generate for March and September bulk updates. Request the latest statement from the lender showing the current market value and the current loan balance amount. Code a manual review outside of March or September only if required.
