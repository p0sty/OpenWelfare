Confirm with practice + Read more ...

Tell the practice contact that when forms are received manually Services Australia will contact them to confirm they sent the form.

  • internet trained, help the practice or organise another time to call back
  • not internet trained, offer to transfer to the AIR Internet helpdesk and give the practice the phone number

  • did not send the form, go to Step 9

  • Subject heading - Manual Lodgement, Provider Number
  • Attach a copy of the form that prompted the call
  • Details of who the Service Officer spoke to, the initial conversation/outcome and reason for not submitting digitally

Check for duplicate registrations + Read more ...

To check for a duplicate registration, select the Detailed Search tab and enter the following details:

  • Postcode range

Check result + Read more ...

Is there a possible duplicate registration?

Do the details match between an organisation(s) listed and the registration being created?

  • Yes, list of possible matches displays. Go to Step 6
  • No, the Organisation Registration Create page displays. Go to Step 7

Note: compare ABN and street name to prevent duplication.

6

Check for matches with listed providers + Read more ...

If there is no match between a provider listed and the registration being created, contact the applicant to confirm whether a change of ownership has occurred. If a change of ownership has occurred, go to Step 7.

If there a match between a provider listed and the registration being created and:

  • an AIR program registration exists:
    • stop processing application
    • contact the applicant to confirm the reason for the submission of a duplicate registration
    • a security check must be done before releasing any registration information
    • release the existing AIR provider number to the contact person over the phone
  • no AIR program registration exists, see Table 4. Note: if some details match and there are unfilled fields on the form, do not amend the form. Create a new registration

7

Organisation Registration Create + Read more ...

Key the details exactly as they are on the form. This includes organisations with ‘The’, numbers and special characters e.g. & or @ in the business name.

On the Organisation tab, enter the following details:

  • Type - select Health Care Practice from the dropdown menu
  • Start Date - over key the pre-populated date with the state/territory health approval date or the signature/receipt date for medical practice provider types
  • ABN - key in the Australian Business Number (ABN) if provided on the form

8

Physical Address + Read more ...

Select the Physical Address tab and record the following details:

  • Locality or Postcode - start to key the postcode or suburb, rapid addressing will list possible matches. Select the locality from the dropdown list
  • Address Line:
    • Key the address details. Rapid addressing will list possible matches
    • Select the address from the dropdown list
    • If the address cannot be found in the rapid addressing, select Manual Entry and key details
    • The Address Summary displays the address details

AIR do not use the Associate and Claiming Channels tabs. Do not complete these sections.

9

Contact details + Read more ...

Select the Contact tab and record the following details:

  • Title - select from dropdown list
  • Surname - key surname given on the form
  • First Name - key first name given on the form
  • Telephone - select country and area code from dropdown menus and key phone number
  • Select Save

The ancillary provider registration form asks ‘L type - Medical practices’ for the principal vaccination provider’s Medicare provider number, but this is not recorded on PDS.

First contact + Read more ...

Attempt to contact the provider or the provider’s authorised representative using the contact details in PDS. Do not use the contacts details provided on the IM005, HW027, HW029 or HW052 forms.

Do not contact provider if processing HW027, HW029 or HW052 forms, as the bank details have been confirmed by Medicare. If this is marked duplicate, check recorded bank details. If they do not match, contact the provider.

If unable to locate any provider contact details in PDS:

  • Enter an appropriate comment in PaNDA (do not escalate the query to AIR Helpdesk). For example, genuine attempt to contact the provider not possible as unable to locate provider in PDS. Unable to return the form
  • Mark the work item as complete, see Work Optimiser for staff
  • Procedure ends here

If located provider contact details:

  • Make a genuine attempt to contact the provider, or their authorised representative
  • Confirm they have sent a request to update their bank account details

Note: if the provider or their representative indicate they did not request an update to their bank details, report the suspected fraud.

Was contact successful and did the provider confirm the update request is genuine?

  • Yes, put comments in PDS and PaNDA, see below. Then continue with the update, go to Step 9

Delegating Bank Details for approval + Read more ...

Service Officers cannot approve or reject bank details they have recorded. Another authorised Service Officer must approve the bank details. An error message displays if a Service Officer tries to approve bank details they have recorded.

To submit Bank Details for approval:

  • Select Send for Delegation
  • A Send for Delegation pop up window displays, select Delegate
  • This will un-assign the work item from the user and automatically capability adjust the work to the delegate’s skill tag. The work item will now show as Delegation Required
  • The delegate will approve and link the bank details, then return the work item to the processing Service Officer
  • The processing Service Officer checks the bank details are linked correctly and finalises in PaNDA. Procedure ends here

Cancel ancillary provider registration + Read more ...

See the Change ancillary provider details table on the Resources page for details that can be updated and how to amend provider details.

To cancel an ancillary provider registration:

  • Select the ancillary provider’s AIR program registration
  • Select Amend. The Program Registration Summary displays
  • Record the following details:
    • Registration End Date - key the end date or select the calendar icon. Enter the date indicated on the request or the date the cancellation request was received if no date is provided
    • Registration End Reason - if the practice is closed, select LOC closed, for all other cancellation reasons, select Other from the End Date Reason dropdown menu
  • Select the Comment tab to record a comment and select Add New to save
  • Select Save twice
  • Finalise the document as per WLM processing guidelines

Do the details match between an organisation listed and the registration being created (Service Officers must compare ABN and street name)?

  • Yes, and the details are the same and an AIR program registration exists:
    • stop processing application
    • contact the applicant to determine the reason for the submission of a duplicate registration

Organisation Registration Create + Read more ...

Key details exactly as they are on the form. This includes organisations with ‘The’, numbers and special characters e.g. & or @ in the business name.

On the Organisation tab, enter the following details:

  • Type - select Health Care Practice from the dropdown menu
  • Start Date - over key the pre-populated date with the state/territory health approval date or the signature/receipt date for medical practice provider types
  • ABN - key in the Australian Business Number (ABN) if provided on the form

6

Physical Address + Read more ...

Select the Physical Address tab, select Manual Entry, and record the following details:

  • Locality or Postcode - start to key the postcode or suburb for the state/territory health jurisdiction as per the below options:
    • Australian Capital Territory enter Canberra, 2601
    • New South Wales (including Norfolk Island) enter Sydney, 2000
    • Northern Territory enter Darwin, 0810
    • Queensland enter Brisbane, 4000
    • South Australia enter Adelaide, 5000
    • Tasmania enter Hobart, 7000
    • Victoria enter Melbourne, 3000
    • Western Australia enter Perth, 6000
  • Address Line:
    • Building or Department – key in Organisation/Business name provided in Question 4 and/or 6 of the IM004 form
    • Street Name - key in Commercial Prov Address

The Address Summary displays the address details.

AIR do not use the Associate and Claiming Channels tabs. Do not complete these sections.
