Create a free text Q888 letter

Table 3

Item

Approved text for Q888 letter

Approved Q888 text

Instructions: Use the text below, one sentence per paragraph and select '2' for all letter options except for selecting '1' at SSALF and PRVACT.

Our records show you have previously received support from a State or

Territory welfare authority.

If this support has stopped or changed, you may be able to get a higher

rate of Youth Allowance.

So that we can check if you can get a higher rate, you will need to get a

letter from your State or Territory welfare authority.

The letter will need to show us if you are receiving any support from the

State or Territory. The letter will also need to show what the support is

for, such as food, housing or help with study or training.

If you were receiving support which has since ceased, the letter should

include the date it ceased.

If you start to receive any type of support in the future, you need to tell

us within 14 days.

If you would like to speak with us about this request, please call 13 24 90.
