Online Advices (OLA) for requesting information

Item

Description

1

List of OLAs

These letters can be used to request information from a customer or third party:

  • QSS32 - Free-text questionnaire for National usage
  • Q146 - Request information from a customer's employer
  • Q608 - Request information from a financial institution regarding a customer's bank accounts
  • Q604 - Request information from a financial institution regarding a customer's loan details

Note: the list is not exhaustive.
