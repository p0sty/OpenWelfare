Assess Part A and B of the IP001 form

Table 1

Step

Action

1

Check form version + Read more ...

Check the form version number on the bottom left corner of the form. Check if this is acceptable in the Practice Incentives forms list.

Is the form version acceptable?

  • Yes, go to Step 2
  • No, return the form to the practice without checking any further details

2

Part A - Eligibility requirements and practice details + Read more ...

Assess the form.

The practice must provide their public liability insurance and professional indemnity cover details at Questions 1 and 2.

If the practice has answered:

  • Yes to any of these questions, check the practice has completed the Company name and Expiry date fields for the public liability insurance at Question 1 (the policy number is optional)
  • No, or left blank any of these questions, check the rest of the form to see if any other details are needed or if the practice is not eligible for the PIP and/or the WIP - Practice Stream before returning the form to the practice

Note:

  • The public liability insurances may be accepted if covered by a State government. See State government public liability insurance on the Resources page
  • If expired public liability insurance details have been provided, confirm that the insurance information is current
    • This information can be verified with an authorised contact person or owner by phone. Add a comment to the practice profile advising the information was verified verbally
    • If unable to contact by phone, check the rest of the form to see if any other details are needed before returning the form to the practice

3

Eligibility requirements - accreditation + Read more ...

The practice must be accredited or registered for accreditation at Question 3 or 4.

If the practice has selected Yes to either accreditation question, check if:

  • The practice has provided a copy of their accreditation certificate, and
  • The details on the accreditation certificate (status of accreditation, practice name and address, current start and end dates) are valid, current and match the practice details on the form

If the practice has indicated No or the details are incomplete,

  • Check the rest of the form to see if any other details are needed, or if the practice is not eligible for the PIP and/or the WIP - Practice Stream
  • Return the form to the practice
  • Record the practice details on the spreadsheet G:SPS_Eastwood/PIP and WIP - Practice Stream New Applications UNABLE to Register Online

If the practice has provided a copy of their accreditation certificate but has not selected Yes on the form, this is acceptable.

4

Program participation + Read more ...

Check which program/s the practice wants to apply for at Question 5.

If the practice has not indicated which program they want to apply for, or they have indicated only 1 -program, check the answers on the application form to see which program/s to register the practice in.

For example, Part C of the application refers to the PIP and Part D of the application refers to the WIP - Practice Stream.

5

Practice details + Read more ...

The practice must provide:

  • Practice name at Question 6,
  • Practice type at Question 7, and
  • Full practice address at Question 8

If the above practice details are incomplete:

  • Check the rest of the form to see if any other details are needed
  • Return the form to the practice

6

Practice contact details and HPOS notifications + Read more ...

The practice must provide a practice phone number at Question 9.

If the above practice detail is missing or incorrect, check the rest of the form to see if any other details are needed before returning the form to the practice.

The practice does not need to provide these details:

  • Fax number and practice email at Question 9
  • Postal address at Question 10
  • Do not want to use HPOS messages at Question 11

If the practice has indicated online notifications but has not provided Registration Authority (RA) numbers for authorised contacts or owners, assess these questions as the practice does not wish to receive notifications through HPOS messages facility. Advise the practice of this in the practice approval letter.

7

Authorised contact person + Read more ...

The practice must provide at least 1 Authorised contact person at Question 12.

Check that:

  • The authorised contact person(s) Title, Family name and First given name are legible, and
  • The signature has been provided

If the above practice details are incomplete:

  • Check the rest of the form to see if any other details are needed
  • Return the form to the practice

The Registration Authority (RA) number is not mandatory.

8

Bank account details + Read more ...

The practice must provide at Question 13:

  • BSB number
  • Account number
  • Account name

If the above practice details are incomplete:

  • Check the rest of the form to see if any other details are needed
  • Return the form to the practice

The practice does not need to provide the Name of bank, building society or credit union.

9

Part B - Additional practice branches + Read more ...

Check if the practice has answered Yes to:

  • Having more than 1 location at Question 14, and
  • That 1 or more general practitioners and/or nurse practitioners from the main practice address also practise at the additional practice branch(es) at Question 15

If Questions 14 and 15 are blank but the practice has additional practice branch details at Question 16, and provided additional practice branch details in Part F, this is acceptable.

If they have answered:

  • Yes, go to Step 10
  • No, see Table 2

10

Address of additional practice branch(es) + Read more ...

The practice must provide the below Additional practice branch details at Question 16

  • Full practice address
  • Accreditation type
  • Accreditation certificate (if accredited or registered for accreditation)

Are the Additional practice branch details complete and the accreditation certificate supplied if applicable?

  • Yes, see Table 2
  • No, see Table 2 to check the rest of the form to see if any other details are needed before returning the form to the practice

Part C - Individual PIP Incentives Indigenous Health Incentive (IHI) + Read more ...

Check if the practice has indicated their eligibility for the Indigenous Health Incentive (IHI) at Questions 17 and 18.

If the practice has selected N/A at Question 18, this is acceptable. The practice needs to show that they are meeting these requirements if they are audited.

If the above responses are No or incomplete, check the rest of the form to see if the practice is applying for any other individual incentives.

2

After Hours Incentive + Read more ...

Check the payment level selected for the After Hours Incentive at Question 20.

  • Not made a selection, continue assessing the form to see if the practice is applying for other individual incentives, go to Step 3
  • Selected more than 1 payment level, the practice can only apply for 1 payment level
    • Continue assessing the form to see if any other details are needed before returning the form to the practice
    • Request confirmation of the After Hours Payment level the practice is applying for. Either request more information (z1736) or advise of approval in PIP (z1597)

3

Procedural General Practitioner Payment + Read more ...

Check if the practice has indicated eligibility for the Procedural General Practitioner payment at Question 21?

  • Yes, this can only be processed if the practice is located in a RRMA classification 3-7, see Table 3
  • No, see Table 3

If the practice is not located in a RRMA classification 3-7, the Procedural GP Payment will not be an available incentive to apply for when completing the application in PIP Online.

Applying or signing up for individual incentives is optional. An application should not be returned to the practice for PIP if they do not apply for any individual incentives.

WIP - Practice Stream participation + Read more ...

Does the practice already participate in the PIP (Question 23), and consent to use the PIP practice information for the WIP - Practice Stream (Question 24)?

Take care when checking these questions. Practices may be unsure of what they should select if applying for both programs at the same time. If the practice has answered No to Question 23 and Yes to Question 24 the application is assessed as PIP consenting.

  • Yes, and the PIP practice ID is:
    • Provided, go to Step 3

3

Professional indemnity cover + Read more ...

Check if the practice has indicated Yes to all nurses, allied health professionals, Aboriginal and Torres Strait Islander health workers and Aboriginal and Torres Strait Islander health practitioners at the practice having the level of professional indemnity insurance specified by the relevant board arrangements at Question 25?

  • Yes, go to Step 4
  • No, the practice is not eligible to participate in the WIP - Practice Stream
    • If the practice is eligible for PIP go to Step 6
    • If a WIP - Practice Stream only application, check the rest of the form to see if any other details are needed or the practice is not eligible for the WIP - Practice Stream
    • Return the form to the practice

4

Practice nurse and health professional details + Read more ...

Check if the practice employs/retains the services of a:

  • Nurse practitioner or registered nurse at Question 26
  • Enrolled nurse at Question 27
  • Aboriginal and Torres Strait Islander health worker/practitioner at Question 28, or
  • Allied health professional at Question 29

If any questions are Yes, go to Step 5.

If all questions are No, the practice is not eligible to participate in the WIP - Practice Stream.

If the practice is eligible for PIP, go to Step 6

If a WIP - Practice Stream only application, return the form to the practice. See Step 7 in Table 4. If all questions are incomplete, check the rest of the form to see if any other details are needed before returning the form to the practice.

If all questions are incomplete, check the rest of the form to see if any other details are needed before returning the form to the practice.

If some questions are incomplete, we can accept:

  • Completed equivalent health professional hours at Question 30 or
  • A letter with an official company letterhead reporting the average weekly hours or an employment contract reporting the average weekly hours of the practice nurse or health professional

5

Practice nurse and health professional details + Read more ...

Check that the practice has completed the below at Question 30 for each health professional or allied health professional:

  • Health professional or allied health professional type and
  • Actual average weekly hours

If the details are missing or incorrect:

  • Check the rest of form to see if any other details are needed
  • Return the form to the practice

The practice does not need to provide these details:

  • Title
  • Family name
  • First given name
  • Unique identifier must be provided if Ahpra number or HPI-I number is selected

6

Part E - Practice ownership details and signatory requirements + Read more ...

The practice must provide:

  • Ownership arrangement type at Question 31
  • Company name at Question 32
    • If the ownership arrangement type is an individual proprietor, the Company name is optional
  • Ownership address at Question 32

If the above practice details are missing or incorrect:

  • Check the rest of form to see if any other details are needed
  • Return the form to the practice

The practice does not need to provide these details:

  • Trading name at Question 32
  • Australian Business Number (ABN) at Question 32
  • The checklist at Question 33

7

Declaration + Read more ...

Check the declaration details on the form at Question 36. There must be:

  • Signatures of the required number of practice owners
  • Full names of all the practice owners
  • Positions held for all the practice owners (unless the owner is an individual proprietor)

For assistance, see the Signatory requirements table on the Resources page.

Note:

The different ownership type boxes in the declaration (Question 36) are designed to direct owners to sign in the correct ownership type and obtain the correct number of signatures

If the declaration (Question 36) has been signed in the incorrect ownership type box, or existing fields have been used instead of adding another page and:

  • it is clear that owners have signed in the wrong boxes, this is acceptable
  • it is not clear, the form must be returned to the practice requesting them to sign in the appropriate areas

In a Body Corporate arrangement, the same person may be both the director and secretary of the corporation. This is acceptable providing it has been clearly stated on the form.

Are the declaration details complete?

  • Yes, see Table 4
  • No, see Table 4 to check if any other details are needed before returning the form to the practice

2

Check provider number details + Read more ...

The practice must provide the provider number for the practitioner at the main practice address at Question 39

If the above practice details are missing or incorrect:

  • Check the rest of the form to see if any other details are needed
  • Return the form to the practice

To confirm the provider number quoted matches the practitioner’s name and main practice address, check the details are correct in the Provider Directory System (PDS).

Log on to the PDS system via the Medicare Portal.

To search for the practitioner, from the Provider Directories menu:

  • Select Provider location
  • Select View from the drop down
  • Key the Provider stem (first 6 digits) in the Provider Stem field
  • Key the Provider location (7th character, alpha or numeric) in the Provider location field
  • Select Search. The Search Results will show

Check the search results to ensure the following details match the form:

  • Provider Number
  • Provider Name
  • Address

If the details do not match:

  • Check the rest of the form to see if any other details are needed
  • Return the form to the practice

Note:

  • The practice name on the form does not need to match the practice name on the PDS
  • If the provider’s address in PDS does not exactly match the address on the form or in PIP but is the same locality an address variance may be accepted

3

Check Provider Start Date + Read more ...

Check the start date at the main practice address has been provided at Question 39.

If the start date on the form is:

  • Before the PDS date, use the PDS start date
    • The PIP/WIP - Practice Stream system will not accept a start date before PDS
    • Add a comment confirming the details once the application has been approved
  • After the PDS date, use the form start date
    • This is the date the practice is advising when the provider commenced
  • Not provided, use the PDS start date

4

Check additional practice branch + Read more ...

Check that the practitioner has completed the additional practice branch details at Question 40.

Yes, check PDS to confirm the provider numbers match the practitioners name and additional practice branch address.

Note: if an additional practice branch provider number at Question 40 is provided and Question 16 in Part B is not complete, search for the practice branch in PIP Online to confirm that the practice branch address is not already registered.

If the additional practice branch is:

  • Already registered in the PIP/WIP - Practice Stream and the practitioner is listed on the practice profile, no action is required
  • Already registered in the PIP/WIP - Practice Stream and the practitioner is not listed on the practice profile, continue to register the practitioner at the main practice address and advise the practice a separate Practice Incentives Individual general practitioner or nurse practitioner details form (IP003) is needed for the additional practice branch details
  • Not registered, check the rest of the form, go to Step 5

5

Procedural general practitioner details + Read more ...

Check if the practitioner has indicated they are a procedural general practitioner at Question 42.

A practice is only eligible for the procedural general practitioner payment if the practice is within a RRMA classification 3-7. If the practice is not within a RRMA classification, 3-7 the general practitioner cannot be added as a procedural general practitioner, go to Step 6.

If the practitioner has indicated:

  • Yes and:
    • Indicated the same provider number listed at either Question 39 or 40, and indicated a Tier at Question 43, go to Step 6
    • Indicated a different provider number to that listed at either Question 39 or 40, check the rest of the form to see if any other details are needed before returning the form to the practice
    • Not indicated a Tier at Question 43, check the rest of the form to see if any other details are needed before returning the form to the practice
  • No or incomplete and:
    • Not selected a Tier at Question 43, questions are not mandatory, go to Step 6
    • Selected a Tier at Question 43, the form must be returned to the practice for the practitioner to confirm their eligibility, check the rest of the form to see if any other details are needed before returning the form to the practice

6

Declaration + Read more ...

Check the declaration details on the form at Question 45.

There must be:

  • The signature of the practitioner
  • Their name
    • Accept the form if there is only a last name
    • If there are no name details, check the signature
    • Accept the form if the name can be identified from the signature

Are the declaration details complete?

The declaration does not need to be dated.

  • Yes, see Table 5
  • No, go to Step 7

PIP Eligibility Check + Read more ...

On the PIP Eligibility Check screen, complete the following details:

  • Date of lodgement
  • Does your practice have public liability insurance? (Yes/ No)
  • Do all medical practitioners at the practice have current professional indemnity cover? (Yes/No)
    • If the practice has no public liability or professional indemnity insurance continue registering the application in to initial status then decline the application
  • Is your practice currently accredited? (Yes/No), or
  • Is your practice currently registered for accreditation? (Yes/No)
    • Select the Accreditation Body - AGPAL, GPA Plus or ACHS
    • Key the Accreditation Number (key as shown on the accreditation certificate)
    • Key the Accreditation Start Date - as shown on the accreditation certificate
    • Key the Accreditation End Date - 12 months from date of approval in the PIP for registered for accreditation. For example, if approval date is 10/10/18 use end date 10/10/19. For accredited practices, use the end date as shown on the accreditation certificate
    • For assistance see the Update accreditation details table in Practice accreditation for PIP and WIP - Practice Stream
  • Practice Name (key as shown at Question 6 of the IP001 form)
  • Practice type (select as shown at Question 7 of the IP001 form)
  • Main Practice Location (key as shown at Question 8 of the IP001 form)
  • Select Check PIP Eligibility
    • Check PIP Eligibility will not be enabled until all the mandatory eligibility check questions are answered
    • If warning message 2307 appears, confirm the warning message and continue to process the form. Address variances has more information

The Eligibility Check will return 1 of the following messages:

  • 'Based on your answers your practice is eligible for PIP'
  • 'Based on your answers your practice is not eligible for PIP'

The status of the application will now be in Initial and a 5 digit Application Number will be assigned to the practice.

  • If the application cannot be processed or other details are needed, go to Step 7
  • If the application can be processed, see Table 6

Contact the practice + Read more ...

Contact a registered owner or authorised contact person via phone. Advise the practice:

  • an IP001 form is received, but the practice is already approved for the PIP and/or WIP – Practice Stream (whichever is applicable)
  • there are discrepancies on the IP001 compared to what is registered on the practice profile
  • the IP001 form is only acceptable at the time of application
  • any changes to practice details must be made via HPOS or the relevant form by the point-in-time date
    • Assist the practice to update their details via HPOS. The Resources page has a link to the HPOS User Guide.
    • Help the practice locate the relevant form. The Resources page has a link to forms.
  • a letter will be sent advising that the practice is already approved for the program(s)

13

Finalise work item + Read more ...

  • Return the form to the practice. Use the Request for more information for the PIP and WIP - Practice Stream (z1736) letter. Advise:
    • the practice is already approved for the program
    • any changes to practice details must be made via HPOS or the relevant form by the point-in-time date
  • Upload the letter and the IP001 form to the practice profile
  • Add a note to the practice profile. Confirm the details of the conversation, including:
    • the owner or authorised contact person
    • the details on the IP001 form that do not match the practice profile
    • If the details were updated via HPOS, or if the relevant form is to be submitted by the point-in-time date
  • Finalise the Work Item as Complete in PaNDA

If the practice is not a PIP consenting practice, upload the letter to both profiles.
