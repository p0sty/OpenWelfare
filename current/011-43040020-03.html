This page contains procedures to assist Service Officers in actioning claims in pend for bulk bill latter day adjustments (LDA).

On this Page:

Assessing adjustments

Delete a claim after assessment

Reset a claim

Update a claim by changing the control line

Assessing adjustments

Procedure to assess bulk bill adjustments

Table 1: This table describes the procedure to assess bulk bill adjustments.

Step

Action

1

Accessing adjustment screen + Read more ...

  • Key DHAI,PROVIDER No,CLAIM ID,DOL=DDMMYY and press [Enter]

2

Background assessing check + Read more ...

The system checks the claim and if background assessing is:

  • Not completed - the error message 3-STATUS is returned
  • Completed - the first DHAC screen for the claim is displayed

Note: when the claim includes more than one adjustment, [F3-NX ADJ] message is displayed.

3

Getting to the next adjustment + Read more ...

Press [F3-NEXT ADJ] to display the next patient adjustment.

Delete a claim after assessment

After assessment of claim and prior to updating, the claim may be deleted.

Procedure to delete a claim after assessment

Table 2: This table describes the procedure to delete a claim after assessment.

Step

Action

1

Accessing screen to delete claim + Read more ...

Key:

  • DHAI,PROVIDER No,CLAIM ID,DOL=DDMMYY,DELETE or
  • DHAC,PROVIDER No,CLAIM ID,DOL=DDMMYY,DELETE

2

Finalising deletion of claim + Read more ...

Press [Enter].

The system returns the message ADJ DEL OK, when the adjustment claim has been deleted, this includes registration of the claim.

Reset a claim

After assessment of the claim and prior to updating, the claim may be reset.

Procedure to reset a claim

Table 3: This table describes the procedure to reset a claim.

Step

Action

1

Accessing screen to reset a claim + Read more ...

Key DHAC,PROVIDER No,CLAIM No,DOL=DDMMYY,RESET.

Press [Enter].

2

Check resetting of claim results + Read more ...

The system resets the claim to the first adjustment ready. This allows the operator to press [F3 NX ADJ] through the claim and amend any errors made, keeping all previously made changes.

Update a claim by changing the control line

Final DHAC screen

When the final adjustment for the claim has been resolved the system displays the message U TO UPDTE. When the message U TO UPDTE is displayed on the DHAC screen, change the control line to complete the processing.

Note: any warning messages remain on the screen.

Procedure to update a claim by changing the control line

Table 4: This table describes the procedure to update a claim by changing the control line.

Step

Action

1

Over keying control line + Read more ...

Over key DHAC with DHAU.

2

Finalising the process + Read more ...

Press [Enter].

The system returns a DHAI screen with the message UPDATE OK and the total result of all adjustments process in the claim is displayed.
