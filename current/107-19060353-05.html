CCS Debt Resource Codes

Table 2

CCS Debt Resource Code

Access Information

ZC.D4.DHS.Debt.Officer

All staff who require access to CCS debts in Process Direct must request the Debt Officer resource code.

This access is View Only.

Staff who require coding access to CCS debts to make changes and updates must also request 1 of the Delegate Access levels below.

ZC.D4.DHS.DEBT.DELG.APS1

ZC.D4.DHS.DEBT.DELG.APS2

ZC.D4.DHS.DEBT.DELG.APS3

ZC.D4.DHS.DEBT.DELG.APS4

ZC.D4.DHS.DEBT.DELG.APS5

ZC.D4.DHS.DEBT.DELG.APS6

ZC.D4.DHS.DEBT.DELG.EL1

ZC.D4.DHS.DEBT.DELG.EL2

ZC.D4.DHS.DEBT.DELG.SES

The Delegate Access grants the staff member full access and coding ability approved for that particular delegation level.

The staff member must also have, or requested the Debt Officer access.

The delegation level requested should be applicable to the APS level the staff member is undertaking. Only 1 level is to be requested.

Provide specific details about why coding access is needed. Do not use acronyms. Insufficient reasoning can lead to rejection of the request.

Staff without the authority to make updates on customer records must request the Debt Officer View Only access. Any Delegate Access requests will be rejected.
