1

Evidence requirements for ACCS (Temporary Financial Hardship) + Read more ...

Customers need to submit evidence before they can submit a combined CCS/ACCS (Temporary Financial Hardship) claim (online or via the ACC). Excludes CCS claims combined with Family Tax Benefit (FTB), Paid Parental Leave (PPL) or Dad and Partner pay (DAP).

Customers can submit some CCS/ACCS claims and ACCS (Temporary Financial Hardship) applications without uploading evidence for:

  • standalone ACCS applications (considered a Change of Circumstances), or
  • CCS/ACCS claims combined with FTB, PPL or DAP

For these applications, customers are advised what evidence is required and the due date on their receipt page. This is deemed a request for information, so a further request is not needed unless additional information is required.

Although customers are advised they have 14 days to provide evidence, an extra 8 days is provided for postage. Therefore:

  • if evidence is not provided within 22 days (14 days + 8 days for postage) of submitting their application,
  • the claim will be taken as not having been made (not effective)

The customer will need to submit a new application if they wish to apply for ACCS (Temporary Financial Hardship). Payment can only be backdated up to 28 days.

If a period of reduced evidence is approved, where the customer or their partner receive an applicable government disaster payment, they will only need to show proof of this payment). No further evidence will be required to demonstrate financial hardship. See the Resources page for a list of approved events and payments.

Evidence must be provided:

  • for an ACCS (Temporary Financial Hardship) application to be effective, and
  • to enable eligibility for ACCS (Temporary Financial Hardship) to be assessed

If the customer has experienced more than 1 Temporary Financial Hardship event, evidence is required for all events included in the application. If not provided for all events, the application will be Not Effective. This is because the application is assessed as a whole, and not separated depending on evidence provided.

Customers may lodge an ACCS (Temporary Financial Hardship) claim in anticipation of an event occurring.

If deemed effective, claims cannot be finalised prior to the event occurring.

Select Documents to display all documentation attached to the record.

Note: before requesting documents, review all evidence provided and on the customer's record that supports the application. This will ensure only 1 letter requesting documents is sent to the customer.

If evidence has been provided or is on the customer's record, confirm it meets the evidence requirements. The Resources page lists suitable evidence for each event.

As well as evidence of the event(s), customers must provide a causal link between the event and their temporary financial hardship. This may include a Commonwealth statutory declaration and supporting evidence of expenses incurred (bank statements, invoices).

Is there enough evidence to assess the ACCS (Temporary Financial Hardship) application?

  • No, go to Step 2

Determine whether further information evidence is to be requested + Read more ...

  • key the transaction ID in the Process Direct inbox, or select the transaction ID from the Transactions icon in the Customer Summary

If the customer:

  • is clearly not eligible for ACCS (Temporary Financial Hardship), (for example the event is more than 6 months in the past, ACCS has already been paid for the event), for:
    • CCS/ACCS (Temporary Financial Hardship) combined claim see Step 1 in Table 3
    • Standalone ACCS (Temporary Financial Hardship) application, see Step 3 in Table 3
  • provided appropriate evidence with their application and/or request for additional evidence, but further information is required to be able to assess the application, go to Step 3
  • provided document/s that do not align with the requested evidence and are unrelated to the ACCS (Temporary Financial Hardship) event, for:
    • CCS/ACCS (Temporary Financial Hardship) combined claim, see Step 1 in Table 3
    • Standalone ACCS (Temporary Financial Hardship) application, see Step 3 in Table 3
  • did not provide documents within the requested timeframe (from original standalone application request or additional information requests), for:
    • CCS/ACCS (Temporary Financial Hardship) combined claim see Step 1 in Table 3
    • Standalone ACCS (Temporary Financial Hardship) application, see Step 3 in Table 3

6

Less than 13 weeks of ACCS (Temporary Financial Hardship) has been paid for 1 or more children + Read more ...

If the customer has been granted ACCS (Temporary Financial Hardship) for less than 13 weeks and they are still experiencing financial hardship due to the same event, they can apply for ACCS (Temporary Financial Hardship) again for the same event

The event must still have occurred within 6 months of the date of the subsequent application. The customer can only be granted the balance of the maximum 13 weeks.

Were all children included in the previous ACCS (Temporary Financial Hardship) assessment?

  • Yes, make note of remaining weeks per child. Go to Step 8
  • No, go to Step 7

7

Children not included in previous ACCS (Temporary Financial Hardship) assessment + Read more ...

A determination of ACCS (Temporary Financial Hardship) is linked to a particular child and a particular customer. If the customer has multiple children, they can select which children the application relates to. The application can be used to make multiple determinations (1 for each child).

If a customer chooses not to select all children, and apply for them at a later date, it is important to note the application for each child must be made within 6 months of the date of the event being claimed for.

Make note of the remaining weeks for each child the customer has indicated as part of the claim.

Coding ACCS Temporary Financial Hardship (TFH) Table + Read more ...

To update or record information relating to the ACCS (Temporary Financial Hardship) application:

  • CCS/ACCS combined claim - select the My Child Care icon and open the ACCS Details twisty and ACCS Temporary Financial Hardship (ACCSTFH) twisty
  • ACCS Stand Alone application - Enter TFH in the SuperKey field to go to the TFH table

Although the customer may not be eligible for ACCS (Temporary Financial Hardship), all mandatory fields for each event are to be completed before finalising the application.

A line will display under the ACCS Temporary Financial Hardship (TFH) Table for each event type. Service Officers are to review evidence associated with each event and complete the relevant fields for each event type.

If an FA086 application form has been lodged, select to record all details from information on the form.

For all applications, review information recorded in comparison to evidence provided. Update information in the following fields as required, including:

  • Application source, will pre-fill if the customer submitted their application online or via an ACC
  • WithDrawn, will default to No. Only select Yes if the customer has requested withdrawal of application or lodged in error and note record as per Table 4, Step 9
  • Event Type, ensure the event type selected aligns with the customer's temporary financial event
  • Event Date, date the event occurred (adjust if required based on evidence provided)
  • Claim Date, defaults to the date the claim/application was submitted. If the customer has lodged an FA086, ensure the date the form was lodged is used
  • Significant Extra Expenses ($), information keyed by the customer. This field will not display for events loss of income or business failure, or loss of child support
  • Significant Extra Expenses Frequency, information keyed by the customer. Note: this field will not display for events loss of income or business failure, or loss of child support
  • Loss/Reduction of Income ($), information keyed by the customer
  • Loss/Reduction of Income Frequency, information keyed by the customer
  • Reduced Ability to Pay Fee, key Yes if you have determined that the customer has had a substantial reduction in their ability to pay child care fees, key No - if no substantial reduction in their ability to pay fees
  • Evidence Provided Code, select the relevant evidence type from the drop down list, or 'Not Provided' if no evidence was submitted
  • Evidence Provided Text, this field will only display when 'Other' evidence type is selected. Key a description of the evidence provided
  • Considered, leave blank if the customer has only indicated 1 event. If the customer has nominated multiple events in their application, select YES against the event(s) being considered to grant ACCS

9

Granting ACCS (Temporary Financial Hardship) - combined claims + Read more ...

Check and correct additional information relevant to a grant of ACCS (Temporary Financial Hardship).

  • Grant, select Yes. This field will only display if evidence has been provided and the Service Officer has confirmed a reduced ability to pay fees. If the customer has experienced multiple events and ACCS is be granted, record 'Yes' against the most significant event
  • ACCS Grant Weeks, select the number of weeks from the dropdown menu. If ACCS (Temporary Financial Hardship) has been paid for the same event, these weeks need to be deducted from the maximum 13 weeks
  • ACCS End Date, defaults to the correct date based on the number of grant weeks keyed

When Yes is keyed, the child details will display. As ACCS is granted at the child level, details for each child in the application must be completed:

  • Child CRN, the child/ren in the application will display
  • Child name, auto populates based on the child's CRN
  • Child Rejection Reason, leave blank when granting
  • Child Start Date, defaults to the date indicated by the customer within the claim, or the system will apply a date based on the Claim Date
  • Child Grant Weeks, key the number of weeks ACCS is payable. If ACCS (Temporary Financial Hardship) has been paid for the child for the same event, these weeks need to be deducted from the maximum 13 weeks. For example, if ACCS has been paid for 6 weeks for the child for the same event, select 7 from the dropdown menu
  • Child End Date, automatically calculates based on the Child Start Date and the number of Child Grant Weeks

To finalise combined CCS/ACCS (Temporary Financial Hardship) applications, see Step 2 in Table 4.

Granting ACCS (Temporary Financial Hardship) - standalone applications + Read more ...

  • ACCS Grant Weeks, select the number of weeks from the dropdown menu. If ACCS (Temporary Financial Hardship) has been paid for the same event, these weeks need to be deducted from the maximum 13 weeks

If the ACCS (Temporary Financial Hardship) has been paid for the child for the same event, these weeks need to be deducted from the maximum 13 weeks. (For example, if ACCS has been paid for 6 weeks for the child for the same event, select 7 from the dropdown menu)

  • End Date, will default to the correct date based on the number of grant weeks keyed
