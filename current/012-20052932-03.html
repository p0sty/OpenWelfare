Open online application for allied health professionals

Table 1

Step

Action

1

Log on to the EHP (eHealth Program) Home Page + Read more ...

Open EHP and WLM.

2

Is the WLM (Workload Management) work item available? + Read more ...

  • Yes, go to Step 3
  • No, go to Step 4

3

Locate application + Read more ...

In EHP:

  • Select Application in the HI Search menu bar
  • Copy and paste the Work Item ID from WLM into the Application ID field
  • Select Search

Procedure ends here. Go to Process online HPI-I application for allied health professionals table to process application.

4

WLM work item is not available + Read more ...

In EHP the Search for individual screen shows.

  • Select Application in the HI Search menu bar
  • Select Pending or Hold (if going back into an application) from the Status drop down list
  • Select Register Allied Health Profession from the Application type drop down list
  • Select Search

To open an application:

  • Select the Application ID

The application will open.

Go to Process online HPI-I application for allied health professionals table to process the application.

Process online HPI-I application for allied health professionals

Table 2: This table gives how to process a Health Professional Online Services (HPOS) application for a Healthcare Provider Identifier – Individual (HPI-I).

Step

Action

1

Open application + Read more ...

Follow the Open online application for allied health professionals table to open the application.

The applicant’s details are prepopulated by Provider Digital Access (PRODA) information and information collected in the application.

2

Check if applicant exists + Read more ...

On the Applicant Details tab, check if the applicant already has a HPI-I.

If the system flags a matching date of birth and surname, open another window and search for an existing record.

Does the applicant already have a HPI-I?

  • Yes. Go to Step 3
  • No. Go to Step 7

3

Check for same professional association and profession + Read more ...

On the Professional Details tab in both windows, does the applicant have the same professional association and profession?

  • Yes. Go to Step 4
  • No:
    • Select the Link with existing individual from the Override option drop down list
    • Copy the HPI-I from the applicant’s existing record, and
    • Paste the HPI-I into the Identifier field
    • Go to Step 5

4

Check end dates on professional association membership + Read more ...

On the Professional Details tab in both windows, are the end dates of the membership with the professional association the same?

  • Yes:
    • Copy the HPI-I from the other window
    • Key ‘applicant has already registered with this professional association/profession and end dates’ in the Comments field
    • Add ‘applicant’s HPI-I is [paste HPI-I]’ in the Comments field
    • Select Reject
    • Procedure ends here. The applicant will receive an automated notification of the outcome of their application in their HPOS Mail Centre
  • No:
    • The applicant is providing evidence of their ongoing membership
    • Go to Step 2 of the Add additional health profession or evidence of ongoing membership with a professional association table

5

Check if professional association or profession exists in HI Service + Read more ...

On the Professional Details tab, is the professional association or profession flagged as not being in the HI Service?

  • Yes:
    • Key ‘[professional association/profession] needs to be checked by Policy’ in the Comments field
    • Note: this doesn’t need to be discussed with the applicant as they get a warning message while completing the application
    • Email Provider HI Service and My Health Record Subject Matter Expert (SME) with the details of the professional association/profession
    • Do not select Save Comment as this closes the application, and it needs to stay open
    • Go to Step 6
  • No. Go to Step 6

6

Check evidence of membership + Read more ...

On the Documents tab, select View to see the document.

Is the document an acceptable form of certified evidence of membership with the applicant’s professional association?

  • Yes. Go to Step 7
  • No:
    • Key ‘need to check document with applicant’ in the Comments field
    • Do not select Save Comment as this closes the application, and it needs to stay open
    • Go to Step 7

7

Check for other issues + Read more ...

On the Applicant Details tab, check the override box for Document is attached if attached document is acceptable. Do not check it if the document is not acceptable.

Are there other issues?

  • Yes:
    • Check the details of the issues, and
    • Override as appropriate, or
    • Key ‘need to check [issue] with applicant’ in the Comments field
    • Go to Step 8
  • No. Go to Step 8

8

Finalise application + Read more ...

Does anything need to be followed up with the applicant or does the professional association/profession need to be checked?

  • Yes:
    • Select Hold to put the application on hold
    • Follow the Contact applicant by phone about online application table for errors the applicant needs to address. Note: do not contact the applicant if the only issue is checking the professional association/profession
    • Procedure ends here
  • No. All issues are resolved. Go to Step 9

9

Approve application + Read more ...

  • Key ‘Approved’ in the Comments field
  • Select Resubmit

The application is approved. The applicant will get an automated notification of the outcome of their application in their HPOS Mail Centre.

Contact applicant by phone about online application

Table 4

Step

Action

1

Open application + Read more ...

  • Open application, see Open online application for allied health professionals table for assistance
  • Check the Comments History tab for issues to discuss with the applicant

2

Call the applicant + Read more ...

Call the applicant using the contact details in the application.

Is the applicant able to talk?

  • Yes:
    • Discuss all issues with the applicant before approving or rejecting the application
    • Go to Step 3
  • No:
    • Follow the Email applicant about online application for allied health professionals table
    • Procedure ends here

3

Is the applicant already registered in the HI Service? + Read more ...

  • Yes:
    • Tell the applicant they are already registered and have a HPI-I
    • If asked and they pass the security check, give them the HPI-I
    • Go to Step 4
  • No. Go to Step 4

See also: Handle call from provider about Healthcare Provider Identifier – Individual (HPI-I).

4

Additional evidence of membership needed? + Read more ...

  • Yes. Ask the applicant to upload additional evidence to their application in HPOS. Go to Step 6
  • No. Go to Step 7

5

Applicant to upload additional evidence later + Read more ...

  • Key ‘applicant to upload additional evidence’ in the Comments field
  • Do not select Save Comment as this closes the application, and it needs to stay open
  • Go to Step 7

6

Uploading additional evidence of membership now + Read more ...

Tell the applicant to go into HPOS and upload their additional evidence to their existing application.

If they don’t know how, talk them through the process. They need to:

  • Log on to their PRODA account
  • Select the Go to service HPOS link
  • Select the My programs tile
  • Select the Healthcare Identifiers and My Health Record tile
  • Select the Healthcare Identifiers – Register Allied Health Professional tile
  • Scroll down the application page, and select the Application ID link to the appropriate application
  • Go to the Documents tab and upload the additional evidence

7

Are there other issues for the Applicant Details tab? + Read more ...

  • Yes:
    • Check the details with the applicant, and
    • Override as appropriate, or
    • Key ‘[issue] unresolved with applicant’ in the Comments field
    • Go to Step 8
  • No. Go to Step 8

8

Finalise application + Read more ...

Have all issues been resolved with the applicant?

  • Yes. Go to Step 9
  • No. If the applicant:
    • can resolve the issues, key ‘applicant will follow up issues’ in the Comments field and select Hold to put the application back into the queue
    • can’t resolve the issues, tell the applicant their application will be rejected and select Reject
    • procedure ends here

9

Approve application + Read more ...

  • Key ‘Approved’ in the Comments field
  • Select Resubmit

The application is approved. The applicant will get an automated notification of the outcome of their application in their HPOS Mail Centre.

Email applicant about online application for allied health professionals

Follow this procedure if unable to contact the applicant by phone.

Table 5

Step

Action

1

Open application + Read more ...

  • Open application, see Open online application for allied health professionals table for assistance
  • Check the Comments History tab for issues to email to the applicant
  • Locate and use the email address in the application and cc [operations email]. Remove the information that’s not needed
  • Create email:
    • in the Subject line, key
      ‘More information for your HPI-I application needed’
  • in the body, key:
    ‘Hello <Title First Name Surname>
    Thank you for your application to register in the Healthcare Identifiers (HI) Service.
    What you need to know’

2

Applicant is already registered in the HI Service + Read more ...

If the applicant is already registered in the Healthcare Identifiers (HI) Service, add the text underneath ‘What you need to know’

‘You are already registered in the HI Service and have a healthcare provider identifier – individual (HPI-I).

What you need to do

You can get your HPI-I by contacting us on [number]. You need to pass a security check before we give you the HPI-I.’

3

Additional documents needed + Read more ...

If the applicant needs to provide additional evidence of membership with a professional association, add the text underneath ‘What you need to know’

Note: do not include the www. In front of the website address. Services Australia does not include links in email for Cyber Security reasons.

‘We need additional evidence of your membership with a professional association. <The documents need to show <that you are a full member><and><the end date of your membership>.><You have uploaded a document that is not related to your application.>

What you need to do

You need to upload additional documents to your application in the Health Professional Online Services (HPOS) by [date].

  • Go to servicesaustralia.gov.au/proda
  • Scroll down and select the Log button
  • Select the Go to service link on the HPOS tile
  • Select the My programs tile
  • Select the Healthcare Identifiers and My Health Record tile
  • Select the Healthcare Identifiers – Register Allied Health Professional tile
  • Scroll down the application page, and select the Application ID link for the appropriate application
  • Go to the Documents tab and upload the additional evidence
  • Select Submit
  • Make sure your details are correct, or update as required
  • Check the ‘declaration’ box

4

Finalising email + Read more ...

To finalise the email include the below at the end.

‘If have any questions about your application, please contact us on [phone number].

More information

Read about the HI Service at servicesaustralia.gov.au/hiservice

[signature block]’

Note: do not include the www. in front of the website address. Service Australia does not include links in email for Cyber Security reasons.

5

Record action in application + Read more ...

  • Send the email
  • Key ‘emailed applicant about issues’, in the Comments field in the application
  • Select Hold to put the application back in the queue
