Medical practitioner prescriber number

Table 1: this table describes how to issue a prescriber number for a medical practitioner when an Application for a Medicare provider number and, or prescriber number for a medical practitioner form (HW019) is received.

Step

Action

1

Assess application details + Read more ...

Check the application to make sure the following fields have been completed:

  • First name and surname
  • Date of birth
  • Existing provider number (if applicable)
  • Indicated prescriber number is needed
  • Declaration must be signed

2

Check eligibility + Read more ...

Check that the medical practitioner:

  • Has a provider number (or is applying for a provider number)
  • Has current medical registration with Ahpra that does not have restrictions to prescribe medications

If an application cannot be processed or if the request is incomplete, see Table 6.

3

Allocate a prescriber number + Read more ...

Allocate a prescriber number in the Provider Directory System (PDS).

Select Qualification tab

In the Prescriber Status field select Prescriber

  • Select Save

4

Issue confirmation letter and finalise + Read more ...

Select the appropriate letter template

  • Z2701 for a prescriber number issued in conjunction with a new provider number or
  • Z2193 for a prescriber number only
  • complete appropriate fields of letter template
  • save letter as a PDF
  • upload letter to PaNDA
  • make sure that the Surname First Name, Provider Number and Case ID fields are completed and correct
  • print the letter and send to the medical practitioner
  • add any appropriate notes to the PaNDA file
  • select Complete
  • Select Request Approved
  • Select Save or Save and Next, as appropriate

Note: start date of the prescriber number on the letter is date of issue or first location start date whichever is later.

Nurse practitioner prescriber number

Table 4: this table describes how to issue a prescriber number for a nurse practitioner when an Application for a Medicare provider number and/or Pharmaceutical Benefits Scheme (PBS) prescriber number for a midwife or nurse practitioner form (2960) is received.

Step

Action

1

Assess application details + Read more ...

Check the application to make sure the following fields have been completed:

  • First name and surname
  • Date of birth
  • Existing provider number (if applicable)
  • Indicated prescriber number is needed
  • Declaration must be signed

Note: make sure the Application for a Medicare provider number and/or PBS prescriber number for a midwife or nurse practitioner form (HW088) has been completed.

2

Check nurse practitioner's details + Read more ...

Check that the nurse practitioner:

  • Has a provider number (or is applying for a provider number)
  • Holds current Ahpra registration as a nurse
  • Has 'Nurse Practitioner' endorsement on their Ahpra registration
  • Go to the Ahpra website and take a snip of the Ahpra registration details. Copy the image into a Word document, convert to a PDF and upload to PaNDA (even if a copy of the Ahpra registration has been supplied with the application)

If an application cannot be processed or if the request is incomplete, see Table 6

3

Move application to the National Office Escalation folder in PaNDA for Delegate approval + Read more ...

In PaNDA:

  • Make sure the Surname, First Name. Provider Number and Case ID fields are complete and correct
  • Select Processing Notes
  • Select Add Processing Note
  • Add a processing note stating “delegate approval required for prescriber number”
  • Select Confirm
  • Select Details
  • Select Unassign
  • Select Recategorised in Reason field
  • Select Provider Registration in Work Type Program
  • Select PROV_NAT_OFFICE_ESC as Work Type
  • Select Submit

When the application is Approved, go to Step 4.

4

Determine the location nurse practitioner works in + Read more ...

When the nurse practitioner works:

  • In private practice - a provider number is issued for billing appropriate MBS items. The specialty code 651 is added to the stem and location. The location provider number may be released to the nurse practitioner or appropriate third party
  • In a government funded clinic/hospital/health service listed on a 19(2) or 19(5) Direction - a provider number is issued for billing appropriate MBS items. The specialty code 651 is added to the stem and location. The location provider number may be released to the nurse practitioner or appropriate third party
  • In public hospital with refer and request rights, or where only prescribing access is required - a provider number is issued for refer and request only. The specialty code 651 is added to the stem and location with the location service ruled to prevent claiming of Medicare benefits when not eligible to claim Medicare benefits. The location provider number may be released to the nurse practitioner or appropriate third party

5

Allocate a prescriber number + Read more ...

Allocate a prescriber number in Provider Directory System (PDS) edit mode.

  • Select Qualification tab
  • In the Prescriber Status field select Prescriber
  • Select Save

Note: where the nurse practitioner has asked for a prescriber number only and a provider record has not previously been created, see Step 13 in the Specialty Code field table in Provider Directory System (PDS) specialty codes for allied health and non-medical health professionals

6

Issue confirmation letter and finalise + Read more ...

Select the appropriate letter template

  • Z2193 for a prescriber number only

  • Complete appropriate fields of letter template
  • Save letter as a PDF
  • Upload letter to PaNDA
  • Make sure that the Surname First Name, Provider Number and Case ID fields are completed and correct
  • Print the letter and send to the medical practitioner
  • Add any appropriate notes to the PaNDA file
  • Select Complete
  • Select Request Approved
  • Select Save or Save and Next, as appropriate

Return inconsistent or incomplete applications

Table 6: this table describes how to return an application where the application cannot be processed or if the request is incomplete.

Step

Action

1

Issuing a letter to the provider + Read more ...

Where the application cannot be processed due to:

  • Application is incomplete - use Z2306 Provider Registration - We need more information letter template
  • Health professional is ineligible to be issued with a prescriber number - use Z2305 Provider Registration - Reject letter template

  • Complete appropriate fields of letter template
  • Print letter and application and post to the health professional
  • Save letter as a PDF
  • Upload letter to PaNDA
  • In PaNDA, make sure the Surname First Name, Provider Number and Case ID fields are completed and correct
  • Add any appropriate notes to the PaNDA file
  • Select Complete
  • Select “Request Rejected” in the Complete Reason field
  • Select Save or Save and Next as appropriate
