Error codes - B status codes - system errors

Table 2: system errors are recording errors in the PBS Claims processing system. Typically, this might be a typographical or a selection error. If the error relates to an incorrect determination, this is an assessment error. See E Assessing errors.

For the process, see Remote Area Aboriginal Health Services (RAAHS).

Error code

Error definition

B01R

Critical

SYSTEM claim registration error

Incorrect:

  • community registration number
  • community name
  • pharmacy approval number
  • date of supply
  • claim reference number

B02R

Critical

SYSTEM claim transaction error

Incorrect or omitted:

  • item code or item code does not match description
  • brand or manufacturer code
  • quantity
  • sub-set quantity
  • total items supplied

B04R

Critical

SYSTEM reason code error

Incorrect reason code.

B14R

Critical

SYSTEM record not updated

  • Failure to complete claim or save record
  • Failure to generate and save the Claims Summary Statement
  • Incorrect naming conventions used for saving the Claims Summary Statement

B15R

Critical

SYSTEM date of lodgement error

Incorrect date of lodgement.

B18R

Non-critical

SYSTEM comments error

Required processing notes not recorded in the system.

Error codes - C status codes - application form errors

Table 3: for the process, see Remote Area Aboriginal Health Services (RAAHS).

Error code

Error definition

C01R

Critical

FORM incomplete or not provided

Claim form:

  • not provided
  • illegible
  • missing information:

    • requested items details

    • pharmacy details

    • declaration details or signature

Error codes - E status codes - assessing errors

Table 5: assessing errors relate to an incorrect determination of entitlement. For example, this might include failure to calculate the correct amount, or exclude ineligible prescriptions. If it is a recording error, these are listed under B - System errors.

For the process, see Remote Area Aboriginal Health Services (RAAHS).

Error code

Error definition

E01R

Critical

ASSESSING entitlement error

Rejected or approved incorrectly.
