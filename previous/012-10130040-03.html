Help access PIP Online and WIP - Practice Stream Online

Table 1

Step

Action

1

Check PRODA account details + Read more ...

To access PIP/WIP - Practice Stream Online the practice must:

  • create a PRODA account, and
  • link HPOS to their PRODA account

Does the practice have access to HPOS through their PRODA account?

  • Yes, go to Step 2
  • No, help the caller to:
    • Create a PRODA account, and
    • Link HPOS to their PRODA account
    • Go to Step 2

See also:

  • Helping users self-manage their PRODA account
  • Helping users create a Provider Digital Access (PRODA) account

2

Link the RA number + Read more ...

Ask the customer for their RA number.

Link the RA number to the practice profile. See Table 2 for assistance.

Tell the customer:

  • they have access to PIP/WIP - Practice Stream Online
  • they can grant online access to other practice staff
    • Direct the customer to the appropriate HPOS User Guide for help. The Resources page has a link
    • Other practice staff do not need to contact Services Australia to link their RA numbers

If the practice is:

  • PIP consenting, the practice needs to update all practice details (except the Quarterly Confirmation Statement (QCS) in WIP - Practice Stream only) in PIP Online
    • the practice will not need to make the same updates in WIP - Practice Stream Online
  • non PIP consenting, the practice needs to update all practice details separately in PIP Online and WIP - Practice Stream Online

See also:

  • Link WIP - Practice Stream non-consenting practices with PIP
  • Quarterly Confirmation Statement (QCS) in WIP - Practice Stream

3

Access PIP/WIP - Practice Stream Online services + Read more ...

Help the customer access PIP/WIP - Practice Stream Online.

  • Select My Programs from the HPOS Homepage
    • The My Programs screen will show
  • Select PIP Online or WIP - Practice Stream Online
    • The PIP or WIP - Practice Stream screen will show
    • A list of all of the PIP or WIP - Practice Stream practice profiles relating to the PRODA account will show

If the practice:

  • is approved for the program
    • select Update from the Actions column, next to the right Practice ID
  • is not approved for the program
    • select Apply Now
    • direct the caller to the appropriate HPOS User Guide for help. The Resources page has a link
    • tell the caller to look at the WIP - Practice Stream Online HPOS User Guide on the Services Australia website for help
  • application is in progress
    • select Continue under In Progress Application
  • application is submitted
    • processing of the application is not yet complete
    • select View PDF or View RTF to review the submitted application

If the customer does not have all functions available (for example: ‘Update’):

  • ask the caller to log out of HPOS in their current browser, and
  • log into HPOS using the Google Chrome browser

Link and remove an RA number

Table 2

Step

Action

1

Log into systems + Read more ...

Log on to these systems:

  • PIP Online, or
  • WIP - Practice Stream Online

2

Search for the practice + Read more ...

Search for the practice in PIP Online or WIP - Practice Stream Online.

On the Practice Search screen:

  • key the Practice ID
  • select Search
    • the Search Results will show

Check if the practice name and address on the practice profile match the details on the form.

If the details match, go to Step 3

If they do not match, check the company or trading name:

  • Select the Practice ID
    • The Practice Summary screen will show
  • Select Practice Details from the Main menu
  • Select Ownership Details from the drop down
    • The Ownership Details screen will show

Do these details match the details on the form?

  • Yes, go to Step 3
  • No, check that the Practice ID keyed is correct.

3

Practice staff system access + Read more ...

Check if the person is registered on the practice profile.

  • Select Practice Staff System Access from the Main menu
    • The Practice Staff System Access screen will show

Is the person registered?

  • Yes, go to Step 4
  • No, ask them to submit the right form:
    • Practice Owner, Practice Incentives Change of practice ownership (IP010) form
    • Authorised contact person, Practice Incentives Change of practice details (IP005) form
    • Provider, Practice Incentives Individual general practitioner, nurse practitioner or health professional details (IP003) form

Procedure ends here.

4

Check for an RA Number + Read more ...

Does the person currently have a linked RA Number?

  • Yes, go to Step 5
  • No, go to Step 6

5

Check registered RA Number + Read more ...

Does the currently linked RA Number match the new RA number provided?

  • Yes, procedure ends here
  • No, go to Step 6

6

Check new RA Number + Read more ...

Check if the new RA number is correct:

  • Search for the PRODA account and/or
  • The Individual-PKI account

Go to Step 7.

7

Check linked RA Number + Read more ...

Does the person currently have a linked RA number?

  • Yes, go to Step 8
  • No, go to Step 9

8

Remove an RA number + Read more ...

To remove an RA number:

  • Find the right person listed under Practice owners, authorised contact persons or providers
  • Select Remove access from the Action column
  • A confirmation message will show
  • Select Yes
  • Repeat this process for each listing under:
    • Practice owners
    • Authorised contact persons, and
    • Providers

9

Link RA Number + Read more ...

To link an RA number:

  • Find the right person listed under Practice owners, authorised contact persons or providers
  • Key the RA number in the RA Number column
  • Select Grant access from the Action column
  • Repeat this process for each listing under:
    • Practice owners
    • Authorised contact persons, and
    • Providers

If error message 2042 shows, decide if an RA number override can occur.

10

PIP consenting practice + Read more ...

Is the practice a PIP consenting practice?

  • Yes, procedure ends here
  • No, log on to WIP - Practice Stream Online
    • repeat Step 5

Note: if the practice is not PIP consenting and has not provided a separate WIP - Practice Stream practice ID, do not make updates in WIP - Practice Stream.

See also: Link WIP - Practice Stream non-consenting practices with PIP
