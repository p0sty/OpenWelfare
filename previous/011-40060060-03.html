Amend program review threshold or hold status details

Procedure on how to amend program review threshold or hold status details

Table 1: This table describes how to amend program review threshold details or hold status for a group.

Step

Action

1

CDMS home page + Read more ...

On the Consumer Directory Maintenance System (CDMS), select the Amend Consumer Details link.

The Amendment Search screen will display.

2

Searching for the customer + Read more ...

  • Conduct an search to identify the customer
  • Select Group, the Amend Medicare Card Details screen will display
  • Select Review, the Amend Program Review Details screen will display

3

Viewing the groups associated with the address + Read more ...

In Threshold Details:

  • select Groups to view the groups currently associated with the address
  • select Return to return to the Amend Program Review Details screen

4

Amending details + Read more ...

In Threshold Details, amend the details as required:

  • Property Usage
  • Threshold
  • Hold Status
  • Comments
  • select Validate (Alt+V)

An information message will display to inform whether the amendment was acceptable or not. Select OK to acknowledge the message. If not acceptable, change the amendment as required.

5

Validating amendments + Read more ...

Scroll down:

  • select Validate (Alt+V)
  • the Confirm Amended Program Review Details screen displays the updated details

6

Changing or printing amended details + Read more ...

Select:

  • Change to change the amended details, go to Step 3
  • Print to print the information on the screen

7

Updating correct details + Read more ...

When the details are correct, select Update (Alt+U). An information message displays advising:

  • the details that have been updated
  • the type of information that was amended

Select OK to acknowledge each message. The Amend Program Review Details screen redisplays with the updated details.

Amend program review status

Procedure on how to amend program review status

Table 2: This table describes how to add or amend the review status of a group with a program review investigation status, such as 'Under Investigation' or 'False Identity'.

Step

Action

1

CDMS home page + Read more ...

On the Consumer Directory Maintenance System (CDMS), select the Amend Consumer Details link.

The Amendment Search screen will display.

2

Searching for the customer + Read more ...

  • conduct an search to identify the customer
  • select Group, the Amend Medicare Card Details screen will display
  • select Review, the Amend Program Review Details screen will display

3

Selecting status + Read more ...

In Program Review Details, select a status in the Review Status field, for example:

  • Under Investigation
  • False Identity

4

Keying comments and validating + Read more ...

In New Program Review Comment:

  • key comments in the Text field
  • if required, key a comment end date comments
  • select Validate (Alt+V)

An information message will display advising whether the amendment was acceptable or not. Select OK to acknowledge the message. If not acceptable, change the amendment as required.

5

Viewing further details + Read more ...

To view further details:

  • select the required record
  • select Group Details

The Group Member Association Details screen redisplays with the Associated Group Members Details section added.

Note: decide from these details if further action is required to move other members from the card before proceeding.

6

Making changes or adding status + Read more ...

Select:

  • Return to make changes
  • Continue to add the status to the group

The Confirm Amended Program Review Details screen displays.

7

Final validation + Read more ...

Scroll down and select Validation (Alt+V)

The Confirm Amended Medicare Card Details screen displays the updated details.

8

Changing or printing amended details + Read more ...

Select:

  • Change to change the amended details, go to Step 3
  • Print to print the information on the screen

9

Updating correct details + Read more ...

When the details are correct, select Update (Alt+U):An information message displays advising:

  • the details that have been updated
  • the type of information that was amended

Select OK to acknowledge each message. The Amend Program Review Details screen redisplays with the updated details.
