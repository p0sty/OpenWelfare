Check the form is complete + Read more ...

The AC019 has 4 parts. To accept the aged care nominee request, Part A is compulsory and only 1 of Part B, C or D needs completing (depending upon the care recipient's circumstances).

Part A - care recipient details:

  • must have all the care recipient's details

Part B - completed when the care recipient is without physical or mental impairment:

  • The care recipient must sign the form
  • If there is no care recipient signature:
    • Services Australia must receive documentation authorising a person to act on the care recipients behalf
  • The nominee must sign the form to accept the arrangement

Part C - completed when the care recipient is physically impaired and they cannot complete the form:

  • The nominee must sign the form to accept the arrangement
  • The Director of Nursing/Hostel Manager/Care Manager must sign the form on behalf of the care recipient

Part D - completed when the care recipient is mentally impaired and they cannot complete the form:

  • The nominee must sign the form to accept the arrangement
  • Services Australia must receive documentation authorising a person to act on the care recipients behalf. Note: the exception is where the partner signs Part D of the form

If a power of attorney (POA) is appointing themselves, another person or professional service organisation as the nominee:

  • Check the POA document for any conditions preventing the appointment. For example:
    • The POA cannot make decisions about financial matters
    • Joint POAs appointed and all POA signatures needed on the nominee form
  • If unsure the POA is valid or has limiting conditions, discuss with:
    • Technical Peer Support (TPS) - Aged Care skilled tagged Service Officers
    • Subject Matter Expert (SME) - ACAH Service Officers
  • Use an internet search and/or the organisation contact number to check a professional service organisation is a:
    • registered professional service, or
    • state, territory or commonwealth government agency

Note: only process a POA request when the relevant POA documentation is provided and the POA conditions are met. This applies to all nominee requests including requests made by a care recipient's partner.

Is the form incomplete, missing relevant signatures and/or authorising documents not provided or conditions not met?

  • Yes, and:
    • form incomplete, missing relevant signatures and/or authorising documents not provided, go to Step 4
    • authorising document conditions not met, see Step 4 in Table 3
  • No, see Table 4 to process an aged care nominee in ACMPS

4

Form incomplete and/or authorising documents not provided to Services Australia + Read more ...

The form must be resubmitted if:

  • incomplete, including missing relevant signatures, and/or
  • authorising documents not provided to Services Australia

If Part B applies (no physical or mental impairment):

  • If the form is incomplete, missing relevant signature and/or authorising documents not provided to Services Australia, return the form to the care recipient
    • If unable to return the form to the care recipient, return it to the proposed nominee. For example, care recipient details missing from the form

If Part C applies (is physically impaired):

  • If the form is missing information from Part A:
    • Return the form to the proposed nominee or the person signing the care recipient's declaration
  • If the form is missing a signature or other information from Part C:
    • Return the form to the person whose section is missing the signature or information (proposed nominee or the person signing the care recipient's declaration)
    • If unable to return the form to the person signing the care recipient's declaration, return it to the proposed nominee. For example, address is unknown
  • If the relevant authorisation document is not provided to Services Australia:
    • Return the form to the proposed nominee

If Part D applies (is mentally impaired):

  • If the form is incomplete, missing relevant signature and/or authorising documentation not provided to Services Australia, return the form to the proposed nominee

Check the form is complete + Read more ...

The form has 4 parts. To accept the aged care nominee request please note the following:

Part A - care recipient (customer) details:

  • must have all the care recipient's details

Part B - nominee details:

  • completed by a person or an organisation
  • must have all the nominee details

Part C - customer declaration and third party authorisation can be signed by:

  • care recipient (customer)
  • Power of Attorney (POA) (a copy of the POA must be provided) or
  • Third Party where the care recipient is unable due to physical or mental disability may sign the form and medical evidence must be provided

Part D - completed by the nominee:

  • The nominee must sign the form to accept the arrangement

If a POA is appointing themselves, another person or professional service organisation as the nominee:

  • Check the POA document for any conditions preventing the appointment. For example:
    • The POA cannot make decisions about financial matters
    • Joint POAs appointed and all POA signatures needed on the nominee form
  • If unsure the POA is valid or has limiting conditions, discuss with:
    • Subject Matter Expert (SME) - ACAH Service Officers

If an organisation is nominated:

  • Check the POA document for any conditions preventing the use of a professional service. For example:
    • The POA cannot make decisions about financial matters
    • Joint POAs appointed and all POA signatures must be on the nominee form
  • Use an Internet search and/or the organisation contact number to confirm the nominee organisation is a:
    • registered professional service, or
    • state, territory or commonwealth government agency

The POA can nominate the following professional services as an aged care nominee:

  • retirement or financial advisors
  • lawyers
  • accountants and tax agents

Note: only process a POA request when the relevant POA documentation is given and the POA conditions are met. This applies to all nominee requests including requests made by a care recipient's partner.

Check the Aged Care Request for a nominee section is complete + Read more ...

A care recipient or POA can appoint a nominee in the Aged Care Calculation of your cost of care form.

The 'Aged Care Request for a nominee' section has 2 parts.

Part A - nominee request completed by the person the assessment is for:

  • Must have all the care recipient's details
  • The care recipient or their POA must sign the form (when care recipient without physical or mental impairment)
  • The Third party authorisation must be signed (if care recipient is physically or mentally impaired)
  • The nominee must sign the form to accept the arrangement
  • If there is no care recipient signature, documentation authorising a person to act on the care recipient's behalf must be given

Part B - to be completed by the nominee for aged care purposes:

  • The nominee must sign the form to accept the arrangement

If a POA is appointing themselves or a professional service organisation as the nominee:

  • Check the POA document for any conditions preventing the appointment. For example:
    • The POA cannot make decisions about financial matters
    • Joint POAs appointed and all POA signatures needed on the nominee form
  • If unsure the POA is valid or has limiting conditions, discuss with:
    • Technical Peer Support (TPS) - Aged Care skilled tagged Service Officers
  • Use an internet search and/or the organisation contact number to check a professional service organisation is:
    • registered professional service, or
    • state, territory or commonwealth government agency

Note: only process a POA request when the relevant POA documentation is given and the POA conditions are met. This applies to all nominee requests including requests made by a care recipient's partner.

Update existing nominee's addresses + Read more ...

  • Select the Addresses tab:
    • select New

    • select Edit List
    • select the correct Address Type from the drop down option in the Address Type field
    • select the correct Address from the drop down option in the Address field

    • There are no mandatory search fields

    • Select the search parameters from the drop down box
    • Copy or remove search rows by clicking the plus (+) or minus (-) icon
    • Selecting Clear removes values in the search fields but retains any new search criteria added
  • Search using the nominee details provided in the request

Search for a person:

  • Key the fields below:
    • Nominee Last Name
    • Nominee First Name
    • DOB
    • Nominee CRN
    • Postal Code
    • Region (Town or State), or
    • a combination of the above

Search for an organisation:

  • Service Officers must check the Organisation nominee ID list on the Resources page, before searching in ACMPS for an ID

    • Key the Nominee CRN field (organisation's CRN), or
    • Key the Organisation field (organisation's name), and
    • Change 'is' to 'contains' (second drop down)
    • Select Search

Note: a nominee organisation can have multiple care recipients linked. To maintain data integrity and quality do not link a care recipient to an organisation record that is created as type AURES.

Large organisations (such as state trustees) may have multiple ACMPS records as the organisation has multiple valid records in Customer First.

State Trustees:

NSW Trustee and Guardian:

  • key Guardian in the Organisation Name field and search using 'contains'

Public Trustee:

  • key Public in the Organisation field and search using 'contains'

    • Instructions for adding a second addresses is found in the next step
  • Address Line 2 (key only if more than 2 address lines)
  • Suburb/Town
  • State
  • Postcode
  • Country - Defaults to Australia: to change to overseas country, select Nominee Type as Overseas Resident

  • E-mail
  • Select Save. The message 'Data has been saved' displays, identifying the record creation is successful
