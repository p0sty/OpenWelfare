Document error status codes

Document error status codes for My Health Record - Provider - quality checking

Table 2: This table lists the document error status codes for My Health Record - Provider - quality checking.

Error status codes

Definition

Processing tip

PCE02

Critical

No documentation

It is an error where there is insufficient documentation provided to support application.

PCE03

Critical

Applicant is not the listed RO/OMO for the Seed Organisation

It is an error when the Applicant is not listed in PPR as the RO or OMO of the Seed organisation.

PCE04

Critical

Application questions not completed in full where applicable

It is an error when the applicable questions have not been fully completed, or completed with incorrect details, for the following forms:

  • standalone: Q1, Q3-9
  • combined: Q1, Q7-9.

PCE05

Critical

Part A of application not attached to Part B where applicable

Part A must be supplied with Part B if applicant has answered only questions 1, 7 to 9 on page 9 of the application.

PCE06

Critical

Declaration not completed or dated on Application form

It is an error if the applicant has not signed or dated the Declaration question 9 on page 9 of the application form for Seed organisation and page 5 for Network organisations.

PCE07

Critical

RO/OMO not contact person on page 2 of participation agreement

It is an error if the RO or OMO is not the designated contact person for the Healthcare Provider Organisation on page 2 of the PA.

PCE08

Critical

ABN/ACN number incomplete or not supplied.

It is an error if the ABN/ACN/ARBN has not been listed on page 2 of the PA.

PCE09

Critical

PA has missing pages

It is an error if the PA has not been submitted in its entirety. The PA must contain 16 pages.

PCE10

Critical

PA has been altered

It is an error if there has been any alterations on any page of the PA without the approval of Health (as directed by Tier 3).

PCE11

Critical

Page 16 of PA not signed/dated

It is an error if the PA has not been signed or dated on page 16.

PCE12

Critical

No PA submitted with Network Application

It is an error if a network is registered without a PA, when a PA is required for the network with a different ABN.

PCE13

Critical

Applicant is not the registered RO/OMO of the Seed Organisation for a Network Application

It is an error when the Applicant is not listed in PPR as the RO or OMO of the Seed organisation for a Network Application.

PCE14

Critical

Network Attachment A or B is not completed with all relevant information

It is an error if Table 1 page 7 or Table 2 page 8 is incomplete.

PCE15

Critical

Incomplete/incorrect HPI-I as listed in application

It is an error if the HPI-I listed on the application is incorrect, incomplete or missing.

PCE16

Critical

No documentation to support add/remove HPI-I by phone

It is an error if no supporting documentation is found for HPI-I linked in PPR as listed in the letter.

PCE17

Critical

RO/OMO signature mismatches the applicant

It is an error if the

  • RO/OMO signs the participation agreement as the delegate, and the signature does not match the signature in Part B of the application form
  • signature does not match the applicant name where applicable

Processing error status codes

Processing error status codes for My Health Record - Provider - quality checking

Table 3: This table lists the processing error status codes for My Health Record - Provider - quality checking.

Error status codes

Definition

Processing tip

PCE20

Critical

HPI-O registered is not same as on documentation

It is an error if the HPI-O registered in PPR is not listed on the application.

PCE21

Critical

Incorrect HPI-I linked to HPI-O

It is an error if the linked HP-I is not listed on the application, when requested as authorised link

PCE22

Critical

Incorrect CSP linked to HPI-O

It is an error if the linked HP-I is not listed on the application

PCE23

Critical

Incorrect RO or OMO selected to process application

It is an error if the incorrect RO/OMO is selected in PPR to process the application. The applicant was not selected.

PCE24

Non-critical

No CRM entry

It is an error if contact was made, if there is no SR entry in CRM matching the HPI-O listed on the application and registered in PPR.

PCE25

Non-critical

SR raised under incorrect HPI-O

The Service Request raised in CRM is not linked to the HPI-O listed on the application and registered in PPR.

PCE26

Non-critical

Incorrect or insufficient information record in SR

It is an error if information in the SR entry in CRM is incorrect or insufficient.

PCE27

Non-critical

CRM not closed off when completed

It is an error if the SR and completed the task has not closed in the CRM.

PCE28

Non-critical

SR reference not recorded in My Health Record Tracking Database

It is an error if the SR number is not entered in My Health Record CRM Ref field of the My Health Record Tracking Database when required.

PCE29

Non-critical

Events not recorded as additional information in My Health Record Tracking Database

It is an error if actions or interaction for the My Health Record application is not recorded as an 'event' in the My Health Record Tracking Database entry.

PCE30

Non-critical

My Health status not changed to 'Closed-completed'

It is an error if the My Health Record status in the My Health Record Tracking Database has not be changed to 'Closed-completed'

PCE31

Non-critical

Electronic application not archived

It is an error if the electronic application has not been placed in the Registered folder.

PCE32

Non-critical

My Health Record Tracking Database not updated

It is an error if the My Health Record Tracking Database has not been completed for the application update

PCE33

Critical

Personal information included in CRM

It is an error if personal information has been included in the CRM record due to privacy implications.

PCE34

Critical

Application not registered in PPR

It is an error if the application has not been updated in the PPR system when applicable.
