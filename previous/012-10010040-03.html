Step

Action

Log into systems and check practice details + Read more ...

Log on to these systems:

  • Practice ID at Question 2 or 3

If the details:

  • match, go to Step 3.

Returning an incomplete form + Read more ...

If the form needs more details:

  • Place the practice payments on hold. Do not do this if the practice ID is missing, incorrect or the practice cannot be identified
  • Return the form to the practice. Use the Request for more information for the PIP and the WIP - Practice Stream (z1736) letter
    • Select the letter option to advise the practice payments are on hold
    • If the practice ID is missing, incorrect or the practice cannot be identified, use the name and new relocated practice address (Question 13) on the form. If the date of relocation (Question 11) is in the future, use the current address (Question 6). If there is no name, use 'Practice Manager'
  • Upload the letter and the IP005 form to the practice profile. Upload the documents to PaNDA only if the practice ID is missing, incorrect or the practice cannot be identified
  • Finalise the work item as Complete, Request Rejected in PaNDA

If the practice is not a PIP consenting practice, upload the letter to both profiles.

Relocate the current practice address + Read more ...

Relocate the practice.

  • Select Practice Details from the Main menu
  • Select Locations from the drop down. The Locations screen will show

Under Current Locations:

  • Find the location that needs to relocate
  • Select the radio button from the Select column
  • Select Relocation. Relocation screen will show
  • Key the below details:
    • Relocation Date (Question 11)
    • New relocated practice address (Question 13). If warning message 2307 appears, confirm the warning message and continue to process the form. Address variances has more information
    • Accreditation details - as per the Accreditation certificate or previously noted details

Under Associated GPs Details:

  • In the Select (if relocating) column, select the relocating practitioner
  • In the New Provider Number column, next to the GP, key each new relocated provider number from the IP003 form
  • In the Provider End Date column, next to the GP
    • Key each new relocated provider number end date (if known)
    • If the end date in PDS is before the date of relocation (Question 11), use the date in PDS
  • Select Save and Exit
