Work types done by Medicare Consumers Program Support + Read more ...

The LPS will confirm if the query is a Medicare Consumers Program Support work type. If escalation is required the LPS will:

  • escalate documents to Medicare Consumers Program Support, or
  • email enquiry where appropriate

Note: applications in PaNDA must be escalated via PaNDA using work type Medicare Eligibility Nat.Office Escalations.

These can include:

  • 410 visa holders
    • issued or applied for before 1 December 1998, and
    • who have never had a Medicare card
  • The removal of a 'Pending Bridging' visa holder
  • Complex enrolments that need to be escalated to the Department of Home Affairs
  • Victims of People Trafficking programme
  • Request to remove children under 15 from a family Medicare card
  • Fourth (or more) Medicare card requests
  • Postcode locality update
  • Person enrolled on the incorrect card
  • Process Record Amendment (PRA) requests
  • Surrogacy arrangement where documentation provided does not meet standard requirements

Face to Face (F2F) Service Officer escalation for Document Verification System (DVS) checks

Table 2

Step

Description

1

Face to Face (F2F) Service Officers who do not hold a DVS licence + Read more ...

  • access DVS request link

2

Submit DVS request + Read more ...

  • select Submit DVS Request
  • enter PaNDA Work Item ID
  • select type of DVS consent
  • complete confirmation messages
  • select Submit

Once the request has been submitted it will appear under ‘My DVS Requests’.

3

DVS request completed + Read more ...

Once DVS request complete, an email will be sent to the processing Service Officer with the outcome.

Match results include:

  • Yes,
    • Verification successful
    • Complete enrolment
  • No,
    • Verification has been unsuccessful (2 attempts to verify have been made)
    • Return the original document to the customer with relevant letter
  • D (details input are recorded with issuer, but cannot be verified)
    • Verification has been unsuccessful
    • Return the original document to the customer with relevant letter
  • S (system issues)
    • Add processing note in PaNDA ‘DVS unsuccessful as system is down, unable to process’
    • Unassign the work item

Service Officers will get an email detailing appropriate action to take when a request is submitted and cannot be processed due to the following reasons:

  • Acceptable identification not provided
  • Scan not legible
  • Form not signed
  • Form does not include DVS consent

Face to Face (F2F) Service Officer escalation

Table 3

Step

Description

1

Try to resolve the enquiry + Read more ...

A F2F Service Officer must try to resolve the customer enquiry first by using:

  • Operational Blueprint
  • staff messages such as Daily Comms
  • comments in the Consumer Directory Maintenance System (CDMS)
  • a VEVO system check

2

Contact Medicare LPS for help + Read more ...

If the F2F Service Officer is unable to resolve the customer’s enquiry call Medicare and Aged Care - Local Peer Support (LPS).

If it is deemed not urgent, the F2F Service Officer escalates the enquiry to LPS. To do this the F2F Service Officer will:

  • Scan all of the enrolment documents
  • In the CDMS:
    • Add a Comment at the Group level (if the customer has a Medicare number). For example, the enrolment has been escalated
    • Add the LPS login ID details in the comment

This work will be finalised by a Medicare Eligibility and Enrolment trained Service Officer or LPS.

If the enquiry is urgent, go to Step 3.

3

Escalate when the enquiry is urgent + Read more ...

If an enquiry requires an immediate response, LPS will call the Medicare Consumers Program Support team for advice.

Note: if the Medicare Consumers Program Support team cannot resolve the enquiry over the phone, refer to Escalations from LPS to Medicare Consumers Program Support.

    • all enquiry details, and

Escalate an urgent enquiry + Read more ...

If an Eligibility LPS needs help with an urgent query, the Eligibility LPS can call Medicare Consumers Program Support.

LPS will explain that the query is urgent and give Medicare Consumers Program Support all of the details.

The Medicare Consumers Program Support will:

  • tell them how to resolve the enquiry, or
  • ask them to send all of the details

Email requests

The Medicare Consumers Program Support will respond to urgent email requests within 2 business days. The team will assess urgent requests and respond to them in order of priority.

The response time may be longer if the case needs external stakeholder advice.

PaNDA requests:

  • Applications escalated via PaNDA must be escalated using work type Medicare Eligibility and enrolment, Nat.Office Escalations
  • Requests to remove children under 15 from a person Medicare card escalate using work type Nat.Office Escalation DCP

Escalate a Department of Child Protection (DCP) or a prisoner enquiry

Table 5

Step

Description

1

How to handle a F2F DCP or prisoner enquiry + Read more ...

F2F Service Officer must:

  • escalate the query to a Medicare and Aged Care - Local Peer Support (LPS)
  • if processing is required scan all of the documents and forms to PaNDA

The DCP or a non-government organisation may make a verbal request for a child’s Medicare card details. Do not give out these details. The DCP or non-government organisation needs to call a Medicare Eligibility trained Service Officer and give their password and agency ID.

Any request from a non-government body must not be actioned, for example, Anglicare or Wesley Mission. Call the Medicare Eligibility LPS team for help.

2

DCP and prisoner queries for a Service Officer + Read more ...

If a Service Officer needs help, call Eligibility LPS.

If LPS asks the Service Officer to send them the enquiry, they must:

  • Forward the details to the LPS via the digitised workflow
  • Details of the enquiry may include:
    • The file name Work Item ID
    • A Medicare card number
    • The applicant’s name

3

DCP and prisoner queries for Medicare Consumers Program Support + Read more ...

If LPS needs help:

  • Call Medicare Consumers Program Support, or
  • escalate via PaNDA using work type Medicare Eligibility Nat.Office Escalations DCP

The Medicare Consumers Program Support will respond to the LPS. A LPS or a Service Officer will complete the enrolment.

Escalate a third Medicare card request

Table 6

Step

Description

1

A third card request + Read more ...

F2F or Service Officers must escalate all third card requests to the Medicare Eligibility LPS team via PaNDA. LPS do not decide and grant third card approvals verbally.

Service Officers must make sure a completed application and all relevant documentation has been provided by the applicant.

Service Officers will escalate to a Medicare Eligibility LPS via PaNDA under the work type:

  • DCP 3rd card request to Medicare Eligibility DCP Escalations.
  • Other 3rd card requests to Medicare Eligibility Escalations

The LPS will:

  • assess and confirm that the request is valid
  • include approval/rejection in PaNDA work item notes
  • re-assign the third card request back to the processing queue for the next Medicare Eligibility trained Service Officer to complete

All 4th card (or more) must be escalated to the Medicare Consumers Program Support team. The LPS must send the request via PaNDA using work type Medicare Eligibility Nat.Office Escalations DCP.

Escalate a Child Dental Benefits Schedule (CDBS) query

Table 7

Step

Description

1

How to handle a F2F CDBS enquiry + Read more ...

A F2F Service Officer must try to resolve the customer enquiry. See Patient eligibility for Child Dental Benefits Schedule (CDBS) in Medicare.

If the F2F Service Officer is unable to resolve, contact a HSDD Eligibility Service Officer:

  • complete the CDBS Eligibility Enquiry Form
  • scan it using a Multi-Function Device (MFD)
  • email to MPS Eligibility

2

Try to resolve the enquiry + Read more ...

The Service Officer must try to resolve the enquiry. If the enquiry cannot be resolved the Service Officer must contact an Eligibility LPS.

If the Medicare Eligibility LPS needs help, they must:

  • complete a CDBS Eligibility Enquiry Form, and
  • complete a CDBS National Office Escalation Form
  • email both to CDBS Medicare Branch National Office
