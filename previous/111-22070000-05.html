Document error status codes

Table 2

Error code

Definition

Processing tip

HI02

Critical

Insufficient or no documentation for Evidence Of Identity (EOI) to support application

It is an error when the EOI documents:

  • are not supplied, and/or
  • do not match the application form

and there is no verification present for the individual’s EOI.

Note: there is no verification needed for a known customer.

HI03

Critical

No documentation to support Responsible Officer (RO) authority to act on behalf of Seed

It is an error when:

  • there is no documentation
  • the documentation is incomplete, and/or
  • the documentation is insufficient to support the RO's legal authority to act on behalf of the Seed organisation.

HI04

Critical

ARIF - Acceptable Referee Identification Form (Part B) section/s incomplete or not signed

It is an error when the ARIF (Part B) is incomplete and/or not signed and not detected by the Service Officer.

HI05

Critical

Document certification insufficient or incomplete

It is an error when the Individual EOI documentation is:

  • not certified, and/or
  • certified by a person other than the acceptable referee.

Note: it is also an error when the business EOI document from ABR is not certified

HI06

Critical

Declaration details incomplete

It is an error when the declaration:

  • signature, and/or
  • date

is missing or incomplete and not detected by the Service Officer.

HI07

Critical

Name details incomplete on application

It is an error when the:

  • surname and/or
  • given name

are incomplete or do not match the EOI documents and there is no verification present.

HI08

Critical

Gender and/or Date of Birth (DOB) incomplete or omitted

It is an error when the:

  • gender and/or
  • date of birth

are incomplete or do not match the EOI documents and there is no verification present.

HI09

Critical

Address Details incomplete or omitted

It is an error when the:

  • business address and/or
  • postal address

are incomplete and there is no verification present.

HI10

Non-critical

Contact Details incomplete or omitted

It is an error when there is incomplete:

  • daytime phone number
  • mobile phone number
  • fax number
  • pager number
  • email address

Note: at least one must be supplied.

HI11

Critical

ABN/ACN Number incomplete or not supplied

It is an error when the:

  • ABN, and/or
  • ACN

is incomplete or not supplied on form and there is no verification present.

HI12

Non-critical

Organisation Service Details incomplete or omitted

It is an error when the:

  • organisation type
  • organisation service type
  • organisation service unit (if applicable)

is incomplete or not supplied on the form.

HI13

Critical

Organisation link details incomplete or omitted

It is an error when there is incomplete or no verification present to link:

  • a Network organisation to the Seed
  • a replacement RO to the Seed
  • an OMO to the Network to Seed

and there is no verification present.

HI14

Non- Critical

PKI/PRODA details incomplete or omitted

It is an error when the:

  • existing PKI/PRODA boxes are not checked (where applicable)
  • existing PKI/PRODA number not supplied (when applicable), and/or
  • processing systems are not stated (when applicable)

and there is no verification present.

HI15

Critical

Professional Membership Association has not been supplied

It is an error when there is insufficient, incorrect or no information of the applicant’s Professional Membership Association.

Processing error status codes

Table 3

Error code

Definition

Processing tip

HI20

Critical

Duplicate created incorrectly

It is an error when a duplicate record has been created for:

  • Responsible Officers (RO)
  • Organisational Maintenance Officers (OMO)
  • Seed
  • Network
  • HPI-I
  • IHI

HI21

Critical

Name keyed incorrectly

It is an error when the name of the:

  • individual
  • organisation (organisation details), and/or
  • title (where applicable)
  • has been keyed incorrectly

HI22

Critical

Gender/DOB keyed or selected incorrectly

It is an error when the:

  • Gender has been selected incorrectly and/or
  • DOB has been keyed incorrectly

HI23

Critical

Deceased details keyed incorrectly or not keyed

It is an error when the deceased notification has not been keyed or keyed incorrectly.

HI24

Critical

ABN/ACN details keyed incorrectly or not keyed

It is an error when the following details have not been keyed, or have been keyed incorrectly:

  • ABN
  • ACN

HI25

Non-Critical

Service Details keyed or selected incorrectly

It is an error when the following details have not been keyed, or have been keyed incorrectly:

  • organisation type
  • organisation service type organisation service unit (if applicable)
  • start date

Note: if the date is not supplied the business rules as per Operational Blueprint must be followed.

HI26

Critical

Address keyed incorrectly

It is an error when the:

  • business address
  • postal address, and/or
  • preferred address indicator
  • has been keyed or selected incorrectly.

Note: capitalisation must be used when keying an address.

HI27

Critical

EOI keyed or selected incorrectly, not updated

It is an error when the EOI document/s (including being a known customer e.g. PRODA, PKI certificate) have not been entered or incorrectly entered in the system.

Note: all EOI documents supplied must be entered.

HI28

Non-Critical

Electronic Communication/Contact Details keyed or selected incorrectly

It is an error when the following details have not been keyed, or have been keyed incorrectly:

  • daytime phone number
  • mobile phone number
  • fax number
  • pager number
  • email address
  • preferred indicator

Note: it is not an error when the preferred indicator has not been ticked on the application form.

HI29

Critical

Healthcare Provider Directory (HPD) entry selected incorrectly

It is an error when the HPD selection on system does not match the request on the form.

HI30

Critical

PRODA account not/incorrectly actioned

It is an error when the PRODA has been linked incorrectly and/or not linked.

HI31

Critical

Link to organisation made incorrectly

It is an error when the:

  • individual has been linked as an RO or OMO to the organisation incorrectly
  • organisation has been linked to a Seed/Network (parent/child) incorrectly
  • HPI-I incorrectly linked to an organisation in error
  • link not created when requested, and/or
  • CSP incorrectly/not linked to an organisation

HII33

Non-Critical

Comments keyed incorrectly/not keyed

It is an error when comments have:

  • been keyed inappropriately, and/or
  • have not been keyed when required

HI34

Critical

EOI Rating actioned incorrectly

It is an error when the EOI rating has been

  • selected incorrectly, or
  • not selected when appropriate

HI35

Critical

Incorrect status applied to a record

It is an error when an organisation or individual record has had an incorrect status applied, such as deactivated rather than retired.

HI36

Critical

Incorrect profession applied to a record or not applied

It is an error where a record has not had a:

  • profession applied
  • additional profession applied
  • correct profession applied

HI37

Critical

Error(s) in Letter to Applicant

It is an error when the letter issued:

  • contains spelling errors
  • contains incorrect details
  • has details left out
  • is on the incorrect letter template

HI38

Non Critical

Letter/documents not uploaded in PANDA

It is an error when the letter/documentation has not been uploaded into PANDA and not detected by the Service Officer.

HI39

Critical

Application approved incorrectly

It is an error when supporting documentation was not supplied but the application is approved.

HI40

Critical

Application rejected incorrectly

It is an error when the application has been rejected yet A2A is acceptable.

HI41

Critical

Professional Association on Membership certificate does not meet HI Service Act

It is an error when the Professional Association no longer meets Policy requirements as per listed in EHP.

HI42

Critical

Professional association on certificate does not match professional association in application

It is an error when the professional association details in the accepted application do not match the details on the certificate.

HI43

Critical

Applicants name on certificate does not match applicant’s name in application

It is an error when the name in the application that has been accepted does not match the name on the certificate.

HI44

Critical

Professional Membership period on certificate does not match membership period on application

It is an error when the end date of membership details in the application does not match the end date membership details on the certificate.

HI45

Non Critical

Membership number on certificate does not match membership number in application

It is an error when the membership number in the application that has been accepted does not match the membership number on the certificate.

HI46

Critical

Change of ownership: Previous HPI-O not retired

It is an error when the previous RO has not completed or supplied HW044.

HI47

Critical

RO/OMO or additional OMO not linked to Organisation

It is an error when additional OMO has not been linked to the Organisation.

HI48

Critical

Change of ownership document not/incorrectly supplied

It is an error when there is not correct/sufficient documentation to prove a Change in Ownership.
