Error status codes for Incentive Programs quality checking, error numbers below INC24

Table 2

Error status codes

Definition

Processing tip

INC02A

Critical

Form incomplete

It is a critical error when all required fields have not been completed on the form and this does affect payment or integrity.

INC02B

Non-critical

Form incomplete

It is a non-critical error/feedback when non-mandatory fields have not been completed on the form and this does not affect payment or integrity.

INC03

Critical

Insufficient supporting documentation supplied

It is an error if there is insufficient documentation to support the request and there is no comment on PaNDA or PIP/WIP-Practice Stream Online to support the action taken.

INC04A

Critical

Declaration incomplete or incorrect signatory.

It is an error when the  Declaration  is:

  • missing the following information, and there is no notation on PIP/WIP – Practice Stream Online:
    • Signature
    • Name
    • Date (where mandatory)
  • not signed by an owner or authorised contact person registered with the practice
  • not signed by the General Practitioner or Nurse Practitioner indicated on the form

Note:  this error code includes when the date of the patient consent is after the date of lodgement.

INC04B

Non-critical

Declaration incomplete or incorrect signatory

It is a non-critical error/feedback where a non-mandatory date has not been completed.

INC05

Critical

University Certification on Teaching Claim form is incomplete or incorrect

It is an error when the  University Declaration  is missing the following information, and there is no notation on PIP/WIP – Practice Stream Online:

  • University name
  • University representative’s full name and signature
  • University declaration date

Or if the university declaration date is:

  • after the date of the first teaching session
  • invalid
  • future dated
  • dated American style

INC06

Critical

Practice Name incorrect

It is an error when the  Practice Name  is keyed incorrectly.

INC07

Critical

Contact Person Details incorrect, including contact not added/removed from the practice profile

It is an error when any of the following  Primary/Secondary Contact Details  are keyed/selected incorrectly, or not keyed/selected/removed from the practice profile:

  • Family Name
  • First Given Name
  • RA Number (if applicable)

INC08A

Critical

Practice Contact Details incorrect

It is an error when any of the following  Practice Contact Details  have been keyed incorrectly or not keyed:

  • Fax number
  • Email address

INC08B

Non-critical

Practice Contact Details incorrect

It is a non-critical error/feedback where a phone number has been keyed incorrectly or not keyed.

INC09

Critical

Practice Address details incorrect

It is an error when any of the following  Address details  have been keyed/selected incorrectly or not updated:

  • postal address
  • main practice location
  • additional location

INC10A

Critical

Practice Accreditation details incorrect

It is an error when any of the following have been keyed/selected incorrectly:

  • Accreditation Status
  • Start and End Dates
  • Address

INC10B

Non-critical

Practice Accreditation details incorrect

It is a non-critical error/feedback when the following has been keyed incorrectly or not keyed:

  • Accreditation Body
  • Accreditation Number

INC11

Critical

Additional practice locations incorrect

It is an error when the additional practice location has been keyed or selected incorrectly.

INC12

Critical

Incorrect Incentive selected

It is an error when an individual PIP incentive, payment tier or level has not been selected or has been selected incorrectly.

INC13A

Critical

Bank details incorrect

It is an error when any of the following  Bank details  have been keyed incorrectly or not updated:

  • BSB
  • Account number

INC13B

Non-critical

Bank details incorrect

It is a non-critical error/feedback when the following has been keyed incorrectly or not keyed:

  • Account name

INC14

Critical

Details of Practice Ownership incorrect

It is an error when any of the following  Practice Ownership  details  have been keyed/selected incorrectly, or not keyed/selected:

  • Arrangement Type
  • Company Name
  • Trading Name
  • Address
  • Declaration

INC15

Critical

Practitioner Start and/or end dates are incorrect

It is an error when the start and/or end date for the following have been keyed incorrectly or not keyed:

  • Provider
  • Nurse Practitioner

INC16

Non-critical

Public Liability insurance details incomplete/incorrect

It is an error when the  Public Liability insurance  has been:

  • selected incorrectly, or
  • not selected

INC18

Critical

Program status field incorrect

It is an error when the closure date in the Practice Program status field has not been keyed correctly when a practice has:

  • ceased operating as a practice
  • ceased operating at a practice branch location

INC19

Critical

Practice incorrectly withdrawn

It is an error when the practice has been incorrectly withdrawn.

INC20

Critical

Relocation or Amalgamation date incorrect

It is an error when the  Relocation  or  Amalgamation  date has been keyed incorrectly.

INC22

Critical

GP ACAI Payments failed

It is an error when a GP ACAI payment has failed because:

  • GP ACAI comments have not been removed from the Provider Directory System
  • new bank details have not been updated on GDES spreadsheet

INC23

Critical

Provider has not been registered or registered incorrectly, including physical location miss-match

It is an error when the provider details:

  • Do not match the physical location or have been registered to the incorrect practice location
  • Have not been registered to a practice location

INC24

Critical

Professional Indemnity insurance expired

It is an error when the Individual GP or nurse practitioner details (IP003) form has been processed and the date of the  Professional Indemnity  has expired.

Error status codes for Incentive Programs quality checking, error numbers INC26 and above

Table 3

Error status codes

Definition

Processing tip

INC26

Critical

RA number has been added to incorrect profile / person

It is an error when the Registration Authority (RA) number or associated person has not been added to the correct profile or keyed incorrectly.

INC28

Critical

Annual/Quarterly statement not updated/incorrect

It is an error when an  Annual/Quarterly Confirmation Statement  has been updated incorrectly or not been updated.

INC29

Critical

Average weekly hours not updated

It is an error when the  Average weekly hours  have been updated incorrectly, or not updated.

INC30

Non-critical

PaNDA work item details are incorrect

It is an error when the  Work Item details  have been keyed incorrectly or not updated:

  • Work classification
  • Practice ID
  • Case ID
  • Completion reason

INC31

Critical

IHI Patient details incorrect or has not been added to the practice profile

It is an error if the:

  • Incorrect patient has been linked to the practice based on the information supplied.
  • Patient has not been added to the profile
  • Information provided has not been keyed

INC32

Non-critical

Aboriginal or Torres Strait Islander indicator incorrect

It is an error when the  Aboriginal or Torres Strait Islander indicator  is not selected correctly.

INC33

Critical

IHI patient information incorrect (doesn't match source documents)

It is an error when the incorrect information has been entered and does not match source documents:

  • IHI chronic disease
  • IHI health check
  • IHI patient consent

INC34

Critical

Incorrect documents uploaded to practice profile or not uploaded to practice profile or PaNDA

It is an error when a form or document has:

  • been uploaded to the incorrect Practice Profile
  • been uploaded and practice has not been clearly identified
  • has not been uploaded to the practice profile or PaNDA

INC36

Critical

Letter incorrect

It is an error when information or content of a letter has not been keyed correctly, or when a letter has been sent via the incorrect communication channel:

  • Letter sent via HPOS and mail was indicated
  • Letter sent via mail and HPOS was indicated

INC37

Non-critical

Outdated form

It is an error when an outdated version of a form has been processed after the appropriate grace period.

INC40

Critical

Incorrect action taken on payment hold

It is an error when a payment hold is not added or removed correctly:

  • Practice details have been updated and the hold has not been removed
  • Payment hold has not been applied when further information has been requested

INC42

Critical

Duplicate claim process leads to a recovery action

It is an error when the information entered results in the incorrect calculation of payments.

INC43

Critical

Work item not processed or processed incorrectly

It is an error when:

  • a work item has not been processed in part or in full
  • the form details do not match the profile
  • Teaching Payment student attendance not entered correctly
