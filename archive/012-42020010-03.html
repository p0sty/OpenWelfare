This page contains steps on how to add or cease acupuncture accreditation for a general practitioner in PDS.

RACGP emails the agency advising of general practitioners eligible for acupuncture accreditation. The below process commences after the email from the RACGP has been received through current workflow arrangements.

Note: email requests received directly from general practitioners requesting this accreditation should not be processed. Escalate to a Programme Support Officer (PSO) who will escalate to a Programme Officer (PO) for further investigation and escalation to the RACGP if necessary.

Requests to cease acupuncture accreditation may be received from either the RACGP or from the general practitioner directly.

How to add acupuncture accreditation

Table 1: this table describes how to add acupuncture accreditation for a general practitioner in PDS.

Step

Action

1

Confirm the general practitioner's details on PDS + Read more ...

  • Open PDS
  • Select Provider, select View
    • to perform a wild card search key the first 3 letters of the last name followed immediately by a %, then
    • key the first 3 letters of the first name followed immediately by a %

Select the Search button.

Do the Details match?

  • Yes, the details match, go to Step 3
  • No, the details do not match, contact a PSO who will escalate to a PO who will email the RACGP if required.

2

Check for Specialty Code + Read more ...

Scroll to the bottom of the Provider Stem record to view the Specialty Codes for the general practitioner.

If acupuncture accreditation (Specialty Code 610) already exists, issue a confirmation letter and go to step 4. Make a note on the request (email or spreadsheet) from RACGP that accreditation has been actioned previously.

If specialty code 610 is not on PDS, go to Step 3.

3

Update PDS + Read more ...

From the Provider Stem record in PDS:

Select:

  • Edit
  • Specialty tab
  • Add Specialty Code
    • Enter Specialty Code: 610
    • Enter Start Date: as indicated on the request from the RACGP, or if not indicated, enter the date the email was received
    • No End Date is required
    • Select Add New
    • Save

4

Preparation of letter + Read more ...

  • Select the letter template - Z2754 Approval and removal of additional accreditation/program item access and complete the general practitioner's details.
  • Save letter as PDF
  • Upload the document to the application in PaNDA

5

Finalise + Read more ...

Once the application has been updated, then:

  • Print the letter and send via post to the general practitioner
  • On the PaNDA file add any notes then, select:
    • Complete
    • Save

How to cease acupuncture accreditation

Step

Action

Request for Removal Received + Read more ...

The request for removal of acupuncture accreditation can be received from either the RACGP or the general practitioner directly.

Confirm general practitioner's details on PDS + Read more ...

  • Select Provider, select View
    • to perform a wild card search key the first 3 letters of the last name followed immediately by a %, then

Select the Search button.

  • Yes, the details match, go to Step 3
  • No, the details do not match:
    • Request received from RACGP: escalate the request to a PSO who will escalate to a PO for further investigation and escalation to the RACGP if required, or
    • Request received from general practitioner: re-categorise the application in PaNDA to the National Office Helpdesk where a PO will email the RACGP if required

3

Update PDS + Read more ...

Select:

  • Edit
  • Specialty tab
  • Amend on the line with Specialty Code 610 (acupuncture accreditation), then:
    • Add end date as advised by the RACGP or, if a specific date has not been requested, use the date the email from RACGP was received
  • Update
  • Notes tab
  • Add New
    • Notes Subject: key Acupuncture accreditation ceased
    • Notes Description: key Acupuncture accreditation ended at RACGP/ general practitioner's request
  • Add New
  • Save
