Triage Tier 1 emails

Table 1

Step

Action

1

Open external email box + Read more ...

Open Microsoft Office Outlook.

Select the relevant PIP or WIP Specialised Provider Services (SPS) email box.

Then select Inbox.

2

Select email + Read more ...

Select the email with the earliest lodgement date. Open and assess the email.

Action the email:

  • Respond, Conduct a security check, go to Table 3
  • Delete, go to Step 3
  • HPOS Notifications, go to Step 5
  • Attention to staff member, go to Step 6
  • Forward to KOFAX, go to Step 7
  • Other programs, go to Step 10
  • Unable to respond, go to Step 12

3

Delete + Read more ...

Delete any non-work related emails.

When an email is received from mail content manager, assess the email. If the email is:

  • Spam, delete the email. Procedure ends here.
  • Work related, go to Step 4

4

Request release of email + Read more ...

Send a request to release the email:

  • Select the mail releases link in the body of the email
  • Key either the PIP or WIP SPS email box in the Bcc field:
  • Key the below details in the body of the email:
    • Logon ID
    • Phone number of operator
    • Date of email
    • Sender’s email
    • Subject line of email
  • Select Send
  • Select Unclassified as the email classification
  • Select Send

File the original and Bcc email in the Mail Release Request folder.

Note: to show the Bcc field, select Options from the task bar. Then select Bcc.

Procedure ends here.

5

HPOS Notifications + Read more ...

If the HPOS notification is from:

  • PIP NO REPLY PRD, go to Step 9
  • PIP <NO REPLY>, go to Step 11

6

Attention to staff member + Read more ...

Forward the email:

  • Select Forward
  • Key the relevant recipient in the To field:
    • Service Delivery staff member, staff member’s name
    • Include the LPS mailbox
    • Select Send
    • Select Unclassified as the email classification
    • Select Send

A list of staff member’s name is available on the shared drive:

  • G:\SPS_Eastwood\Contacts

Go to Step 11.

7

Check attachments + Read more ...

Forward all requests to update practice details or PDS records to KOFAX.

Is there an attachment?

  • Yes, go to Step 8
  • No, go to Step 9

8

Check PDF + Read more ...

Is the attachment in PDF format?

  • Yes, go to Step 9
  • No, respond to the practice using the NON PDF email template. Go to Table 4.

9

Forward to KOFAX + Read more ...

Forward the email to KOFAX.

  • Select Forward
  • Send to the Incentives KOFAX team
  • Select Unclassified as the email classification
  • Select Send

Go to Step 11.

10

Other programs + Read more ...

Check the email to see which program the email relates to.

Forward the email to the relevant program:

  • Select Forward
  • Key the other program’s email address in the To field
    • Search Useful contact information for Services Australia contacts
  • Select Send
  • Select Unclassified as the email classification
  • Select Send

If unable to identify the program, file the email in the Tier 2 folder. Procedure ends here.

Go to Step 11.

11

File the email + Read more ...

File the email in the relevant folder:

  • HPOS notifications, and is:
    • Forwarded to KOFAX, KOFAX – Auto Forward folder
    • PIP <NO REPLY>, Tier 2 folder
  • Attention to staff member, forwarded to staff member folder
  • Attention PSO, forwarded to PSO
  • Form, KOFAX – Auto Forward folder
  • Duplicate Statement, KOFAX – Auto Forward folder
  • RA Number request, KOFAX – Auto Forward folder
  • Recovery of over payments, PSO box – forwarded for action \ recovery
  • Other programs, other programs

Procedure ends here.

12

Unable to respond + Read more ...

Add a comment to the email:

  • Select Action
  • Select Edit Message
  • Key the following details:
    • assessing steps taken
    • other relevant information
  • Close the email
    • A warning message will show
  • Select Yes

File the email in the Tier 2 folder.

3

ACAI security check + Read more ...

The ACAI is payments made to practitioners. Details can only be provided directly to the practitioner.

To check the practitioner:

  • Log on to the Provider Directory System (PDS)
  • Select Provider
  • Select View
  • Key Provider Stem (first 6 digits) in the Search ID field
  • Select Provider Stem from the ID Type drop down
  • Select Search

Does the practitioner name match the details in the email?

  • Yes, go to Table 4
  • No, go to Step 4

4

Security check failed + Read more ...

Respond to the email:

  • Locate the appropriate email template:
    • PIP and WIP – Practice Stream, adding an authorised contact (enquiry not from ACP)
    • ACAI, PIP ACAI email address mismatch
  • Copy and paste the template into the body of the email
  • Complete the missing template details where necessary
  • Key either the relevant PIP, WIP or ACAI SPS email box in the Bcc field:
  • Select Send
  • Select Unclassified as the email classification
  • Select Send

File the original and Bcc email in the Customer Enquiries folder.

Note: to show the Bcc field, select Options from the task bar. Then select Bcc.

Triage HPOS Mail Centre

Table 5

Step

Action

1

Log on to systems + Read more ...

Log on to HPOS Support

  • Select Mail Centres from the Main menu
  • Select PIP or WIP – Practice Stream from the drop down:
    • A list of outstanding HPOS messages will show

2

Select message + Read more ...

Select the message with the earliest lodgement date. Open and assess the message.

Action the email according to the following:

  • Respond, go to Step 3
  • Forward to external email box, go to Step 6

3

Conduct a security check + Read more ...

Identify the RA number and name of the sender in the From field.

Check that these details are from an owner or authorised contact person of the practice:

  • Search for the practice
  • Select Practice Staff System Access from the Main menu
    • The practice Staff System Access screen will show

Do these details match the details on the practice profile?

  • Yes, go to Step 4
  • No, go to Step 5

4

Send HPOS message response + Read more ...

Send a response to the HPOS message:

  • Select Reply
  • Copy and paste an appropriate email template in the body of the message. If a template is not available:
    • use information from guidelines, website or other relevant references
    • copy and paste the relevant information if necessary
    • make sure the information is correct, makes sense and answers the query
  • Upload any attachments (if applicable)
  • Select Send

For assistance with wording, contact Local Peer Support (LPS).

Note: when copy and pasting an email template, the formatting may not transfer. Check that formatting is correct before sending.

Go to Step 7

5

Security check failed + Read more ...

Respond to the email:

  • Select Reply
  • Locate the relevant email template
  • Copy and paste the template in the body of the message
  • Complete the missing template details where necessary
  • Select Send

Go to Step 7.

6

  • Key either the PIP or WIP SPS email box address:
  • Select Forward

  • Select Move
