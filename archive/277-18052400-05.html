When to use each letter

Q805

Shared Care Grant (Agreed full %)

Use the letter to advise carers of the shared care decision based on either evidence provided from all carers or agreement between all carers.

Q806

Shared care Grant - No third party response

Use the letter to advise carers of a shared care decision based on evidence provided by one of the carers only. For example where the other carer has not responded to a request for evidence.

Q807

SSA Shared Care Principal Carer Determination

Send the letter to all carers in a shared care arrangement, where at least one carer is claiming/receiving an income support payment. The letter advises which carer is determined as the principal carer of the child/children.

Q808

SSA Couple Principal Carer Determination

Send the letter to members of a couple advising which parent is determined as the principal carer of the child/children.

Q810

Shared Care Dispute - Request for information

Use this letter to request information and/or evidence about shared care arrangements from any carer.

Q999

Manual letter (only use to request information not covered in script letters)

Use the letter to request evidence from non-mutual customers or to notify customers of a rejected care assessment. See the Resources page in Notification and assessment of shared care arrangements for Family Tax Benefit (FTB) for more information and approved content.

Q888

Manual letter (only use to request information not covered in script letters)

Use the letter to request evidence for Child Care Subsidy (CCS) and additional CCS customers.

Note: free text will be required. Obtain Service Support Officer or Team Leader approval for text. See Creating a Q999 or Q888 letter.

Q777

Request Documents guided procedure (online FTB claims)

Use the letter to request additional information required to assess a Families online claim. Use the Request Documents Q777 guided procedure.
