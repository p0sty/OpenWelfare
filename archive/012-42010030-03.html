Medical practitioner prescriber number

Table 1: This table describes how to issue a prescriber number for a medical practitioner when an Application for a Medicare provider number and, or prescriber number for a medical practitioner form (HW019) is received.

Step

Action

1

Assess application details + Read more ...

Check the application to ensure the following fields have been completed:

  • First name and surname
  • Date of birth
  • Existing provider number (if applicable)
  • Indicated prescriber number is required
  • Declaration must be signed. The date signed is optional

Note: make sure the Application for a Medicare provider number and, or prescriber number for a medical practitioner form (HW019) has been completed.

2

Check eligibility + Read more ...

Check that the medical practitioner:

  • Has a provider number (or is applying for a provider number)
  • Has current medical registration with Ahpra that does not have restrictions to prescribe medications

If an application cannot be processed or if the request is incomplete, see Table 6.

3

Allocate a prescriber number + Read more ...

Allocate a prescriber number in the Provider Directory System (PDS) edit mode. Select:

  • Qualification tab
    • Prescriber in Prescriber Status field
  • Select Validate
  • Select Save

4

Issue confirmation letter and finalise + Read more ...

Select the Z2193 PBS prescriber number and complete the medical professional's details:

  • Print the letter
  • Convert the letter into PDF format
  • Upload the letter to PaNDA
  • Make sure the PaNDA primary, secondary and Case ID fields are completed and correct
  • Add any appropriate notes to the PaNDA file
  • Change PaNDA file to Complete
  • Select Save

Nurse practitioner prescriber number

Forward application in PaNDA for Delegate approval + Read more ...

In PaNDA, use the 'Send for Delegation' function.

4

Determine the location nurse practitioner works in + Read more ...

When the nurse practitioner works:

  • In private practice - a provider number is issued for billing appropriate MBS items. The specialty code 651 is added to the stem and location. The location provider number may be released to the nurse practitioner or appropriate third party
  • In a government funded clinic/hospital/health service listed on a 19(2) or 19(5) Direction - a provider number is issued for billing appropriate MBS items. The specialty code 651 is added to the stem and location. The location provider number may be released to the nurse practitioner or appropriate third party
  • In public hospital with refer and request rights - a provider number is issued for refer and request only. The specialty code 651 is added to the stem and location with the location service ruled to prevent claiming of Medicare benefits when not eligible to claim Medicare benefits. The location provider number may be released to the nurse practitioner or appropriate third party
  • In public hospital where only a prescriber number is required - a 'silent' provider number is issued to activate the prescriber number. The specialty code 651 is added to the stem and location with the location service ruled to prevent claiming of Medicare benefits when not eligible. The location provider number is not to be released as it is for internal PBS purposes only

5

Allocate a prescriber number + Read more ...

Allocate a prescriber number in Provider Directory System (PDS) edit mode.

When processing a prescriber number for a nurse practitioner who is working in private practice, at a location on a 19(2) or 19(5) Direction or in a public hospital with refer and request only, select:

  • Qualification tab
    • Prescriber in Prescriber Status field
  • Select Validate
  • Select Save

Where the nurse practitioner has requested a prescriber number only and a provider record has not previously been created:

Create stem

  • Complete title, surname, first name, other given name (if applicable), date of birth, gender, contact details (where the nurse practitioner has selected these details to be recorded) and qualifications as per provider number application
  • Select Prescriber in Prescriber Status field under Qualification tab
  • Complete Registration Number as per Ahpra, selecting Current from the Status drop down menu
  • Registration Start Date is the date of processing
  • Registration End Date is left blank
  • Key specialty codes 510 and 651 with the start date being date of processing
  • Select Save to save the stem

Create 'silent' provider number

  • Address tab
    • Select Manual Entry
    • Key Office Use Only in Building or Department
    • The hospital/health service or practice name in Street Name
    • Key Suburb and Postcode as per provider application
  • Contact tab
    • Location Start Date to be date of processing
  • Loc-Spec codes tab
    • Select Add New
    • Key 651 in the Specialty Code Descriptor
    • Key date of processing in Start Date
    • Select Add
    • Select Cancel

Add Service Rule

  • Loc Service Rules tab
    • Select Add Service Rule
    • Key 618 in the Service Rule Number
    • Select Yes from the Activity Code drop down menu
    • Key date of processing in Start Date
    • Select Add New
    • Select Cancel
  • Select Save

6

Issue confirmation letter and finalise + Read more ...

  • Select the Z2193 PBS prescriber number and complete the nurse practitioner's details:Print the letter
  • Convert the letter into PDF format
  • Upload the letter to PaNDA
  • Make sure the PaNDA primary, secondary and Case ID fields are completed and correct
  • Add any appropriate notes to the PaNDA file

Return inconsistent or incomplete applications

Table 6: This table describes how to return an application where the application cannot be processed or if the request is incomplete.

Step

Action

1

Issuing a letter to the provider + Read more ...

Where the application cannot be processed due to:

  • Application is incomplete - use Z2306 Provider Registration - We need more information letter template
  • Health professional is ineligible to be issued with a prescriber number - use Z2305 Provider Registration - Reject (Z2305.1903) letter template

Complete the relevant fields in the letter.

Note: there may be more than one reason the application cannot be processed. It is important that, where possible, all reasons are communicated to the applicant.

2

Finalising the application + Read more ...

  • Print the letter and application to post to the health professional
  • Convert the letter into PDF format
  • Upload letter to PaNDA
  • Add any appropriate notes to the PaNDA file
  • Select Complete
  • Select Save
