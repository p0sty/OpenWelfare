Information displayed on ICVCs

Table 1

Item

Field

Description

1

Visible Digital Seal (VDS)

QR code containing individual’s personal and vaccination details. Assists authorities to verify document is unaltered. Is scanned by a QR reader and checked against Australia’s ePassport Country Signing Certificate Authority public key.

2

Issued by [country]

3 digit country code – defaults to Australia (AUS)

3

Version [numeral]

Version control indicates number of requested to date

4

UCVI number/code

Document code

5

Name of the holder

Customer’s name. Format is LAST NAME, FIRST NAME

6

Date of birth

Customer’s date of birth. Format is YYYY-MM-DD

7

Sex

Customer’s sex. Options are M, F or X

8

Passport number

Document number of passport customer has provided

9

Vaccine or prophylaxis

Vaccine code – COVID-19

10

Vaccine brand

Vaccine brand name/s given

11

Disease or agent targeted

Disease – COVID-19

12

Date of vaccination

Date/s vaccine given

13

Dose number

Vaccine dose number

14

Country of vaccination

Country vaccinations were given. This is a mandatory field. If the country is missing, the vaccination provider must update the AIR.

15

Administering centre

The vaccination provider type

16

Vaccine batch number

The batch number of the vaccine given. This is a mandatory field. If the batch number is missing it will show as ‘not recorded’ and the vaccination provider must update the AIR

Reason

DVS results

Table 3: details for Service Officers who verify a document in the Document Verification Service (DVS) and get a Y, N, D or S result.

Result

Description

Y - Matched

Yes, the document is verified.

N - Not matched

Not verified. Check the coding to ensure there are no typos.

If this cannot be verified, escalate to a PSO.

D - Data out of range (document is not able to be electronically verified or is invalid)

The details input are recorded with the issuer but the document cannot be verified.

S - System availability error

System issues. Try again in a few minutes.

If there is a system outage a Network News Update (NNU) Newsflash will be published.
