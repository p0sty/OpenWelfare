Document error status codes

Table 3

Error status codes

Definition

Processing tip

V02

Non-critical

Claim header service provider details incomplete or omitted

It is an error when, on the claim header, the service provider’s details are incomplete or omitted and there is no verification present.

See also V12 for service provider details omitted from voucher or claim form.

V03

Critical

Claim header or form not signed

It is an error when the claim header or form is not signed.

Note: for medical manual claims only - it is not an error when there is no header supplied, or a blank header has been attached to a claim by Services Australia staff.

V04

Critical

Voucher or claim form not signed by veteran when appropriate

It is an error when the veteran (or responsible person) does not sign the voucher or claim form (when DVA stationery used) or the provider has not notated or certified that the veteran is unable to sign.

V07

Non-critical

Item and item description is omitted

It is an error when the item and item description are omitted.

Note: for allied health, item must be supplied.

V08

Non-critical

Amount claimed omitted (when required)

Hospital/Medical

It is an error when the amount claimed is omitted from the voucher or claim form and there is no verification present.

Allied

This error code is only used for Allied Health Fee by Negotiation items which require a dollar amount to be quoted on the voucher.

Note: for allied health default fee items and LMO attendance items 1 to 99 Medical only: It is not an error where a provider does not write the amount claimed on the voucher. Claims for default fee items are processed even if an amount claimed is not written on the voucher. (It is not uncommon, particularly around the start of new fee schedule time, such as the start of the year, for providers not to write an amount claimed for default fee items).

V09

Non-critical

Requesting or referral details omitted

It is an error when the requesting and/or referral:

  • date
  • period of referral
  • provider

are omitted when required and there is no verification present.

V12

Critical

Voucher or claim form - service provider’s details omitted or incomplete

It is an error when the service provider’s details are omitted from the voucher or claim form (when DVA stationery is used).

Note: it is not an error when the:

  • service provider’s details are omitted (or a different provider name (Locum) were present on the voucher) where valid details are on the claim header
  • service voucher states a different provider name (locum) where the provider has completed a claim header

See also V02 for service provider details omitted from claim header.

V13

Critical

Kilometres travelled omitted

It is an error when the kilometres travelled are omitted and there is no verification present.

V16

Non-critical

Separation indicator incomplete

It is an error when the separation indicator on the claim form is incomplete or inconsistent and there is no verification present.

V18

Critical

ICD 10 code omitted

It is an error when the ICD 10 code is omitted when required and there is no verification present.

V20

Critical

No documents to support payment

It is an error when a claim has been processed:

  • without supporting documentation
  • where the documentation used was not an original copy

If documentation is recovered after quality checking is complete, the error code must not be reversed.

Note: Team Leader/PSO must confirm the service and try to retrieve the documents. Where confirmation of the service or retrieval of the documents cannot be achieved, escalate to the VAP Helpdesk.

Financial impact of VAP errors

Errors identified in the VAP quality checking can have a financial impact where a benefit is affected and results in a resubmission or adjustment. The financial impact can be an under or overpayment.

If the error detected at quality checking results in an under or overpayment, the difference between the original benefit paid and the correct benefit paid becomes the amount used to calculate ’financial impact’, see the example below.

See also Other financial impact examples
