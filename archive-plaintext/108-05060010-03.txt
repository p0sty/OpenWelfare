| 1                                 | Updating the grandfathered status |
|                                   | for an existing income stream     |
|                                   | which has lost its grandfathered  |
|                                   | status + Read more ...            |
|                                   |                                   |
|                                   | When a customer with a            |
|                                   | grandfathered income stream has   |
|                                   | their income support payment      |
|                                   | suspended, the system will        |
|                                   | automatically change the category |
|                                   | 2 grandfathered income stream to  |
