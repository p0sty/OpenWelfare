This page contains e-mail templates for communicating with VAP Helpdesk,
links to Veterans' Affairs Processing Quality Checking Procedures,
information about how a health practitioner or facility can repay an
overpaid amount, Electronic Funds Transfer (EFT) bank details for
Veterans' Affairs Processing (VAP), areas of responsibilities for
recovery of VAP overpayments and a link to VAP recovery reason codes and
Prefix. Links to this page may be unavailable to external audiences.

Repay an overpaid amount to VAP

Table 1: This table describes the ways a health practitioner or facility
can repay an overpaid amount to VAP.

+-----------------------------------+-----------------------------------+
| Payment type                      | Description                       |
+-----------------------------------+-----------------------------------+
| Money orders and bank cheques     | Accepting money orders and bank   |
|                                   | cheques                           |
|                                   |                                   |
|                                   | Money orders and bank cheques     |
|                                   | should be issued to Medicare      |
|                                   | Veterans' Treatment Benefits.     |
|                                   | Money orders/cheques issued to    |
|                                   | Medicare, Services Australia,     |
|                                   | Department of Veterans' Affairs   |
|                                   | or Veterans' Affairs Processing   |
|                                   | will also be accepted.            |
+-----------------------------------+-----------------------------------+
| Repayment Cheque or Returned      | Processing Repayment Cheque or    |
| Cheque Advice                     | Returned Cheque Advice            |
|                                   |                                   |
|                                   | The Department of Veterans'       |
|                                   | Affairs (DVA) will forward        |
|                                   | cheques to Medicare, attached to  |
|                                   | a Repayment Cheque or Returned    |
|                                   | Cheque Advice to action.          |
|                                   |                                   |
|                                   | These cheques are processed the   |
|                                   | same as cheques received directly |
|                                   | from a health practitioner or     |
|                                   | facility.                         |
+-----------------------------------+-----------------------------------+
| Electronic funds transfer (EFT)   | Repaying an overpayment via EFT   |
|                                   |                                   |
|                                   | Health practitioners or           |
|                                   | facilities can repay an           |
|                                   | overpayment by EFT using the EFT  |
|                                   | bank details for VAP.             |
|                                   |                                   |
|                                   | When the Department of Veterans'  |
|                                   | Affairs (DVA) receives a payment  |
|                                   | via EFT it will be forwarded to   |
|                                   | Medicare and the site will be     |
|                                   | notified. These payments are      |
|                                   | processed the same as EFT         |
|                                   | payments received directly from a |
|                                   | health practitioner or facility.  |
+-----------------------------------+-----------------------------------+
