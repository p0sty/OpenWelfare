|                                   |     student start date of the new |

|                                   |     payments)                     |

|                                   |     for correct                   |
|                                   |                                   |
|                                   | Coding for Secondary Boarding     |
|                                   | students                          |

|                                   |     update the Student End Date   |

|                                   |     student commenced at the new  |
|                                   |     school                        |
|                                   | -   in the Action field, key 'C'  |
|                                   |     for correct and enter         |
|                                   | -   navigate to EDCH screen, type |
|                                   |     EDC in NXT field and press    |
|                                   |     [Enter] to generate a new EDC |
|                                   |     page                          |
|                                   | -   key the Participation Start   |
|                                   |     Date (PSD) and Student Start  |
|                                   |     Date (SSD) as the Actual date |
|                                   |     the student commenced at the  |
|                                   |     new school and key all other  |
|                                   |     relevant fields               |
|                                   | -   navigate to the EYSB screen,  |
|                                   |     update the Date of Event with |
|                                   |     the day after the student     |
|                                   |     actually ceased at the        |
|                                   |     previous school, remove all   |
|                                   |     boarding /tuition amounts and |
|                                   |     key NPB                       |

|                                   |     display adjusted (ARO) and an |
|                                   |     (OTP) payment lines           |
