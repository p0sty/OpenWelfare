Changing study details to cancel payment when ceasing studies

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Changing study details to cancel  |
|                                   | payment when ceasing studies   +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | If a YA or Austudy (Student)      |
|                                   | ceases full-time study they must  |
|                                   | notify the agency within 14 days  |
|                                   | (Notification Period) to receive  |
|                                   | backdating of JobSeeker Payment   |
|                                   | (JSP) or YA (Job seeker) if they  |
|                                   | are eligible. If they fail to     |
|                                   | notify cessation of full time     |
|                                   | study within 14 days, YA or       |
|                                   | Austudy will be cancelled from    |
|                                   | study end date + 1 day. Payment   |
|                                   | of JSP/YA (Job seeker) may only   |
|                                   | commence from the date the job    |
|                                   | seeker claim was submitted.       |
|                                   |                                   |
|                                   | As the customer is advising they  |
|                                   | are no longer studying full time, |
|                                   | they are no longer eligible for   |
|                                   | payment.                          |
|                                   |                                   |
|                                   | Update the customer record as     |
|                                   | follows:                          |
|                                   |                                   |
|                                   | go to Education Course History    |
|                                   | (EDCH) screen and correct the     |
|                                   | Student End Date: field, to:      |
|                                   |                                   |
|                                   | -   the date the student last     |
|                                   |     studied, or                   |
|                                   | -   the date the student ceased   |
|                                   |     to have an intention to       |
|                                   |     return to study, or           |
|                                   | -   the date the student is no    |
|                                   |     longer undertaking a          |
|                                   |     sufficient study load to      |
|                                   |     qualify for payment. That is, |
|                                   |     the day before their study    |
|                                   |     load reduced                  |
|                                   |                                   |
|                                   | insert a Newstart Education       |
|                                   | Status (NES) entry:               |
|                                   |                                   |
|                                   | -   Event Date: field, code the   |
|                                   |     student end date plus 1 day   |
|                                   | -   Student Status: field, code   |
|                                   |     'NST' (not a full-time        |
|                                   |     student)                      |
|                                   | -   Education Level Attained:     |
|                                   |     field, update if necessary    |
|                                   |                                   |
|                                   | For customers under 22 years that |
|                                   | do not want to continue to        |
|                                   | receive YA as a jobseeker:        |
|                                   |                                   |
|                                   | -   go to the Benefit Action (BA) |
|                                   |     screen                        |
|                                   | -   Service Reason: field, select |
|                                   |     the customers payment type    |
|                                   | -   Action: field, CAN - Cancel   |
|                                   | -   Reason: field 'NST' - not a   |
|                                   |     full-time student             |
|                                   | -   Effect Date: field, new study |
|                                   |     end date plus 1 day.          |
|                                   |                                   |
|                                   | Note: customers 22 years of age   |
|                                   | or over will not require manual   |
|                                   | cancellation via the BA screen.   |
|                                   | These customers will have their   |
|                                   | payment automatically cancelled   |
|                                   | from the updates made to NES and  |
|                                   | EDCH.                             |
|                                   |                                   |
|                                   | Record the details of the         |
|                                   | decision to cancel on a DOC.      |
|                                   |                                   |
|                                   | If the customer has contacted:    |
|                                   |                                   |
|                                   | within their Notification Period, |
|                                   | and they want to receive further  |
|                                   | financial assistance:             |
|                                   |                                   |
|                                   | -   for a job seeker payment, see |
|                                   |      Student to Job seeker        |
|                                   |     transfers                     |
|                                   | -   for an alternative income     |
|                                   |     support, see  Contact in      |
|                                   |     relation to an intended claim |
|                                   |     (CLK)                         |
|                                   |                                   |
|                                   | outside their Notification Period |
|                                   | and want to receive further       |
|                                   | financial assistance, they must   |
|                                   | lodge a new claim. Go to Step 2   |
+-----------------------------------+-----------------------------------+
| 2                                 | Debt incurred for failing to      |
|                                   | notify + Read more ...            |
|                                   |                                   |
|                                   | Determine if the customer has     |
|                                   | incurred a debt for failing to    |
|                                   | notify of ceasing studies, see:   |
|                                   |                                   |
|                                   | -   Raising debts resulting from  |
|                                   |     a failure to report or notify |
|                                   |     and                           |
|                                   | -   Raising debts for Centrelink  |
|                                   |     administered payments for     |
|                                   |     students                      |
|                                   |                                   |
|                                   | Record details of the decision to |
|                                   | cancel and any other action taken |
|                                   | on a DOC.                         |
