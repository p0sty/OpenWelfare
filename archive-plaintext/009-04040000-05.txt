Multiple claims lodged

Table 5

+-----------------------------------+-----------------------------------+
| Qualifying care receivers         | Example                           |
+-----------------------------------+-----------------------------------+
| Nil                               | Customer qualifies for Nil of the |
|                                   | care receivers claimed for.       |
|                                   |                                   |
|                                   | Before rejecting the claims check |
|                                   | if any of the following are       |
|                                   | applicable:                       |
|                                   |                                   |
|                                   | -   Carer Payment (CP) (child)    |
|                                   |     when caring for 2 to 4        |
|                                   |     children each with a          |
|                                   |     disability or medical         |
|                                   |     condition (combined care)     |
|                                   | -   Eligibility for Carer Payment |
|                                   |     (CP) under the Lower Adult    |
|                                   |     Disability Assessment (ADAT)  |
|                                   |     Tool                          |
|                                   | -   Carer Payment (CP) (child)    |
|                                   |     when caring for a disabled    |
|                                   |     adult and 1 or 2 children     |
|                                   |     with a disability or medical  |
|                                   |     condition (multiple care)     |
|                                   |                                   |
|                                   | Rejection is to be completed for  |
|                                   | both/ all receivers claimed for.  |
|                                   |                                   |
|                                   | Carer has lodged CP claim for     |
|                                   | more than one care receiver       |
|                                   |                                   |
|                                   | -   Multiple CP claim is to be    |
|                                   |     made 'Not required'           |
|                                   | -   SOA is to be created for each |
|                                   |     care receiver and 'Rejected'  |
|                                   |     one at a time                 |
|                                   |                                   |
|                                   | Carer is current on CP            |
|                                   |                                   |
|                                   | -   Claim for the new care        |
|                                   |     receiver is to be made 'Not   |
|                                   |     required'                     |
|                                   | -   Claim is considered           |
|                                   |     'Rejected'                    |
|                                   | -   Review CP in respect of the   |
|                                   |     existing care receiver        |
+-----------------------------------+-----------------------------------+
| One                               | Carer has lodged CP claim for     |
|                                   | more than one care receiver and   |
|                                   | qualifies for one of the care     |
|                                   | receivers claimed for.            |
|                                   |                                   |
|                                   | -   Original claim is to be made  |
|                                   |     'Not required'                |
|                                   | -   Create SOA and 'Reject' for   |
|                                   |     the care receiver/s that do   |
|                                   |     not qualify                   |
|                                   | -   Create SOA and 'Grant' for    |
|                                   |     the qualifying care receiver  |
+-----------------------------------+-----------------------------------+
| More than one                     | Customer:                         |
|                                   |                                   |
|                                   | -   qualifies for more than one   |
|                                   |     care receivers claimed for,   |
|                                   |     or                            |
|                                   | -   is in receipt of CP, and      |
|                                   |     qualifies for CP for the new  |
|                                   |     care receiver claimed for     |
|                                   |                                   |
|                                   | Contact customer to confirm care  |
|                                   | receiver being claimed for.       |
|                                   |                                   |
|                                   | Carer has lodged CP claim for     |
|                                   | more than one care receiver       |
|                                   |                                   |
|                                   | -   Withdraw claim for multiple   |
|                                   |     care receivers                |
|                                   | -   Process claim for care        |
|                                   |     receiver being claimed for    |
|                                   |                                   |
|                                   | Carer is current on CP            |
|                                   |                                   |
|                                   | -   Withdraw claim for new care   |
|                                   |     receiver, or                  |
|                                   | -   Process change in care        |
|                                   |     receiver (cancelling original |
|                                   |     care receiver and processing  |
|                                   |     claim for new care receiver)  |
