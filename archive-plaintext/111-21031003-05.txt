Error codes - E status codes - assessing errors

Table 5. assessment errors relate to an incorrect eligibility
assessment. For example, this might include failure to recognise the
correct issue date or to exclude ineligible prescriptions.

Recording errors are listed under B PBS Safety Net Claim system errors.

Processes can be found in Pharmaceutical Benefits Scheme (PBS) Safety
Net processing and enquiries.

+-----------------------------------+-----------------------------------+
| Error codes                       | Error definition                  |
+-----------------------------------+-----------------------------------+
| E01S                              | Entitlement error.                |
|                                   |                                   |
| Critical                          | -   Not eligible - did not meet   |
|                                   |     the eligibility criteria or   |
|                                   |     threshold for Safety Net      |
|                                   |     registration type             |
|                                   | -   Eligible - rejected in error  |
+-----------------------------------+-----------------------------------+
| E02S                              | Date of issue error.              |
|                                   |                                   |
| Non-Critical                      | Eligible, but:                    |
|                                   |                                   |
|                                   | -   incorrectly determined date   |
|                                   |     of issue from manual review,  |
|                                   |     or                            |
|                                   | -   failed to correct the Safety  |
|                                   |     Net Claims system (SNCS)      |
|                                   |     where issue date returned is  |
|                                   |     before 1 July 2022            |
