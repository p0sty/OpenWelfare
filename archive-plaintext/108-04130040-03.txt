| 5                                 | Urban address + Read more ...     |
|                                   |                                   |
|                                   | An urban address identifies,      |
|                                   | labels or locates real estate     |
|                                   | such as houses (including rural   |
|                                   | home sites on less than two       |
|                                   | hectares), flats, units schools,  |
|                                   | parks, businesses, reserves,      |
|                                   | caravan parks, addresses accessed |
|                                   | by water, public amenities in     |
|                                   | urban areas, shopping centres.    |
|                                   |                                   |
|                                   | Urban areas are traditionally     |
|                                   | subdivided into lots. Sites       |
|                                   | within urban areas are usually    |
|                                   | identified by a site address      |
|                                   | which has address numbers         |
|                                   | allocated according to the        |
|                                   | typical subdivision lot size for  |
|                                   | that area. Numbers are allocated  |
|                                   | for anticipated future lots. Lots |
|                                   | are combined for large areas      |
|                                   | hence ranged addresses numbers. A |
|                                   | suffix (e.g. A, B) on the address |
|                                   | number usually indicates          |
|                                   | insufficient numbers were         |
|                                   | available for newly subdivided    |
|                                   | land, where expected number has   |
|                                   | been allocated.                   |
|                                   |                                   |
|                                   | While most councils in Australia  |
|                                   | have adopted national address     |
|                                   | standards, there is no            |
|                                   | requirement for councils to       |
|                                   | renumber existing urban           |
|                                   | properties and some legacy        |
|                                   | addresses will present a problem. |
|                                   | In addition, for new subdivisions |
|                                   | or in other instances where road  |
|                                   | numbers have not yet been         |
|                                   | allocated, allowances must be     |

|                                   |                                   |
|                                   | A rural address will most likely  |

|                                   |                                   |
|                                   | Rural areas currently use a       |
|                                   | mixture of property names, lot    |
|                                   | numbers and delivery points (e.g. |

|                                   | address is for a mail delivery    |
|                                   | point which may not be on or near |
|                                   | the property. Note: postal        |

|                                   | recorded in the address field for |
|                                   | a real estate asset e.g. RMB or   |

|                                   |                                   |
|                                   | In recent years, local, state and |
|                                   | federal government authorities    |
|                                   | have been cooperating to address  |
|                                   | all rural properties with a       |
|                                   | unique number determined by the   |
|                                   | distance of that property's       |
|                                   | (main) egress from the logical    |
|                                   | starting point (datum point) of   |
|                                   | the road, with odd numbers to the |
|                                   | left, even to right. Not all      |
|                                   | states have completed this unique |
|                                   | numbering. Customers who own      |
|                                   | rural properties which have been  |
|                                   | allocated an address under the    |
|                                   | Rural Addressing project would    |
|                                   | have been notified of their new   |
|                                   | address by their local council or |
|                                   | shire authority. The 'new'        |
|                                   | address would also be included in |
|                                   | the rates notices.                |
|                                   |                                   |
|                                   | This address should be used in    |
|                                   | preference to all legacy          |

|                                   | numbers have not yet been         |
|                                   | allocated. The address field must |
