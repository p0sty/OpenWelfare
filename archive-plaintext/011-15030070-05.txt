Table 1

+-----------------------------------+-----------------------------------+
| Reason                            | Use this option when…             |
+-----------------------------------+-----------------------------------+
| Current NOPB exists – verified    | a verified NOPB is currently      |
|                                   | valid and the claimant is seeking |
|                                   | another NTC.                      |
+-----------------------------------+-----------------------------------+
| Current NOPB exists – deemed      | a deemed NOPB is currently valid  |
|                                   | and the claimant is seeking       |
|                                   | another NTC.                      |
+-----------------------------------+-----------------------------------+
| Insufficient reason or medical    | a claimant is seeking to amend    |
| evidence for failing to submit    | either the Care Costs or Medical  |
| the statement by the due date –   | Services without evidence (as per |
| NOC deemed                        | current process). This option is  |
|                                   | not restricted to NOC deemed –    |
|                                   | this wording will be updated in   |
|                                   | the future.                       |
+-----------------------------------+-----------------------------------+
| Failed to submit the statement    | the claimant is seeking to amend  |
| within 2 years from DOJS – NOC    | a deemed NOC more than 2 years    |
| deemed                            | from the date of judgment or      |
|                                   | settlement.                       |

Pre judgment or settlement – evidence and delegations to amend notices

Table 3: outlines the evidence and delegations required to amend a
notice of past benefits (NOPB) at pre judgment or settlement stage.

+-----------------------------------+-----------------------------------+
| Reason for amendment              | Delegation and evidence required  |
+-----------------------------------+-----------------------------------+
| Valid NOPB all services taken     | Generally, a notice of refusal    |
|                                   | will be issued in these           |
|                                   | circumstances, however a Service  |
|                                   | Delivery APS5 or above can        |
|                                   | approve to process on a case by   |
|                                   | case basis:                       |
|                                   |                                   |
|                                   | a recompleted statement by        |
|                                   | claimant with                     |
|                                   |                                   |
|                                   | written evidence from the         |
|                                   | compensation payer or claimant’s  |
|                                   | solicitor that the claim cannot   |
|                                   | be finalised without an accurate  |
|                                   | notice and must be endorsed by:   |
|                                   |                                   |
|                                   | -   a judge and the date has been |
|                                   |     set, or                       |
|                                   | -   the Public Trustee            |
|                                   |                                   |
|                                   | See Escalations and case notes    |
|                                   | for more information on           |
|                                   | escalating a work item to the     |
|                                   | Service Support Officer (APS5).   |
+-----------------------------------+-----------------------------------+
| Valid verified NOPB               | Consider a review if:             |
|                                   |                                   |
|                                   | -   there are processing errors,  |
|                                   |     Service Delivery APS1 can     |
|                                   |     approve                       |
|                                   | -   there is an invoice/receipt   |
|                                   |     from compensation payer that  |
|                                   |     they have paid the account,   |
|                                   |     Service Delivery APS 3 can    |
|                                   |     approve                       |
|                                   | -   care costs do not relate,     |
|                                   |     escalate to Programme         |
|                                   |     Management with the letter    |
|                                   |     from the treating doctor. See |
|                                   |     Escalations and case notes    |
|                                   |     for more information on       |
|                                   |     escalating a work item to     |
|                                   |     Program Management (APS5)     |
|                                   |                                   |
|                                   | Otherwise amend at expiry of      |
|                                   | notice or post settlement by      |
|                                   | processing new statement by       |
|                                   | claimant.                         |
+-----------------------------------+-----------------------------------+
| Expired NOPB                      | Service Delivery APS 3 can        |
|                                   | approve to process a statement to |
|                                   | remove previously verified        |
|                                   | services with a clear statement   |
|                                   | by the claimant.                  |
|                                   |                                   |
|                                   | If unsure, consult a Programme    |
|                                   | Support Officer.                  |
|                                   |                                   |
|                                   | Note: this is not an amended      |
|                                   | NOPB. It is completed in          |
|                                   | statement by claimant processing. |
