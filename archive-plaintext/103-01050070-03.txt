This page contains information on the steps to take when a customer, who
is subject to Income Management, requests the return of a payment sent
to a Third Party Organisation (TPO).

|                                   |     for recall of IM funds table  |

|                                   |     completed prior to commencing |
|                                   |     this process. See Step 1 in   |
|                                   |     the Workflow for recall of IM |
|                                   |     funds table                   |
|                                   | -   If the customer does not wish |
