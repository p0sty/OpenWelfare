|                                   | Is contact with the administrator |
|                                   | or liquidator required?           |

|                                   |                                   |

| 9                                 | Formal Proof of Debt (Form 535) + |
|                                   | Read more ...                     |
|                                   |                                   |

|                                   | return to list Services Australia |

|                                   |                                   |
|                                   | The Form 535 is not standardised  |

|                                   |                                   |

|                                   | it to the liquidator:             |
|                                   |                                   |

|                                   |     debt. See Step 4 in Table 3   |

|                                   | Note: dividends can still apply   |

|                                   | the dividends even if the debt is |
|                                   | written off under insolvency.     |
