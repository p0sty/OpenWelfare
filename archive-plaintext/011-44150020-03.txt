customer is with another person, on speaker phone, or already identified
with family and domestic violence concerns, continue with the current
business conversation. Otherwise, go to the Family and domestic violence
procedure to conduct the risk identification and referral process.

Table 6

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |

| 1                                 | When is a child shown on a third  |
|                                   | card + Read more ...              |
|                                   |                                   |
|                                   | The Medicare enrolment system     |
|                                   | currently allows a child to be    |
|                                   | shown on 2 Medicare cards but     |
|                                   | sometimes it is clear a child     |
|                                   | should be shown on a third card.  |
|                                   |                                   |
|                                   | Children can be included on a     |
|                                   | third Medicare card where there   |
|                                   | are extenuating circumstances.    |
|                                   |                                   |
|                                   | This could be where a child       |
|                                   | already enrolled on 2 Medicare    |
|                                   | cards, has now been placed in     |
|                                   | Out-Of-Home-Care or in the care   |
|                                   | of a grandparent or person other  |
|                                   | than the parent.                  |
|                                   |                                   |
|                                   | These applications must be        |
|                                   | escalated.                        |
+-----------------------------------+-----------------------------------+
| 2                                 | Third card requests must be       |
|                                   | approved + Read more ...          |
|                                   |                                   |
|                                   | A Medicare Enrolment and          |
|                                   | Eligibility PSO must approve      |
|                                   | third card requests. The PSO must |
|                                   | assess the request and all        |
|                                   | documents to determine if they    |
|                                   | can create a third card.          |
|                                   |                                   |
|                                   | Before creating a third card      |
|                                   | check the court order as this     |
|                                   | will generally be quite specific  |
|                                   | about the responsibility, if any, |
|                                   | of both parents or if no contact  |
|                                   | is to occur. In these cases,      |
|                                   | there is no reason for the child  |
|                                   | to remain on that Medicare card   |
|                                   | (Office of Legal Counsel has      |
|                                   | provided this policy).            |
