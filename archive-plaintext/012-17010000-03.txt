This page contains information on how to perform a security check when a
provider calls the Veteran's Affairs Processing (VAP) National Enquiry
Line, responding to provider requests for duplicate statements, and
referrals and escalations.

Provider security check for VAP

Table 1: This table outlines how Service Officers and Programme Support
Officers perform a security check when contacted via the VAP National
Enquiry Line by a health professional or their delegate, practice staff
or authorised representative from a hospital, residential care facility
or approved provider organisation.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Advise security check             |
|                                   | requirement + Read more ...       |
|                                   |                                   |
|                                   | Advise the caller a security      |
|                                   | check must be conducted before    |
|                                   | releasing any information.        |
|                                   |                                   |
|                                   | Request caller details:           |
|                                   |                                   |
|                                   | -   provider number               |
|                                   | -   full name of the provider     |
|                                   | -   address for the provider      |
|                                   |     number supplied (the nearest  |
|                                   |     corner and suburb are         |
|                                   |     acceptable)                   |
|                                   |                                   |
|                                   | If the caller identifies          |
|                                   | themselves as DVA staff, no       |
|                                   | personal information of a         |
|                                   | veteran, provider or service is   |
|                                   | to be released. DVA staff should  |
|                                   | be contacting their own DVA       |
|                                   | General Enquiry line. Services    |
|                                   | Australia can assist DVA staff    |
|                                   | with general enquiries where the  |
|                                   | VAP Operator does not need to     |
|                                   | check any client or provider      |
|                                   | information on the system.        |
+-----------------------------------+-----------------------------------+
| 2                                 | Check information provided + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Check that the information        |
|                                   | provided matches details held in  |
|                                   | the VAP provider directory system |
|                                   | on the Mainframe Customer         |
|                                   | Information Control System (CICS) |
|                                   | screen by keying 'VPUI' and the   |
|                                   | provider number stem.             |
|                                   |                                   |
|                                   | Are details correct?              |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, the Service Officer       |
|                                   |     should advise the caller they |
|                                   |     cannot give them any          |
|                                   |     information as they did not   |
|                                   |     meet the requirements of the  |
|                                   |     security check. If the caller |
|                                   |     disputes this, the call       |
|                                   |     should be transferred to the  |
|                                   |     Team Coach. Procedure ends    |
|                                   |     here                          |
+-----------------------------------+-----------------------------------+
| 3                                 | Identify if caller is the         |
|                                   | provider + Read more ...          |
|                                   |                                   |
|                                   | Identify if the caller is the     |
|                                   | provider, or another person       |
|                                   | (practice staff, delegate or      |
|                                   | authorised representative of a    |
|                                   | hospital, residential care        |
|                                   | facility or approved provider     |
|                                   | organisation).                    |
|                                   |                                   |
|                                   | Is the caller the provider?       |
|                                   |                                   |
|                                   | -   Yes, proceed with the         |
|                                   |     enquiry. Procedure ends here  |
|                                   | -   No, go to Step 4              |
+-----------------------------------+-----------------------------------+
| 4                                 | Caller is not the provider + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | If the caller identifies as       |
|                                   | practice staff or delegate, ask   |
|                                   | the caller:                       |
|                                   |                                   |
|                                   | -   'Do you have permission from  |
|                                   |     the provider to request       |
|                                   |     information about the claim?’ |
|                                   |                                   |
|                                   | If the caller identifies as being |
|                                   | from a hospital, residential care |
|                                   | facility or approved provider     |
|                                   | organisation, ask the caller:     |
|                                   |                                   |
|                                   | -   'Are you an authorised        |
|                                   |     representative of the         |
|                                   |     organisation?'                |
|                                   |                                   |
|                                   | Does the caller have permission   |
|                                   | or are they an authorised         |
|                                   | representative?                   |
|                                   |                                   |
|                                   | -   Yes, proceed with call.       |
|                                   |     Procedure ends here           |
|                                   | -   No, the Service Officer       |
|                                   |     should advise the caller they |
|                                   |     cannot give them any          |
|                                   |     information. If the caller    |
|                                   |     disputes this, the call       |
|                                   |     should be escalated to the    |
|                                   |     Team Coach. Procedure ends    |
|                                   |     here                          |

Issuing duplicate statements to a DVA provider or their delegate

Table 2: This table describes the steps to follow to check that a
statement can be issued to a provider, and the exceptional circumstances
when a duplicate statement can be faxed by a Programme Support Officer
(PSO).

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Ensure provider cannot access via |
|                                   | electronic channel + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | Is the provider able to access    |
|                                   | statements via an electronic      |
|                                   | channel such as online claiming   |
|                                   | or DVA Webclaim?                  |
|                                   |                                   |
|                                   | -   Yes, advise the provider how  |
|                                   |     they can access statements.   |
|                                   |     See DVA Webclaim for          |
|                                   |     Veterans' Affairs Processing  |
|                                   |     (VAP)                         |
|                                   | -   No, go to Step 2              |
+-----------------------------------+-----------------------------------+
| 2                                 | Check date of issue + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | Check the date of issue for the   |
|                                   | requested statement.              |
|                                   |                                   |
|                                   | Has a reasonable delivery time    |
|                                   | elapsed (approximately 10 working |
|                                   | days)?                            |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, advise the provider to    |
|                                   |     wait several more days.       |
|                                   |     Procedure ends here           |
+-----------------------------------+-----------------------------------+
| 3                                 | Confirm address + Read more ...   |
|                                   |                                   |
|                                   | Confirm the address where the     |
|                                   | original statement was sent. This |
|                                   | may involve checking any pay      |
|                                   | group links. Advise the provider. |
|                                   |                                   |
|                                   | If an address change is required, |
|                                   | advise the provider of action     |
|                                   | required by them to update their  |
|                                   | mailing address and redirect the  |
|                                   | provider to appropriate area      |
|                                   | (Provider registration or DVA for |
|                                   | DVA only providers).              |
|                                   |                                   |
|                                   | If an address change has been     |
|                                   | completed after the original      |
|                                   | statement was issued, a manual    |
|                                   | statement will need to be         |
|                                   | produced and the address amended. |
|                                   | See Veterans' Affairs Processing  |
|                                   | (VAP) for contact details if      |
|                                   | needed.                           |
|                                   |                                   |
|                                   | For escalation details, see Item  |
|                                   | 1 in Referrals and escalations    |
|                                   | table.                            |
+-----------------------------------+-----------------------------------+
| 4                                 | Create duplicate statement + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Create a VPSI document            |
|                                   |                                   |
|                                   | -   In Mainframe Customer         |
|                                   |     Information Control System    |
|                                   |     (CICS) screen, key the        |
|                                   |     control line VCEI,            |
|                                   |     PROVIDER#,CLAIMID             |
|                                   | -   Press [Enter]                 |
|                                   |                                   |
|                                   | Claim details history will        |
|                                   | display.                          |
|                                   |                                   |
|                                   | -   Over key VCEI with 'VPSI'     |
|                                   | -   Add 'DOL=ddmmyy' to the end   |
|                                   |     of the control line           |
|                                   | -   Add ',D' to the end of        |
|                                   |     'DOL=DDMMYY,D'                |
|                                   | -   VPSI,PROVIDER#,CLAIM          |
|                                   |     ID,DOL=DDMMYY,D and press     |
|                                   |     [Enter]                       |
|                                   |                                   |
|                                   | Claim Statement will display.     |
|                                   |                                   |
|                                   | For more information on actioning |
|                                   | duplicate provider statements,    |
|                                   | see Payments for Veterans’        |
|                                   | Affairs Processing (VAP).         |
|                                   |                                   |
|                                   | -    For single and multiple      |
|                                   |     statements, see the Reissue   |
|                                   |     single and multiple           |
|                                   |     statements table              |
|                                   | -    For reissue of statements    |
|                                   |     using VCEI, see the Access    |
|                                   |     claims over 6 months for a    |
|                                   |     provider table                |
+-----------------------------------+-----------------------------------+
| 5                                 | Faxing duplicate statements +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Programme Support Officers only   |
|                                   |                                   |
|                                   | Copy claim statement into fax     |
|                                   | template:                         |
|                                   |                                   |
|                                   | -   open the fax duplicate        |
|                                   |     statement template. See       |
|                                   |     Resources page for a link.    |
|                                   | -   copy entire statement from    |
|                                   |     Mainframe                     |
|                                   | -   paste into the fax duplicate  |
|                                   |     statement template            |
|                                   |                                   |
|                                   | To add multiple pages of the      |
|                                   | statement to the template, in     |
|                                   | Mainframe go to the relevant page |
|                                   | of the Claim Statement:           |
|                                   |                                   |
|                                   | -   highlight the required lines  |
|                                   |     to copy                       |
|                                   | -   select Edit > Copy Append     |
|                                   |     from the dropdown list        |
|                                   | -   repeat if required for each   |
|                                   |     additional statement page on  |
|                                   |     screen                        |
|                                   | -   paste at the end of the fax   |
|                                   |     duplicate statement template  |
|                                   |                                   |
|                                   | Confirm the provider’s fax number |
|                                   | and print and send duplicate      |
|                                   | statement.                        |
