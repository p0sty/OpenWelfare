PPL details for customers with children born or entering care both
before and on or after 1 July 2023.

PPL and DAP rates

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Parental Leave Pay (PPL) + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | The PPL rate is based on the      |
|                                   | national minimum wage rate. From  |
|                                   | 1 July 2023, the rate of PPL is   |
|                                   | $882.75 per week.                 |
|                                   |                                   |
|                                   | PPL:                              |
|                                   |                                   |
|                                   | is based on the national minimum  |
|                                   | wage and paid to eligible         |
|                                   | customers                         |
|                                   |                                   |
|                                   | -   for up to 18 weeks (90        |
|                                   |     payable days) for a child     |
|                                   |     born or adopted before 1 July |
|                                   |     2023                          |
|                                   | -   for up to 20 weeks (100       |
|                                   |     payable days) for a child     |
|                                   |     born or adopted on or after 1 |
|                                   |     July 2023                     |
|                                   |                                   |
|                                   | is a taxable payment and is       |
|                                   | therefore included in a person's  |
|                                   | Adjusted Taxable Income (ATI)     |
|                                   |                                   |
|                                   | is not subject to income          |
|                                   | management                        |
|                                   |                                   |
|                                   | is counted as assessable ordinary |
|                                   | income for calculating the rate   |
|                                   | of payment for income support     |
|                                   | payments                          |
|                                   |                                   |
|                                   | The PPL income test applies to    |
|                                   | eligibility and does not affect   |
|                                   | the rate of PPL.                  |
|                                   |                                   |
|                                   | A rate change applies from the    |
|                                   | start of an employee's first pay  |
|                                   | period on or after 1 July each    |
|                                   | year. The system automatically    |
|                                   | calculates and adjusts a          |
|                                   | customer's record to account for  |
|                                   | the increase.                     |
|                                   |                                   |
|                                   | Instalments of PPL that crossover |
|                                   | the 1 July increase will likely   |
|                                   | result in the need for payment of |
|                                   | arrears. Arrears (including PPL   |
|                                   | funds distributed to an employer  |
|                                   | prior to 1 July) will be          |
|                                   | delivered with the next available |
|                                   | payment.                          |
|                                   |                                   |
|                                   | For children born or adopted:     |
|                                   |                                   |
|                                   | before 1 July 2023, PPL is made   |
|                                   | up of:                            |
|                                   |                                   |
|                                   | -   a PPL period of up to 12      |
|                                   |     weeks (60 payable days), plus |
|                                   | -   30 Flexible PPL days          |
|                                   |                                   |
|                                   | on or after 1 July 2023, 100 PPL  |
|                                   | days                              |
|                                   |                                   |
|                                   | PPL period                        |
|                                   |                                   |
|                                   | For customers with children that  |
|                                   | are born or adopted before 1 July |
|                                   | 2023, the PPL period is based on  |
|                                   | a standard Monday to Friday       |
|                                   | working week. It is paid for each |
|                                   | weekday in the Paid Parental      |
|                                   | Leave period even if the          |
|                                   | customer:                         |
|                                   |                                   |
|                                   | -   usually only works a few days |
|                                   |     a week, or                    |
|                                   | -   on a weekend                  |
|                                   |                                   |
|                                   | It must be paid as a single       |
|                                   | continuous block within 1 year of |
|                                   | the child’s birth or adoption.    |
|                                   |                                   |
|                                   | The maximum PPL period is 12      |
|                                   | weeks (60 payable weekdays), if   |
|                                   | the child’s birth or adoption on  |
|                                   | or after 1 July 2020.             |
|                                   |                                   |
|                                   | Flexible PPL days                 |
|                                   |                                   |
|                                   | Customers with children that are  |
|                                   | born or adopted before 1 July     |
|                                   | 2023 can access 30 Flexible PPL   |
|                                   | days. These can be paid on        |
|                                   | weekends and weekdays. Flexible   |
|                                   | PPL days must be paid within 2    |
|                                   | years of the child’s birth or     |
|                                   | adoption.                         |
|                                   |                                   |
|                                   | PPL days                          |
|                                   |                                   |
|                                   | Customers with children born or   |
|                                   | adopted on or after 1 July 2023,  |
|                                   | the maximum PPL entitlement to    |
|                                   | PPL is 100 days. The PPL period   |
|                                   | will be removed, meaning all 100  |
|                                   | days are to be treated as         |
|                                   | ‘flexible’ PPL days where they    |
|                                   | can be accessed as:               |
|                                   |                                   |
|                                   | -   a single block                |
|                                   | -   all scattered days            |
|                                   | -   any combination of smaller    |
|                                   |     blocks and scattered days     |
|                                   |                                   |
|                                   | PPL days can be received for any  |
|                                   | day of the week (Monday to        |
|                                   | Sunday).                          |
|                                   |                                   |
|                                   | A nominated start date for the    |
|                                   | PPL period will no longer be      |
|                                   | required and PPL days must be     |
|                                   | paid within 2 years of the        |
|                                   | child’s birth or adoption.        |
|                                   | However at least one PPL day must |
|                                   | be taken before the child’s first |
|                                   | birthday or first anniversary of  |
|                                   | care.                             |
+-----------------------------------+-----------------------------------+
| 2                                 | Dad and Partner Pay (DAP) + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | DAP can be paid for a child born  |
|                                   | or adopted before 1 July 2023.    |
|                                   |                                   |
|                                   | The DAP rate is based on the      |
|                                   | National Minimum Wage rate. From  |
|                                   | 1 July 2023, the current rate of  |
|                                   | DAP is $882.75 per week.          |
|                                   |                                   |
|                                   | The DAP income test applies to    |
|                                   | eligibility and does not affect   |
|                                   | the rate of DAP.                  |
|                                   |                                   |
|                                   | DAP:                              |
|                                   |                                   |
|                                   | -   is based on the national      |
|                                   |     minimum wage and paid for up  |
|                                   |     to 2 weeks to eligible        |
|                                   |     customers                     |
|                                   | -   is a taxable payment and is   |
|                                   |     therefore included in a       |
|                                   |     person's Adjusted Taxable     |
|                                   |     Income (ATI)                  |
|                                   | -   is not subject to income      |
|                                   |     management                    |
|                                   |                                   |
|                                   | Income from DAP is counted as     |
|                                   | assessable ordinary income for    |
|                                   | calculating the rate of payment   |
|                                   | for income support payments.      |
|                                   |                                   |
|                                   | DAP is based on a standard Monday |
|                                   | to Friday working week. It is     |
|                                   | paid for each weekday in the DAP  |
|                                   | period (up to 2 weeks), even if   |
|                                   | the customer usually works a few  |
|                                   | days a week, or on a weekend.     |
|                                   |                                   |
|                                   | DAP will be paid at the beginning |
|                                   | of the DAP period as long as the  |
|                                   | customer's claim is finalised by  |
|                                   | that date.                        |
|                                   |                                   |
|                                   | Payment will be made into the     |
|                                   | customer's nominated bank account |
|                                   | and Pay as You Go (PAYG)          |
|                                   | withholdings will be deducted     |
|                                   | from the payment at a rate of 15% |
|                                   | unless another amount is advised  |
|                                   | by the customer (0% - 50%).       |
|                                   |                                   |
|                                   | No other deductions from DAP are  |
|                                   | allowed.                          |
|                                   |                                   |
|                                   | For multiple births or multiple   |
|                                   | children adopted as part of the   |
|                                   | same adoption process, DAP is     |
|                                   | only payable for 1 child.         |
