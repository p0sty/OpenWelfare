NEIS Pre-Training/Small Business Training

Table 1

+-----------------------------------+-----------------------------------+
| Step                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Customers undertaking NEIS        |
|                                   | Pre-Training/Small Business       |
|                                   | Training + Read more ...          |
|                                   |                                   |
|                                   | Customers wanting to start a      |
|                                   | small business or who already run |
|                                   | a micro-business can access NEIS  |
|                                   | Pre-Training/Small Business       |
|                                   | Training. This accredited         |
|                                   | training course helps job seekers |
|                                   | gain a nationally recognised      |
|                                   | qualification and is appropriate  |
|                                   | to starting and/or running a      |
|                                   | small business. Depending on the  |
|                                   | type of proposed new business, or |
|                                   | existing micro business, the      |
|                                   | NEIS/Self-Employment provider     |
|                                   | will refer the job seeker to      |
|                                   | either a:                         |
|                                   |                                   |
|                                   | -   Certificate III in            |
|                                   |     Entrepreneurship and New      |
|                                   |     Business (BSB30220)           |
|                                   | -   Certificate IV in             |
|                                   |     Entrepreneurship and New      |
|                                   |     Business (BSB40320)           |
|                                   | -   Micro Business Skill Set      |
|                                   |     (BSBSS00102)                  |
|                                   | -   New Business Ventures Skill   |
|                                   |     Set (BSBSS00103)              |
|                                   | -   Small Business Management     |
|                                   |     Skill Set (BSBSS00104)        |
|                                   |                                   |
|                                   | During the training, job seekers  |
|                                   | will develop a Business Plan,     |
|                                   | which will need to be approved by |
|                                   | their NEIS/Self-Employment        |
|                                   | provider.                         |
|                                   |                                   |
|                                   | When a JobSeeker Payment (JSP),   |
|                                   | Youth Allowance (job seeker)      |
|                                   | (YA), Parenting Payment Single    |
|                                   | (PPS) or Special Benefit (SpB)    |
|                                   | job seeker with mutual obligation |
|                                   | requirements, or Disability       |
|                                   | Support Payment (DSP) job seeker  |
|                                   | with Compulsory Participation     |
|                                   | Requirements commences NEIS       |
|                                   | Pre-Training/Small Business       |
|                                   | Training, the activity will       |
|                                   | auto-display on the Activity and  |
|                                   | Exemption Summary (AEX) page in   |
|                                   | Process Direct with the code      |
|                                   | 'NEI'.                            |
|                                   |                                   |
|                                   | To view an approved activity in   |
|                                   | Process Direct:                   |
|                                   |                                   |
|                                   | -   Key 'AEX' in the Super Key,   |
|                                   |     select Enter                  |
|                                   | -   The Activity and Exemption    |
|                                   |     Summary (AEX) will display    |
|                                   |                                   |
|                                   | Job seekers will be able to       |
|                                   | attend the NEIS                   |
|                                   | Pre-Training/Small Business       |
|                                   | Training activity, prior to the   |
|                                   | commencement date and record      |
|                                   | their attendance as part of their |
|                                   | points based activity (PBA). Job  |
|                                   | seekers undertaking NEIS          |
|                                   | Pre-Training/Small Business       |
|                                   | Training may be eligible for      |
|                                   | EdEP. See Education Entry Payment |
|                                   | (EdEP) for information on         |
|                                   | eligibility and coding            |
|                                   | requirements.                     |
|                                   |                                   |
|                                   | Participation in NEIS             |
|                                   | Pre-Training/Small Business       |
|                                   | Training fully satisfies the      |
|                                   | customer's requirements and the   |
|                                   | customer will continue to receive |
|                                   | their income support payment for  |
|                                   | the duration of their NEIS        |
|                                   | Pre-Training/Small Business       |
|                                   | Training.                         |
+-----------------------------------+-----------------------------------+
| 2                                 | Voluntary job seekers             |
|                                   | participating in NEIS             |
|                                   | Pre-Training/Small Business       |
|                                   | Training + Read more ...          |
|                                   |                                   |
|                                   | When a customer in receipt of Age |
|                                   | Pension, Parenting Payment        |
|                                   | Partnered (PPP), PPS (with a      |
|                                   | child aged under 6), or Carer     |
|                                   | Payment (CP), commences in NEIS   |
|                                   | Pre-Training/Small Business       |
|                                   | Training, the activity will       |
|                                   | auto-create a Manual Follow-up    |
|                                   | (MFU) action on the customer's    |
|                                   | record. Staff should select this  |
|                                   | activity and cancel it as no      |
|                                   | action is required.               |
|                                   |                                   |
|                                   | These customers are not subject   |
|                                   | to requirements. Their            |
|                                   | participation in NEIS             |
|                                   | Pre-Training/Small Business       |
|                                   | Training will have no impact on   |
|                                   | their payment eligibility.        |
