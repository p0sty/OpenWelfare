+-----------------------------------+-----------------------------------+
| Step                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Customers undertaking Small       |
|                                   | Business Training + Read more ... |
|                                   |                                   |
|                                   | Customers wanting to start a      |
|                                   | small business or who already run |
|                                   | a micro-business can access Small |
|                                   | Business Training. This           |
|                                   | accredited training course helps  |
|                                   | job seekers gain a nationally     |
|                                   | recognised qualification and is   |
|                                   | appropriate to starting and/or    |
|                                   | running a small business.         |
|                                   | Depending on the type of proposed |
|                                   | new business, or existing micro   |
|                                   | business, the Self-Employment     |
|                                   | Assistance provider will refer    |
|                                   | the job seeker to either a:       |
|                                   |                                   |
|                                   | -   Certificate III in            |
|                                   |     Entrepreneurship and New      |
|                                   |     Business (BSB30220)           |
|                                   | -   Certificate IV in             |
|                                   |     Entrepreneurship and New      |
|                                   |     Business (BSB40320)           |
|                                   | -   Micro Business Skill Set      |
|                                   |     (BSBSS00102)                  |
|                                   | -   New Business Ventures Skill   |
|                                   |     Set (BSBSS00103)              |
|                                   | -   Small Business Management     |
|                                   |     Skill Set (BSBSS00104)        |
|                                   |                                   |
|                                   | During the training, job seekers  |
|                                   | will develop a Business Plan,     |
|                                   | which will need to be approved by |
|                                   | their Self-Employment Assistance  |
|                                   | provider.                         |
|                                   |                                   |
|                                   | When a JobSeeker Payment (JSP),   |
|                                   | Youth Allowance (job seeker)      |
|                                   | (YA), Parenting Payment Single    |
|                                   | (PPS - not PNX) or Special        |
|                                   | Benefit (SpB) job seeker with     |
|                                   | mutual obligation requirements    |
|                                   | commences Small Business          |
|                                   | Training, the Department of       |
|                                   | Employment and Workplace          |
|                                   | Relations (DEWR) send an advice   |
|                                   | to notify Services Australia. The |
|                                   | activity will auto-display on the |
|                                   | Activity and Exemption Summary    |
|                                   | (AEX) screen in Process Direct    |
|                                   | with the activity type code       |
|                                   | 'Small Business Training'.        |
|                                   |                                   |
|                                   | To view an approved activity in   |
|                                   | Process Direct:                   |
|                                   |                                   |
|                                   | -   Key 'AEX' in the Super Key,   |
|                                   |     select Enter                  |
|                                   | -   The Activity and Exemption    |
|                                   |     Summary (AEX) will display    |
|                                   |     activity type 'Small Business |
|                                   |     Training' with start and end  |
|                                   |     dates as per DEWR's advice    |
|                                   |                                   |
|                                   | For Disability Support Payment    |
|                                   | (DSP) customers with              |
|                                   | participation requirements and    |
|                                   | participating in Small Business   |
|                                   | Training, the AEX screen will not |
|                                   | display the 'Small Business       |
|                                   | Training' activity. Service       |
|                                   | Officers will need to access the  |
|                                   | Participation Summary page in     |
|                                   | Process Direct, then select the   |
|                                   | Activity type/Non-applied         |
|                                   | exemptions tab to view this       |
|                                   | activity. In addition, Service    |
|                                   | Officers can negotiate the DSP    |
|                                   | Participation Plan and include    |
|                                   | 'Small Business                   |
|                                   | Training/Coaching' as required.   |
|                                   | For details on negotiating DSP    |
|                                   | Participation Plan, see           |
|                                   | Disability Support Pension (DSP)  |
|                                   | Participation Plan.               |
|                                   |                                   |
|                                   | Job seekers will be able to       |
|                                   | attend the Small Business         |
|                                   | Training activity, prior to the   |
|                                   | Small Business Coaching           |
|                                   | commencement date, and record     |
|                                   | their attendance as part of their |
|                                   | points based activity (PBA). Job  |
|                                   | seekers undertaking Small         |
|                                   | Business Training may be eligible |
|                                   | for EdEP. See Education Entry     |
|                                   | Payment (EdEP) for information on |
|                                   | eligibility and coding            |
|                                   | requirements.                     |
|                                   |                                   |
|                                   | Participation in Small Business   |
|                                   | Training fully satisfies the      |
|                                   | customer's requirements and the   |
|                                   | customer will continue to receive |
|                                   | their income support payment for  |
|                                   | the duration of their Small       |
|                                   | Business Training. The customer   |
|                                   | remains provider managed and      |
|                                   | their employment services         |
|                                   | provider should negotiate a Job   |
|                                   | Plan with the customer and        |
|                                   | include the Small Business        |
|                                   | Training activity.                |

+-----------------------------------+-----------------------------------+

| 1                                 | Job seekers under JSP             |
|                                   | conditions - Small Business       |
|                                   | Coaching + Read more ...          |
|                                   |                                   |
|                                   | DEWR send an advice to notify     |
|                                   | Services Australia when a job     |
|                                   | seeker in receipt of JobSeeker    |
|                                   | Payment (JSP), Youth Allowance    |
|                                   | (YA) and Special Benefit (SpB)    |
|                                   | (under JSP conditions) has        |
|                                   | commenced in Small Business       |
|                                   | Coaching under Self-Employment    |
|                                   | Assistance. The Small Business    |
|                                   | Coaching activity will            |
|                                   | auto-display on the Activity and  |
|                                   | Exemption Summary (AEX) screen.   |
|                                   | If the customer is also receiving |
|                                   | the 'Self-Employment Allowance',  |
|                                   | the details will display on the   |
|                                   | Other Government Payments (OGP)   |
|                                   | screen.                           |
|                                   |                                   |
|                                   | JSP and SpB job seekers who are   |
|                                   | undertaking Small Business        |
|                                   | Coaching and eligible for a       |
|                                   | partial rate of income support    |
|                                   | payment, will be deemed as fully  |
|                                   | meeting their mutual obligation   |
|                                   | requirements. As such, Small      |
|                                   | Business Coaching participants    |
|                                   | (apart from those with DES or PNX |
|                                   | providers) are managed by         |
|                                   | Services Australia and will be    |
|                                   | exited from their Workforce       |
|                                   | Australia Service provider's      |
|                                   | caseload, as they will not be     |
|                                   | required to attend any provider   |
|                                   | appointments. Service Officers    |
|                                   | must negotiate a new Job Plan     |
|                                   | with the Small Business Coaching  |
|                                   | participant.                      |
+-----------------------------------+-----------------------------------+
| 2                                 | JSP, YA and SpB job seekers in    |
|                                   | receipt of Self-Employment        |
|                                   | Allowance + Read more ...         |
|                                   |                                   |
|                                   | When the Small Business Coaching  |
|                                   | activity notification is received |
|                                   | from DEWR, the job seeker's       |
|                                   | income support payable rate       |
|                                   | (including add-ons) will be       |
|                                   | reduced by the amount of          |
|                                   | Self-Employment Allowance         |
|                                   | received (direct deduction,       |
|                                   | dollar-for-dollar).               |
|                                   | Self-Employment Allowance is paid |
|                                   | at the 'single, 22 or over, with  |
|                                   | no children' rate of JobSeeker    |
|                                   | Payment (JSP) which will often be |
|                                   | more than the job seeker's rate   |
|                                   | of income support, and will       |
|                                   | usually reduce the income support |
|                                   | rate to nil.                      |
|                                   |                                   |
|                                   | If the job seeker is receiving a  |
|                                   | higher rate of JSP or SpB, they   |
|                                   | may have an ongoing entitlement   |
|                                   | to a partial rate of income       |
|                                   | support payment in addition to    |
|                                   | Self-Employment Allowance.        |
|                                   |                                   |
|                                   | Where a customer's rate is        |
|                                   | reduced to nil due to combination |
|                                   | of the Self-Employment Allowance  |
|                                   | and earnings, their income        |
|                                   | support payment status may        |
|                                   | display as 'current at nil rate   |
|                                   | (CUR/CZR)'. Income support        |
|                                   | payment will auto cancelled after |
|                                   | 6 consecutive fortnights of nil   |
|                                   | rate.                             |
|                                   |                                   |
|                                   | Services Australia's system will  |
|                                   | automatically code the OGP screen |
|                                   | with the event start date, amount |
|                                   | and payment frequency of          |
|                                   | Self-Employment Allowance as per  |
|                                   | advice received from DEWR. No     |
|                                   | coding is required by Service     |
|                                   | Officers.                         |
|                                   |                                   |
|                                   | Prior to 3 December 2022, some    |
|                                   | JSP or SpB customers in receipt   |
|                                   | of Self-Employment Allowance who  |
|                                   | were eligible for higher rates    |
|                                   | were coded with a manual top-up   |
|                                   | rate of income support. For       |
|                                   | information about continuing to   |
|                                   | pay manual rate of JSP or SpB as  |
|                                   | well as Self-Employment           |
|                                   | Allowance, see the Resources      |
|                                   | page.                             |
|                                   |                                   |
|                                   | JSP, YA and SpB job seekers who   |
|                                   | receive Self-Employment Allowance |
|                                   | from DEWR and that results in     |
|                                   | cancellation of income support    |
|                                   | payment from Services Australia,  |
|                                   | may be eligible for a Low Income  |
|                                   | Health Care Card (LIC). These job |
|                                   | seekers are required to lodge a   |
|                                   | claim for the LIC in order to     |
|                                   | receive the LIC.                  |
|                                   |                                   |
|                                   | Create a Display On Access DOC    |
|                                   | using reason code OTH, stating    |
|                                   | the job seeker is undertaking     |
|                                   | Small Business Coaching and fully |
|                                   | meeting their mutual obligation   |
|                                   | requirements.                     |
+-----------------------------------+-----------------------------------+
| 3                                 | JSP, YA and SpB job seekers -     |
|                                   | Small Business Coaching           |
|                                   | Suspensions + Read more ...       |
|                                   |                                   |
|                                   | When a JSP, YA or SpB job seeker  |
|                                   | is suspended from Small Business  |
|                                   | Coaching, an advice will be sent  |
|                                   | from DEWR and the Small Business  |
|                                   | Coaching end date will be         |
|                                   | auto-extended by 13 weeks on the  |
|                                   | AEX screen where the income       |
|                                   | support status is still current.  |
|                                   |                                   |
|                                   | If a JSP, YA or SpB job seeker    |
|                                   | attends a service centre to claim |
|                                   | or enquires about an income       |
|                                   | support payment due to Self       |
|                                   | Employment Allowance suspension,  |
|                                   | the Service Officer will need to  |
|                                   | contact the provider to confirm   |
|                                   | the job seeker's Small Business   |
|                                   | Coaching suspension reason. If    |
|                                   | the suspension is due to a        |
|                                   | medical condition, and it is      |
|                                   | confirmed that Self-Employment    |
|                                   | Allowance is suspended and all    |
|                                   | other eligibility rules are met,  |
|                                   | Service Officers can grant or     |
|                                   | reassess the rate of the income   |
|                                   | support payment for the period of |
|                                   | the Small Business Coaching       |
|                                   | suspension.                       |
|                                   |                                   |
|                                   | If the suspension is due to other |
|                                   | reasons, such as not complying    |
|                                   | with Small Business Coaching      |
|                                   | requirements, Service Officers    |
|                                   | should reject the claim because   |
|                                   | the job seeker is receiving a     |
|                                   | precluding payment and refer the  |
|                                   | job seeker back to their          |
|                                   | Self-Employment provider.         |
|                                   |                                   |
|                                   | Service Officers should document  |
|                                   | any discussions with              |
|                                   | Self-Employment providers and     |
|                                   | scan any documentation provided   |
|                                   | by the job seeker, which confirms |
|                                   | suspension of participation in    |
|                                   | Small Business Coaching.          |
|                                   |                                   |
|                                   | For information about Small       |
|                                   | Business Coaching participants    |
|                                   | claiming income support payments  |
