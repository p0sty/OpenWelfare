This page contains an example of the 28 day count for the start date of
the Paid Parental Leave (PPL) period, links to relevant forms and
publications, the Newborn Supplement (NBS) maximum rate map, and
information about claim signature requirements.

Example - 28 day count for start date of PPL period for a child born or adopted before 1 July 2023

This table contains an example of applying the 28 day start date of the
PPL period.

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Twenty eight day count for start  |
|                                   | date of PPL period for a child    |
|                                   | born or adopted before 1 July     |
|                                   | 2023                              |
|                                   |                                   |
|                                   | Jenny's baby Gabrielle is born on |
|                                   | 20 May 2016. Jenny lodges a PPL   |
|                                   | claim for Gabrielle on 25 May     |
|                                   | 2016, which is pended awaiting    |
|                                   | proof of birth.                   |
|                                   |                                   |
|                                   | Jenny uploads the proof of birth  |
|                                   | online on 17 June 2016.           |
|                                   |                                   |
|                                   | The count of 28 days begins the   |
|                                   | day after Gabrielle's birth,      |
|                                   | which will be 21 May 2016 and     |
|                                   | ends, 28 days later on 17 June    |
|                                   | 2016 at midnight. Jenny has       |
|                                   | submitted all the required        |
|                                   | documentation within the 28 day   |
|                                   | time frame for a determination to |
|                                   | be made. Jenny has met all the    |
|                                   | eligibility requirements and as   |
|                                   | requested, Jenny's PPL period     |
|                                   | will begin from Gabrielle's       |
|                                   | birthdate of 20 May 2016.         |
+-----------------------------------+-----------------------------------+
