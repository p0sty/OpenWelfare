|                                   | portion of an instalment paid to  |

|                                   | raised against the responsible    |
|                                   | organisation payee. The ABSTUDY   |
|                                   | Payee/Debtor Details (EYPY)       |
|                                   | screen will hold the organisation |
|                                   | details and the debt activity     |
|                                   | will be automatically transferred |
|                                   | to the debtor organisation.       |
|                                   |                                   |
|                                   | Term-in-advance boarding          |
|                                   | allowance paid to a boarding      |
|                                   | institution prior to 1 January    |
|                                   | 2019                              |
|                                   |                                   |
|                                   | If a debt amount occurs from the  |
|                                   | term-in-advance amount paid to    |
|                                   | the boarding institution, a debt  |
|                                   | amount should be raised only for  |
|                                   | the amount that relates to the    |
|                                   | period that a boarding            |
|                                   | institution does not retain the   |
|                                   | away from home payments. Record a |
|                                   | DOC on both the student and board |
|                                   | organisation record of action     |
|                                   | undertaken. See the Resources     |
|                                   | page for information about        |
|                                   | term-in-advance payments.         |
|                                   |                                   |
|                                   | In this circumstance, whilst the  |
|                                   | boarding institution, or Hostel,  |
|                                   | is entitled to retain the way     |
|                                   | from home entitlements already    |
|                                   | received via a term-in-advance    |
|                                   | payment for a period after the    |
|                                   | student discontinues, the student |
|                                   | is deemed to have ceased          |
|                                   | eligibility to the away from home |
|                                   | rate from the date their          |
|                                   | circumstances changed.            |
|                                   |                                   |
|                                   | The student is not entitled to    |
|                                   | any fortnightly residual boarding |
|                                   | allowance from the date they are  |
|                                   | deemed to have ceased eligibility |
|                                   | for the away from home rate.      |
|                                   | Residual boarding allowance       |
|                                   | components will need to be ceased |
|                                   | by coding Rate Component Override |
|                                   | (EPRC) screen.                    |
|                                   |                                   |
|                                   | Access the EPRC screen:           |
|                                   |                                   |
|                                   | use the field (?) help to key the |
|                                   | relevant values:                  |
|                                   |                                   |
|                                   | -   start date for the override   |
|                                   |     in the Start Date field (for  |
|                                   |     example, student end date +   |
|                                   |     1)                            |
|                                   | -   end date in the End Date      |
|                                   |     field (if non-standard rate   |
|                                   |     will be for a defined period, |
|                                   |     for example, end of the       |
|                                   |     term-in-advance entitlement   |
|                                   |     period)                       |
|                                   | -   override type in the          |
|                                   |     Component field               |
|                                   | -   the reason in the Reason      |
|                                   |     field                         |
|                                   | -   the amount in the Amount      |
|                                   |     field (which will be $0.00)   |
|                                   | -   Source and DOR fields, and    |
|                                   |     press [Enter]                 |
|                                   |                                   |
|                                   | finalise activity on Assessment   |
|                                   | Results (AR) screen, ensuring     |
|                                   | that the residual boarding amount |
|                                   | has ceased from the correct date, |
|                                   | and                               |
|                                   |                                   |
|                                   | record a DOC advising reason for  |
|                                   | decision                          |
|                                   |                                   |
