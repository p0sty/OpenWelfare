Initiating EIIP customer and partner interventions

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Initiating the intervention +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Before starting the EIIP          |
|                                   | conversation with the customer,   |
|                                   | the IRS intervention must be      |
|                                   | initiated STA (Started).This will |
|                                   | make sure that all updates to the |
|                                   | customer record can attribute to  |
|                                   | the intervention.                 |
|                                   |                                   |
|                                   | If the customer has been:         |
|                                   |                                   |
|                                   | -   contacted by the Outbound     |
|                                   |     Dialler, go to Step 2         |
|                                   | -   sent a Request for            |
|                                   |     information (IRSRFI) letter,  |
|                                   |     go to Step 3                  |
+-----------------------------------+-----------------------------------+
| 2                                 | Initiation - customer contacted   |
|                                   | by Outbound Dialler + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | If the IRS review has not been    |
|                                   | initiated, it will show:          |
|                                   |                                   |
|                                   | -   Review type: EII              |
|                                   | -   Review method: PHO or FIL     |
|                                   | -   Review status: REG            |
|                                   | -   Review origin: TDF or DE or   |
|                                   |     EUD                           |
|                                   |                                   |
|                                   | To initiate the intervention,     |
|                                   | record contact details in IRS.    |
|                                   |                                   |
|                                   | Go to Step 4.                     |
+-----------------------------------+-----------------------------------+
| 3                                 | Initiation - inbound contact or   |
|                                   | successful outbound call          |
|                                   | attempt + Read more ...           |
|                                   |                                   |
|                                   | As an IRSRFI letter has been      |
|                                   | issued to the customer, the       |
|                                   | intervention has already been     |
|                                   | initiated.                        |
|                                   |                                   |
|                                   | Record inbound contact details in |
|                                   | IRS.                              |
+-----------------------------------+-----------------------------------+
| 4                                 | Does the customer have a          |
|                                   | partner? + Read more ...          |
|                                   |                                   |
|                                   | -   Yes, go to Step 5             |
|                                   | -   No, see Assessing and         |
|                                   |     finalising EIIPs. Procedure   |
|                                   |     ends here                     |
+-----------------------------------+-----------------------------------+
| 5                                 | Partner intervention + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | It is important to take into      |
|                                   | account the partner's             |
|                                   | circumstances.                    |
|                                   |                                   |
|                                   | Initiating the partner's          |
|                                   | intervention will make sure all   |
|                                   | updates to customer and partner   |
|                                   | records can attribute to the      |
|                                   | interventions.                    |
|                                   |                                   |
|                                   | This is done by updating IRCD for |
|                                   | the partner record:               |
|                                   |                                   |
|                                   | from the customer's record, key   |
|                                   | 'SR' (swap record) in Next: to go |
|                                   | to the partner's record           |
|                                   |                                   |
|                                   | go to the Review Summary (IRRS)   |
|                                   | screen and 'K' the partner's EIIP |
|                                   |                                   |
|                                   | the Review Contact Summary IRCS   |
|                                   | screen will present               |
|                                   |                                   |
|                                   | -   if the intervention needs to  |
|                                   |     be initiated (create and      |
|                                   |     close the contact on the IRCD |
|                                   |     screen), or                   |
|                                   | -   if it is already STA, close   |
|                                   |     the contact on the IRCD       |
|                                   |     screen                        |
|                                   |                                   |
|                                   | from the partner's record, key    |
|                                   | 'SR' in Next: to go back to       |
|                                   | customer's record                 |
|                                   |                                   |
|                                   | Manual partner interventions      |
|                                   |                                   |
|                                   | Do not load a manual partner      |
|                                   | intervention if the partner was   |
|                                   | not getting a Centrelink payment  |
|                                   | during the determined scope       |
|                                   | period of the intervention.       |
|                                   |                                   |
|                                   | If the customer has a partner and |
|                                   | no partner (PTR) intervention is  |
|                                   | loaded, and the customer received |
|                                   | a Centrelink payment, create a    |
|                                   | manual partner review (MPR). See  |
|                                   | Adding a Manual Partner           |
|                                   | Intervention.                     |
+-----------------------------------+-----------------------------------+
| 6                                 | Partner contacts + Read more ...  |
|                                   |                                   |
|                                   | If the partner's intervention:    |
|                                   |                                   |
|                                   | does not contain match data,      |
|                                   | there is no need for the EIIP     |
|                                   | conversation                      |
|                                   |                                   |
|                                   | does contain match data,          |
|                                   | undertake the EIIP process with   |
|                                   | both customer and partner before  |
|                                   | completing the interventions      |
|                                   |                                   |
|                                   | -   This may or may not be at the |
|                                   |     same time                     |
|                                   | -   The education message must be |
|                                   |     given to both before          |
|                                   |     completing the IRS            |
|                                   |     interventions                 |
|                                   | -   The CO who undertakes the     |
|                                   |     second contact is responsible |
|                                   |     for completing both IRS       |
|                                   |     interventions                 |
|                                   |                                   |
|                                   | If the partner calls in response  |
|                                   | to the IRSRFI letter, and has no  |
|                                   | match data:                       |
|                                   |                                   |
|                                   | -   Update the partner's IRCD     |
|                                   |     screen. Key 'Y' in the        |
|                                   |     Expected Response Date field. |
|                                   | -   Advise the partner that the   |
|                                   |     customer still needs to       |
|                                   |     contact for the early         |
|                                   |     intervention discussion       |
|                                   |                                   |
|                                   | See Assessing and finalising      |
|                                   | EIIPs.                            |
