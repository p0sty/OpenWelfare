Help individuals get ready for vaccination

Table 3

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Do not give health or clinical    |
|                                   | advice + Read more ...            |
|                                   |                                   |
|                                   | If an individual asks if they     |
|                                   | should get vaccinated, or when    |
|                                   | and where they can get            |
|                                   | vaccinated:                       |
|                                   |                                   |
|                                   | -   do not provide advice about   |
|                                   |     vaccines                      |
|                                   | -   tell them they can go to the  |
|                                   |     online COVID-19 Vaccine       |
|                                   |     Clinic Finder                 |
|                                   | -   direct them to the National   |
|                                   |     coronavirus and COVID-19      |
|                                   |     helpline or australia.gov.au  |
|                                   |     for the latest information    |
|                                   |                                   |
|                                   | To help individuals get ready for |
|                                   | COVID-19 vaccination:             |
|                                   |                                   |
|                                   | -   individuals enrolled in       |
|                                   |     Medicare, go to Step 2        |
|                                   | -   individuals who are not       |
|                                   |     eligible for Medicare, go to  |
|                                   |     Step 3                        |
+-----------------------------------+-----------------------------------+
| 2                                 | Individuals who are enrolled in   |
|                                   | Medicare + Read more ...          |
|                                   |                                   |
|                                   | Individuals who are eligible for  |
|                                   | Medicare can do 3 things to get   |
|                                   | ready:                            |
|                                   |                                   |
|                                   | -   get a myGov account           |
|                                   | -   link Medicare and update      |
|                                   |     their contact details         |
|                                   |                                   |
|                                   | Trained Service Officers can help |
|                                   | individuals by:                   |
|                                   |                                   |
|                                   | -   telling them how to get a     |
|                                   |     myGov account. See How users  |
|                                   |     create a myGov account and    |
|                                   |     link services                 |
|                                   | -   giving them a linking code to |
|                                   |     link Medicare to myGov. See   |
|                                   |     Using the myGov Staff Linking |
|                                   |     Application                   |
|                                   | -   telling them How to get proof |
|                                   |     of your COVID-19 vaccination  |
|                                   |     in Medicare online account or |
|                                   |     the Express Plus Medicare     |
|                                   |     mobile app                    |
|                                   | -   checking their contact        |
|                                   |     details for Medicare are      |
|                                   |     correct. See Amend address in |
|                                   |     the Consumer Directory        |
|                                   |     Maintenance System (CDMS) for |
|                                   |     Medicare                      |
+-----------------------------------+-----------------------------------+
| 3                                 | Individuals who are not eligible  |
|                                   | for Medicare + Read more ...      |
|                                   |                                   |
|                                   | Tell individuals who are not      |
|                                   | eligible for Medicare that they   |
|                                   | can get proof of their            |
|                                   | vaccinations online by:           |
|                                   |                                   |
|                                   | -   getting an Individual         |
|                                   |     Healthcare Identifier (IHI).  |
|                                   |     They can request an IHI       |
|                                   |     online by accessing the IHI   |
|                                   |     service in myGov or           |
|                                   |     completing the Request an     |
|                                   |     Individual Healthcare         |
|                                   |     Identifier form (MS003)       |
|                                   | -   using the IHI service or      |
|                                   |     registering for a My Health   |
|                                   |     Record using their IHI. The   |
|                                   |     Australian Digital Health     |
|                                   |     Agency is responsible for My  |
|                                   |     Health Record. See Register   |
|                                   |     for a My Health Record for    |
|                                   |     how to register               |
|                                   |                                   |
|                                   | Trained Service Officers can help |
|                                   | individuals by:                   |
|                                   |                                   |
|                                   | -   checking the individual's     |
|                                   |     details in the AIR are        |
|                                   |     correct. To update, see Amend |
|                                   |     individual registration on    |
|                                   |     the Australian Immunisation   |
|                                   |     Register                      |
|                                   | -   reading the IHI general       |
|                                   |     enquiries- frequently asked   |
|                                   |     questions                     |
