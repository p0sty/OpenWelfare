This page contains information on how to generate a customer receipt
number when Cuba has not generated one automatically.

Record a customer receipt number

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Start point - record a customer   |
|                                   | receipt number + Read more ...    |
|                                   |                                   |
|                                   | Start point - Customer Receipt    |
|                                   | Number window.                    |
+-----------------------------------+-----------------------------------+
| 2                                 | Generate a customer receipt       |
|                                   | number + Read more ...            |
|                                   |                                   |
|                                   | -   Select the Generate Receipt   |
|                                   |     Number button. This opens the |
|                                   |     Generate Receipt Number       |
|                                   |     dialogue box                  |
|                                   | -   Select the appropriate option |
|                                   |     from the Type drop down list  |
|                                   | -    Select the OK button. This   |
|                                   |     will generate the receipt     |
|                                   |     number and return you to the  |
|                                   |      Customer Receipt Number      |
|                                   |     window.                       |
+-----------------------------------+-----------------------------------+
