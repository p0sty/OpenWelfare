+-----------------------------------+-----------------------------------+

|                                   | HPOS (Form Upload) received       |

|                                   |                                   |
|                                   | Select the link to HPOS Support   |
|                                   | in PaNDA, check the file is       |
|                                   | correct and respond with          |
|                                   | incorrect category wording. See   |
|                                   | the Resources page for the        |
|                                   | wording.                          |
|                                   |                                   |
|                                   | See Table 9 for information on    |
|                                   | trouble shooting.                 |
+-----------------------------------+-----------------------------------+
| 2                                 | Is the prescription to be         |
|                                   | processed by DVA/RPBS? + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | When a Department of Veterans'    |
|                                   | Affairs (DVA) file number is      |

|                                   |                                   |
|                                   | Select the folder to save the     |
|                                   | file into                         |
|                                   |                                   |
|                                   | Name the file 'DVA prescription'  |
|                                   | and select Save                   |
|                                   |                                   |
|                                   | Open 'File Explorer' and locate   |
|                                   | the file                          |
|                                   |                                   |
|                                   | Right click on the file and       |
|                                   | select Send to > Mail recipient.  |

|                                   | email address                     |

|                                   |     selected as                   |
|                                   |     SENSITIVE:PERSONAL before     |
|                                   |     selecting 'Send'              |
|                                   |                                   |
|                                   | Delete file from File Explorer    |
|                                   |                                   |

|                                   | the HPOS item by using the 9      |
|                                   | digit HPOS reference number to    |
|                                   | locate the item and send a reply. |
|                                   | See the Resources page for the    |
|                                   | required wording to be included   |
|                                   | in the email response             |
|                                   |                                   |
|                                   | In PaNDA, add a Processing note   |
|                                   | on the file indicating action     |
|                                   | taken and change work status to   |
|                                   | 'Completed'                       |
|                                   |                                   |
|                                   | Procedure ends here.              |

|                                   | Prescriptions received by mail    |
|                                   |                                   |

|                                   |     page for the correct DVA      |

Process to reissue a returned written Authority prescription

Table 8: this table describes how Service Officers process an Authority
prescription returned to Services Australia from Australia Post.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Check if the Authority            |
|                                   | prescription should be returned   |
|                                   | to patient or prescriber + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | If the Authority prescription     |
|                                   | indicates:                        |
|                                   |                                   |
|                                   | -   return to patient, go to Step |
|                                   |     2                             |
|                                   | -   return to prescriber, go to   |
|                                   |     Step 3                        |
+-----------------------------------+-----------------------------------+
| 2                                 | Authority prescription indicates  |
|                                   | return to patient - check if      |
|                                   | address correct + Read more ...   |
|                                   |                                   |
|                                   | Check the patient's address on    |
|                                   | the Authority prescription        |
|                                   | matches the address on the        |
|                                   | envelope.                         |
|                                   |                                   |
|                                   | -   If address does not match,    |
|                                   |     create a new envelope. Go to  |
|                                   |     Step 4                        |
|                                   | -   If address matches, fill in   |
|                                   |     Returned Authority            |
|                                   |     Application Letter. The       |
|                                   |     Resources page contains a     |
|                                   |     link to this letter. Go to    |
|                                   |     Step 4                        |
+-----------------------------------+-----------------------------------+
| 3                                 | Authority prescription indicates  |
|                                   | return to prescriber - check if   |
|                                   | address correct + Read more ...   |
|                                   |                                   |
|                                   | Check the prescriber's address on |
|                                   | the Authority prescription        |
|                                   | matches the address on the        |
|                                   | envelope.                         |
|                                   |                                   |
|                                   | -   If address does not match,    |
|                                   |     create a new envelope. Go to  |
|                                   |     Step 5                        |
|                                   | -   If address matches, contact   |
|                                   |     prescriber for new address    |
|                                   |     details to forward the        |
|                                   |     prescription                  |
|                                   | -   If address does not match and |
|                                   |     the doctor's contact details  |
|                                   |     on the script are invalid go  |
|                                   |     to Step 4                     |
+-----------------------------------+-----------------------------------+
| 4                                 | Confirm Prescriber Details in     |
|                                   | Provider Directory System (PDS) + |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | -   Log into the Medicare         |
|                                   |     Portal - eBusiness Gateway    |
|                                   | -   From the dropdown select the  |
|                                   |     PDS, then submit              |
|                                   | -   From the Provider Directories |
|                                   |     open Provider                 |
|                                   | -   View                          |
|                                   | -   Basic search                  |
|                                   | -   From the ID Type dropdown     |
|                                   |     select Prescriber Number      |
|                                   | -   Type in prescriber number in  |
|                                   |     the Search ID field and       |
|                                   |     search                        |
|                                   | -   If prescriber not found refer |
|                                   |     to a Local Peer Support (LPS) |
|                                   | -   If prescriber found and       |
|                                   |     contact details are recorded  |
|                                   |     call the prescriber           |
|                                   | -   If prescriber found but no    |
|                                   |     contact details recorded      |
|                                   |     refer to a LPS                |
+-----------------------------------+-----------------------------------+
| 5                                 | Search for and view an            |
|                                   | authority + Read more ...         |
|                                   |                                   |
|                                   | Access Online PBS Authorities -   |
|                                   | Search Authorities                |
|                                   |                                   |
|                                   | In the Medicare - eBusiness       |
|                                   | Gateway:                          |
|                                   |                                   |
|                                   | -   Log on to PBS Authority       |
|                                   |     Application System from the   |
|                                   |     eBusiness System dropdown     |
|                                   | -   Submit and select Search      |
|                                   |     Authorities                   |
|                                   | -   Enter the search criteria     |
|                                   |                                   |
|                                   | There are 4 Search types, each    |
|                                   | with mandatory or minimum fields. |
|                                   | Mandatory fields are marked with  |
|                                   | an *. Explanatory text will       |
|                                   | display to indicate minimum       |
|                                   | fields                            |
|                                   |                                   |
|                                   | The search types include:         |
|                                   |                                   |
|                                   | -   Medicare Number               |
|                                   | -   Patient (name, DOB)           |
|                                   | -   Authority, and                |
|                                   | -   Prescriber                    |
|                                   |                                   |
|                                   | The default Search type is        |
|                                   | Medicare Number. Enter the        |
|                                   | patient’s Medicare number if this |
|                                   | is the Search type being used     |
|                                   |                                   |
|                                   | If another Search type is to be   |
|                                   | used, select from the Search type |
|                                   | dropdown                          |
|                                   |                                   |
|                                   | To search, select [Enter] or      |
|                                   | Search                            |
|                                   |                                   |
|                                   | Select from the search result     |
|                                   |                                   |
|                                   | Authorities matching the search   |
|                                   | criteria will display             |
|                                   |                                   |
|                                   | To View an authority, select View |
|                                   |                                   |
|                                   | Key the following information     |
|                                   | into the Authority Comments:      |
|                                   |                                   |
|                                   | Add 'RTS - what action has been   |
|                                   | taken'. Action taken can be:      |
|                                   |                                   |
|                                   | -   Rx now forwarded to           |
|                                   |     prescriber                    |
|                                   | -   Incorrect address on          |
|                                   |     envelope, resent to patient   |
|                                   | -   Incorrect address on          |
|                                   |     envelope, resent to           |
|                                   |     prescriber                    |
|                                   |                                   |
|                                   | Select Save                       |
