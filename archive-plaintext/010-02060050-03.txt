| 4                                 | One Time Access Code (OTAC)       |
|                                   | rules + Read more ...             |
|                                   |                                   |
|                                   | An OTAC:                          |
|                                   |                                   |
|                                   | -   is unique to a                |
|                                   |     school/boarding institution,  |
|                                   |     the term/year and the ABSTUDY |
|                                   |     Payment on Advice process     |
|                                   | -   will allow multiple entries   |
|                                   |     to the eService but will      |
|                                   |     expire once information is    |
|                                   |     submitted to Services         |
|                                   |     Australia                     |
|                                   |                                   |
|                                   | Schools/boarding institutions can |
|                                   | save information in the Education |
|                                   | Institutions eService and should  |
|                                   | be encouraged to continually save |
|                                   | data as entered.                  |
|                                   |                                   |
|                                   | The ABSTUDY Payment on Advice     |
|                                   | Student Input screen will allow   |
|                                   | for all information to be         |
|                                   | completed. However, a timeout     |
|                                   | message will display after 15     |
|                                   | minutes of non-keying. If a       |
|                                   | timeout message occurs, then the  |
|                                   | information coded in the eService |
|                                   | will be lost. Schools/boarding    |
|                                   | institutions will need to reuse   |
|                                   | their OTAC to log back into the   |
|                                   | eService and re-code the          |
|                                   | information.                      |
|                                   |                                   |
|                                   | No further changes can be made to |
|                                   | the data once it has been         |
|                                   | submitted to Services Australia.  |
|                                   | Any amendments will need to be    |
|                                   | submitted to Services Australia   |
|                                   | using the SY106 - First or last   |
|                                   | day of attendance for ABSTUDY     |
|                                   | Away from home students form,     |
|                                   | available on the Services         |
|                                   | Australia website. See the        |
|                                   | Resources page for a copy of the  |
|                                   | Payment on Advice OTAC letter.    |
|                                   |                                   |
|                                   | If Services Australia issues a    |
|                                   | new OTAC (if OTAC was lost or a   |
|                                   | new OTAC is issued for the next   |
|                                   | term), then the previous OTAC     |
|                                   | will expire.                      |
