PPL employer debt raising and recovery

Table 1

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Debt identification + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | Services Australia raise and      |
|                                   | recover PPL debts from employees  |
|                                   | and/or employers where an excess  |
|                                   | payment has occurred.             |
|                                   |                                   |
|                                   | Third parties (individuals or     |
|                                   | organisations) may also have a    |
|                                   | debt raised against them for any  |
|                                   | PPL received in error. See        |
|                                   | Requesting refunds from third     |
|                                   | parties.                          |
+-----------------------------------+-----------------------------------+
| 2                                 | Determining who owes the debt +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | The Paid Parental Leave Employer  |
|                                   | Processing Team (PPLEPT) issue a  |
|                                   | Q318 letter to the employer. This |
|                                   | confirms the location of the PPL  |
|                                   | funding amounts for the debt      |
|                                   | period.                           |
|                                   |                                   |
|                                   | Where exceptional circumstances   |
|                                   | have occurred, such as the death  |
|                                   | of the PPL claimant, Paid         |
|                                   | Parental Leave Employer           |
|                                   | Processing Team (PPLEPT) staff    |
|                                   | will:                             |
|                                   |                                   |
|                                   | -   make 2 genuine attempts to    |
|                                   |     contact the employer, whilst  |
|                                   |     maintaining the privacy and   |
|                                   |     confidentiality of the        |
|                                   |     customer                      |
|                                   | -   request the employer to       |
|                                   |     urgently return the letter to |
|                                   |     minimise debt impacts         |
|                                   | -   issue the Q318 letter         |
|                                   |                                   |
|                                   | The Q318 letter asks:             |
|                                   |                                   |
|                                   | -   what payments the claimant    |
|                                   |     received                      |
|                                   | -   how much tax the claimant     |
|                                   |     paid, and                     |
|                                   | -   has the tax been remitted to  |
|                                   |     the Australian Taxation       |
|                                   |     Office (ATO)                  |
|                                   |                                   |
|                                   | Employers must supply this        |
|                                   | information in writing within 14  |
|                                   | days. Issue a new letter if the   |
|                                   | employer advises they made an     |
|                                   | error.                            |
|                                   |                                   |
|                                   | When the employer provides PPL,   |
|                                   | raise the debt against the person |
|                                   | who has the money. If the         |
|                                   | employer:                         |
|                                   |                                   |
|                                   | -   has not yet paid the money to |
|                                   |     the employee, the debt is     |
|                                   |     recoverable from the employer |
|                                   | -   has paid the money to the     |
|                                   |     employee, the money is        |
|                                   |     recoverable from the employee |
+-----------------------------------+-----------------------------------+
| 3                                 | Employer debts - how they may     |
|                                   | occur + Read more ...             |
|                                   |                                   |
|                                   | Raise a debt against the employer |
|                                   | when:                             |
|                                   |                                   |
|                                   | -   there was an advance of PPL   |
|                                   |     for a period where PPL is no  |
|                                   |     longer payable                |
|                                   | -   they have failed to provide   |
|                                   |     the PPL to the employee       |
|                                   | -   they have not made the agency |
|                                   |     aware of a change in          |
|                                   |     circumstances                 |
|                                   | -   the employer receives an      |
|                                   |     incorrect amount of PPL as a  |
|                                   |     result of an administrative   |
|                                   |     error                         |
|                                   | -   the agency receives an        |
|                                   |     outcome from the Fair Work    |
|                                   |     Ombudsman (FWO)               |
|                                   | -   there is a duplicate payment  |
|                                   | -   the debt is due to fraud      |
|                                   |     and/or misrepresentation, for |
|                                   |     example in relation to the    |
|                                   |     Income Test                   |
|                                   |                                   |
|                                   | Note: raise the whole amount as a |
|                                   | debt if the employer fails to     |
|                                   | provide information requested in  |
|                                   | the Q318.                         |
|                                   |                                   |
|                                   | For debt benefit types and        |
|                                   | components applicable to the PPL  |
|                                   | claim and debt period see         |
|                                   | Resources.                        |
+-----------------------------------+-----------------------------------+
| 4                                 | Employer debts + Read more ...    |
|                                   |                                   |
|                                   | Check how much employer debt is   |
|                                   | owed.                             |
|                                   |                                   |
|                                   | An employer is to repay any PPL   |
|                                   | still held by them. This includes |
|                                   | any withheld Pay As You Go (PAYG) |
|                                   | deductions which have not been    |
|                                   | remitted to the ATO.              |
+-----------------------------------+-----------------------------------+
| 5                                 | Employer debts - how they are     |
|                                   | recovered + Read more ...         |
|                                   |                                   |
|                                   | The agency has the following      |
|                                   | options for recovering a PPL debt |
|                                   | from an employer:                 |
|                                   |                                   |
|                                   | -   cash repayments               |
|                                   | -   garnishee                     |
|                                   | -   legal proceedings (civil      |
|                                   |     recovery)                     |
|                                   |                                   |
|                                   | See Recovering debts from         |
|                                   | organisations.                    |
|                                   |                                   |
|                                   | Note: PPL employers/organisations |
|                                   | can make payments by electronic   |
|                                   | funds transfer as a last resort.  |
|                                   | See Garnishee of continuing       |
|                                   | income to recover debts.          |
+-----------------------------------+-----------------------------------+
| 6                                 | Further notice of debt letters +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | A letter issues when:             |
|                                   |                                   |
|                                   | -   a PPL employer debt remains   |
|                                   |     outstanding for 28 days after |
|                                   |     initial notice of debt        |
|                                   | -   an existing debt payment      |
|                                   |     arrangement becomes more than |
|                                   |     seven days overdue            |
+-----------------------------------+-----------------------------------+

PPL customer debt raising and recovery

Table 2

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Customer debts - how they may     |
|                                   | occur + Read more ...             |
|                                   |                                   |
|                                   | PPL debts may be raised if:       |
|                                   |                                   |
|                                   | customer was not eligible for     |
|                                   | payment due to:                   |
|                                   |                                   |
|                                   | -   ceasing to be a permanent     |
|                                   |     resident                      |
|                                   | -   payment made after death      |
|                                   | -   working on a PPL day          |
|                                   | -   not having care of a child on |
|                                   |     a PPL day                     |
|                                   | -   duplicate payment             |
|                                   |                                   |
|                                   | for a child born or adopted       |
|                                   | before 1 July 2023:               |
|                                   |                                   |
|                                   | -   no longer being the primary   |
|                                   |     carer                         |
|                                   | -   PPL payability ceasing due to |
|                                   |     return to work                |
|                                   | -   transfer of payment           |
|                                   |                                   |
|                                   | fraud and/or misrepresentation    |
|                                   |                                   |
|                                   | when an employer is responsible   |
|                                   | for delivering PPL and any        |
|                                   | flexible PPL scheduled within 12  |
|                                   | months of the child birth or      |
|                                   | adoption                          |
+-----------------------------------+-----------------------------------+
| 2                                 | How much will be raised? + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | The employee is to repay all      |
|                                   | amounts. This includes Pay As You |
|                                   | Go (PAYG) deductions which have   |
|                                   | been remitted to the ATO.         |
|                                   |                                   |
|                                   | For debt benefit types and        |
|                                   | components applicable to the PPL  |
|                                   | claim and debt period see         |
|                                   | Resources.                        |
+-----------------------------------+-----------------------------------+
| 3                                 | Account payable - customer debt + |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | For PPL customer debts, key the   |
|                                   | total amount of all PPL amounts   |
|                                   | paid to the customer in the       |
|                                   | Amount Received field, regardless |
|                                   | of whether the payment was        |
|                                   | delivered by the employer or the  |
|                                   | agency.                           |
|                                   |                                   |
|                                   | When the PAYG withholdings amount |
|                                   | has been paid, it must be         |
|                                   | included in the account payable   |
|                                   | free text.                        |
|                                   |                                   |
|                                   | Note: do not include amounts paid |
|                                   | to the employer that were not     |
|                                   | passed on to the customer. These  |
|                                   | amounts are raised as employer    |
|                                   | debts.                            |
+-----------------------------------+-----------------------------------+
| 4                                 | How they are recovered + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | A customer can have their debt    |
|                                   | offset against the PPL payable to |
|                                   | their current partner or other    |
|                                   | secondary claimant. This is for   |
|                                   | cases where payments are          |
|                                   | retrospective. In this situation, |
|                                   | the secondary claimant has to     |
|                                   | request to offset the debt.       |
|                                   |                                   |
|                                   | The agency has the following      |
|                                   | options for recovering a PPL debt |
|                                   | from an employee:                 |
|                                   |                                   |
|                                   | -   cash repayments               |
|                                   | -   withholding repayments from   |
|                                   |     current Centrelink payments   |
|                                   | -   offsetting against PPL        |
|                                   |     arrears, where these occur in |
|                                   |     the same activity             |
|                                   | -   garnishee                     |
|                                   | -   legal proceedings (civil      |
|                                   |     recovery)                     |
|                                   |                                   |
|                                   | See Debt recovery for Centrelink. |
+-----------------------------------+-----------------------------------+
| 5                                 | Further information for           |
|                                   | customers + Read more ...         |
|                                   |                                   |
|                                   | Debt finalised in the current     |
|                                   | financial year                    |
|                                   |                                   |
|                                   | For payments delivered by the     |
|                                   | agency, customers can request a   |
|                                   | refund of the PAYG withholding    |
|                                   | from the agency. See Refunding    |
|                                   | Australian Taxation Office (ATO)  |
|                                   | excess tax deductions.            |
|                                   |                                   |
|                                   | For payments delivered by their   |
|                                   | employer, customers can request a |
|                                   | refund of the PAYG withholding    |
|                                   | amount from their employer. On    |
|                                   | their income tax return,          |
|                                   | customers must report the total   |
|                                   | PAYG withheld and total taxable   |
|                                   | income.                           |
|                                   |                                   |
|                                   | For more information, see         |
|                                   | Parental Leave Pay (PPL) debts    |
|                                   | between employers and employees.  |
|                                   |                                   |
|                                   | If portions of the debt were      |
|                                   | delivered by both the agency and  |
|                                   | their employer, see Parental      |
|                                   | Leave Pay (PPL) debts between     |
|                                   | employers and employees.          |
|                                   |                                   |
|                                   | Debt finalised after the end of   |
|                                   | the financial year in which it    |
|                                   | occurred but tax return not       |
|                                   | lodged                            |
|                                   |                                   |
|                                   | The customer will need to:        |
|                                   |                                   |
|                                   | -   recover the PAYG amount from  |
|                                   |     the Australian Taxation       |
|                                   |     Office (ATO) when they        |
|                                   |     complete their tax return     |
|                                   | -   make an agreement with the    |
|                                   |     agency allowing them to repay |
|                                   |     the PAYG amount upon receipt  |
|                                   |     from the ATO, and             |
|                                   | -   report the amended total PAYG |
|                                   |     withheld and total taxable    |
|                                   |     information from the revised  |
|                                   |     payment summary               |
|                                   |                                   |
|                                   | For more information, see         |
|                                   | Parental Leave Pay (PPL) debts    |
|                                   | between employers and employees.  |
|                                   |                                   |
|                                   | Debt finalised after the end of   |
|                                   | the financial year in which the   |
|                                   | tax return has been lodged        |
|                                   |                                   |
|                                   | The customer will need to lodge   |
|                                   | an amended assessment application |
|                                   | to the ATO advising of the        |
|                                   | revised payment summary.          |
|                                   |                                   |
|                                   | Note: when the agency does not    |
|                                   | provide the payment the employer  |
|                                   | is responsible for amending the   |
|                                   | payment summary. The Initial      |
|                                   | Notice of Debt states this        |
|                                   | information on it.                |
|                                   |                                   |
|                                   | For more information, see Step 4  |
|                                   | in Financial year and Australian  |
|                                   | Taxation Office (ATO) details     |
|                                   | when a PPL debt was actioned      |
|                                   | table.                            |
+-----------------------------------+-----------------------------------+
| 6                                 | Writing off debts + Read more ... |
|                                   |                                   |
|                                   | PPL debts may be written off as   |
|                                   | per current provisions.           |
+-----------------------------------+-----------------------------------+
| 7                                 | Waiver of debts + Read more ...   |
|                                   |                                   |
|                                   | PPL debts may be waived as per    |
|                                   | current provisions.               |
|                                   |                                   |
|                                   | Use the waiver reason code of     |
|                                   | (LTW). This is PPL specific.      |
|                                   |                                   |
|                                   | Apply the waiver where:           |
|                                   |                                   |
|                                   | -   the outstanding balance of a  |
|                                   |     PPL debt is equal to or more  |
|                                   |     than $50, but less than $200, |
|                                   |     and                           |
|                                   | -   it is not cost effective to   |
|                                   |     recover the remaining amount  |
|                                   |                                   |
|                                   | This waiver reason is only for    |
|                                   | debt management staff. Do not use |
|                                   | during debt raising as a waiver   |
|                                   | reason.                           |

PPL debt offsetting, exemptions, and refunds

|                                   | Families Level 2 Helpdesk.        |
+-----------------------------------+-----------------------------------+

|                                   |                                   |
|                                   | Regular living and medical        |
|                                   | expenses are not grounds for a    |
|                                   | refund/exemption, requests must   |

|                                   | exceptional or unforeseen.        |
|                                   |                                   |
|                                   | Is there sufficient evidence?     |
|                                   |                                   |
|                                   | Yes, refer to National Families   |
|                                   | and Child Care Level 2 Policy     |
|                                   | Helpdesk for further assessment.  |
|                                   | Go to Step 5                      |
|                                   |                                   |
|                                   | No:                               |
|                                   |                                   |
|                                   | -   Advise referrer that          |
|                                   |     insufficient information is   |

|                                   | -   If customer cannot provide    |
|                                   |     any further information they  |
|                                   |     do not meet the requirements  |

|                                   | -   Details of the unforeseen or  |
|                                   |     exceptional circumstances     |
