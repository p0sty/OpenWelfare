Reconnection strategies for customers

Reconnection strategies for customers who are vulnerable, at risk or in
immediate danger.

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Reconnection strategies for       |
|                                   | customers who are vulnerable and  |
|                                   | at risk + Read more ...           |
|                                   |                                   |
|                                   | -   Further letter sent to the    |
|                                   |     customer to request they      |
|                                   |     contact Services Australia    |
|                                   |     immediately                   |
|                                   | -   SMS to customer requesting    |
|                                   |     they contact Services         |
|                                   |     Australia immediately         |
|                                   | -   Further attempts to contact   |
|                                   |     the person with permission to |
|                                   |     enquire (PPE) to make         |
|                                   |     arrangements for the customer |
|                                   |     to contact Services Australia |
+-----------------------------------+-----------------------------------+
| 2                                 | Reconnection strategies for       |
|                                   | customers who may be in immediate |
|                                   | danger + Read more ...            |
|                                   |                                   |
|                                   | -   Engage a social worker to     |
|                                   |     arrange contact with the      |
|                                   |     customer. Options for contact |
|                                   |     include contact with agencies |
|                                   |     such as the Department of     |
|                                   |     Housing, local police, a      |
|                                   |     financial organisation or     |
|                                   |     other third parties (as       |
|                                   |     permitted legislation) etc.   |
|                                   | -   Contact by appropriate        |
|                                   |     Services Australia specialist |
|                                   |     (for example, Community       |
|                                   |     Engagement Officers (CEO),    |
|                                   |     Indigenous Services Officer   |
|                                   |     (ISO), Customer Critical      |
|                                   |     Response Team (CCRT))         |
+-----------------------------------+-----------------------------------+
