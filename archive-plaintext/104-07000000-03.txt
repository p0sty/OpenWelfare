| Unfavourable decision             | Communicating unfavourable        |
|                                   | decisions + Read more ...         |
|                                   |                                   |
|                                   | It is important to make genuine   |
|                                   | attempts to contact a customer to |
|                                   | give them the chance to           |
|                                   | understand the decision and       |
|                                   | provide new information or        |
|                                   | evidence where appropriate.       |
|                                   |                                   |
|                                   | Clearly explaining a decision can |
|                                   | prevent rework through            |
|                                   | complaints, reviews and appeals   |
|                                   | or objections, and repeat contact |
|                                   | from the customer. It also gives  |
|                                   | the agency the chance to correct  |
|                                   | simple errors. Explaining         |
|                                   | decisions can help prevent or     |
|                                   | de-escalate incidents by          |
|                                   | respectfully considering the      |
|                                   | customer’s views and, if          |
|                                   | appropriate, quick referrals or   |
|                                   | follow up.                        |

|                                   | customer of a decision about      |
|                                   | their payment or claim:           |

|                                   |     the relevant policy or        |
|                                   |     legislation; this enables you |
|                                   |     to explain the reason(s) for  |
|                                   |     the decision in a confident   |
|                                   |     manner                        |
|                                   | -   Make sure any commitments are |
|                                   |     followed up                   |
|                                   | -   Anticipate the customer might |
|                                   |     interrupt. Stop talking to    |

| De-escalation techniques          | De-escalating the interaction +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Where a customer interaction      |
|                                   | escalates, respond in a way       |
|                                   | intended to bring the discussion  |
|                                   | back to a respectful interaction. |
|                                   | Always address the customer’s     |
|                                   | emotion before starting to        |
|                                   | explain, find a solution or fix   |
|                                   | the issue:                        |
|                                   |                                   |
|                                   | Let the customer explain their    |
|                                   | situation. Do not interrupt,      |
|                                   | correct or disagree with them     |
|                                   |                                   |
|                                   | Build rapport using good          |
|                                   | listening skills, empathising and |
|                                   | acknowledging their situation.    |
|                                   | Show that you hear what they are  |
|                                   | saying, ‘That would be so hard    |
|                                   | for you’ or ‘I couldn’t imagine   |
|                                   | what you are coping with right    |
|                                   | now.’                             |
|                                   |                                   |
|                                   | Maintain self-control and model   |
|                                   | behaviour you want to see/hear    |
|                                   |                                   |
|                                   | Make it hard for the customer to  |
|                                   | be angry with you by being        |
|                                   | patient and courteous             |
|                                   |                                   |
|                                   | When a customer starts agreeing   |
|                                   | with your empathy comments, for   |
|                                   | example by responding with        |
|                                   | 'exactly', it is hard for them to |
|                                   | turn around and say 'No, you      |
|                                   | don’t understand' or 'How would   |
|                                   | you know what it’s like?'         |
|                                   |                                   |
|                                   | Only move forward once the        |
|                                   | customer knows that their         |
|                                   | emotions have been acknowledged.  |
|                                   | If the customer cannot respond to |
|                                   | the following logically, their    |
|                                   | emotions are still too high:      |
|                                   |                                   |
|                                   | -   'I want to help, perhaps we   |
|                                   |     can explore some options to   |
|                                   |     support you'                  |
|                                   | -   'What can I do to make this   |
|                                   |     better?'                      |
|                                   |                                   |
|                                   | After trying the above techniques |
|                                   | and if the customer’s behaviour   |
|                                   | is frightening or intimidating,   |
|                                   | deliver an assertive request for  |
|                                   | them to stop. This is about       |
|                                   | stopping the behaviour, not       |
|                                   | changing the customer’s emotional |
|                                   | state.                            |
|                                   |                                   |
|                                   | Deliver the first request at a    |
|                                   | slightly lower volume than you    |
|                                   | would normally speak and speak    |
|                                   | slower to create a calming        |
|                                   | effect. Address the customer’s    |
|                                   | emotion first, before starting to |
|                                   | explain, find a solution, or fix  |
|                                   | the issue.                        |
|                                   |                                   |
|                                   | Helpful responses include:        |
|                                   |                                   |
|                                   | -   Try to reduce swearing,       |
|                                   |     yelling or abuse, by asking,  |
|                                   |     'Can we bring this down a     |
|                                   |     bit?' or 'Can we slow things  |
|                                   |     down a bit?'                  |
|                                   | -   'When you continue to         |
|                                   |     swear/yell/speak over         |
|                                   |     me/blame/threaten/use your    |
|                                   |     tone to intimidate, I can’t   |
|                                   |     help you.'                    |
|                                   | -   'I need you to stop           |
|                                   |     swearing/lower your           |
|                                   |     voice/slow down/use a more    |
|                                   |     respectful tone so I can help |
|                                   |     you.'                         |
|                                   | -   If the customer begins        |
|                                   |     escalating, do not raise your |
|                                   |     voice; get their attention by |
|                                   |     saying their name             |
|                                   |                                   |
|                                   | Avoid responses that could        |
|                                   | escalate the interaction, for     |
|                                   | example:                          |
|                                   |                                   |
|                                   | -   ‘I understand your situation’ |
|                                   | -   ‘I can imagine’               |
|                                   | -   ‘Sorry I’m not allowed to’    |
|                                   | -   ‘Unfortunately’               |
|                                   | -   ‘I’m just trying my best’     |
|                                   | -   ‘Please can you just calm     |
|                                   |     down’                         |
|                                   | -   ‘Do you realise that the      |
|                                   |     other customers don’t act     |
|                                   |     like you?’                    |
|                                   | -   ‘I’m just trying to do my     |
|                                   |     job'                          |
|                                   |                                   |
|                                   | De-escalation check + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | The customer has de-escalated     |
|                                   | when:                             |
|                                   |                                   |
|                                   | -   They are paraphrasing and     |
|                                   |     mirroring your calm behaviour |
|                                   | -   Their tone, volume and pace   |
|                                   |     of voice has reduced          |
|                                   | -   They agree with you/start     |
|                                   |     nodding                       |
|                                   | -   There is respectful tone/eye  |
|                                   |     contact                       |
|                                   | -   They apologise for their      |
|                                   |     behaviour                     |
|                                   | -   They stay on track and are    |
|                                   |     solution focussed             |
|                                   |                                   |
|                                   | If the customer’s emotions become |
|                                   | high again, repeat the            |
|                                   | de-escalation techniques before   |
|                                   | continuing to resolve the reason  |
|                                   | for the contact.                  |

| End phone call                    | Ending/discontinuing customer     |
|                                   | phone calls + Read more ...       |
|                                   |                                   |
|                                   | Sometimes de-escalation may not   |
|                                   | work. In such cases, end the      |
|                                   | call.                             |
|                                   |                                   |
|                                   | Do not remove headset before      |
|                                   | ending a phone call.              |
|                                   |                                   |
|                                   | Call recording continues until a  |
|                                   | call is ended. Anything recorded  |
|                                   | is deemed to be information the   |
|                                   | customer has supplied to the      |
|                                   | agency. Removing the headset      |
|                                   | before the call ends means        |
|                                   | valuable information may be       |
|                                   | missed.                           |
|                                   |                                   |
|                                   | Do not end the call if the        |
|                                   | customer is:                      |
|                                   |                                   |
|                                   | -   talking about suicide         |
|                                   | -   talking about harm (self or   |
|                                   |     others)                       |
|                                   | -   making a threat (staff or     |
|                                   |     agency)                       |
|                                   |                                   |
|                                   | Warn the customer before ending a |
|                                   | call                              |
|                                   |                                   |
|                                   | Explain to the customer the call  |
|                                   | will be discontinued if their     |
|                                   | behaviour continues. Inform the   |
|                                   | customer that the agency expects  |
|                                   | its staff to be treated with      |
|                                   | respect and courtesy and it is    |
|                                   | difficult to help when a customer |
|                                   | is not acting accordingly.        |
|                                   |                                   |
|                                   | Example                           |
|                                   |                                   |
|                                   | ‘I may not be able to continue    |
|                                   | this call with you because you    |
|                                   | are (quote behaviour). This is    |
|                                   | making it hard for me to help     |
|                                   | you. If you continue to           |
|                                   | (behaviour) I will end the call   |
|                                   | without any further warning and   |
|                                   | discuss this with my manager/team |
|                                   | leader.’                          |
|                                   |                                   |
|                                   | If the behaviour continues,       |
|                                   | inform the customer the call is   |
|                                   | ending. End the call politely.    |

|                                   | escalates further, or if a staff  |
