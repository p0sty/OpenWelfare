+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Check if death processing         |
|                                   | completed + Read more ...         |
|                                   |                                   |
|                                   | Has death already been processed? |
|                                   |                                   |
|                                   | Yes:                              |
|                                   |                                   |
|                                   | -   Undo death action             |
|                                   | -   Continue processing the       |
|                                   |     claim, go to Step 2           |
|                                   |                                   |
|                                   | No, go to Step 2                  |
+-----------------------------------+-----------------------------------+
| 2                                 | Complete activity on the same     |
|                                   | day + Read more ...               |
|                                   |                                   |
|                                   | The new claim activity, including |
|                                   | the adjustment of arrears and the |
|                                   | subsequent death processing must  |
|                                   | be done on the same day.          |
|                                   |                                   |
|                                   | All coding to correctly assess    |
|                                   | the claim (income, assets and     |
|                                   | residence) must be done in the    |
|                                   | new claim activity and before the |
|                                   | claim is finalised, the arrears   |
|                                   | must be zeroed from the date of   |
|                                   | event (DOV) to prevent any debt   |
|                                   | to the deceased customer's        |
|                                   | estate.                           |
|                                   |                                   |
|                                   | The DOV used to zero the arrears  |
|                                   | depends on if the claimant was    |
|                                   | single/partnered.                 |
|                                   |                                   |
|                                   | Was the customer single or in a   |
|                                   | relationship with a non-pensioner |
|                                   | spouse?                           |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, go to Step 4              |
+-----------------------------------+-----------------------------------+
| 3                                 | Death of a single pensioner       |
|                                   | (single customer or pensioner     |
|                                   | with non-pensioner spouse) + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | A single customer is entitled to  |
|                                   | be paid until the first           |
|                                   | Entitlement Period End Day (EPED) |
|                                   | after their date of death. For    |
|                                   | pensioners in I environment, this |
|                                   | is always "4" on the calendar.    |
+-----------------------------------+-----------------------------------+
| 4                                 | Death of one member of a          |
|                                   | pensioner couple + Read more ...  |
|                                   |                                   |
|                                   | Lump Sum Bereavement Payment      |
|                                   | (LBP), which is the difference    |
|                                   | between the combined couple rate  |
|                                   | and the new single rate, is paid  |
|                                   | up to and including the next 7    |
|                                   | EPEDs. If the combined couple     |
|                                   | rate has continued for the full   |
|                                   | bereavement period then no LBP is |
|                                   | payable and the arrears from the  |
|                                   | new claim are zeroed from the     |
|                                   | Bereavement Period end date       |
|                                   | (BPED) +1.                        |
|                                   |                                   |
|                                   | For more information, see:        |
|                                   |                                   |
|                                   | -   Calculation of a Lump Sum     |
|                                   |     Bereavement Payment (LBP)     |
|                                   | -   Death of one member of a      |
|                                   |     pensioner couple              |
|                                   | -   Death of both members of a    |
|                                   |     couple                        |
|                                   | -   Death of a single pensioner   |
