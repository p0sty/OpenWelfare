Urgent payment of a Stoma or ParaQuad claim (Force Payment indicator)

Table 4: This table describes how Service Officers prioritise a Stoma or
ParaQuad claim for urgent payment. Consult with the Operational Policy &
Legacy Programs Section within the Pharmaceutical Benefits Branch prior
to prioritising payment.

Note: the Force Payment Indicator can be used on claims submitted via
Health Professional Online Services (HPOS) or USB as they are
automatically at 'Hold Payment' status. Force payment can also applied
to manual claims before they reach a status of status of 'Generate
Statement'.

| Step                              | Action                            |
+-----------------------------------+-----------------------------------+

|                                   |     Details                       |
|                                   | -   Key the association’s         |
|                                   |     approval ID into the ASN      |
|                                   |     field and claim period number |
|                                   |     into the CPN field            |
|                                   | -   Select the channel the claim  |
|                                   |     was received – either CTS or  |
|                                   |     Manual (noting the CTS        |
|                                   |     channel includes claims       |
|                                   |     received via USB and HPOS)    |
