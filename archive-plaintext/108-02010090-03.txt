|                                   | Notice of Assessment (NOA)        |

|                                   | A superannuation withdrawal under |
|                                   | the First Home Super Saver (FHSS) |

|                                   | the Parental Income Test.         |

|                                   | withdrawals is different between  |
|                                   | Services Australia and the ATO:   |
|                                   |                                   |

|                                   |     reflected on an individual's  |

|                                   | -   A customer supplying a NOA    |
|                                   |     who has had a superannuation  |
|                                   |     withdrawal under the FHSS     |
|                                   |     will need to advise the       |
|                                   |     amount considered taxable.    |
|                                   |     This can be manually excluded |
|                                   |     for the agency's purposes     |
|                                   |                                   |
