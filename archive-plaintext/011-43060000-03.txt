|                                   |     check for Medicare health     |
|                                   |     professionals                 |
|                                   | -   Perform telephone security    |

|                                   |                                   |

|                                   | detailsdisplay above the service  |

+-----------------------------------+-----------------------------------+
| 3                                 | Status of a Claim or              |
|                                   | Application + Read more ...       |
|                                   |                                   |
|                                   | Service Officers can help callers |
|                                   | enquiring about the status of a   |
|                                   | claim or application. It is       |
|                                   | important that a level of         |
|                                   | assurance is provided to the      |
|                                   | caller about their claim or       |
|                                   | application.                      |
|                                   |                                   |
|                                   | This does not include details     |
|                                   | about internal processes.         |
|                                   |                                   |
|                                   | For example, the caller does not  |
|                                   | need to know about digitised      |
|                                   | electronic workflow systems, fax  |
|                                   | to email services, scanning       |
|                                   | solutions or KPIs.                |
|                                   |                                   |
|                                   | Advise the caller that the agency |
|                                   | endeavours to process claims and  |
|                                   | applications as quickly as        |
|                                   | possible but it can be impacted   |
|                                   | by peaks in demand                |
|                                   |                                   |

|                                   | the agency by searching the       |
|                                   | workflow                          |
|                                   |                                   |

|                                   | processing dates.                 |
+-----------------------------------+-----------------------------------+
| 4                                 | ECLIPSE claim enquiries + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | ECLIPSE claims can be uniquely    |
|                                   | identified on the patient history |
|                                   | by the claim ID allocated to the  |
|                                   | services. ECLIPSE claims have a % |
|                                   | symbol at the end of the claim ID |
|                                   | and the claim IDs are made up of  |
|                                   | the following characters: NNNNN%. |
|                                   |                                   |
|                                   | To access the claim ID, Service   |
|                                   | Officers must press [F2].         |
|                                   |                                   |
|                                   | If an ECLIPSE claim is rejected   |
|                                   | in full, the claim details do not |
|                                   | write to the patient history.     |
|                                   | Claims that are only partially    |
|                                   | rejected or fully paid will write |
|                                   | to the patient history.           |
|                                   |                                   |
|                                   | Service Officers can resolve      |
|                                   | enquiries relating to an ECLIPSE  |
|                                   | claim (claims with a payee code   |
|                                   | 8) rejection/assessment where the |
|                                   | Medicare reason code is a 3 digit |
|                                   | number, for example, 179. Where   |
|                                   | the rejected services are not     |
|                                   | present on the patient history,   |
|                                   | Service Officers must ask the     |
|                                   | caller what reason code was used  |
|                                   | to reject the claim.              |
|                                   |                                   |
|                                   | Service Officers may be able to   |
|                                   | determine why the claim was       |
|                                   | rejected by referring to the list |
|                                   | of reason codes.                  |
|                                   |                                   |
|                                   | Service Officers must warm        |
|                                   | transfer enquiries for ECLIPSE    |
|                                   | claims rejected with a 4 digit    |
|                                   | return code to the eBusiness      |
|                                   | Service Centre.                   |
|                                   |                                   |
|                                   | When an ECLIPSE claim is rejected |
|                                   | in full, the rejected services do |
|                                   | not appear on the patient's       |
|                                   | history. To determine the reason  |
|                                   | why the service was rejected,     |
|                                   | Service Officers must ask the     |
|                                   | caller what the rejected reason   |
|                                   | code was used to reject the       |
|                                   | claim. Service Officers may be    |
|                                   | able to determine why the claim   |
|                                   | was rejected from the list of     |
|                                   | Medicare reason codes.            |
|                                   |                                   |
|                                   | See: Indicators, codes, modifiers |
|                                   | and control lines for claims      |
|                                   | processing in Medicare            |
|                                   |                                   |
|                                   | If unable to find the reason for  |
|                                   | rejection, Service Officers must  |
|                                   | escalate to a Programme Support   |
|                                   | Officer (PSO).                    |
+-----------------------------------+-----------------------------------+
| 5                                 | Rejection enquiry + Read more ... |

|                                   |                                   |
|                                   | Service Officers may be able to   |
|                                   | determine why the claim was       |
|                                   | rejected by referring to the list |
|                                   | of reason codes.                  |
|                                   |                                   |
|                                   | See: Indicators, codes, modifiers |

|                                   |     rules, restrictions, time     |
|                                   |     dependencies                  |
|                                   |                                   |
|                                   | See: General assessing            |

|                                   |     documentation or provide more |
|                                   |     direction on how to resubmit  |

|                                   |                                   |
|                                   | If unable to find the reason for  |
|                                   | rejection, Service Officers must  |
|                                   | escalate to a Programme Support   |
|                                   | Officer (PSO).                    |

Simplified Billing Manual claims

manual claims.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Manual claims + Read more ...     |
|                                   |                                   |
|                                   | Private health insurers and       |
|                                   | billing agents submit manual      |
|                                   | Simplified Billing claims on the: |
|                                   |                                   |
|                                   | -   Simplified Billing manual     |
|                                   |     claims batch header form      |
|                                   |     (HW091), and                  |
|                                   | -   include an invoice/account    |
|                                   |                                   |
|                                   | Staff can identify manual         |
|                                   | Simplified Billing claims by the  |
|                                   | claim header. All claims have     |
|                                   | these attached                    |
|                                   |                                   |
|                                   | Service Officers must make sure   |
|                                   | all manual Simplified Billing     |
|                                   | claims submitted for payment      |
|                                   | comply with the policy            |
|                                   | requirements, before processing   |
|                                   | the claim                         |
|                                   |                                   |
|                                   | All fields on the claim header    |
|                                   | are mandatory. Return claims to   |
|                                   | the private health insurer or     |
|                                   | billing agent, and send More      |
|                                   | information is needed for your    |
|                                   | Simplified Billing claim (z2162), |
|                                   | if the claim header:              |
|                                   |                                   |
|                                   | is incomplete                     |
|                                   |                                   |
|                                   | has incorrect details, or         |
|                                   |                                   |
|                                   | has more than 50 items/services   |
|                                   | per batch header                  |
|                                   |                                   |
|                                   | Do not split the claim under a    |
|                                   | second date of lodgement, or key  |
|                                   | a claim if the claim header is    |
|                                   | not attached                      |
|                                   |                                   |
|                                   | The agency will not accept any    |
|                                   | other forms. For example a        |
|                                   | Simplified Billing or ECLIPSE     |
|                                   | adjustment claim form (HW023).    |
|                                   | Exception: if a late lodgement    |
|                                   | application is lodged for         |
|                                   | services over 2 years old,        |
|                                   | private health insurers and       |
|                                   | billing agents can lodge a        |
|                                   | Simplified Billing manual claims  |
|                                   | batch header form (HW091)         |
|                                   |                                   |
|                                   | For claims with a date of         |
|                                   | lodgement from 1 March 2019,      |
|                                   | accept the following forms of     |
|                                   | signature:                        |
|                                   |                                   |
|                                   | -   handwritten signature         |
|                                   | -   image of an electronic        |
|                                   |     signature                     |
|                                   | -   digital image or stamp of the |
|                                   |     private health insurer or     |
|                                   |     billing agent’s logo          |
|                                   |                                   |
|                                   | To identify the person sending    |
|                                   | the batch header, the sender’s    |
|                                   | name must be complete             |
|                                   |                                   |
|                                   | If a health professional lodges a |
|                                   | simplified billing claim to the   |
|                                   | agency:                           |
|                                   |                                   |
|                                   | -   return to the health          |
|                                   |     professional, and             |
|                                   | -   send Your Simplified Billing  |
|                                   |     claim cannot be processed     |
|                                   |     (Z2160)                       |
|                                   |                                   |
|                                   | Addresses to use when returning a |
|                                   | claim                             |
|                                   |                                   |
|                                   | Use the return address details    |
|                                   | below for:                        |
|                                   |                                   |
|                                   | -   private health insurers or    |
|                                   |     billing agents, Resources     |
|                                   |     Billing Agent/Private Health  |
|                                   |     Insurer contact details       |
|                                   | -   a health professional, see    |
|                                   |     the health professional       |
|                                   |     address listed in the         |
|                                   |     Provider Directory System     |
|                                   |     (PDS)                         |
|                                   |                                   |
|                                   | The Resources page has a link to  |
|                                   | the Simplified Billing letter     |
|                                   | templates.                        |

|                                   | -   Claim Number is the number    |

|                                   | Services Australia - DD MM YY     |
|                                   |                                   |
|                                   | -   Customer Class:               |
|                                   | -   MB for billing agent claims,  |

| 4                                 | Key Simplified Billing claim      |
|                                   | details + Read more ...           |

|                                   |     keying to ensure that         |

|                                   |     background operator           |
|                                   | -   pat - the patient’s first     |
|                                   |     name or IRN                   |
|                                   | -   item - the item number        |
|                                   |     allocated on the invoice      |
|                                   | -   DOS - the date of service     |
|                                   |     listed on the account. This   |
|                                   |     field is also for the         |
|                                   |     referral date                 |
|                                   | -   no - this field is used to    |
|                                   |     denote the number of fields   |
|                                   |     for radiotherapy and/or the   |
|                                   |     number of patients the health |
|                                   |     professional has seen when    |
|                                   |     claiming certain consult      |
|                                   |     items, for example, item 33,  |
|                                   |     40                            |
|                                   | -   Prov Num - the Servicing      |
|                                   |     Providers Provider number     |
|                                   | -   charge - the amount the       |
|                                   |     health professional has       |
|                                   |     charged for the service       |
|                                   | -   SCP - the Specimen Collection |
|                                   |     Point that has to accompany   |
|                                   |     pathology items (for example, |
|                                   |     SAO1 S055)                    |
|                                   | -   LSPN - the Location Specific  |
|                                   |     Practice Number that has to   |
|                                   |     accompany diagnostic items    |
|                                   |     (for example LSPN 123456)     |
|                                   | -   R3 - an indication of Rule 3  |
|                                   |     Exemption can be recorded by  |
|                                   |     keying Y in the R3 field      |
|                                   | -   SD - self deemed indicator    |
|                                   | -   RSN - service operators can   |
|                                   |     force a claim into a PEND     |
|                                   |     status by keying PND in this  |
|                                   |     field                         |
|                                   | -   press [Enter]                 |
|                                   |                                   |
|                                   | To access LSPN, SCP, SD, RSN or   |
|                                   | R3 fields, press [F6] from the    |
|                                   | data entry screen.                |
|                                   |                                   |

|                                   | service, the following fields may |

|                                   |                                   |

|                                   | -   Pat                           |
|                                   | -   Item                          |
|                                   | -   DOS                           |
|                                   | -   Prov Num                      |
|                                   | -   SCP                           |
|                                   | -   LSPN                          |
|                                   | -   SD                            |
|                                   | -   R3                            |
