| 1                                 | Service Officer has been          |
|                                   | allocated a SEE related work      |
|                                   | item + Read more ...              |
|                                   |                                   |
|                                   | The following work item is        |
|                                   | created when a job seeker is      |
|                                   | identified as potentially         |
|                                   | benefiting from a referral to     |
|                                   | Skills for Education and          |
|                                   | Employment (SEE) when the Job     |
|                                   | Seeker Snapshot is run, and the   |
|                                   | Service Officer confirms that the |
|                                   | referral is required:             |
|                                   |                                   |
|                                   | -   ZALL_REF_GEN (JSR/MFU with    |
|                                   |     ‘SEE’ keyword)                |
|                                   |                                   |
|                                   | Note: make sure the job seeker    |
|                                   | meets the eligibility criteria    |
|                                   | before referring them.            |
|                                   |                                   |
|                                   | ZALL work items are allocated and |
|                                   | actioned to Service Officers      |
|                                   | within Process Direct. Service    |
|                                   | Officers should consider the      |
|                                   | information provided within the   |
|                                   | relevant Work Item under the      |
|                                   | Snapshot button in Process Direct |
|                                   | or on the Activity List (AL)      |
|                                   | screen in Customer First.         |
|                                   |                                   |
|                                   | When the activity is actioned in  |
|                                   | Process Direct, Service Officers  |
|                                   | must return to Customer First and |
|                                   | check the AL screen to make sure  |
|                                   | the Manual Follow Up (MFU)        |
|                                   | activity is completed.            |
|                                   |                                   |
|                                   | See Customer First navigation,    |
|                                   | common screens and functions.     |
|                                   |                                   |
|                                   | Where a job seeker requires an    |
|                                   | Interpreter to undertake the      |
|                                   | phone contact, see Contacting an  |
|                                   | on-demand telephone interpreter   |
|                                   | (Interpreter Connect).            |
|                                   |                                   |
|                                   | If a JSR 1st attempt to contact   |
|                                   | Fast Note has already been        |
|                                   | recorded on the Document List, go |
|                                   | to Step 2.                        |
|                                   |                                   |
|                                   | Service Officers allocated one of |
|                                   | the above work items must contact |
|                                   | the job seeker to discuss the     |
|                                   | referral. 2 genuine attempts to   |
|                                   | contact must be made. If the      |
|                                   | contact number recorded on the    |
|                                   | job seeker's record is a mobile   |
|                                   | number, and they have registered  |
|                                   | for SMS messaging, create a       |
|                                   | pre-call notification (SMS) only  |
|                                   | using Desktop Electronic          |
|                                   | Messaging Capability (DEMC).      |
|                                   | After sending the SMS, wait 1 or  |
|                                   | 2 minutes before making the       |
|                                   | outbound call attempt.            |
|                                   |                                   |
|                                   | If the job seeker’s telephone     |
|                                   | details are not recorded,         |
|                                   | incorrect or outdated:            |
|                                   |                                   |
|                                   | -   manually cancel the MFU       |
|                                   |     activity from the AL screen   |
|                                   |     in Process Direct, or         |
|                                   | -   complete the MFU in Customer  |
|                                   |     First                         |
|                                   |                                   |
|                                   | Was contact with the job seeker   |
|                                   | successful?                       |
|                                   |                                   |
|                                   | Yes, go to Step 3                 |
|                                   |                                   |
|                                   | No, hold the work item to the     |
|                                   | next calendar day, and:           |
|                                   |                                   |
|                                   | -   record details about the      |
|                                   |     outbound contact attempt      |
|                                   |     using the JSR 1st attempt to  |
|                                   |     contact Fast Note in Customer |
|                                   |     First so the next Service     |
|                                   |     Officer can action the second |
|                                   |     contact attempt and finalise  |
|                                   |     the activity                  |
|                                   | -   select the ‘Display on        |
|                                   |     access’ box                   |
|                                   | -   check ‘OTH-Urgent free text   |
|                                   |     message exists’ displays in   |
|                                   |     the Reason field              |
|                                   | -   the Expiry Date field should  |
|                                   |     default to the maximum date   |
|                                   |     allowed (60 days from today)  |
|                                   | -   procedure ends here           |
+-----------------------------------+-----------------------------------+
| 2                                 | Second outbound attempt to        |
|                                   | contact job seeker + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | Make a second genuine attempt to  |
|                                   | contact the job seeker if the     |
|                                   | first attempt was unsuccessful.   |
|                                   |                                   |
|                                   | If the contact number recorded on |
|                                   | the job seeker’s record is a      |
|                                   | mobile number, and they have      |
|                                   | registered for SMS messaging      |
|                                   | create a pre-call notification    |
|                                   | (SMS) only using Desktop          |
|                                   | Electronic Messaging Capability   |
|                                   | (DEMC). Following creation of the |
|                                   | SMS, wait 1 or 2 minutes before   |
|                                   | making the outbound attempt.      |
|                                   |                                   |
|                                   | Was the contact successful?       |
|                                   |                                   |
|                                   | Yes, go to Step 3                 |
|                                   |                                   |
|                                   | No:                               |
|                                   |                                   |
|                                   | -   Close the JSR 1st attempt to  |
|                                   |     contact Display on Access     |
|                                   |     (DOA) DOC                     |
|                                   | -   Annotate the JSR 1st Attempt  |
|                                   |     to contact DOC, stating that  |
|                                   |     the 2nd attempt to contact    |
|                                   |     was unsuccessful              |
|                                   | -   As SEE is a voluntary program |
|                                   |     prior to the job seeker       |
|                                   |     accepting a referral,         |
|                                   |     manually cancel the MFU       |
|                                   |     activity from the AL screen   |
|                                   |     in Process Direct, or         |
|                                   |     complete the MFU in Customer  |
|                                   |     First                         |
|                                   | -   Procedure ends here           |

|                                   |     Supplement (PES)              |
|                                   |                                   |
|                                   | See Skills for Education and      |
|                                   | Employment (SEE).                 |
|                                   |                                   |

|                                   | the following screens in Process  |

|                                   | referral process. This assists    |
|                                   | with determining eligibility:     |

|                                   | Citizenship and Visa sub class:   |

|                                   | -   the Legal Residency Details   |

|                                   |                                   |
|                                   | Permanent visa details:           |

|                                   |                                   |
|                                   | End date of the Newly Arrived     |

|                                   | -   the Period of Australian      |
|                                   |     Presence screen will display  |
|                                   |                                   |
|                                   | If the job seeker has mutual      |

|                                   |     Direct, or                    |
|                                   | -   Return to Customer First and  |
|                                   |     check the AL screen to make   |
|                                   |     sure the activity is          |
|                                   |     completed                     |
|                                   | -   See Customer First            |
|                                   |     navigation, common screens    |
|                                   |     and functions                 |

| 8                                 | Arrange a Pre-Training Assessment |
|                                   | appointment with the SEE          |
|                                   | Provider + Read more ...          |
|                                   |                                   |
|                                   | For all referrals to a local      |
|                                   | provider, arrange an appointment  |
|                                   | with the SEE provider for the     |
|                                   | Pre-Training Assessment. This is  |
|                                   | mandatory for all non-distance    |
|                                   | education SEE referrals.          |
|                                   |                                   |
|                                   | Participation in the SEE          |
|                                   | Pre-training assessment and       |
|                                   | training is compulsory if         |
|                                   | assessed as suitable for the      |
|                                   | program. Make sure the job seeker |
|                                   | understands the compliance        |
|                                   | consequences if they do not       |
|                                   | attend their Pre-Training         |
|                                   | Assessment appointment.           |
|                                   |                                   |
|                                   | Service Officers should not       |
|                                   | discuss the job seeker's          |
|                                   | circumstances with the provider.  |
|                                   | Only disclose enough information  |
|                                   | to book the appointment.          |
|                                   |                                   |
|                                   | Check if the job seeker has a     |
|                                   | silent phone number. If they do,  |
|                                   | seek the job seeker’s consent to  |
|                                   | give the number to the provider   |
|                                   | over the phone, as the referral   |
|                                   | system will exclude it from the   |
|                                   | online referral                   |
|                                   |                                   |
|                                   | Phone the provider and make a     |
|                                   | Pre-Training Assessment           |
|                                   | appointment using the contact     |
|                                   | details located in the Referral   |
|                                   | Summary after the referral has    |
|                                   | been completed                    |
|                                   |                                   |
|                                   | Tell the job seeker of the:       |
|                                   |                                   |
|                                   | -   appointment date              |
|                                   | -   appointment time              |
|                                   | -   appointment location, and     |
|                                   | -   consequences of not attending |
|                                   |     or to contact the SEE         |
|                                   |     provider to reschedule their  |
|                                   |     appointment if they are       |
|                                   |     unable to attend              |
