This page contains lists of Clinical psychology items and descriptions,
Psychology, Neuropsychology, Social Worker (mental health), Occupational
therapy (mental health) items and descriptions and Clinical psychology
business rules, Psychology business rules, Neuropsychology business
rules, Social Worker (mental health) business rules, and Occupational
therapy (mental health) business rules, fact sheets and links to
external websites. Links on this page may be unavailable to external
audiences.

Business Rules for Clinical psychology

Table 6:

+-----------------------------------+-----------------------------------+
| Item                              | Rule                              |
+-----------------------------------+-----------------------------------+
| 1                                 | The type and number of treatments |
|                                   | provided under a referral are     |
|                                   | based on clinical need. There is  |
|                                   | no limit on number of             |
|                                   | consultations claimed in a 12     |
|                                   | month referral period.            |
+-----------------------------------+-----------------------------------+
| 2                                 | Treatment in residential aged     |
|                                   | care facilities (RACF) high care  |
|                                   | and public hospitals requires     |
|                                   | prior approval.                   |
+-----------------------------------+-----------------------------------+
| 3                                 | After eight sessions of trauma    |
|                                   | focussed therapy a Case Review    |
|                                   | US07 and prior approval are       |
|                                   | required before any further       |
|                                   | treatment.                        |
+-----------------------------------+-----------------------------------+
| 4                                 | Providers are to comply with      |
|                                   | billing arrangements as outlined  |
|                                   | in the Notes for Allied Mental    |
|                                   | Health Care Providers available   |
|                                   | on DVA’s Allied health            |
|                                   | professional’s website. See       |
|                                   | External websites for a link to   |
|                                   | the DVA Allied health             |
|                                   | professional site and scroll down |
|                                   | to locate this information.       |

+-----------------------------------+-----------------------------------+

+-----------------------------------+-----------------------------------+

|                                   |     in the relevant allied fee    |
|                                   |     schedule has been claimed     |
|                                   | -   Any number from 1 to 11       |
|                                   |     subsequent consultation items |

|                                   |     for each initial consultation |
