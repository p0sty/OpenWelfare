This page contains information on processing CSHC online claims,
assisted customer claims and paper claims.

Staff trained in Process Direct select the Process Direct tab, staff not
yet trained select Customer First.

|                                   | financial year the customer has   |
|                                   | declared their income and the     |
|                                   | date of receipt (DOR) of the      |
|                                   | claim determines the reference    |
|                                   | tax year                          |
|                                   |                                   |

|                                   |     TNA or NRL, must be within 2  |

|                                   |     the financial year ended,     |
|                                   |     e.g. for 2020/21, code 2021   |
|                                   |                                   |
|                                   | Members of a couple need to       |
|                                   | provide information relating to   |
|                                   | the same financial year           |
|                                   |                                   |
|                                   | The reference tax year for an     |
|                                   | estimate (EST) is the current     |
|                                   | financial year. Record the year   |
|                                   | that the current financial year   |
|                                   | is to end, for example, for       |
|                                   | 2021/22 code 2022. After the      |
|                                   | estimate (EST) and income details |
|                                   | have been recorded, a second      |
|                                   | entry is required for the         |
|                                   | reference year of the NOA using   |
|                                   | type TNA and the date before the  |
|                                   | claim DOV, that is, DOV - 1 day   |
|                                   |                                   |

|                                   |                                   |

|                                   |     below the tax free threshold  |
|                                   |     or as a result of an ATO tax  |

|                                   |     when the customer is required |
|                                   |     to lodge. For example, the    |
|                                   |     customer has income above the |
|                                   |     tax lodgement threshold, or   |
|                                   |     had tax withheld from any     |
|                                   |     income in that financial      |
|                                   |     year.                         |
|                                   | -   EST – Estimate customer (or   |
|                                   |     partner) declared due to an   |
|                                   |     acceptable condition, they    |
|                                   |     are providing an estimate of  |
|                                   |     income for the current        |
