Examples of using proof of birth for Centrelink and Medicare

Table 1

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Jane is not claiming payments and |
|                                   | is lodging the form for 'Medicare |
|                                   | enrolment only 'on the Newborn    |
|                                   | Child Declaration (FA081).        |
|                                   |                                   |
|                                   | Jane's partner has claimed Dad    |
|                                   | and Partner Pay (DAP).            |
|                                   |                                   |
|                                   | Use the FA081 for proof of birth  |
|                                   | to finalise the partner's         |
|                                   | claim(s).                         |
+-----------------------------------+-----------------------------------+
| 2                                 | At the time of lodgement, Carol   |
|                                   | is lodging the form for 'Medicare |
|                                   | enrolment only' on the FA081.     |
|                                   |                                   |
|                                   | Two months later, Carol lodged a  |
|                                   | claim for Paid Parental Leave.    |
|                                   |                                   |
|                                   | Use the proof of birth to         |
|                                   | finalise the claim.               |
+-----------------------------------+-----------------------------------+
| 3                                 | Simona lodged the back page of    |
|                                   | the FA081 (Doctor/Midwife - Proof |
|                                   | of Birth declaration). This is to |
|                                   | support a claim for Family Tax    |
|                                   | Benefit (FTB). A Medicare Details |
|                                   | DOC creates.                      |
|                                   |                                   |
|                                   | Medicare staff can use the back   |
|                                   | page as proof of birth for the    |
|                                   | Medicare enrolment. The customer  |
|                                   | may need to give more details on  |
|                                   | a Medicare enrolment form.        |
+-----------------------------------+-----------------------------------+
| 4                                 | Betty completes the 'Add newborn  |
|                                   | child' service and uploads the    |
|                                   | back page of the FA081. The       |
|                                   | Doctor/Midwife â€“ Proof of Birth   |
|                                   | declaration has been fully        |
|                                   | completed, but the top section of |
|                                   | the back page is incomplete.      |
|                                   |                                   |
|                                   | As the newborn child's name has   |
|                                   | been provided in the 'Add newborn |
|                                   | child' service, the back page     |
|                                   | still serves as acceptable proof  |
|                                   | of birth for Centrelink purposes. |
+-----------------------------------+-----------------------------------+
| 5                                 | Susan has participated in the     |
|                                   | Newborn Enrolment Trial and has   |
|                                   | been advised by the hospital that |
|                                   | they will electronically confirm  |
|                                   | the birth of her child with       |
|                                   | Medicare and Centrelink. Susan    |
|                                   | has been advised due to this she  |
|                                   | does not need to lodge the FA081  |
|                                   | but has called to confirm this    |
|                                   | information.                      |
|                                   |                                   |
|                                   | Advise the customer to complete   |
|                                   | the 'Add newborn child' service   |
|                                   | online or the 'Add child' service |
|                                   | via the Express Plus Centrelink   |
|                                   | mobile app and follow any prompts |
|                                   | in this service. See Newborn      |
|                                   | Enrolment Trial for more          |
|                                   | information.                      |
