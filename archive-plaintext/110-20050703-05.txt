Categorisation of group members' debts

Table 5

+-----------------------------------+-----------------------------------+
| Category type                     | Eligibility and outcome           |
+-----------------------------------+-----------------------------------+
| Category 1                        | Not eligible:                     |
|                                   |                                   |
|                                   | Group members whose debts were    |
|                                   | based on averaged ATO income      |
|                                   | information and have made no      |
|                                   | repayments will:                  |
|                                   |                                   |
|                                   | -   have their averaged debt      |
|                                   |     zeroed                        |
|                                   | -   not get a refund or a         |
|                                   |     settlement payment            |
|                                   |                                   |
|                                   | Scenario:                         |
|                                   |                                   |
|                                   | Customer notified of debt but     |
|                                   | they DID NOT make any repayments  |
|                                   | to this debt. The debt has been   |
|                                   | zeroed.                           |
|                                   |                                   |
|                                   | The customer is not eligible for  |
|                                   | disbursement as no monies were    |
|                                   | paid to or held by the agency for |
|                                   | any period of time.               |
+-----------------------------------+-----------------------------------+
| Category 2                        | Eligible:                         |
|                                   |                                   |
|                                   | Group members whose debts were    |
|                                   | wholly or partially based on      |
|                                   | averaged ATO income information   |
|                                   | and had made a repayment towards  |
|                                   | those debts will receive a        |
|                                   | settlement payment.               |
|                                   |                                   |
|                                   | This payment is in addition to    |
|                                   | the refund already received.      |
|                                   |                                   |
|                                   | Scenario:                         |
|                                   |                                   |
|                                   | Customer notified of debt AND     |
|                                   | made repayments to this debt.     |
|                                   |                                   |
|                                   | This debt has been refunded and   |
|                                   | zeroed.                           |
|                                   |                                   |
|                                   | The customer is eligible for      |
|                                   | disbursement based on:            |
|                                   |                                   |
|                                   | -   the amount paid towards the   |
|                                   |     debt                          |
|                                   | -   the timeframe these monies    |
|                                   |     were held by the agency       |
|                                   | -   when the monies were refunded |
+-----------------------------------+-----------------------------------+
| Category 3                        | Eligible:                         |
|                                   |                                   |
|                                   | Group members whose debts were    |
|                                   | initially based on averaged ATO   |
|                                   | income information, but were      |
|                                   | recalculated using evidence such  |
|                                   | as:                               |
|                                   |                                   |
|                                   | -   payslips or                   |
|                                   | -   bank statements,              |
|                                   | -   and had overpaid the          |
|                                   |     recalculated debt             |
|                                   |                                   |
|                                   | These group members will receive  |
|                                   | a settlement payment for the      |
|                                   | overpaid amount of their          |
|                                   | recalculated debt, whether the    |
|                                   | overpayment was refunded or       |
|                                   | applied to pay a valid debt.      |
|                                   |                                   |
|                                   | This group will not receive an    |
|                                   | additional refund under the       |
|                                   | government's announcement of 29   |
|                                   | May 2020. Any original            |
|                                   | overpayment would have been       |
|                                   | refunded at the time of           |
|                                   | recalculation.                    |
|                                   |                                   |
|                                   | Scenario:                         |
|                                   |                                   |
|                                   | Initial ATO based debt $950,      |
|                                   | after recalculation with evidence |
|                                   | debt dropped to $800 AND customer |
|                                   | had paid above the $800.          |
|                                   |                                   |
|                                   | The correct debt has been repaid  |
|                                   | and the customer is eligible for  |
|                                   | disbursement based on the amount  |
|                                   | paid above the $800, the          |
|                                   | timeframe these monies were held  |
|                                   | by the Agency and when the monies |
|                                   | were refunded.                    |
|                                   |                                   |
|                                   | Not eligible:                     |
|                                   |                                   |
|                                   | Group members whose debts were    |
|                                   | initially based on averaged ATO   |
|                                   | income information, but were      |
|                                   | recalculated using:               |
|                                   |                                   |
|                                   | -   payslips or                   |
|                                   | -   bank statements,              |
|                                   | -   and had not overpaid the      |
|                                   |     recalculated debt             |
|                                   |                                   |
|                                   | These group members will not      |
|                                   | receive a refund or a settlement  |
|                                   | payment.                          |
|                                   |                                   |
|                                   | Scenario:                         |
|                                   |                                   |
|                                   | Initial ATO based debt $950,      |
|                                   | after recalculation with evidence |
|                                   | debt dropped to $800 AND customer |
|                                   | paid the $800 or less.            |
|                                   |                                   |
|                                   | The customer is not eligible for  |
|                                   | disbursement as no monies above   |
|                                   | the recoverable amount were paid  |
|                                   | to or held by the agency for any  |
|                                   | period of time.                   |
|                                   |                                   |
|                                   | Scenario:                         |
|                                   |                                   |
|                                   | Debt initially generated based on |
|                                   | ATO income information, but       |
|                                   | customer gave supporting          |
|                                   | evidence:                         |
|                                   |                                   |
|                                   | -   payslips for the debt to be   |
|                                   |     recalculated, and             |
|                                   | -   had made repayments to the    |
|                                   |     value of the recalculated     |
|                                   |     debt or less                  |
|                                   |                                   |
|                                   | The customer is not eligible for  |
|                                   | disbursement as no monies above   |
|                                   | the recoverable amount were paid  |
|                                   | to or held by the agency for any  |
|                                   | period of time.                   |
+-----------------------------------+-----------------------------------+
