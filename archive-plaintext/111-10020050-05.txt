Suggested text for letters and phone calls

This table describes the type of text to use for letters and phone calls
when a document cannot be located. Note: only a Technical Peer Support
(TPS) staff member, Service Support Officer or Team Leader can draft
free text (Q999/Q888) letters .

+-----------------------------------+-----------------------------------+
| Item                              | Letter text or response for phone |
|                                   | calls                             |
+-----------------------------------+-----------------------------------+
| 1                                 | Customer indicates they have      |
|                                   | provided information, but it has  |
|                                   | not been recorded                 |
|                                   |                                   |
|                                   | 'We have followed all possible    |
|                                   | processes to find your document,  |
|                                   | and we have been unable to locate |
|                                   | it.'                              |
+-----------------------------------+-----------------------------------+
| 2                                 | If the required information can   |
|                                   | be collected by phone             |
|                                   |                                   |
|                                   | 'If it suits you, I can collect   |
|                                   | the required information from you |
|                                   | over the phone right now to       |
|                                   | ensure the process is completed   |
|                                   | as promptly as possible. The      |
|                                   | information I require is ...'     |
+-----------------------------------+-----------------------------------+
| 3                                 | If the required information must  |
|                                   | be provided in writing or as      |
|                                   | original documents                |
|                                   |                                   |
|                                   | 'To ensure the processing is      |
|                                   | completed as promptly as          |
|                                   | possible, you will need to        |
|                                   | provide this information again.   |
|                                   | The information I require is…'    |
+-----------------------------------+-----------------------------------+
