This page contains the procedure to be used by Service Officers, who are
conducting a Disbursement interview for customers exiting Vulnerable
Welfare Payment Recipient (VWPR) Income Management. Portions of this
process must only be completed by the Income Management Policy helpdesk.

| 2                                 | Reason for Income Management      |
|                                   | ending + Read more ...            |
|                                   |                                   |
|                                   | A customer must be exited from    |
|                                   | VWPR Income Management if any of  |
|                                   | the following circumstances       |
|                                   | occur:                            |
|                                   |                                   |
|                                   | They have an excluded payment     |
|                                   | nominee for Income Management     |
|                                   | purposes                          |
|                                   |                                   |
|                                   | Their category H welfare payment  |
|                                   | has cancelled                     |
|                                   |                                   |
|                                   | For customers automatically       |
|                                   | identified for VWPR due to their  |
|                                   | circumstances:                    |
|                                   |                                   |

|                                   |     exclusion from Income         |
|                                   |     Management, or                |

|                                   |     Income Management as they no  |

|                                   |                                   |
|                                   | For customers assessed for VWPR   |
|                                   | by a social worker:               |

|                                   |     expires, or                   |

|                                   |     their situation and           |

|                                   |     vulnerable so the Notice is   |

|                                   |     natural end date (for         |

|                                   |     managed and will continue to  |

|                                   | When conducting a Disbursement    |

|                                   | income managed within the next 60 |
|                                   | days. Service Officers should     |
|                                   | consider:                         |
|                                   |                                   |

|                                   | be eligible for another measure   |

|                                   | current measure ends (by checking |
|                                   | the Income Management Summary     |
|                                   | screen)                           |
|                                   |                                   |
|                                   | Where Income Management is ending |
|                                   | due to their eligible payment     |
|                                   | cancelling, consider the reason   |

|                                   | example:                          |
|                                   |                                   |
|                                   | -   CAN/OTH - check for           |
|                                   |     cancellation reason. If       |
|                                   |     reason is unknown, it is      |
|                                   |     likely the customer will be   |
|                                   |     income managed within 60 days |
|                                   | -   CAN/CLR - check for details   |
|                                   |     of why the customer requested |
|                                   |     cancellation of their         |
|                                   |     payment. If the reason is     |
|                                   |     unknown, it is likely the     |
|                                   |     customer will income managed  |
|                                   |     again within 60 days          |
|                                   | -   CAN/DNL or CAN/FRP - it is    |
