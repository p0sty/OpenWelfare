| 4                                 | Risk based model + Read more ...  |
|                                   |                                   |
|                                   | A risk based model for assessment |
|                                   | of real estate values             |
|                                   | automatically assesses a          |
|                                   | customer’s individual             |
|                                   | circumstances to determine the    |
|                                   | appropriate ongoing method of     |
|                                   | assessment for their property.    |
|                                   | This may be:                      |
|                                   |                                   |
|                                   | -   acceptance of the customer's  |
|                                   |     estimated value               |
|                                   | -   a check of the customer's     |
|                                   |     estimated value               |
|                                   |                                   |
|                                   | The risk based model will be      |
|                                   | applied to most properties, such  |
|                                   | as houses, units and townhouses,  |
|                                   | and the system will automatically |
|                                   | initiate the process to determine |
|                                   | if a customer’s estimate is       |
|                                   | acceptable or a valuation is      |
|                                   | required.                         |
|                                   |                                   |
|                                   | Other properties are also         |
|                                   | included such as:                 |
|                                   |                                   |
|                                   | -   RET: retail premises          |
|                                   | -   CMP: commercial premises      |
|                                   | -   IDP: industrial premises      |
|                                   | -   FRM: farm                     |
|                                   | -   MGD: market garden            |
|                                   | -   HBF: hobby farm               |
|                                   | -   BHB: bush block               |
|                                   | -   HMB: home block on greater    |
|                                   |     than 2 hectares               |
|                                   | -   VAC: vacant blocks            |
|                                   |                                   |
|                                   | Not included in the risk based    |
|                                   | model                             |
|                                   |                                   |
|                                   | Some property is not included in  |
|                                   | the risk based model and will     |
|                                   | require a valuation request to    |
|                                   | ensure the customers assets are   |
|                                   | assessed correctly. These can     |
|                                   | include:                          |
|                                   |                                   |
|                                   | -   overseas properties           |
|                                   | -   property that has been gifted |
|                                   | -   a person's home when they     |
|                                   |     enter residential care        |
|                                   | -   property considered as Other  |
|                                   |     (OTH) such as car parks and   |
|                                   |     marina berths                 |
|                                   | -   properties that have never    |
|                                   |     been advised and historical   |
|                                   |     valuations are required       |
|                                   | -   self-contained properties     |
|                                   |     (SCF)                         |
|                                   |                                   |
|                                   | Go to step 5                      |

| 11                                | Submission of a valuation request |

|                                   |                                   |
|                                   | Some real estate asset valuations |
|                                   | require a submission using a      |
|                                   | Services Australia Valuation      |
|                                   | Request (SA470) form. These       |
|                                   | submissions are required for real |
|                                   | estate:                           |
|                                   |                                   |
|                                   | -   not included in the risk      |
|                                   |     based model                   |
|                                   | -   requiring historical          |
|                                   |     valuation dates               |
|                                   |                                   |
|                                   | See Resources for a link to the   |
|                                   | Services Australia Valuation      |
|                                   | Request (SA470) form.             |
|                                   |                                   |
|                                   | Staff should refer to an APS5 or  |
|                                   | above to determine if a valuation |
|                                   | request is for a valid reason.    |
|                                   | This includes determining if it   |
|                                   | is approved when the              |
|                                   | circumstances are not listed      |
|                                   | below.                            |
|                                   |                                   |
|                                   | If a valuation request is         |
|                                   | required complete the SA470 in an |
|                                   | editable format, see Resources    |
|                                   | for instructions:                 |
|                                   |                                   |
|                                   | -   Include the reason for the    |
|                                   |     valuation request in the      |
|                                   |     SA470t.                       |
|                                   | -   DOC the customer record with  |
|                                   |     the reason for the valuation  |
|                                   |     request                       |
|                                   | -   Include Urgent in the summary |
|                                   |     line and reason in the body   |
|                                   |     of the email, if the request  |
|                                   |     is urgent                     |
|                                   | -   Email the SA470 to the        |
|                                   |     Valuation Helpdesk            |
|                                   |                                   |
|                                   | Circumstances where a valuation   |
|                                   | may be required                   |
|                                   |                                   |
|                                   | Overseas real estate              |
|                                   |                                   |
|                                   | A valuation is required when:     |
|                                   |                                   |
|                                   | -   the property is in United     |
|                                   |     Kingdom, United States of     |
|                                   |     America or New Zealand        |
|                                   | -   the customer's estimate       |
