FAQs - Energy Bill Relief Fund

Table 6

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Question: How much is the rebate? |
|                                   |                                   |
|                                   | Answer:                           |
|                                   |                                   |
|                                   | -   Each state and territory      |
|                                   |     government has determined the |
|                                   |     eligibility, frequency and    |
|                                   |     amount of the rebate          |
|                                   | -   Customers can find more       |
|                                   |     details about the rebate and  |
|                                   |     eligibility at energy.gov.au  |
+-----------------------------------+-----------------------------------+
| 2                                 | Question: When will customers get |
|                                   | the rebate?                       |
|                                   |                                   |
|                                   | Answer:                           |
|                                   |                                   |
|                                   | -   Each state and territory      |
|                                   |     government has determined the |
|                                   |     eligibility, frequency and    |
|                                   |     amount of the rebate          |
|                                   | -   Depending on the state or     |
|                                   |     territory, customers might    |
|                                   |     get bill relief on a single   |
|                                   |     bill or spread across         |
|                                   |     multiple bills                |
|                                   | -   Customers can find more       |
|                                   |     details about the rebate and  |
|                                   |     eligibility at energy.gov.au  |
+-----------------------------------+-----------------------------------+
| 3                                 | Question: Can customers receive   |
|                                   | the rebate for an address other   |
|                                   | than their home address, for      |
|                                   | example a temporary or term       |
|                                   | address?                          |
|                                   |                                   |
|                                   | Answer:                           |
|                                   |                                   |
|                                   | -   Customers can only get one    |
|                                   |     bill relief for their         |
|                                   |     principal place of residence  |
|                                   | -   The Energy Bill Relief        |
|                                   |     Consent workflow              |
|                                   |     automatically populates the   |
|                                   |     customer’s home address       |
|                                   |     (coded HOM on the AD screen). |
|                                   |     This is the address Services  |
|                                   |     Australia will provide to the |
|                                   |     customer’s electricity        |
|                                   |     retailer                      |
|                                   | -   Customers can find more       |
|                                   |     details about the rebate and  |
|                                   |     eligibility at energy.gov.au  |
+-----------------------------------+-----------------------------------+
| 4                                 | Question: Can customers receive   |
|                                   | the rebate if the electricity     |
|                                   | bill is in their partner’s name?  |
|                                   |                                   |
|                                   | Answer:                           |
|                                   |                                   |
|                                   | -   Each household can only get   |
|                                   |     one bill relief rebate, even  |
|                                   |     if there is more than one     |
|                                   |     eligible person living there  |
|                                   | -   To receive the rebate,        |
|                                   |     customers must be the account |
|                                   |     holder or named on the        |
|                                   |     account and receive an        |
|                                   |     eligible concession card or   |
|                                   |     payment                       |
|                                   | -   Customers can find more       |
|                                   |     details about the rebate and  |
|                                   |     eligibility at energy.gov.au  |
+-----------------------------------+-----------------------------------+
| 5                                 | Question: Can customers sharing   |
|                                   | accommodation receive the rebate? |
|                                   |                                   |
|                                   | Answer:                           |
|                                   |                                   |
|                                   | -   Each household can only get   |
|                                   |     one bill relief rebate, even  |
|                                   |     if there is more than one     |
|                                   |     eligible person living there  |
|                                   | -   If the customer is not the    |
|                                   |     account holder or named on    |
|                                   |     the account, they should      |
|                                   |     speak with the person who is  |
|                                   |     the account holder            |
|                                   | -   Customers can find more       |
|                                   |     details about the rebate and  |
|                                   |     eligibility at energy.gov.au  |
+-----------------------------------+-----------------------------------+
| 6                                 | Question: What if I live in a     |
|                                   | caravan park or apartment         |
|                                   | building that shares one          |
|                                   | electricity provider? (Embedded   |
|                                   | network)                          |
|                                   |                                   |
|                                   | Answer:                           |
|                                   |                                   |
|                                   | -   Embedded or on-supplied       |
|                                   |     networks are private          |
|                                   |     electricity networks that     |
|                                   |     serve multiple customer       |
|                                   |     premises (or lots) within a   |
|                                   |     building or self-contained    |
|                                   |     site (such as a caravan park, |
|                                   |     apartment building or         |
|                                   |     retirement home or village)   |
|                                   | -   Customers in embedded         |
|                                   |     networks cannot provide       |
|                                   |     consent through the online    |
|                                   |     workflow or staff assisted    |
|                                   |     process                       |
|                                   | -   Each state or territory has a |
|                                   |     specific process for          |
|                                   |     customers who are in embedded |
|                                   |     networks. Direct the customer |
|                                   |     to go to energy.gov.au and    |
|                                   |     locate their state or         |
|                                   |     territory government's        |
|                                   |     embedded network process      |
+-----------------------------------+-----------------------------------+
| 7                                 | Question: Will customers using    |
|                                   | pre-paid electricity meters       |
|                                   | (tokens, e-meters and secure      |
|                                   | meters) receive the rebate?       |
|                                   |                                   |
|                                   | Answer:                           |
|                                   |                                   |
|                                   | -   The approach for customers    |
|                                   |     who use pre-paid electricity  |
|                                   |     meters such as tokens,        |
|                                   |     e-meters and secure meters    |
|                                   |     has been determined by each   |
|                                   |     state and territory           |
|                                   |     government                    |
|                                   | -   Customers can find more       |
|                                   |     details about the rebate and  |
|                                   |     eligibility at energy.gov.au  |
+-----------------------------------+-----------------------------------+
| 8                                 | Question: What if the customer    |
|                                   | has changed address?              |
|                                   |                                   |
|                                   | Answer:                           |
|                                   |                                   |
|                                   | -   Use the Change of Address     |
|                                   |     workflow and apply it to      |
|                                   |     their record before           |
|                                   |     completing the Energy Bill    |

|                                   | -   Customers can only get the    |
|                                   |     energy bill relief for their  |
|                                   |     principal place of residence  |
|                                   | -   Customers can find more       |
|                                   |     details about the Energy Bill |
|                                   |     Relief Fund and other state   |
|                                   |     and territory concessions at  |
|                                   |     energy.gov.au                 |
+-----------------------------------+-----------------------------------+

|                                   | cancelled, suspended, newly       |
