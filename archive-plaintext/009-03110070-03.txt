| 3                                 | Cancellations of CA can be        |
|                                   | automatic or manual + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | For cancellations that are:       |
|                                   |                                   |
|                                   | -   automatic, the carer's (or    |
|                                   |     care receiver's)              |
|                                   |     circumstances have changed    |
|                                   |     and as a result of that       |
|                                   |     change the carer is no longer |
|                                   |     qualified for payment.        |
|                                   | -   manual, a Service Officer     |
|                                   |     makes a decision to cancel a  |
|                                   |     carer's payment and runs the  |
|                                   |     Carer cancellation and        |
|                                   |     suspension workflow to cancel |
|                                   |     CA. For example, the carer's  |
|                                   |     record has been suspended for |
|                                   |     3 months and now needs to be  |
|                                   |     cancelled:                    |
|                                   | -   a manual cancellation may     |
|                                   |     cancel all payments of CA to  |
|                                   |     the carer for all care        |
|                                   |     receivers                     |
|                                   | -   if cancellation is due to a   |
|                                   |     carer completing the online   |
|                                   |     task within 14 days following |
|                                   |     a change of address, manual   |
|                                   |     cancellation must occur using |
|                                   |     Process Direct. See Step 3 in |
|                                   |     Table 2                       |
|                                   |                                   |
|                                   | Key all carer circumstances to    |
|                                   | allow the payment to auto cancel  |
|                                   | before considering a manual       |
|                                   | cancellation using the workflow.  |
|                                   | An auto cancellation will ensure  |
|                                   | the correct letter is sent to the |
|                                   | carer. It will also ensure        |
|                                   | Services Australia has the most   |
|                                   | up-to-date record of the carer    |
|                                   | and carer receiver's              |
|                                   | circumstances. This may be        |
|                                   | important if their CA payment is  |
|                                   | subsequently re-granted.          |
|                                   |                                   |
|                                   | The most common reasons for an    |
|                                   | auto-cancellation include:        |
|                                   |                                   |
|                                   | permanent cessation of care:      |
|                                   |                                   |
|                                   | -   see Care receiver permanently |
|                                   |     enters or                     |
|                                   |     temporarily/permanently       |
|                                   |     leaves an institution         |
|                                   | -   care receiver leaves care of  |
|                                   |     the carer. See Step 8 in      |
|                                   |     Table 2                       |
|                                   | -   carer ceases to provide care  |
|                                   |     for care receiver(s). See     |
|                                   |     Step 8 in Table 2             |
|                                   |                                   |
|                                   | temporary cessation of care, in   |
|                                   | most cases, payment will only     |
|                                   | cancel once the 63 days of        |
|                                   | respite care have been used up.   |
|                                   | If the carer is in gaol or        |
|                                   | psychiatric confinement for:      |
|                                   |                                   |
|                                   | -   more than 13 weeks because    |
|                                   |     they have been charged with   |
|                                   |     an offence, respite days      |
|                                   |     cannot be used, and the       |
|                                   |     payment must be cancelled     |
|                                   |     reason IMP                    |
|                                   | -   13 weeks or less, the payment |
|                                   |     is suspended reason IMP. If   |
|                                   |     the carer is still confined   |
|                                   |     after 13 weeks, the suspended |
|                                   |     payment will auto-cancel from |
|                                   |     the date of confinement. See: |
|                                   | -   Carer is absent from the      |
|                                   |     caring situation              |
|                                   | -   Care receiver is absent from  |
|                                   |     the caring situation          |
|                                   | -   Continuation of respite after |
|                                   |     the expiry of hospitalisation |
|                                   | -   Overseas absences             |
|                                   |                                   |
|                                   | death of a carer                  |
|                                   |                                   |
|                                   | death of a child care receiver    |
|                                   | and the effect on CA              |
|                                   |                                   |
|                                   | death of adult care receiver and  |
|                                   | the effect on CA                  |
|                                   |                                   |
|                                   | CA income review processing.      |
|                                   | Cancellation can occur due to     |
|                                   | 'Failed to reply to               |
|                                   | correspondence', excess income    |
|                                   | and estimate not accepted         |
|                                   |                                   |
|                                   | For:                              |
|                                   |                                   |
|                                   | -   other automatic               |
|                                   |     cancellations, go to Step 4   |
|                                   | -   manual cancellations, go to   |
|                                   |     Step 5                        |
