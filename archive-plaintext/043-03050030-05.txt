Assessment of commission income for employees after 7 December 2020

+-----------------------------------+-----------------------------------+
| Type of commission used           | Assessment                        |
+-----------------------------------+-----------------------------------+
| Amount paid in each entitlement   | Assess income paid in fortnight   |
| period (EP) is known.             | as income in 1 period (IOP). The  |
|                                   | customer reports the amount paid  |
|                                   | in the EP. Customer will be a     |
|                                   | statement reporter.               |
+-----------------------------------+-----------------------------------+
| Regular payments are equal to or  | Assess income paid in the         |
| less than 2 weeks apart, but the  | entitlement period (EP) as income |
| period the commission covers is   | longer than one period (LOP). The |
| greater than one entitlement      | customer will need to provide the |
| period (EP).                      | start date and the end date of    |
|                                   | the period the commission was     |
|                                   | paid for.                         |
+-----------------------------------+-----------------------------------+
| Regular payments of commission    | Assess using LOP frequency for    |
| income are greater than 2 weeks   | the number of days the payment    |
| apart. Period commission is paid  | represents from the start of the  |
| for is greater than one           | entitlement period (EP) in which  |
| entitlement period (EP).          | it is paid. For example if the    |
|                                   | payment is for a period of 31     |
|                                   | days, assess as lump sum income   |
|                                   | for 31 days from the entitlement  |
|                                   | period start date (EPSD). The     |
|                                   | customer will need to provide the |
|                                   | start date and the end date of    |
|                                   | the period the commission was     |
|                                   | paid for.                         |
+-----------------------------------+-----------------------------------+
| One-off or ad hoc where period    | Assess using LOP frequency for    |
| paid is known and is greater than | the number of days the commission |
| one entitlement period (EP).      | was paid for, up to a maximum of  |
|                                   | 52 weeks. For example, if the     |
|                                   | commission represents a period of |
|                                   | 15 months code the start and      |
|                                   | dates of the 15 month period and  |
|                                   | the system will assess for 52     |
|                                   | weeks from the entitlement period |
|                                   | start date (EPSD) of the period   |
|                                   | in which it was paid.             |
+-----------------------------------+-----------------------------------+
| One-off or ad hoc where the       | Assess using LOP frequency for a  |
| period paid is unknown.           | period of up to 52 weeks starting |
|                                   | from the entitlement period start |
|                                   | date (EPSD). If the period cannot |
|                                   | be determined after discussions   |
|                                   | with the customer and a Local     |
|                                   | Peer Support (LPS), have your LPS |
|                                   | refer the case to the Level 2     |
|                                   | Policy Helpdesk.                  |
+-----------------------------------+-----------------------------------+
