Risks to a customer's privacy have been identified. See Separating
Safely - protecting personal details to make sure the customer's
personal details are safe before progressing.

This is a Family and Domestic Violence Intervention Point. If the
customer is with another person, on speakerphone, or already identified
with family and domestic violence concerns, continue with the current
business conversation. Otherwise, go to the Family and domestic violence
procedure to conduct the risk identification and referral process.

|                                   | -   SS284/SS293 form(s)           |
|                                   | -   Centrelink records of the     |

|                                   |     including the DL screen and   |
|                                   |     scanned documents             |

|                                   | whether the customer is a MoC or  |
|                                   | not.                              |
|                                   |                                   |
|                                   | Aspects to consider:              |
|                                   |                                   |
|                                   | -   Is there any family and       |
|                                   |     domestic violence identified? |
|                                   |     The presence of family and    |
|                                   |     domestic violence may         |
|                                   |     indicate a person is not a    |
|                                   |     member of a couple and must   |
|                                   |     be considered when making a   |
|                                   |     decision about a customer’s   |
|                                   |     relationship status. This     |
|                                   |     includes retrospective        |
|                                   |     assessments required due to   |
|                                   |     claims for Crisis Payment     |
|                                   |     (CrP) as a result of family   |
|                                   |     and domestic violence         |

|                                   |     requested which has not been  |
|                                   |     provided (for example, the    |
|                                   |     customer failed to respond to |
