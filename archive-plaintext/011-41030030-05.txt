|                                   |                                   |
|                                   | Note: the customer needs to be    |
|                                   | eligible for the FTB Part A       |

+-----------------------------------+-----------------------------------+

CDBS eligibility requirements

Table 2: This table describes the qualifying payments for either
parents/guardian or children/teenagers.

+-----------------------------------+-----------------------------------+
| Item                              | Qualifying payments               |
+-----------------------------------+-----------------------------------+
| 1                                 | Parent/guardian or their partner  |
|                                   | gets:                             |
|                                   |                                   |
|                                   | -   FTB Part A                    |
|                                   | -   Parenting Payment (PP)        |
|                                   | -   Double Orphan Pension (DOP)   |
+-----------------------------------+-----------------------------------+
| 2                                 | Child/teenager gets:              |
|                                   |                                   |
|                                   | -   Family Tax Benefit (FTB) Part |
|                                   |     A                             |
|                                   | -   Youth Allowance (YA)          |
|                                   | -   ABSTUDY                       |
|                                   | -   Disability Support Pension    |
|                                   |     (DSP)                         |
|                                   | -   Parenting Payment (PP)        |
|                                   | -   Special Benefit (SpB)         |
|                                   | -   Carer Payment (CP)            |
+-----------------------------------+-----------------------------------+
