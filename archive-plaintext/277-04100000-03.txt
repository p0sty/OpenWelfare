| 1                                 | Determine appropriate notice to   |
|                                   | issue + Read more ...             |
|                                   |                                   |
|                                   | Does the customer have an         |
|                                   | overpayment debt?                 |
|                                   |                                   |
|                                   | -   Yes, if enforcement action is |
|                                   |     required on an overpayment,   |
|                                   |     see Debt repayment            |
|                                   | -   No, consider which type of    |
|                                   |     notice is appropriate         |
|                                   |                                   |
|                                   | Issue the notice through Cuba to  |
|                                   | serve the notice. Consider        |
|                                   | whether it is appropriate to      |
|                                   | issue an s72A notice on an        |
|                                   | organisation with a Ceased,       |
|                                   | Insolvent or Duplicate status.    |
|                                   |                                   |
|                                   | Note: a warning message will      |
|                                   | appear if the customer has a      |
|                                   | current alias. The alias may need |
|                                   | to be included on the notice/s.   |
|                                   |                                   |
|                                   | Notices for collection through    |
|                                   | third parties are issued under    |
|                                   | s72A of the Child Support         |
|                                   | (Registration and Collection) Act |
|                                   | 1988. An s72A notice may be       |
|                                   | issued for:                       |
|                                   |                                   |
|                                   | -   a lump sum, go to Step 2      |
|                                   | -   a periodic amount, go to Step |
|                                   |     3                             |
|                                   | -   cents in the dollar, go to    |
|                                   |     Step 4                        |

| 2                                 | Lump sum notices + Read more ...  |
|                                   |                                   |
|                                   | Lump sum notices are appropriate  |
|                                   | when the source of funds is:      |
|                                   |                                   |
|                                   | -   greater than $100. Unless the |
|                                   |     case has ended or has a low   |
|                                   |     liability and arrears have    |
|                                   |     accrued over a substantial    |
|                                   |     period of time                |
|                                   | -   available once only, for      |
|                                   |     example, proceeds from a sale |
|                                   |     of a house, compensation      |
|                                   |     payout, and                   |
|                                   | -   likely to become unavailable  |
|                                   |     in the future. For example,   |
|                                   |     funds in a bank account       |
