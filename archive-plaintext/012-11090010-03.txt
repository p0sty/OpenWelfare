Identify if a practice is PIP consenting

Table 1

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Check the PIP Registered Practice |
|                                   | status + Read more ...            |
|                                   |                                   |
|                                   | Log on to WIP – Practice Stream   |
|                                   | Online.                           |
|                                   |                                   |
|                                   | On the Practice Search screen:    |
|                                   |                                   |
|                                   | -   key the Practice ID           |
|                                   | -   select Search. The Search     |
|                                   |     Results will show             |
|                                   | -   select the Practice ID. The   |
|                                   |     Practice Summary screen will  |
|                                   |     show                          |
|                                   |                                   |
|                                   | If the PIP Registered Practice    |
|                                   | and consent to use PIP data for   |
|                                   | WIP – Practice Stream status is:  |
|                                   |                                   |
|                                   | -   Yes, the practice is PIP      |
|                                   |     Consenting                    |
|                                   | -   No, the practice is not PIP   |
|                                   |     Consenting                    |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 2                                 | Check if the PIP and WIP –        |
|                                   | Practice Stream practice IDs      |
|                                   | match + Read more ...             |
|                                   |                                   |
|                                   | Log onto these systems:           |
|                                   |                                   |
|                                   | -   PIP Online, and               |
|                                   | -   WIP – Practice Stream Online  |
|                                   |                                   |
|                                   | On the Practice Search screen:    |
|                                   |                                   |
|                                   | -   key the Practice ID           |
|                                   | -   select Search. The Search     |
|                                   |     Results will show             |
|                                   |                                   |
|                                   | If no Search Results show, the    |
|                                   | practice is not PIP consenting.   |
|                                   |                                   |
|                                   | Check if the below WIP – Practice |
|                                   | Stream details match the details  |
|                                   | on the PIP practice profile:      |
|                                   |                                   |
|                                   | -   Practice ID                   |
|                                   | -   Practice name, and            |
|                                   | -   Practice address              |
|                                   |                                   |
|                                   | If the details:                   |
|                                   |                                   |
|                                   | -   match, the practice is PIP    |
|                                   |     consenting.                   |
|                                   | -   do not match, the practice is |
|                                   |     not PIP consenting.           |

Assess a PIP consenting request

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Log into systems and check        |
|                                   | practice details + Read more ...  |
|                                   |                                   |
|                                   | Log on to these systems:          |
|                                   |                                   |
|                                   | -   PIP Online, and               |
|                                   | -   WIP – Practice Stream Online  |
|                                   |                                   |
|                                   | Assess the practice details on    |
|                                   | the correspondence. The practice  |
|                                   | must provide a written request:   |
|                                   |                                   |
|                                   | -   on official company           |
|                                   |     letterhead, or                |
|                                   | -   via email                     |
|                                   |                                   |
|                                   | The correspondence must include:  |
|                                   |                                   |
|                                   | -   the practice name             |
|                                   | -   the practice address, and     |
|                                   | -   both PIP and WIP – Practice   |
|                                   |     Stream practice IDs           |
|                                   |                                   |
|                                   | If the above details are missing, |
|                                   | return the correspondence to the  |
|                                   | practice. Go to Step 8.           |
+-----------------------------------+-----------------------------------+
| 2                                 | Search for the practice + Read    |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Search for the practice in PIP    |
|                                   | Online and WIP – Practice Stream  |
|                                   | Online.                           |
|                                   |                                   |
|                                   | On the Practice Search screen:    |
|                                   |                                   |
|                                   | -   key the Practice ID           |
|                                   | -   select Search. The Search     |
|                                   |     Results will show             |
|                                   |                                   |
|                                   | If no Search Results show, the    |
|                                   | practice ID is incorrect, return  |
|                                   | the request to the practice. go   |
|                                   | to Step 8.                        |
|                                   |                                   |
|                                   | Check if the practice name and    |
|                                   | address on the practice profile   |
|                                   | match the details on the          |
|                                   | correspondence. Note: an address  |
|                                   | variance may be accepted.         |
|                                   |                                   |
|                                   | If the details:                   |
|                                   |                                   |
|                                   | match, go to Step 3               |
|                                   |                                   |
|                                   | do not match, check the company   |
|                                   | or trading name:                  |
|                                   |                                   |
|                                   | -   Select the Practice ID. The   |
|                                   |     Practice Summary screen will  |
|                                   |     show                          |
|                                   | -   Select Practice Details from  |
|                                   |     the Main menu                 |
|                                   | -   Select Ownership Details from |
|                                   |     the drop down. The Ownership  |
|                                   |     Details screen will show      |
|                                   |                                   |
|                                   | Do these details match the        |
|                                   | details on the form?              |
|                                   |                                   |
|                                   | -   Yes, go to Step 3             |
|                                   | -   No, return the correspondence |
|                                   |     to the practice, go to Step 8 |
+-----------------------------------+-----------------------------------+
| 3                                 | Practice Status + Read more ...   |
|                                   |                                   |
|                                   | On the Practice Search screen,    |
|                                   | check the Search Results Status.  |
|                                   |                                   |
|                                   | Does the status show as Approved? |
|                                   |                                   |
|                                   | -   Yes, go to Step 4             |
|                                   | -   No, return the correspondence |
|                                   |     to the practice, go to Step 8 |
+-----------------------------------+-----------------------------------+
| 4                                 | PIP consenting practices + Read   |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Is the practice a PIP consenting  |
|                                   | practice?                         |
|                                   |                                   |
|                                   | -   Yes, finalise the Work Item   |
|                                   |     as Complete, Already complete |
|                                   |     in PaNDA                      |
|                                   | -   No, go to Step 5              |
+-----------------------------------+-----------------------------------+
| 5                                 | Check the signature block + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Check the signature block details |
|                                   | on the request. There must be:    |
|                                   |                                   |
|                                   | -   the signature of at least one |
|                                   |     registered owner or an        |
|                                   |     authorised contact person     |
|                                   |     Do not check these details if |
|                                   |     the practice ID is missing,   |
|                                   |     incorrect or the practice     |
|                                   |     cannot be identified          |
|                                   | -   their full name               |
|                                   |     If there are no name details, |
|                                   |     check the signature. Accept   |
|                                   |     the form if the name can be   |
|                                   |     identified from the signature |
|                                   |                                   |
|                                   | Are the declaration details       |
|                                   | complete?                         |
|                                   |                                   |
|                                   | -   Yes, go to Step 6             |
|                                   | -   No, check the rest of the     |
|                                   |     correspondence to see if any  |
|                                   |     other details are needed and  |
|                                   |     return the correspondence to  |
|                                   |     the practice                  |
+-----------------------------------+-----------------------------------+
| 6                                 | Ownership details + Read more ... |
|                                   |                                   |
|                                   | The ownership details in the WIP  |
|                                   | – Practice Stream practice        |
|                                   | profile must match the PIP        |
|                                   | practice profile.                 |
|                                   |                                   |
|                                   | -   Select Practice Details from  |
|                                   |     the Main menu                 |
|                                   | -   Select Ownership details from |
|                                   |     the drop down. The Ownership  |
|                                   |     Details screen will show      |
|                                   |                                   |
|                                   | Check that these ownership        |
|                                   | details match:                    |
|                                   |                                   |
|                                   | -   Ownership type                |
|                                   | -   Company name                  |
|                                   | -   Trading name                  |
|                                   | -   ABN                           |
|                                   | -   Ownership address, and        |
|                                   | -   Owners/Partners/Associates    |
|                                   |                                   |
|                                   | Do the ownership details match?   |
|                                   |                                   |
|                                   | -   Yes, go to Step 7             |
|                                   | -   No or other details are       |
|                                   |     needed. Go to Step 8          |
+-----------------------------------+-----------------------------------+
| 7                                 | Escalate PIP consenting request   |
|                                   | and finalise work item + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | -   Escalate the request to the   |
|                                   |     Program Support Officer (PSO) |
|                                   |     via the Incentives Escalation |
|                                   |     Database                      |
|                                   | -   Upload the request to the     |
|                                   |     practice profile.             |
|                                   | -   Finalise the Work Item as     |
|                                   |     Complete in PaNDA             |
+-----------------------------------+-----------------------------------+
| 8                                 | Returning incomplete requests +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | If the correspondence needs more  |
|                                   | details:                          |
|                                   |                                   |
|                                   | Return the correspondence to the  |
|                                   | practice. If the correspondence   |
|                                   | is received via:                  |
|                                   |                                   |
|                                   | -   Official company letterhead,  |
|                                   |     use the Request for more      |
|                                   |     information for the PIP and   |
|                                   |     the WIP – Practice Stream     |
|                                   |     (z1736) letter. If the        |
|                                   |     practice ID is missing,       |
|                                   |     incorrect or the practice     |
|                                   |     cannot be identified, use the |
|                                   |     address on the request        |
|                                   | -   Email, reply to the original  |
|                                   |     email using the appropriate   |
|                                   |     email template                |
|                                   |                                   |
|                                   | Upload the letter or email and    |
|                                   | the correspondence to the         |
|                                   | practice profile. Upload the      |
|                                   | documents to PaNDA only if the    |
|                                   | practice ID is missing, incorrect |
|                                   | or the practice cannot be         |
|                                   | identified                        |
|                                   |                                   |
|                                   | Finalise the Work Item as         |
|                                   | Complete, Request rejected in     |
|                                   | PaNDA                             |

PIP consenting request escalation for Program Support Officers (PSO)

Table 3

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Notify the Program Management     |
|                                   | team + Read more ...              |
|                                   |                                   |
|                                   | Send an email to Program          |
|                                   | Management (Payments team)        |
|                                   | advising of the PIP consenting    |
|                                   | request. See                      |
|                                   |                                   |
|                                   | suggested email wording on the    |
|                                   | Resources page.                   |
|                                   |                                   |
|                                   | This can take several weeks to    |
|                                   | complete as PIP consenting        |
|                                   | changes are only actioned during  |
|                                   | specific periods of each          |
|                                   | calculation quarter.              |
+-----------------------------------+-----------------------------------+

Notify a practice of completed PIP consenting request - Program Support Officers (PSO)

Table 4

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Notification from Program         |
|                                   | Management + Read more ...        |
|                                   |                                   |
|                                   | Go to the PSO mailbox and find    |
|                                   | the email from the Program        |
|                                   | Management (WIP – Practice Stream |
|                                   | Payments team).                   |
|                                   |                                   |
|                                   | The Subject is Practices          |
|                                   | requesting to become PIP          |
|                                   | consenting (or similar)           |
|                                   |                                   |
|                                   | The email gives these details:    |
|                                   |                                   |
|                                   | -   New WIP – Practice Stream     |
|                                   |     practice ID                   |
|                                   | -   Previous WIP – Practice       |
|                                   |     Stream practice ID            |
|                                   | -   Practice Name, and            |
|                                   | -   Further action needed         |
+-----------------------------------+-----------------------------------+
| 2                                 | Notify the practice + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | Use the suggested email wording   |
|                                   | on the Resources page to send an  |
|                                   | email to the practice or call to  |
|                                   | confirm these details:            |
|                                   |                                   |
|                                   | -   The practice is PIP           |
|                                   |     consenting                    |
|                                   | -   The new WIP – Practice Stream |
|                                   |     practice ID and Practice name |
|                                   | -   The WIP – Practice Stream     |
|                                   |     practice ID now matches the   |
|                                   |     PIP practice ID               |
|                                   | -   The next WIP – Practice       |
|                                   |     Stream Quarterly Confirmation |
|                                   |     Statement (QCS) will not be   |
|                                   |     generated                     |
|                                   | -   Confirm the practice’s Health |
|                                   |     Professional details via      |
|                                   |     return email for the next WIP |
|                                   |     – Practice Stream quarter     |
|                                   | -   The new WIP – Practice Stream |
|                                   |     profile now reflects the same |
|                                   |     details as their PIP profile  |
|                                   | -   Make most practice updates to |
|                                   |     their PIP profile without     |
|                                   |     making the same update to     |
|                                   |     their WIP – Practice Stream   |
|                                   |     profile                       |
|                                   |                                   |
|                                   | Note: only call a practice if no  |
|                                   | email address is available. Add a |
|                                   | comment to the practice profile   |
|                                   | advising details of the call.     |
