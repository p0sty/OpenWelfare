Determine, calculate and raise overpayments

Table 5: For Compliance Officers (CO)

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Check if a Recovery Fee will      |
|                                   | apply + Read more ...             |
|                                   |                                   |
|                                   | If the intervention includes      |
|                                   | verification of gross employment  |
|                                   | income paid including income from |
|                                   | self-employment, determine if the |
|                                   | Recovery Fee will apply.          |
|                                   |                                   |
|                                   | The recovery fee will only be     |
|                                   | applied if the customer has not   |
|                                   | met their obligations. For        |
|                                   | example, they failed to provide   |
|                                   | information relating to their     |
|                                   | income confirmation.              |
|                                   |                                   |
|                                   | The recovery fee will not be      |
|                                   | applied if the customer fulfilled |
|                                   | their obligations. For example,   |
|                                   | they provided information         |
|                                   | relating to their income          |
|                                   | confirmation.                     |
|                                   |                                   |
|                                   | Note: employment income debts     |
|                                   | with a period of 14 days or less  |
|                                   | will be automatically raised on   |
|                                   | the completion of the income      |
|                                   | activity. The Recovery Fee is not |
|                                   | applied to auto raised debts. The |
|                                   | customer will receive the         |
|                                   | standard debt advice as part of   |
|                                   | the automatic process.            |
+-----------------------------------+-----------------------------------+
| 2                                 | Determine debt and calculate +    |
|                                   | Read more ...                     |
|                                   |                                   |
