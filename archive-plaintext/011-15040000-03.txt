|                                   | the My Work page:                 |
|                                   |                                   |
|                                   | -   select the Work Item ID (tick |

+-----------------------------------+-----------------------------------+
| 2                                 | Re-categorise work item + Read    |
|                                   | more ...                          |

|                                   | -   MCRS_NORA_NOC_APPRVL for NORA |

|                                   | -   MCRS_PRIORITY_SO4 for         |
|                                   |     priority processing           |
|                                   | -   MCRS_REVIEW_EXT for review of |
|                                   |     extension decision by an APS4 |
|                                   | -   MCRS_REVIEW_REFUSAL for       |
|                                   |     review of a refusal decision  |
|                                   |     by an APS5                    |
|                                   | -   MCRS_SSO5_ESC for escalations |
|                                   |     to Service Support Officer    |

|                                   |                                   |
|                                   | select Submit                     |
|                                   |                                   |
|                                   | Note: escalation of work to       |
|                                   | relevant work groups must occur   |
|                                   | as per individual process tables. |

| 3                                 | Escalating work to Service        |
|                                   | Officer (SO5) or Programme        |
|                                   | Management + Read more ...        |
|                                   |                                   |
|                                   | When a claim is referred to a     |
|                                   | Service Officer (SO5) or          |
|                                   | Programme Management for advice:  |
|                                   |                                   |
|                                   | -   complete the escalation       |
|                                   |     template, see Resources for a |
|                                   |     link, and                     |
|                                   | -   save a copy of the template   |
|                                   |     in a case note in the claim,  |
|                                   |     see Table 2                   |
|                                   |                                   |
|                                   | Note: do not send an email to     |
|                                   | Programme Management as they no   |
|                                   | longer need an email to be sent.  |

Add a case note

Table 2

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Locate Add case note + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | In an Enquiry case, select Case   |
|                                   | note + in the right hand utility  |
|                                   | panel, or                         |
|                                   |                                   |
|                                   | Via Restricted Claim Actions:     |
|                                   |                                   |
|                                   | -   search for the WIN, under     |
|                                   |     Search results                |
|                                   | -   select Actions from the       |
|                                   |     dropdown menu next to the WIN |
|                                   | -   select Record case note       |
|                                   |                                   |
|                                   | The Add case note window appears  |
+-----------------------------------+-----------------------------------+
| 2                                 | Complete Add case note + Read     |
|                                   | more ...                          |
|                                   |                                   |
|                                   | -   Select a type of case note    |
|                                   |     from the Type heading         |
|                                   |     dropdown menu                 |
|                                   | -   Select the most relevant case |
|                                   |     note heading                  |
|                                   | -   Open the standard case note   |
|                                   |     template, see Resources for a |
|                                   |     link                          |
|                                   | -   Copy and paste the relevant   |
|                                   |     case note into the note       |
|                                   |     section                       |
|                                   | -   Enter the required details    |
|                                   | -   Select Submit                 |

|                                   | -   Select the Case Notes tab     |
|                                   |     from the View Compensation    |
|                                   |     Details screen to view a list |

|                                   |     results select the WIN        |
|                                   |     hyperlink                     |
|                                   | -   Select the Case Notes tab     |
|                                   |     from the View Compensation    |
|                                   |     Details screen to view a list |
|                                   |     of all available case notes   |
|                                   | -   Select any case note title    |
|                                   |     from the list to view the     |
|                                   |     content of a case note        |
|                                   |                                   |
|                                   | Note: case notes added to an open |
|                                   | Enquiry case do not show in the   |
|                                   | Case notes tab until the Enquiry  |
|                                   | case is finalised.                |
