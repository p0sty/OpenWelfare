PPL details for customers with children born or entering care both
before and on or after 1 July 2023.

This page contains information on assessing claims for Parental Leave
Pay (PPL) and/or Family Tax Benefit (FTB) from birth mothers who
relinquish the care of their child.

PPL and FTB claims from birth mothers relinquishing care of child

| 4                                 | Cessation of care of the child +  |

|                                   | birth may still be eligible for   |

|                                   | purposes.                         |

|                                   | work.                             |
|                                   |                                   |
|                                   | For children born before 1 July   |
|                                   | 2023, birth mothers may receive   |
|                                   | PPL up to 18 weeks following the  |
|                                   | birth of their child if they:     |
|                                   |                                   |
|                                   | -   nominate a start date no      |
|                                   |     later than the child's date   |
|                                   |     of birth                      |
|                                   | -   claim and provide relevant    |
|                                   |     documents within the 28 days  |
|                                   |     following the birth           |
|                                   |                                   |
|                                   | As the birth mother cannot        |
|                                   | receive PPL more than 18 weeks    |
|                                   | after the childâ€™s birth, tell     |
|                                   | them to either:                   |
|                                   |                                   |
|                                   | -   connect all of their Flexible |
|                                   |     PPL days to the end of their  |
|                                   |     PPL period, or                |
|                                   | -   claim consecutive Flexible    |
|                                   |     PPL days after their PPL      |
|                                   |     Period                        |
|                                   |                                   |
|                                   | For children born on or after 1   |
|                                   | July 2023, birth mothers may      |
|                                   | receive up to 20 weeks of PPL, in |
|                                   | addition to the gaining           |
|                                   | parent/carer.                     |
|                                   |                                   |
|                                   | For the birth mother, PPL will    |
|                                   | only be payable during the first  |
|                                   | 20 weeks (140 calendar days)      |
|                                   | after the birth of the child.     |
|                                   |                                   |
|                                   | If they claim and provide         |
|                                   | relevant documents more than 100  |
|                                   | days following the birth, PPL may |
|                                   | only start within 100 days from   |
|                                   | the date all information is       |
|                                   | received. This means they may     |
|                                   | receive less than the full        |
|                                   | entitlement.                      |
|                                   |                                   |
|                                   | Has the customer lodged an        |
|                                   | effective claim and provided all  |
|                                   | relevant documents within 28 days |
|                                   | of the birth of the child?        |
|                                   |                                   |
|                                   | Yes, confirm the customer has     |
|                                   | selected the child's date of      |
|                                   | birth as their nominated start    |
|                                   | date for PPL. Go to Step 5        |
|                                   |                                   |
|                                   | No, the customer may not receive  |
|                                   | the full PPL entitlement          |
|                                   |                                   |
|                                   | -   Confirm the customer has      |
|                                   |     selected the date of claim as |
|                                   |     their start date              |
|                                   | -   If the customer has selected  |
|                                   |     a later date, try to call the |
|                                   |     customer as a later start     |
|                                   |     date will prevent them        |
|                                   |     receiving the full PPL        |
|                                   |     entitlement                   |
|                                   | -   Go to Step 5                  |
