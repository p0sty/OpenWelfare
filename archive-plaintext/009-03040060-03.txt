|                                   | The percentage of the total care  |

|                                   | the Shared Care %: field. For     |

|                                   | hours care and carer B, provides  |

|                                   | if both carers claim payment. If  |
|                                   | only one carer claims, that carer |
|                                   | receives 100%.                    |

|                                   | percentage of care provided must  |

|                                   | carer with the greater amount of  |
|                                   | care is the primary carer.        |

|                                   | Care: field should be coded with  |
|                                   | one of the following, depending   |

|                                   |     Care is provided on a daily   |
|                                   |     basis and at least 20 hours   |
|                                   |     care is provided per week     |

|                                   |     provided on a daily basis but |
|                                   |     less than 20 hours care is    |
