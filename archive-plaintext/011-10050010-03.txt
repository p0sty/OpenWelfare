This page contains information and procedures on how AIR Service
Officers handle vaccination provider registration enquiries and amend
AIR program registration details in PDS.

On this page:

Record or update Medicare provider bank details for AIR

Approve provider bank details for AIR

Link provider bank details for AIR

Process a Location ID/Minor ID/Software ID

Nurse/Nurse practitioner provider number requests for the AIR

Table 1

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Check nurse/nurse practitioner    |
|                                   | qualifications + Read more ...    |
|                                   |                                   |
|                                   | A nurse/nurse practitioner must   |
|                                   | have certain qualifications to    |
|                                   | register with Medicare and obtain |
|                                   | a Medicare provider number.       |
|                                   |                                   |
|                                   | If the nurse/nurse practitioner:  |

|                                   |     provider number, they can use |

|                                   |     the AIR or send vaccination   |
|                                   |     details                       |
|                                   | -   already has a Medicare        |
|                                   |     provider number but does not  |
|                                   |     know the details, go to Step  |
|                                   |     2                             |
|                                   | -   does not know if they have    |
|                                   |     the necessary qualifications  |
|                                   |     to register as a nurse/nurse  |
|                                   |     practitioner, go to Step 2    |
|                                   | -   knows they are not eligible   |
|                                   |     to register as a nurse/nurse  |
|                                   |     practitioner with Medicare,   |
|                                   |     go to Step 3                  |

|                                   | line + Read more ...              |

|                                   | be eligible to register as a      |
|                                   | Medicare provider or may already  |
|                                   | be registered. Transfer the call  |
|                                   | to the Medicare provider          |
|                                   | enquiries line for further        |
|                                   | assistance.                       |
|                                   |                                   |
|                                   | Procedure ends here.              |
+-----------------------------------+-----------------------------------+
| 3                                 | Refer caller to their state or    |
|                                   | territory health department +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | A nurse/nurse practitioner        |
|                                   | employed by an organisation that  |

|                                   | state or territory health         |
|                                   | department for further advice.    |

|                                   | Australian Immunisation Register  |
|                                   | (AIR).                            |

|                                   | This will display the Program     |
|                                   | Registration Create/Amend page.   |
+-----------------------------------+-----------------------------------+

|                                   | ...                               |

|                                   | On the Basic Search tab:          |
|                                   |                                   |
|                                   | -   key the recognised            |
|                                   |     vaccination provider’s stem   |

+-----------------------------------+-----------------------------------+
| 3                                 | Provider bank details + Read more |
|                                   | ...                               |

|                                   | Verbal advice from a vaccination  |

|                                   | sections.                         |
|                                   |                                   |
|                                   | Are bank details already          |
|                                   | recorded?                         |
|                                   |                                   |
|                                   | -   Yes, go to Step 4             |
|                                   | -   No, go to Step 6              |
+-----------------------------------+-----------------------------------+
| 4                                 | Provider bank details already     |
|                                   | recorded + Read more ...          |
|                                   |                                   |
|                                   | If the bank details on the form   |
|                                   | have been added previously and    |
|                                   | end dated, do not remove the end  |
|                                   | date. Create the bank details as  |
|                                   | a new entry and send for          |
|                                   | delegation.                       |
|                                   |                                   |
|                                   | If bank details are recorded in:  |
|                                   |                                   |
|                                   | Current payment instruction       |
|                                   |                                   |
|                                   | -   check the bank details are    |
|                                   |     correct                       |
|                                   | -   if correct, no further action |
|                                   |     required. Finalise the        |
|                                   |     document as per WLM           |
|                                   |     processing guidelines         |
|                                   |                                   |
|                                   | Future payment instruction, check |
|                                   | that:                             |
|                                   |                                   |
|                                   | -   there is an end date under    |
|                                   |     the Current Payment           |
|                                   |     Instruction                   |
|                                   | -   the bank details recorded in  |
|                                   |     Future payment instruction    |
|                                   |     are correct                   |
|                                   | -   the start date is after the   |
|                                   |     end date recorded under the   |
|                                   |     Current payment instruction   |
|                                   | -   finalise the document as per  |
|                                   |     WLM processing guidelines     |
|                                   |                                   |
|                                   | If the bank details recorded in   |
|                                   | Current Payment Instruction or    |
|                                   | Future Payment Instruction are    |
|                                   | not up to date, go to Step 5      |
+-----------------------------------+-----------------------------------+
| 5                                 | Provider bank details recorded +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | If there are bank details         |
|                                   | recorded:                         |
|                                   |                                   |
|                                   | go to Current payment instruction |
|                                   |                                   |
|                                   | from the drop down list, check    |
|                                   | for the bank details provided on  |
|                                   | the form. If the details are:     |

|                                   | -   not listed, go to Step 6      |
|                                   | -   listed, see Link provider     |
|                                   |     bank details table            |

|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Medicare provider bank details    |
|                                   | must not be updated without       |
|                                   | verbally confirming the request   |
|                                   | with the provider or the          |
|                                   | provider’s authorised             |
|                                   | representative.                   |
|                                   |                                   |
|                                   | Two genuine attempts must be made |
|                                   | to contact the provider or their  |
|                                   | authorised representative.        |
|                                   |                                   |
|                                   | Do not contact provider if        |
|                                   | processing HW027, HW029 or HW052  |
|                                   | forms as the bank details have    |
|                                   | been confirmed by Medicare.       |
|                                   |                                   |
|                                   | Attempt to contact the provider   |
|                                   | or the provider’s authorised      |
|                                   | representative using the contact  |
|                                   | details in PDS. Do not use the    |
|                                   | contacts details provided on the  |
|                                   | IM005 form                        |
|                                   |                                   |
|                                   | -   If unable to locate any       |
|                                   |     provider contact details in   |
|                                   |     PDS, enter an appropriate     |
|                                   |     comment in PaNDA (do not      |
|                                   |     escalate the query to the     |
|                                   |     helpdesk). For example,       |
|                                   |     genuine attempt to contact    |
|                                   |     the provider not possible as  |
|                                   |     unable to locate provider in  |
|                                   |     PDS. Unable to return the     |
|                                   |     form                          |
|                                   |                                   |
|                                   | Mark the work item as complete,   |
|                                   | see Work Optimiser for staff      |
|                                   |                                   |
|                                   | Procedure ends here               |
+-----------------------------------+-----------------------------------+
| 7                                 | First contact + Read more ...     |
|                                   |                                   |
|                                   | Make a genuine attempt to contact |
|                                   | the provider, or their authorised |
|                                   | representative. Confirm they have |
|                                   | sent a request to update their    |
|                                   | bank account details.             |
|                                   |                                   |
|                                   | Note: if the provider or their    |
|                                   | representative indicate they did  |
|                                   | not request an update to their    |
|                                   | bank details, report the          |
|                                   | suspected fraud.                  |
|                                   |                                   |
|                                   | Was contact successful and did    |
|                                   | the provider or their authorised  |
|                                   | representative confirm the update |
|                                   | request is genuine?               |
|                                   |                                   |
|                                   | Yes, put comments in PDS and      |
|                                   | PaNDA, see below. Then continue   |
|                                   | with the update, go to Step 11    |
|                                   |                                   |
|                                   | -   date and time of contact      |
|                                   | -   contact successful            |
|                                   | -   name of any person spoken to  |
|                                   | -   Outcome of conversation       |
|                                   |                                   |
|                                   | No, contact was unsuccessful, go  |
|                                   | to Step 8                         |
+-----------------------------------+-----------------------------------+

|                                   | Read more ...                     |
|                                   |                                   |
|                                   | If first contact was              |
