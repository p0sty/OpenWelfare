| 1                                 | Claims 20 week (100 PPL days)     |
|                                   | continuous payment - child born   |
|                                   | on or after 1 July 2023           |
|                                   |                                   |
|                                   | Jane is single and gives birth to |
|                                   | child Michael on 1 January 2024.  |
|                                   | Jane has lodged an effective      |
|                                   | early claim for PPL claim prior   |
|                                   | to Michael's birth and indicated  |
|                                   | Michael is being adopted out.     |
|                                   | Jane:                             |
|                                   |                                   |
|                                   | -   provides proof of birth for   |
|                                   |     Michael shortly after the     |
|                                   |     birth                         |
|                                   | -   confirms that Michaels birth  |
|                                   |     has been registered with      |
|                                   |     Births. Deaths and Marriages  |
|                                   | -   confirms that care has been   |
|                                   |     relinquished to Michael's     |
|                                   |     adoptive parents              |
|                                   | -   nominates the date of birth   |
|                                   |     as the first PPL day, and     |
|                                   | -   chooses to receive all 100    |
|                                   |     PPL days in a single block    |
|                                   |                                   |
|                                   | Jane is eligible for PPL from 1   |
|                                   | January 2024 to 17 May 2024.      |
|                                   |                                   |
|                                   | Jane would not be eligible for    |
|                                   | PPL for any period on or after 20 |
|                                   | May 2024, as this is more than 20 |
|                                   | weeks after the birth.            |
|                                   |                                   |
|                                   | Jane is eligible to receive 100   |
|                                   | PPL days as one continuous        |
|                                   | period.                           |
|                                   |                                   |
|                                   | If applicable, an employer        |
|                                   | determination will be made and    |
|                                   | PPL will be delivered via Jane’s  |
|                                   | employer                          |
+-----------------------------------+-----------------------------------+
| 2                                 | Claims 20 week continuous         |
|                                   | payment - child born on or after  |
|                                   | 1 July 2023                       |
|                                   |                                   |
|                                   | Kate is partnered and gives birth |
|                                   | to child Luke on 1 February 2024  |
|                                   | and relinquishes care to the      |
|                                   | commissioning parents on that     |
|                                   | day. Kate lodges an effective PPL |
|                                   | claim on 30 March 2024.           |
|                                   |                                   |
|                                   | Kate’s PPL can start from the     |
|                                   | date of birth of the child. PPL   |
|                                   | claims for children born 1 July   |
|                                   | 2023 or later can be backdated up |
|                                   | to 100 days and starts from the   |
|                                   | date the effective claim and      |
|                                   | supporting documents are lodged.  |
|                                   |                                   |
|                                   | Kate cannot be paid past 9 April  |
|                                   | 2024 under the relinquishing care |
|                                   | rules as this is more than 20     |
|                                   | weeks after the birth.            |
|                                   |                                   |
|                                   | Kate elects to claim 100 PPL days |
|                                   | as a continuous payment.          |
|                                   |                                   |
|                                   | Kate is paid 100 all PPL days     |
|                                   | regardless of being partnered at  |
|                                   | the time of claim due to being    |
|                                   | assessed under the relinquishing  |
|                                   | care provisions.                  |
|                                   |                                   |
|                                   | Kate is unable to share PPL days  |
|                                   | with her partner.                 |

| 3                                 | Claims 20 week continuous         |
|                                   | payment - child born on or after  |
|                                   | 1 July 2023                       |
|                                   |                                   |
|                                   | Maeve gives birth to child Benny  |
|                                   | on 1 February 2024 and            |
|                                   | relinquishes care to the adoptive |
|                                   | parents the following day.        |
|                                   |                                   |
|                                   | Maeve lodged an effective claim   |
|                                   | with all required documents and   |
|                                   | chose to receive PPL in a         |
|                                   | continuous block for 20 weeks. 5  |
|                                   | PPL days per week Mon-Fri.        |
|                                   |                                   |
|                                   | On 10 March, Maeve contacts to    |
|                                   | notify a day of upcoming work on  |
|                                   | Monday 11 March 2024. As a PPL    |
|                                   | day cannot be paid for a day      |
|                                   | where more than one hour of work  |
|                                   | is performed, Maeve nominates to  |
|                                   | change the PPL for 11 March 2024  |
|                                   | to the Saturday 16 March instead. |
|                                   |                                   |
|                                   | From 11 March Maeve is no longer  |
|                                   | receiving PPL in a continuous     |
|                                   | block (5 day week) as this block  |
|                                   | has been broken by the work day.  |
|                                   |                                   |
|                                   | Maeve will need to nominate the   |
|                                   | dates for the remaining PPL days. |
|                                   | This can be done by using the PPL |
|                                   | online service.                   |
|                                   |                                   |
|                                   | Maeve can still claim the full    |
|                                   | 100 days by accessing some PPL    |
|                                   | days on the weekend. However, PPL |
|                                   | days cannot be paid past 9 April  |
|                                   | 2024 under the relinquishing care |
|                                   | rules, as this is more than 20    |
|                                   | weeks after the birth.            |

| 4                                 | Partnered Birth mother sharing    |
|                                   | PPL - child born on or after 1    |
|                                   | July 2023                         |
|                                   |                                   |
|                                   | Sarah and Sam are partnered.      |
|                                   | Sarah is a surrogate birth mother |
|                                   | for a family member.              |
|                                   |                                   |
|                                   | Sarah gives birth to child Max on |
|                                   | 1 February 2024.                  |
|                                   |                                   |
|                                   | As part of the surrogacy          |
|                                   | agreement Sarah and Sam are       |
|                                   | caring for the child for 14 days  |
|                                   | before they relinquish care of    |
|                                   | the child to the commissioning    |
|                                   | parents.                          |
|                                   |                                   |
|                                   | Sarah contacts to discuss whether |
|                                   | they can claim and share PPL days |
|                                   | during this period as both Sarah  |
|                                   | and her partner Sam will be       |
|                                   | taking time off work to care for  |
|                                   | the child.                        |
|                                   |                                   |
|                                   | As Sarah and Sam had not          |
|                                   | relinquished care of the child    |
|                                   | during the 14 day period and      |
|                                   | would both be meeting the care    |
|                                   | requirements for the child, they  |
|                                   | may be able to claim and share    |
|                                   | PPL if they meet all the          |
|                                   | eligibility requirements.         |
|                                   | However, once care of the child   |
|                                   | is relinquished, only Sarah as    |
|                                   | the birth mother would be able to |
|                                   | continue to receive any further   |
|                                   | PPL days, and only in the 20-week |
|                                   | period immediately following the  |
|                                   | birth.                            |

|                                   |     25 April 2023                 |
|                                   | -   Connected period 26 April     |
|                                   |     2023 to 6 June 2023           |

|                                   | Jamie is eligible to receive 18   |
|                                   | weeks PPL as one continuous       |
|                                   | period.                           |
