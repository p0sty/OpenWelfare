Crisis Payment scenarios

IM extension for VWPR IM (Crisis Payment) customers

This table describes scenarios when IM may be extended for VWPR IM
(Crisis Payment) customers.

+-----------------------------------+-----------------------------------+
| Item                              | Scenario                          |
+-----------------------------------+-----------------------------------+
| 1                                 | Customer moves out of declared IM |
|                                   | area                              |
|                                   |                                   |
|                                   | If a customer moves out of a      |
|                                   | declared IM area while their      |
|                                   | 12-month VWPR IM period is        |
|                                   | active, they will remain on VWPR  |
|                                   | IM as per the existing            |
|                                   | portability rules.                |
|                                   |                                   |
|                                   | If, within the first 13 weeks of  |
|                                   | the customer's VWPR IM period,    |
|                                   | they receive a second, or         |
|                                   | subsequent Crisis Payment because |
|                                   | of release from prison or         |
|                                   | psychiatric confinement, the      |
|                                   | customer's VWPR IM end date will  |
|                                   | be extended to 12 months from the |
|                                   | date of their most recent Crisis  |
|                                   | Payment.                          |
|                                   |                                   |
|                                   | Note: this rule can also apply to |
|                                   | customers who no longer reside in |
|                                   | a declared IM area.               |
+-----------------------------------+-----------------------------------+
| 2                                 | Automatic restoration of IM       |
|                                   |                                   |
|                                   | If, at the time that IM is        |
|                                   | reinstated, the customer had been |
|                                   | granted a second, or subsequent   |
|                                   | Crisis Payment because of release |
|                                   | from prison or psychiatric        |
|                                   | confinement within the last 13    |
|                                   | weeks, the customer’s VWPR IM end |
|                                   | date will be extended to 12       |
|                                   | months from the date of their     |
|                                   | most recent Crisis Payment.       |
|                                   |                                   |
|                                   | However, if a customer's trigger  |
|                                   | payment for IM is cancelled, and  |
|                                   | subsequently restored within 13   |
|                                   | weeks, the customer will then     |
|                                   | commence enhanced IM as a new     |
|                                   | customer..                        |
+-----------------------------------+-----------------------------------+
