|                                   |     alternatively, they may have  |
|                                   |     an exemption from the NARWP   |
|                                   |     in certain circumstances      |
|                                   |                                   |
|                                   | The length of the NARWP changed   |
|                                   | from 1 January 2019 and is        |
|                                   | determined by the grant date of   |
|                                   | the customer's first qualifying   |
|                                   | residence visa, SCV or specified  |
|                                   | visa subclass that allows         |

|                                   |                                   |

|                                   | (104 week) NARWP if:              |

|                                   |     or 837, or                    |
|                                   | -   they hold a temporary partner |
|                                   |     visa (subclasses 309 and      |
|                                   |     820), or                      |
|                                   | -   their first permanent         |
|                                   |     residence visa was granted    |
|                                   |     before 1 January 2019         |
|                                   |                                   |
|                                   | Otherwise, the customer is        |
|                                   | required to serve a 4 year (208   |
|                                   | week) NARWP                       |
|                                   |                                   |
|                                   | Note: if the holder of a SCV or   |
|                                   | temporary partner visa            |
|                                   | (subclasses 309 and 820) is       |
|                                   | granted a permanent visa on or    |
|                                   | after 1 January 2019, they will   |
|                                   | become subject to a 4 year (208   |
|                                   | week) NARWP for the LIC. The      |
|                                   | calculation of the NARWP still    |
|                                   | commences from the application    |
|                                   | date for temporary partner visa   |
|                                   | (subclasses 309 and 820), and     |
|                                   | does not restart upon             |
|                                   | commencement of the permanent     |
|                                   | visa.                             |
|                                   |                                   |
|                                   | Residence requirements for FST    |
|                                   |                                   |
|                                   | To qualify for an FST the         |
|                                   | claimant must:                    |
|                                   |                                   |
|                                   | -   be an Australian resident, or |
|                                   | -   the holder of an SCV residing |
|                                   |     in Australia                  |
|                                   |                                   |
|                                   | For residence requirements that   |
|                                   | apply to the FST foster child,    |
|                                   | see Health Care Cards (HCC) for   |
|                                   | foster children.                  |
|                                   |                                   |
|                                   | Residence requirements for EHC    |
|                                   |                                   |
|                                   | To qualify for an EHC the         |
|                                   | claimant must:                    |
|                                   |                                   |
|                                   | -   be an Australian resident,    |
|                                   |     and                           |
|                                   | -   satisfy a 2 year (104 week)   |
|                                   |     or 4 year (208 week) NARWP or |
|                                   |     alternatively, they may have  |
|                                   |     an exemption from the NARWP   |
|                                   |     in certain circumstances      |
|                                   |                                   |
|                                   | The length of the NARWP changed   |
|                                   | from 1 January 2019 and is        |
|                                   | determined by the grant date of   |
|                                   | the customer's first permanent    |
|                                   | residence visa.                   |
|                                   |                                   |
|                                   | The customer will have a 2 year   |
|                                   | (104 week) NARWP if:              |
|                                   |                                   |
|                                   | -   their first permanent         |
|                                   |     residence visa was granted    |
|                                   |     before 1 January 2019, or     |
|                                   | -   they hold or have held a visa |
|                                   |     subclass 115, 117, 835 or 837 |
|                                   |                                   |
|                                   | Otherwise, the customer is        |
|                                   | required to serve a 4 year (208   |
|                                   | week) NARWP                       |
|                                   |                                   |
|                                   | To remain qualified for the       |
|                                   | concession card, the customer     |
|                                   | must also remain an Australian    |
|                                   | resident, the holder of an SCV    |
|                                   | residing in Australia or the      |
|                                   | holder of an SpB eligible         |
|                                   | temporary visa, as per the        |
|                                   | residence requirements specific   |
|                                   | to the card above.                |
|                                   |                                   |
|                                   | The new claim workflow will       |
|                                   | present the Immigration Enquiry   |
|                                   | (RSIMME) screen. Ensure the       |
|                                   | customer's country of birth and   |
|                                   | any travel documents are          |
|                                   | recorded. The system will         |
|                                   | automatically link to the         |
|                                   | Department of Home Affairs        |
|                                   | information and update the        |
|                                   | customer's residence and movement |
|                                   | information. If more information  |
|                                   | is required, see Activating the   |
|                                   | Department of Home Affairs        |
|                                   | datalink and contingency          |
|                                   | procedures if datalink is         |
|                                   | unavailable.                      |
