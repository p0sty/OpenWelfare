Feedback

Use the Forms, Letters and Electronic Messaging Feedback webform to:

-   send feedback about layout, format, readability or spelling errors
-   suggest improvements to letters, forms or messages

Use mySupport to report issues with letters about incorrect content. For
example, wrong information or 'Text Not Found', non-enabled History IDs,
or the incorrect print status code is displaying:

-   search for the Enterprise Outbound Correspondence (EOC) issue
    webform or use key words (such as SMS, email, messages, letters,
    communications)
-   complete and submit the form. Make a note of the receipt number or
    SD reference number
