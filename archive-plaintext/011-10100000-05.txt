| 1                                 | Determine claim lodgement method  |
|                                   |                                   |
|                                   | In AIR Mainframe:                 |
|                                   |                                   |

|                                   | -   Key the provider number in    |

|                                   |     select [Enter]                |

|                                   | The Claim ID column provides      |
|                                   | information regarding the         |
|                                   | lodgement channel used (and       |
|                                   | therefore available) for each     |

|                                   |     Software                      |
|                                   | -   Claim ID with a $ means       |
|                                   |     provider has lodged data      |
|                                   |     using Practice Management     |
|                                   |     Software that supports web    |
|                                   |     services                      |
|                                   | -   Claim ID with a * or no       |
|                                   |     symbol mean a claim that has  |
|                                   |     been keyed by an AIR Service  |
|                                   |     Officer (manual)              |
