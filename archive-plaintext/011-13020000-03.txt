This page contains information and procedures for staff handling
enquiries relating to the External Breast Prostheses Reimbursement
Program (EBPRP).

Manage claim lodgement and payment delay enquiries

Table 3: This table describes how to respond to payment delays or
confirmation that the EBPRP claim has been received.
Note: Smart Centre and service centre staff do not have access to the
EBPRP system and must complete steps 1-4 below before transferring the
call to the EBPRP Processing team.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Obtain Medicare card number +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Obtain the customer’s Medicare    |
|                                   | card number, and:                 |
|                                   |                                   |
|                                   | -   Proof of Identity             |
|                                   | -   Amend/update customer details |
|                                   |     in CDMS                       |
+-----------------------------------+-----------------------------------+
| 2                                 | Check Consumer Directory          |
|                                   | Maintenance System (CDMS) + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | Check:                            |
|                                   |                                   |
|                                   | -   the Personal Comments field   |
|                                   |     in CDMS and locate the EBPRP  |
|                                   |     receiving/processing comment  |
|                                   | -   customer restrictions in CDMS |
|                                   |     when verifying the Medicare   |
|                                   |     card                          |
|                                   |                                   |
|                                   | Follow the appropriate procedures |
|                                   | with the message received.        |
+-----------------------------------+-----------------------------------+
| 3                                 | Check receipt of claim details +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | If receipt of claim details are:  |
|                                   |                                   |
|                                   | -   Present, advise customer if   |
|                                   |     the claim has been processed  |
|                                   | -   If no comment or any other    |
|                                   |     comments have been left for a |
|                                   |     rejection or discretionary    |
|                                   |     request, warm transfer the    |
|                                   |     call to the EBPRP Processing  |
|                                   |     team. Note: do not discuss    |
|                                   |     rejections or discretionary   |
|                                   |     request with customer         |
|                                   | -   Not present, go to Step 4     |
+-----------------------------------+-----------------------------------+
| 4                                 | Request further information +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | If the claim has not been         |
|                                   | receipted in the Personal         |
|                                   | Comments field in CDMS, ask the   |
|                                   | customer:                         |
|                                   |                                   |
|                                   | -   the approximate date of       |
|                                   |     lodgement                     |
|                                   | -   whether the claim was lodged  |
|                                   |     at a service centre (and if   |
|                                   |     so, which office), by mail or |
|                                   |     via email                     |
|                                   | -   the preferred contact number  |
|                                   |                                   |
|                                   | Note: the preferred telephone     |
|                                   | contact number might differ from  |
|                                   | the details recorded in CDMS.     |
+-----------------------------------+-----------------------------------+
| 5                                 | Contact EBPRP Processing team +   |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Contact EBPRP Processing team and |
|                                   | provide the above details.        |
+-----------------------------------+-----------------------------------+
| 6                                 | Follow instructions from EBPRP    |
|                                   | Processing team + Read more ...   |
|                                   |                                   |
|                                   | Follow instructions given by the  |
|                                   | EBPRP Processing team Service     |
|                                   | Officer.                          |
+-----------------------------------+-----------------------------------+

Financial hardship or requests for urgent payments

Table 5: This table describes how Service Officers manage financial
hardship or urgent payment requests.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |

| 1                                 | Handling the enquiry + Read more  |

|                                   | circumstances. Warm transfer the  |

|                                   | team so they can evaluate the     |
|                                   | claim.                            |
|                                   |                                   |
|                                   | For:                              |

|                                   | EBPRP Processing team, go to Step |
|                                   | 5                                 |

|                                   | other Service Officers accepting  |

|                                   |                                   |
|                                   | Advise the customer their claim   |
|                                   | may be processed urgently and     |
|                                   | paid via EFT into their bank      |
|                                   | account.                          |
|                                   |                                   |
|                                   | Procedure ends here for phone     |
|                                   | enquiries.                        |
+-----------------------------------+-----------------------------------+
| 3                                 | Enquiries in person + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | Tell the customer that:           |
|                                   |                                   |
|                                   | -   their claim may be processed  |

|                                   |     their bank account            |

|                                   |                                   |

|                                   | PaNDA using the following         |

|                                   | claim + Read more ...             |
|                                   |                                   |
|                                   | This step is for EBPRP Processing |
|                                   | team only.                        |
|                                   |                                   |
|                                   | Process the claim on the same day |
|                                   | of receipt of the emailed         |
|                                   | documents from the service        |
|                                   | centre.                           |

Complaints about EBPRP

Table 7: This table describes how the EBPRP Processing team respond to
complaints received about EBPRP.

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Service Officer receives a        |
|                                   | complaint via phone + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | Service Officers are to warm      |
|                                   | transfer all complaints to the    |
|                                   | EBPRP Processing team with the    |
|                                   | following information:            |
|                                   |                                   |
|                                   | -   ensure Service Officer has    |
|                                   |     identified the customer and   |
|                                   |     completed house keeping       |
|                                   | -   customer's contact details    |
|                                   |     (daytime phone and email)     |
|                                   | -   Medicare card number          |
|                                   | -   time and date of call         |
|                                   | -   full details of the complaint |
+-----------------------------------+-----------------------------------+
| 2                                 | EBPRP Processing team action +    |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | If a customer wants to lodge a    |
|                                   | formal complaint:                 |
|                                   |                                   |
|                                   | -   the EBPRP Processing team are |
|                                   |     to register the complaint in  |
|                                   |     the Customer First Tool, as   |
|                                   |     per the instructions in       |
|                                   |     Managing customer complaints  |
|                                   |     and feedback, or              |
|                                   | -   Service Officers are to       |
|                                   |     direct them to the Services   |
|                                   |     Australia website where they  |
|                                   |     can record the complaint      |
+-----------------------------------+-----------------------------------+
| 3                                 | EBPRP Processing team + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | If the complaint:                 |
|                                   |                                   |
|                                   | -   has been recorded through     |
|                                   |     complaints and feedback, it   |
|                                   |     will be allocated by the      |
|                                   |     Feedback and Complaints team  |
|                                   |     to the appropriate area       |
|                                   | -   relates to a policy decision, |
|                                   |     escalate to the Program       |
|                                   |     Management team               |
|                                   | -   relates to an operational     |
|                                   |     matters, direct it to the     |
|                                   |     EBPRP team leader             |
|                                   |                                   |
|                                   | Apply internal procedures to      |
|                                   | resolve the complaint.            |
