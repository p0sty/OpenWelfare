The Customer Management Approach (CMA) provides a core service offer
through a set of common work activities that all staff must complete. It
also defines the areas that undertake more specialised activities and
areas considered enabling areas or exceptions to the CMA.

Exceptions to the CMA process

Table 3

+-----------------------------------+-----------------------------------+
| Item                              | Exception area                    |
+-----------------------------------+-----------------------------------+
| 1                                 | Complaints, External Complaints - |
|                                   | Child Support (ExCCS) + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | -   Level 2 Complaints and ExCCS  |
|                                   |     facilitate resolution of      |
|                                   |     customer complaints and other |
|                                   |     escalated issues for the      |
|                                   |     Minister, Government and the  |
|                                   |     Australian Community in       |
|                                   |     consultation with team        |
|                                   |     leaders and Service Officers. |
|                                   |     These will generally not lock |
|                                   |     customers                     |
|                                   | -   Level 2 Complaints and ExCCS  |
|                                   |     officers will use discretion  |
|                                   |     to determine the best way to  |
|                                   |     resolve a customer complaint. |
|                                   |     If an officer takes ownership |
|                                   |     of an issue, the officer will |
|                                   |     lock the customer and         |
|                                   |     complete all common           |
|                                   |     activities, including Minimum |
|                                   |     Collection Activity           |
+-----------------------------------+-----------------------------------+
| 2                                 | Decision makers, General and Part |
|                                   | 6A Objections + Read more ...     |
|                                   |                                   |
|                                   | -   Decision makers consider      |
|                                   |     Change of assessment (COA)    |
|                                   |     applications and decide if    |
|                                   |     the assessment must change.   |
|                                   |     The decision making process   |
|                                   |     involves considering all      |
|                                   |     circumstances of the case to  |
|                                   |     determine if a change should  |
|                                   |     be made                       |
|                                   | -   Decision makers lock          |
|                                   |     customers for the duration of |
|                                   |     finalising a COA application, |
|                                   |     with the exception of         |
|                                   |     customers already locked to   |
|                                   |     specialised areas. All        |
|                                   |     internal decision makers will |
|                                   |     undertake at least Minimum    |
|                                   |     Collection Activity and refer |
|                                   |     other common or specialised   |
|                                   |     activities to either          |
|                                   |     Mainstream Services Teams or  |
|                                   |     the owning business area for  |
|                                   |     resolution. Contracted        |
|                                   |     decision makers are exempt    |
|                                   |     from minimum collection       |
|                                   |     activity                      |
|                                   | -   Objections Officers provide a |
|                                   |     formal full merit review for  |
|                                   |     customers who believe Child   |
|                                   |     Support have made a mistake   |
|                                   |     or applied the law            |
|                                   |     incorrectly                   |
|                                   | -   the principle of independence |
|                                   |     from being an 'original       |
|                                   |     decision maker' is            |
|                                   |     fundamental to providing a    |
|                                   |     fair and impartial review     |
|                                   |     process                       |
|                                   | -   General Objections Officers   |
|                                   |     and Part 6A Objections        |
|                                   |     Officers will lock customers. |
|                                   |     All Objections Officers will  |
|                                   |     undertake at least Minimum    |
|                                   |     Collection Activity and refer |
|                                   |     other common or specialised   |
|                                   |     activities to either          |
|                                   |     Mainstream Services Teams or  |
|                                   |     the owning business area for  |
|                                   |     resolution                    |
|                                   |                                   |
|                                   | Note: full Customer Management    |
|                                   | Approach (CMA) process applies to |
|                                   | general enquiry telephony         |
|                                   | contribution. Change of           |
|                                   | Assessment Internal Decision      |
|                                   | Makers, General Objection         |
|                                   | Officers and Part 6A Objection    |
|                                   | Officers will undertake full CMA  |
|                                   | action and retain relevant        |
|                                   | activities for work generated     |
|                                   | from these telephony shifts.      |
|                                   |                                   |
|                                   | The exception where CoA and       |
|                                   | Objection staff undertake minimum |
|                                   | collection only, applies to       |
|                                   | Change of Assessment or           |
|                                   | Objections customers where there  |
|                                   | is an active                      |
|                                   | application/objection.            |
+-----------------------------------+-----------------------------------+
| 3                                 | Integrated Care + Read more ...   |
|                                   |                                   |
|                                   | Integrated Care teams complete    |
|                                   | change in care decisions for      |
|                                   | Services Australia                |
|                                   |                                   |
|                                   | Integrated Care teams will:       |

|                                   | -   lock customers in line with   |
|                                   |     existing instructions in the  |

|                                   |     Activity and common           |
|                                   |     activities while a care       |
