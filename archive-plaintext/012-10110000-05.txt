Teaching Payment FAQs

+-----------------------------------+-----------------------------------+
| Item                              | Description                       |
+-----------------------------------+-----------------------------------+
| 1                                 | Question: if the university       |
|                                   | representative’s certification    |
|                                   | date is in the middle of or after |
|                                   | all date of sessions, is this     |
|                                   | acceptable?                       |
|                                   |                                   |
|                                   | Answer: escalate to the Local     |
|                                   | Peer Support (LPS) for            |
|                                   | assessment.                       |
+-----------------------------------+-----------------------------------+
| 2                                 | Question: if a practice submits a |
|                                   | teaching claim the IP006.2011     |
|                                   | version of the form, is this      |
|                                   | acceptable?                       |
|                                   |                                   |
|                                   | Answer: yes, because the          |
|                                   | information contained in the form |
|                                   | is consistent with the current    |
|                                   | version.                          |
+-----------------------------------+-----------------------------------+
| 3                                 | Question: can we process a claim  |
|                                   | if the university is not listed   |
|                                   | in PIP online?                    |
|                                   |                                   |
|                                   | Answer: escalate to the LPS for   |
|                                   | assessment.                       |
+-----------------------------------+-----------------------------------+
| 4                                 | Question: can a practice claim    |
|                                   | sessions prior to 1 January 2015? |
|                                   |                                   |
|                                   | Answer: no, claims prior to 1     |
|                                   | January 2015 are not eligible for |
|                                   | payment.                          |
+-----------------------------------+-----------------------------------+
| 5                                 | Question: can a practice claim    |
|                                   | sessions for an intern or         |
|                                   | overseas student?                 |
|                                   |                                   |
|                                   | Answer: no, teaching sessions for |
|                                   | interns or overseas students do   |
|                                   | not meet the eligibility          |
|                                   | criteria.                         |
+-----------------------------------+-----------------------------------+
| 6                                 | Question: can a practice claim    |
|                                   | for a student studying a course   |
|                                   | at an Australian university that  |
|                                   | is preparing them for entry into  |
|                                   | the Australian medical            |
|                                   | profession, but the session is    |
|                                   | part of the student’s non-core    |
|                                   | curriculum, such as an elective   |
|                                   | GP placement?                     |
|                                   |                                   |
|                                   | Answer: escalate to the LPS for   |
|                                   | assessment:                       |
|                                   |                                   |
|                                   | -   Program Management (policy    |
|                                   |     team) will assess these on a  |
|                                   |     case by cases basis           |
|                                   | -   The practice must submit all  |
|                                   |     of the details and any        |
|                                   |     supporting documents          |
+-----------------------------------+-----------------------------------+
| 7                                 | Question: can a practice claim    |
|                                   | for a student undertaking their   |
|                                   | PhD or course OTHER than an       |
|                                   | undergraduate or graduate course? |
|                                   |                                   |
|                                   | Answer: no, a student undertaking |
|                                   | a PhD course attending a teaching |
|                                   | session is not eligible to claim  |
|                                   | a teaching payment.               |
+-----------------------------------+-----------------------------------+
| 8                                 | Question: can a practice claim    |
|                                   | more than 2 three-hour sessions,  |
|                                   | per GP daily?                     |
|                                   |                                   |
|                                   | Answer: no, a practice can only   |
|                                   | claim a maximum of 2 sessions,    |
|                                   | per GP.                           |
|                                   |                                   |
|                                   | If a practice submits a claim     |
|                                   | with more than 2 sessions signed  |
|                                   | by a GP for one date:             |
|                                   |                                   |
|                                   | -   process the 2 sessions, and   |
|                                   | -   send a letter to the practice |
|                                   |     advising only 2 claims can be |
|                                   |     processed                     |
+-----------------------------------+-----------------------------------+
| 9                                 | Question: can a practice claim    |
|                                   | more than 2 three-hour sessions   |
|                                   | per student daily?                |
|                                   |                                   |
|                                   | Answer: yes, the number of        |
|                                   | sessions the practice can claim   |
|                                   | per student is not limited.       |
+-----------------------------------+-----------------------------------+
