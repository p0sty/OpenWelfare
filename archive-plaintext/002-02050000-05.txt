Historical asset tests

From 1 September 2018 to 11 June 2020, the farm asset test threshold was
$5 million. The FHA assets test for this period included 2 components.
Even if a customer qualified for FHA, it was only payable if both of the
following assets tests were satisfied:

-   Non-farm Assets Test (Tier 1) included the value of all non-farm and
    liquid assets. Net values of each asset were used. The net value of
    a non-farm asset is calculated by deducting the value of any charge
    or encumbrance over that asset from the gross asset value. The
    thresholds are the same as the allowance assets thresholds
-   Net Farm Assets Test (Tier 2) included the net value of all farm
    assets. Calculate the net value of all farm assets by adding up the
    value of all liabilities related to farm assets and deducting that
    amount from the total value of all farm assets. The value of Tier 2
    assets could not exceed the FHA farm assets value limit

|                                   | by multiple assets                |
|                                   |                                   |
|                                   | A farmer has a loan of $1.8       |
|                                   | million secured jointly against:  |
|                                   |                                   |
|                                   | -   farmland with a gross value   |
|                                   |     of $1.85 million              |
|                                   | -   a water entitlement valued at |
|                                   |     $700,000, and                 |
|                                   | -   a holiday home valued at      |
|                                   |     $150,000                      |
|                                   |                                   |
|                                   | The combined total value of the   |
|                                   | security assets is $2.7 million.  |

|                                   | 700,000 + $150,000 = $2.7         |
|                                   | million)                          |
|                                   |                                   |
|                                   | To determine how much of the loan |
|                                   | relates to each asset, divide the |
|                                   | amount of the loan by the value   |
|                                   | of the secured assets.            |
|                                   |                                   |
|                                   | (That is, $1.8 million in         |
|                                   | loans/$2.7 million in secured     |
|                                   | assets = 0.6667).                 |
|                                   |                                   |
|                                   | Hence, approximately 67% of the   |
|                                   | value of each asset is held as    |
|                                   | security for the loan. The value  |
|                                   | of each asset will be reduced by  |
|                                   | the amount of the loan that       |
|                                   | relates to that asset. In this    |
|                                   | example, the gross value of each  |
|                                   | asset will be reduced by          |
|                                   | approximately 67%.                |
|                                   |                                   |
|                                   | Use the following formula to find |
|                                   | the net value of the asset;       |
|                                   |                                   |
|                                   | Gross asset value - (proportion   |
|                                   | of the loan that relates to the   |
|                                   | asset X asset value)              |
|                                   |                                   |
|                                   | Note: use 4 decimal places        |
|                                   |                                   |
|                                   | So the net values of the assets   |
|                                   | in this example are:              |
|                                   |                                   |
|                                   | Farm land: $1.85 million -        |
|                                   | (0.67*$1.85 million) = $610,500   |
|                                   |                                   |
|                                   | Water Asset: $700,000 -           |
|                                   | (0.67*$700,000) = $231,000        |
|                                   |                                   |
|                                   | Holiday Home $150,000 - (0.67 *   |
|                                   | $150,000) = $49,500               |

|                                   |                                   |
|                                   | If the ability to sell or trade a |

|                                   |                                   |
|                                   | To ensure unintended impacts are  |

Coding claims lodged before and after 5 April 2017

Where a customer and partner have both lodged a claim for Farm Household
Allowance (FHA) and 1 claim is lodged before 5 April 2017 while the
other is lodged on or after 5 April 2017, the customer and partner are
assessed under the assets test applicable to the date of lodgement of
their claims. This means there will be 2 different assets coding applied
to the record.

Coding for shares - claims lodged before and after 5 April 2017

Where a customer and partner have both lodged a claim for Farm Household
Allowance (FHA) and one claim is lodged before 5 April 2017 while the
other is lodged on or after 5 April 2017, the customer and partner are
assessed under the assets test applicable to the date of lodgement of
their claims. This means there will be 2 different assets coding applied
to the record.
