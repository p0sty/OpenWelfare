This page contains the procedures on how to register to participate in
the My Health Record system and how to confirm Responsible Officer (RO)
or Organisation Maintenance Officer (OMO) credentials.

Register for the My Health Record system in HPOS

Table 1: This table describes how Service Officers help ROs and Seed
OMOs register to participate in the My Health Record system through
Health Professional Online Services (HPOS).

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Navigate to Health Professional   |
|                                   | screen + Read more ...            |
|                                   |                                   |
|                                   | Go to the Services Australia      |
|                                   | website. The Resources page       |
|                                   | contains a link to the website.   |
|                                   |                                   |
|                                   | Select the Health professionals   |
|                                   | link.                             |
|                                   |                                   |
|                                   | The For health professionals      |
|                                   | screen is displayed.              |
+-----------------------------------+-----------------------------------+
| 2                                 | Generate Prepare to Register      |
|                                   | screen + Read more ...            |
|                                   |                                   |
|                                   | -   Select the HPOS LOGON button  |
|                                   |     on the top right of the       |
|                                   |     screen. The HPOS home page is |
|                                   |     displayed                     |
|                                   | -   Select the My Health Record   |
|                                   |     Registration link from the    |
|                                   |     left side menu. The Prepare   |
|                                   |     to Register screen is         |
|                                   |     displayed with a link to the  |
|                                   |     My Health Record website for  |
|                                   |     more information for health   |
|                                   |     professionals                 |
|                                   |                                   |
|                                   | Note: this screen lists all seed  |
|                                   | organisations representing the    |
|                                   | hierarchies that the RO/Seed OMO  |
|                                   | can act on behalf of.             |
+-----------------------------------+-----------------------------------+
| 3                                 | View network map and set access   |
|                                   | flags + Read more ...             |
|                                   |                                   |
|                                   | Select Seed HPI-O number or Seed  |
|                                   | Organisation name.                |
|                                   |                                   |
|                                   | The Healthcare Provider           |
|                                   | Organisation Registration screen  |
|                                   | is displayed with a copy of the   |
|                                   | network map for that              |
|                                   | organisation.                     |
|                                   |                                   |
|                                   | Set all access flags to yes or    |
|                                   | no.                               |
|                                   |                                   |
|                                   | Note:                             |
|                                   |                                   |
|                                   | -   Access flag for Seed          |
|                                   |     Organisation is always set to |
|                                   |     yes                           |
|                                   | -   For information on access     |
|                                   |     flags, visit the Registration |
|                                   |     overview page on the My       |
|                                   |     Health Record website (see    |
|                                   |     Resources for link). The      |
|                                   |     Resources page also contains  |
|                                   |     a link to the Australian      |
|                                   |     Digital Health Agency website |
+-----------------------------------+-----------------------------------+
| 4                                 | Save or cancel registration       |
|                                   | changes + Read more ...           |
|                                   |                                   |
|                                   | Save or cancel registration       |
|                                   | changes. If changes are:          |
|                                   |                                   |
|                                   | -   cancelled, RO/OMO is returned |
|                                   |     to main HPOS screen           |
|                                   | -   saved, the Registration       |
|                                   |     Confirmation screen is        |
|                                   |     displayed                     |
|                                   | -   reset, the registration       |
|                                   |     screen reverts to the status  |
|                                   |     of the session when it was    |
|                                   |     accessed                      |
+-----------------------------------+-----------------------------------+
| 5                                 | Generate My Health Record system  |
|                                   | Registered Organisations screen + |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Select Manage Authorisation Links |
|                                   | on the HPOS home page main menu   |
|                                   | on the left side or on the        |
|                                   | Healthcare Provider Organisation  |
|                                   | Registration screen.              |
|                                   |                                   |
|                                   | The My Health Record system       |
|                                   | Registered Organisations screen   |
|                                   | is displayed.                     |
|                                   |                                   |
|                                   | Note: this screen lists all the   |
|                                   | organisations (seed or Network)   |
|                                   | that the RO/OMO can act for to    |
|                                   | maintain authorisation links.     |
+-----------------------------------+-----------------------------------+
| 6                                 | Generate Manage Authorisation     |
|                                   | Links screen + Read more ...      |
|                                   |                                   |
|                                   | Select Healthcare Provider        |
|                                   | Identifier - Organisation (HPI-O) |
|                                   | number or Organisation name.      |
|                                   |                                   |
|                                   | The Manage Authorisation Links    |
|                                   | screen is displayed with a list   |
|                                   | of provider individuals who are   |
|                                   | already linked, if any exist.     |
+-----------------------------------+-----------------------------------+
| 7                                 | Enter each Healthcare Provider    |
|                                   | Identifier - Individual (HPI-I) + |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Enter each individual healthcare  |
|                                   | professional’s HPI-I separately   |
|                                   | by:                               |
|                                   |                                   |
|                                   | -   keying HPI-I in the Enter     |
|                                   |     HPI-I Number field            |
|                                   | -   selecting Search              |
+-----------------------------------+-----------------------------------+
| 8                                 | Confirm authorisation links +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Select Confirm in pop up window   |
|                                   | to confirm authorisation links.   |
|                                   |                                   |
|                                   | Note: selecting Cancel will not   |
|                                   | save changes and RO/OMO is        |
|                                   | returned to the Manage            |
|                                   | Authorisation links screen.       |
|                                   |                                   |
|                                   | Deactivate authorisation links    |
|                                   |                                   |
|                                   | To deactivate authorisation       |
|                                   | links, select Deactivate          |
|                                   | Authorisation Links.              |
+-----------------------------------+-----------------------------------+

Confirm RO or OMO credentials in the PPR

Table 2: This table describes how Service Officers search for a
Healthcare Provider Identifier - Organisation (HPI-O) in the Provider
Participation Register (PPR)and request details for the RO or OMO when
the details in the PPR do not match the Application to Register a Seed
Organisation (HW018 formerly 2978) or Application to establish list of
authorised healthcare provider individuals (HA0916).

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Access the PPR + Read more ...    |
|                                   |                                   |
|                                   | Complete Access the Provider      |
|                                   | Participation Register (PPR) in   |
|                                   | Provider Participation Register   |
|                                   | (PPR).                            |
+-----------------------------------+-----------------------------------+
| 2                                 | Search Organisations + Read more  |
|                                   | ...                               |
|                                   |                                   |
|                                   | Under Navigation menu options,    |
|                                   | select Search Organisations       |
|                                   |                                   |
|                                   | Is the HPI-O number on the        |
|                                   | application form that is being    |
|                                   | processed?                        |
|                                   |                                   |
|                                   | Yes, key the HPI-O number into    |
|                                   | the Search bar                    |
|                                   |                                   |
|                                   | -   select Search                 |
|                                   | -   go to Step 3                  |
|                                   |                                   |
|                                   | No, key the Organisation Name     |
|                                   | from the form into the Search bar |
|                                   |                                   |
|                                   | -   select Search                 |
|                                   | -   go to Step 3                  |
+-----------------------------------+-----------------------------------+
| 3                                 | Match address details + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | Matched Records table in the      |
|                                   | Search Organisations screen.      |
|                                   |                                   |
|                                   | Do the online address details     |
|                                   | match the form address details?   |
|                                   |                                   |
|                                   | -   Yes, if HPI-O number is not   |
|                                   |     on the electronic application |
|                                   |     form, write the HPI-O number  |
|                                   |     in a speech bubble in the     |
|                                   |     electronic application form.  |
|                                   |     Go to Step 4                  |
|                                   | -   No, return application to     |
|                                   |     Subject Matter Expert (SME)   |
|                                   |     for action. Procedure ends    |
|                                   |     here.                         |
+-----------------------------------+-----------------------------------+
| 4                                 | View HPI-O details + Read more    |
|                                   | ...                               |
|                                   |                                   |
|                                   | View the correct HPI-O details in |
|                                   | the Search Organisations screen.  |
|                                   |                                   |
|                                   | Select the corresponding          |
|                                   | Registrations or Authorisation    |
|                                   | Links button.                     |
|                                   |                                   |
|                                   | Select Officer screen is          |
|                                   | displayed. If more than one       |
|                                   | RO/OMO exists, a list is          |
|                                   | displayed.                        |
+-----------------------------------+-----------------------------------+
| 5                                 | Check RO/OMO details against      |
|                                   | application form + Read more ...  |
|                                   |                                   |
|                                   | Do the RO/OMO details in Select   |
|                                   | Officer screen match the          |
|                                   | application form details?         |
|                                   |                                   |
|                                   | Yes, select radio button for the  |
|                                   | RO/OMO and select Continue.       |
|                                   | Procedure ends here               |
|                                   |                                   |
|                                   | No, if the applicant has applied  |
|                                   | as the:                           |
|                                   |                                   |
|                                   | -   RO, go to Step 6              |
|                                   | -   OMO, go to Step 7             |
|                                   |                                   |
|                                   | Note: if HPI-O details are keyed  |
|                                   | incorrectly, select Clear and     |
|                                   | start again.                      |
+-----------------------------------+-----------------------------------+
| 6                                 | Confirm RO identity + Read more   |
|                                   | ...                               |
|                                   |                                   |
|                                   | Email the HI Service and ask them |
|                                   | to confirm the evidence of        |
|                                   | identity (EOI) status in the HI   |
|                                   | Service. See Healthcare           |
|                                   | Identifiers (HI) service.         |
|                                   |                                   |
|                                   | Go to Step 9, then return to Step |
|                                   | 8 when there is a response to the |
|                                   | email sent to HI Service.         |
|                                   |                                   |
|                                   | Note: the RO should always be a   |
|                                   | known customer as EOI would       |
|                                   | initially be established in the   |
|                                   | HI Service.                       |
+-----------------------------------+-----------------------------------+
| 7                                 | Follow up OMO details + Read more |
|                                   | ...                               |
|                                   |                                   |
|                                   | Call the applicant and advise     |
|                                   | them that they are not listed in  |
|                                   | the HI Service as the authorised  |
|                                   | OMO.                              |
|                                   |                                   |
|                                   | Advise them that to register as   |
|                                   | the OMO of the organisation, the  |
|                                   | authorised RO or another          |
|                                   | authorised OMO of the             |
|                                   | organisation will need to         |
|                                   | register the OMO via HPOS, or by  |
|                                   | signed paper application -The     |
|                                   | Resources page contains a link to |
|                                   | the application. If the caller    |
|                                   | (applying as OMO) wishes to       |
|                                   | discuss their missing OMO         |
|                                   | credentials, advise the caller    |
|                                   | that:                             |
|                                   |                                   |
|                                   | -   they will have to discuss     |
|                                   |     with the RO or another OMO of |
|                                   |     the organisation, and         |
|                                   | -   the RO/OMO will need to       |
|                                   |     contact Tier 2 Healthcare     |
|                                   |     Identifiers Service to        |
|                                   |     clarify registration details  |
|                                   |     for the applicant             |
|                                   | -   go to Step 9                  |
|                                   |                                   |
|                                   | If the caller disagrees with the  |
|                                   | record details, warm transfer     |
|                                   | them to the SME for follow up     |
|                                   | with Tier 3. Go to Step 9.        |
+-----------------------------------+-----------------------------------+
| 8                                 | Determine if the RO details are   |
|                                   | confirmed by the HI Service +     |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | Are the RO details confirmed by   |
|                                   | the HI Service?                   |
|                                   |                                   |
|                                   | -   Yes, go to Step 9, then       |
|                                   |     return to Step 5 if still in  |
|                                   |     the Select Officer screen,    |
|                                   |     otherwise return to Step 1    |
|                                   | -   No, return application to SME |
|                                   |     for action, go to Step 9      |
+-----------------------------------+-----------------------------------+
| 9                                 | Record all details in the         |
|                                   | Customer Relationship Manager     |
|                                   | (CRM) + Read more ...             |
|                                   |                                   |
|                                   | Record details of all             |
|                                   | interactions in the CRM.          |
+-----------------------------------+-----------------------------------+
