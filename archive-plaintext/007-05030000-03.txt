Eligibility criteria for DAP

| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Paid Parental Leave scheme + Read |
|                                   | more ...                          |
|                                   |                                   |
|                                   | DAP is paid under the Paid        |
|                                   | Parental Leave scheme for a child |
|                                   | born or adopted before 1 July     |
|                                   | 2023. The payment is for eligible |
|                                   | working fathers or other partners |
|                                   | (including adopting parents) who  |
|                                   | are caring for a child. DAP is    |
|                                   | paid at the rate of the national  |
|                                   | minimum wage for a maximum of 2   |
|                                   | weeks (in a lump sum).            |
|                                   |                                   |
|                                   | Note: DAP claims lodged by birth  |
|                                   | mothers on behalf of their        |
|                                   | partners will be made not         |
|                                   | effective (NEF) as birth mothers  |
|                                   | are not an eligible person to     |
|                                   | claim DAP.                        |
|                                   |                                   |
|                                   | Stillborn or deceased child       |
|                                   |                                   |
|                                   | Customers may receive DAP for a   |

|                                   | would otherwise have been         |
|                                   | eligible. All eligibility         |
|                                   | requirements must be met except:  |
|                                   |                                   |
|                                   | -   if the child was stillborn    |
|                                   |     they are not required to      |
|                                   |     register the child's birth    |

|                                   |     in care                       |

|                                   | particular day. The customer may  |
|                                   | be the primary carer of the child |
|                                   | or providing care for the child   |
|                                   | at the same time as another       |
|                                   | person (for example, the child's  |
|                                   | birth mother) who is providing    |
|                                   | primary care for the child.       |
|                                   |                                   |
|                                   | A customer may still be eligible  |
|                                   | for DAP if they are not caring    |
|                                   | for the child for all or part of  |

|                                   |     steps to have the child       |
|                                   |     returned to their care        |

|                                   |     partner, is the child's legal |
|                                   |     parent or is otherwise        |
|                                   |     legally responsible for the   |
|                                   |     child                         |
|                                   | -   if the child is in the care   |
|                                   |     of another legal parent - the |
|                                   |     customer, or the customer’s   |
|                                   |     partner, has a court order or |
|                                   |     a parenting plan stating the  |
|                                   |     child is to live with them    |
|                                   | -   another person has not        |
|                                   |     already claimed or been paid  |
|                                   |     DAP for the child for the     |
|                                   |     same period                   |
