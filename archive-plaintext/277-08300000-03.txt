|                                   |     liability registered in       |
|                                   |     accordance with section 18A   |
|                                   |     of the Child Support          |
|                                   |     (Registration and Collection) |
|                                   |     Act 1988, excluding a debt    |
|                                   |     arising from a New Zealand    |
|                                   |     child support liability       |

|                                   | be removed through this process:  |

|                                   | -   consolidated revenue debts    |
|                                   | -   overpayments                  |
|                                   |                                   |
|                                   | An entry will only be obsolete    |
|                                   | if:                               |
|                                   |                                   |
|                                   | -   the child support case has    |
|                                   |     ended, and                    |
|                                   | -   a non-pursuit of debt         |
|                                   |     decision was made less than 6 |
|                                   |     months ago, or                |
|                                   | -   the case meets the criteria   |
|                                   |     for non-pursuit of debt, and  |
|                                   | -   the outstanding child support |
|                                   |     cannot be collected because   |
|                                   |     the payer is deceased 2 or    |
|                                   |     more years or the estate is   |
|                                   |     finalised, or                 |
|                                   | -   the liability is a registered |
|                                   |     overseas maintenance          |
|                                   |     liability and the payer and   |
|                                   |     payee are not residents of    |
|                                   |     Australia                     |

Contact payee and complete obsolete entry submission

Table 2:

+-----------------------------------+-----------------------------------+
| Step                              | Action                            |
+-----------------------------------+-----------------------------------+
| 1                                 | Contact payee + Read more ...     |
|                                   |                                   |
|                                   | Contact the payee in all cases    |
|                                   | and if required the Overseas      |
|                                   | Authority in International cases. |
|                                   | Refer the case to the             |
|                                   | International Central Authority   |
|                                   | Team to contact the Overseas      |
|                                   | Authority. Check with Team Leader |
|                                   | if unsure how to refer the        |
|                                   | customer.                         |
|                                   |                                   |
|                                   | Discuss the following with the    |
|                                   | payee or Overseas Authority:      |
|                                   |                                   |
|                                   | explain Child Support is          |
|                                   | considering deleting their case   |
|                                   | and removing the outstanding      |
|                                   | child support from the Register   |
|                                   | because the case has ended and    |
|                                   | there is no avenue to collect the |
|                                   | debt, that is, Child Support      |
|                                   | considers the entry is obsolete   |
|                                   |                                   |
|                                   | if the case is deleted (deemed    |
|                                   | obsolete) there will no longer be |
|                                   | a debt to Child Support           |
|                                   |                                   |
|                                   | for International cases:          |
|                                   |                                   |
|                                   | -   advise the liability will     |
|                                   |     cease to be enforced in       |
|                                   |     Australia but may continue to |
|                                   |     be owed in the originating    |
|                                   |     jurisdiction                  |
|                                   | -   explain that if the payer or  |
|                                   |     payee return to Australia a   |
|                                   |     new application can be made   |
|                                   |     and if Child Support          |
|                                   |     determine the application is  |
|                                   |     valid, may include any        |
|                                   |     arrears that remain           |
|                                   |     outstanding at that point in  |
|                                   |     time                          |
|                                   | -   ask if either parent is       |
|                                   |     returning to live in          |
|                                   |     Australia in the near future. |
|                                   |     If the payee advises the      |
|                                   |     payer is returning to         |
|                                   |     Australia, request supporting |
|                                   |     evidence if available         |
|                                   |                                   |
|                                   | for DCM cases, advise that our    |
|                                   | decision to remove the debt from  |
|                                   | the Register will not prevent the |
|                                   | payee from taking action          |
|                                   | privately to recover outstanding  |
|                                   | child support                     |
|                                   |                                   |
|                                   | ask if they can provide any       |
|                                   | additional information Child      |
|                                   | Support should take into account  |
|                                   | when making the decision          |
|                                   |                                   |
|                                   | Document information provided by  |
|                                   | the customer or their             |
|                                   | representative in a notepad on    |
|                                   | the Case window and include in    |
|                                   | the Obsolete Entry submission     |
|                                   | macro.                            |
|                                   |                                   |
|                                   | If unable to contact the payee or |
|                                   | Overseas Authority by telephone,  |
|                                   | create a unique letter using the  |
|                                   | Obsolete entry letters macro:     |
|                                   |                                   |
|                                   | for DCM cases:                    |
|                                   |                                   |
|                                   | -   to the payee - UQ120.00 - S41 |
|                                   |     DCM Pre Decision letter, or   |
|                                   |                                   |
|                                   | for International cases:          |
|                                   |                                   |
|                                   | -   to the payee - UQ120.02 - S41 |
|                                   |     International Pre Decision    |
|                                   |     letter, and/or                |
|                                   | -   to the Overseas Authority -   |
|                                   |     UQ120.05 - S41 Central        |
|                                   |     Authority Pre Decision letter |
|                                   |                                   |
|                                   | in the unique letter request the  |
|                                   | payee/Overseas Authority contact  |
|                                   | Child Support within:             |
|                                   |                                   |
|                                   | -   14 days if they have          |
|                                   |     CSAOnline access, see Access  |
|                                   |     and troubleshooting Child     |
|                                   |     Support self service and      |
|                                   |     Using Child Support self      |
|                                   |     service                       |
|                                   | -   60 days if they are an        |
|                                   |     International customer, or    |
|                                   | -   28 days for all other         |
|                                   |     customers                     |
|                                   |                                   |
|                                   | Create a S41 Obsolete Entry       |
|                                   | Contact Req intray with a due     |
|                                   | date of 14, 28 or 60 days and     |
|                                   | record the name and date of the   |
|                                   | letter in the intray notepad.     |

|                                   | considered                        |
|                                   |                                   |
|                                   | a non-pursuit of debt decision is |
