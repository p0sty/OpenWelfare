This page contains information on managing compulsory Quarterly
Participation Interviews (QTI) for job seekers assessed as having
Partial Capacity to Work (PCW) and Temporary Reduced Work Capacity
(TRWC) (0-7 or 8-14 hours per week).

Triggering and recording a QTI requirement for a job seeker with PCW or TRWC

| 5                                 | QTI + Read more ...               |
|                                   |                                   |
|                                   | A QTI is a compulsory requirement |
|                                   | for job seekers assessed as       |
|                                   | having a:                         |
|                                   |                                   |
|                                   | -   Partial Capacity to Work      |
|                                   |     (PCW) of 0-7 or 8-14 hours    |
|                                   |     per week, or                  |
|                                   | -   Temporary Reduced Capacity to |
|                                   |     Work (TRWC) of 0-7 or 8-14    |
|                                   |     hours per week.               |
|                                   |     Note: if the job seeker is    |
|                                   |     only assessed as having a     |
|                                   |     TRWC of 0-7 or 8-14 hours per |
|                                   |     week for less than 12 weeks,  |
|                                   |     they will only be required to |
|                                   |     attend the QTI Initial        |
|                                   |     appointment                   |
|                                   |                                   |
|                                   | A job seeker will be identified   |
|                                   | as requiring a QTI when:          |
|                                   |                                   |
|                                   | -   an Employment Services        |
|                                   |     Assessment (ESAt) report or a |
|                                   |     Job Capacity Assessment (JCA) |
|                                   |     report determines the job     |
|                                   |     seeker has a PCW or TRWC of   |
|                                   |     0-14 hours per week,          |
|                                   |     including TRWC for less than  |
|                                   |     12 weeks duration, and        |
|                                   | -   the ESAt/JCA has been         |
|                                   |     accepted and recorded by the  |
|                                   |     agency                        |
|                                   |                                   |
|                                   | Go to Step 6.                     |
+-----------------------------------+-----------------------------------+
| 6                                 | Triggering the QTI requirement +  |
|                                   | Read more ...                     |
|                                   |                                   |
|                                   | -   The ESAt report is accepted,  |
|                                   |     triggering the start of the   |
|                                   |     QTI requirements              |
|                                   | -   The Service Component 707     |
|                                   |     (QTI Initial Phone) or 708    |
|                                   |     (QTI Initial Face to Face)    |
|                                   |     will automatically start when |
|                                   |     the ESAt/JCA report is        |
|                                   |     accepted. The Initial QTI     |
|                                   |     appointment type is           |
|                                   |     automatically booked for 2    |
|                                   |     weeks after the ESAt/JCA      |
|                                   |     acceptance date. Note: the    |
|                                   |     QTI Initial appointment also  |
|                                   |     applies to job seekers who    |
|                                   |     are assessed with only TRWC   |
|                                   |     of 0-7 or 8-14 hours per week |
|                                   |     for less than 12 weeks        |
|                                   | -   Once the QTI Initial          |
|                                   |     interview is completed with   |
|                                   |     all relevant actions coded    |
|                                   |     correctly (for example, QTI   |
|                                   |     activity is coded on the      |
|                                   |     Activity and Exemption        |
|                                   |     Summary (AEX) screen), the    |
|                                   |     ongoing QTI interview will be |
|                                   |     scheduled for job seekers     |
|                                   |     assessed as PCW or TRWC of    |
|                                   |     0-14 hours for 12 weeks or    |
|                                   |     more. Service Component 710   |
|                                   |     (QTI Ongoing Phone) or 711    |
|                                   |     (QTI Ongoing Face to Face)    |
|                                   |     will be scheduled every 12    |
|                                   |     weeks from the ESAt/JCA       |
|                                   |     acceptance date (not from the |
|                                   |     date of the Initial QTI       |
|                                   |     interview), if it is still    |
|                                   |     required                      |
|                                   | -   Note: the ESAt report will    |
|                                   |     have a ‘submitted’ status for |
|                                   |     28 days but this does not     |
|                                   |     affect the commencement of    |
|                                   |     QTI requirements. The job     |
|                                   |     seeker’s work capacity and    |
|                                   |     QTI requirements are          |
|                                   |     effective from the date the   |
|                                   |     ESAt is accepted              |
|                                   | -   If the job seeker has already |
|                                   |     lodged a medical certificate, |
|                                   |     follow the process to assess  |
|                                   |     the medical certificate and   |
|                                   |     update the Medical            |
|                                   |     Certificate (MC) screen to    |
|                                   |     either grant or not grant a   |
|                                   |     Temporary Incapacity (INP)    |
|                                   |     exemption                     |
|                                   |                                   |
|                                   | An automatic letter is generated  |
|                                   | for the job seeker advising them  |
|                                   | of their:                         |
|                                   |                                   |
|                                   | -   mutual obligation             |
|                                   |     requirements, and             |
|                                   | -   requirement to negotiate a    |
|                                   |     Job Plan with the agency      |
|                                   |                                   |
|                                   | Go to Step 7.                     |
+-----------------------------------+-----------------------------------+

| 10                                | Conduct QTI + Read more ...       |
|                                   |                                   |
|                                   | During the interview, discuss the |
|                                   | following:                        |
|                                   |                                   |
|                                   | The ESAt assessment and whether   |
|                                   | it is still current and valid, or |
|                                   | if the customer’s circumstances   |
|                                   | have changed                      |
|                                   |                                   |
|                                   | If the job seeker advises a       |
|                                   | change in circumstances, re-run   |
|                                   | the Job Seeker Snapshot and refer |
|                                   | for a new ESAt if appropriate.    |
|                                   | This includes where:              |
|                                   |                                   |
|                                   | -   there has been a change in    |
|                                   |     circumstances                 |
|                                   | -   the QTI requirement has       |
|                                   |     expired for a job seeker with |
|                                   |     a TRWC and the job seeker is  |
|                                   |     likely to be eligible for     |
|                                   |     QTI, as it appears they still |
|                                   |     have a work capacity of less  |
|                                   |     than 15 hours per week        |
|                                   |                                   |
|                                   | Assistance recommendations from   |
|                                   | their last ESAt/JCA               |
|                                   |                                   |
|                                   | Social participation options      |
|                                   | where the job seeker has been     |
|                                   | assessed as having a TRWC         |
|                                   |                                   |
|                                   | Encourage voluntary job search.   |
|                                   | Include voluntary participation   |
|                                   | with a Workforce Australia,       |
|                                   | Disability Employment Services    |
|                                   | (DES) or Community Development    |
|                                   | Program (CDP) provider. Note:     |
|                                   | voluntary participation with a    |
|                                   | provider does not end the QTI     |
|                                   | requirement                       |
|                                   |                                   |
|                                   | Any work the job seeker might be  |
|                                   | doing as paid employment. It      |
|                                   | could meet the job seeker’s       |
|                                   | mutual obligation requirements    |
|                                   | and they may no longer be         |
|                                   | required to attend a QTI          |
|                                   |                                   |
|                                   | Voluntary work and other          |
|                                   | community assistance options      |
|                                   |                                   |
|                                   | Provide information on the        |
|                                   | benefits of work (such as Working |
|                                   | Credit and Mobility Allowance)    |
|                                   |                                   |
|                                   | Outline the benefits of planning  |
|                                   | for the future and explain        |
|                                   | assistance services like          |
|                                   | Financial Information Services    |
|                                   |                                   |
|                                   | Outline the disadvantages of long |
|                                   | term income support dependence,   |
|                                   | and                               |
|                                   |                                   |
|                                   | Explore eligibility for other     |
|                                   | income support payments           |
|                                   |                                   |
|                                   | If the job seeker has a TRWC,     |
|                                   | check when this period ends as    |
|                                   | the job seeker may no longer be   |
|                                   | required to attend QTIs and will  |
|                                   | need to re-engage with a          |
|                                   | Workforce Australia, DES or CDP   |
|                                   | provider. A referral will be      |
|                                   | required when the TRWC period     |
|                                   | ends                              |
|                                   |                                   |
|                                   | Check the job seeker knows the    |
|                                   | date of their next quarterly      |
|                                   | participation interview and when  |
|                                   | other reviews or appointments     |
|                                   | will be                           |
|                                   |                                   |
|                                   | Select the appropriate radio      |
|                                   | button once a discussion topic is |
|                                   | addressed.                        |
|                                   |                                   |
|                                   | Go to Step 11.                    |

|                                   | Check the customer’s reporting    |

|                                   |                                   |

|                                   | notify the agency of any          |
|                                   | reduction in their working hours  |

|                                   | Australia for the duration of     |

+-----------------------------------+-----------------------------------+
| 13                                | Job seeker is not meeting         |
|                                   | requirements through paid work or |
|                                   | self-employment + Read more ...   |

|                                   | the Job Plan.                     |
|                                   |                                   |
