|                                   | -   stability of accommodation    |
|                                   | -   personal issues               |
|                                   | -   behavioural issues            |
|                                   | -   work history                  |
|                                   | -   geographical location         |
|                                   | -   family and caring             |
|                                   |     responsibilities              |
|                                   |                                   |
|                                   | Leaving work voluntarily and      |
|                                   | accessing or increasing the       |
|                                   | reliance on a participation       |
|                                   | payment is a serious              |
|                                   | consideration. It is generally    |
|                                   | not reasonable for a job seeker   |
|                                   | to leave work for factors within  |
|                                   | their control.                    |
|                                   |                                   |
|                                   | For these reasons, the assessment |
|                                   | of whether the voluntary          |
|                                   | unemployment was reasonable is    |
|                                   | usually related to whether the    |
|                                   | work was untenable or not, that   |
|                                   | is, the work had become           |
|                                   | unsuitable or it was unreasonable |
|                                   | to remain in that job due to work |
|                                   | related factors.                  |
|                                   |                                   |
|                                   | Further considerations to         |
|                                   | determine if the job seeker left  |
|                                   | work voluntarily may include:     |
|                                   |                                   |
|                                   | -   any efforts made by the job   |
|                                   |     seeker to retain their        |
|                                   |     employment, for example, did  |
|                                   |     they discuss issues they were |
|                                   |     experiencing with management  |
|                                   |     and if so what action was     |
|                                   |     taken                         |
|                                   | -   did they notify anyone of     |
|                                   |     their decision to leave       |
|                                   |     employment (include names,    |
|                                   |     position and contact details) |
|                                   |                                   |
|                                   | Factors related to personal       |
|                                   | reasons outside of the job that   |
|                                   | do not make the work unreasonable |
|                                   | include:                          |
|                                   |                                   |
|                                   | -   wishing to relocate to be     |
|                                   |     near children, a new partner  |
|                                   |     or re-join an ex-partner      |
|                                   | -   wishing to relocate to a more |
|                                   |     agreeable climate or          |
|                                   |     location, or                  |
|                                   | -   wanting to take up a sporting |
|                                   |     commitment or relocate        |
|                                   |     pending retirement            |
|                                   |                                   |
|                                   | Such reasons are not generally    |
|                                   | regarded as being reasonable for  |
|                                   | leaving work. A person would      |
|                                   | usually be expected to obtain new |
|                                   | work to accommodate their         |
|                                   | external interests without having |
|                                   | to rely on a participation        |
|                                   | payment.                          |
|                                   |                                   |
|                                   | Similarly, leaving work because   |
|                                   | the work or contract was going to |
|                                   | be ceasing in the near future is  |
|                                   | not regarded as reasonable where  |
|                                   | the job seeker did not have       |
|                                   | another job to go to.             |
|                                   |                                   |

|                                   | Assessing reasonable excuse for   |
|                                   | non-compliance with mutual        |
|                                   | obligation requirements.          |

| 6                                 | Misconduct as an employee + Read  |
|                                   | more ...                          |
|                                   |                                   |
|                                   | When assessing the employer's     |
|                                   | reason for terminating            |
|                                   | employment, it is important to    |
|                                   | ensure the reason for termination |
|                                   | was for misconduct at work or in  |
|                                   | relation to the employment and    |
|                                   | not because of the employee's     |
|                                   | general behaviour.                |
|                                   |                                   |
|                                   | It would generally not be         |
|                                   | appropriate to apply compliance   |
|                                   | action if:                        |
|                                   |                                   |
|                                   | -   the job seeker's individual   |
|                                   |     circumstance led to an        |
|                                   |     inability to produce          |
|                                   |     satisfactory work             |
|                                   | -   the job seeker's personal     |
|                                   |     circumstances resulted in an  |
|                                   |     inability to control their    |
|                                   |     behaviour or to understand    |
|                                   |     that such behaviour was not   |
|                                   |     appropriate                   |
|                                   | -   dismissal was due to a very   |
|                                   |     minor breach of conduct (such |
|                                   |     as being minimally late for   |
|                                   |     work on a single occasion),   |
|                                   |     the job seeker was not warned |
|                                   |     of dismissal, and was         |
|                                   |     genuinely and reasonably      |
|                                   |     unaware that their action or  |
|                                   |     inaction would result in      |
|                                   |     dismissal. However, some      |
|                                   |     incidents that may seem       |
|                                   |     minor, such as being late,    |
|                                   |     can have significant          |
|                                   |     repercussions for businesses  |
|                                   |     and their reputations         |
|                                   |                                   |
|                                   | It would be appropriate to apply  |
|                                   | compliance action where there is  |
|                                   | sufficient evidence to establish  |
|                                   | that dismissal was a result of:   |
|                                   |                                   |
|                                   | -   a deliberate failure to       |
|                                   |     produce/deliver a reasonable  |
|                                   |     amount of work                |
|                                   | -   unauthorised absences from    |
|                                   |     duty without good reason      |
|                                   | -   improper behaviour or         |
|                                   |     practices, such as theft,     |
|                                   |     assault or harassment of      |
|                                   |     other employees or customers  |
|                                   | -   actions that cause serious    |
|                                   |     risk to the health or safety  |
|                                   |     of other employees or         |
|                                   |     customers                     |
|                                   | -   actions that threaten the     |
|                                   |     reputation, viability or      |
|                                   |     profitability of the business |
|                                   |                                   |
|                                   | If the job seeker is seeking an   |
|                                   | independent investigation by an   |
|                                   | external body into the reasons    |
|                                   | for the termination of            |
|                                   | employment, this process is       |
|                                   | outside the unemployment failure  |
|                                   | investigation and decision.       |
|                                   |                                   |
|                                   | PST-skilled Service Officers or   |
|                                   | Service Officers with a UNPP      |
|                                   | security resource must complete   |
|                                   | their investigation using the     |
|                                   | evidence available at the time of |
|                                   | the decision.                     |
|                                   |                                   |
|                                   | If the external body subsequently |
|                                   | changes the facts in relation to  |
|                                   | the termination of the job        |
|                                   | seeker's employment, the customer |
|                                   | can apply for a formal review of  |
|                                   | the unemployment failure          |
|                                   | decision.                         |

| 7                                 | Contacting employers for further  |
|                                   | evidence + Read more ...          |
|                                   |                                   |
|                                   | Considering any previous contact  |
|                                   | by another Service Officer,       |
|                                   | contact with the employer may be  |
|                                   | required to establish the facts.  |
|                                   | This is particularly when extra   |
|                                   | evidence from the job seeker      |
|                                   | contradicts statements made by    |
|                                   | the employer.                     |
|                                   |                                   |
|                                   | The Service Officer should        |
|                                   | attempt to contact the employer   |
|                                   | by phone. This ensures a timely   |
|                                   | investigation and provides        |
|                                   | opportunity for matters raised to |
|                                   | be clarified with additional      |
|                                   | questions if required.            |
|                                   |                                   |
|                                   | Further contact with an employer  |
|                                   | may be required:                  |
|                                   |                                   |
|                                   | -   to determine that the reason  |
|                                   |     for dismissal was for         |
|                                   |     misconduct at work and not    |
|                                   |     because of the employee's     |
|                                   |     general behaviour             |
|                                   | -   to clarify inconsistencies in |
|                                   |     the evidence gathered         |
|                                   |                                   |
|                                   | If phone contact cannot be        |
|                                   | established with the employer,    |
|                                   | and evidence is likely to have an |
|                                   | impact on the outcome of the      |
|                                   | decision, further evidence should |
|                                   | be requested from the jobs        |
|                                   | seeker.                           |
